Device Drivers In OS/2 Warp Server for SMP

Existing device drivers run on OS/2 Warp Server for SMP without modification provided that they use the device helper (DevHlp) routines to access the interrupt controller. Device drivers which access the interrupt controller directly, or which mask and unmask interrupts directly through the controller, will need to be changed.
