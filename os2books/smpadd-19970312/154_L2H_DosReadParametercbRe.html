DosRead Parameter - cbRead

    cbRead (ULONG) - input

      The length, in bytes, of pBuffer.

      This is the number of bytes to be read. A value of zero is treated as a null operation as is not considered an error.

      In the raw file system, this must be a multiple of the sector size (512).
