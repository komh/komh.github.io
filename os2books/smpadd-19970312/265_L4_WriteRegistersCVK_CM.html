Write Registers (CVK_CMD_WREG)

Parameters:

    Cmd

      CVK_CMD_WREG
    TID
      Slot of thread in whose registers are to be changed.
    Reg
      Registers for thread. Only the following registers are affected: EAX, EBX, ECX, EDX, EDI, ESI, EBP, ESP, EIP, CS, DS, ES, FS, GS, SS, and EFLAGS.

    Returns:

    Cmd

      Return code
