DSK_SETDEVICEPARAMS (43h) - Drive Unit

    Drive Unit

      The field is used only when this IOCtl is issued without using a previously allocated file handle and when Command Information is set to 01. A file handle of -1 must be used. Note that media in the drive is not required. Drive Unit values are 0=A, 1=B, 2=C, and so forth.
