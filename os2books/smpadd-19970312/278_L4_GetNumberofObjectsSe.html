Get Number of Objects/Segments in Module (CVK_CMD_OBJCOUNT)

Parameters:

    Cmd

      CVK_CMD_OBJCOUNT
    Value
      MTE handle for module about which information is desired.

    Returns:

    Value

      Number of objects/segments in module. 0 returned if the module is a forwarder module, which contains no objects or segments, or a resource module, which contains only resource objects and segments.
