DSK_FORMATVERIFY (45h) - Number of Sectors

    Number of Sectors

      The number of sectors on the track being formatted. On return from a multitrack request, if a defective spot is found on the media before the multitrack format operation can complete, this field indicates the number of tracks remaining (to be formatted) on this multitrack format request.
