Hook logging 3 2-byte buffers

Note that STRACE POST processing removes the data item length information and simply lists each entry in little-endian (reverse byte) order. 

  803   12    1   254:3419280201    2   2600 0100 0002
  | |    |    |    |                |   |    |    |
  | |    |    |    |                |   |    |    +------- Data item 3
  | |    |    |    |                |   |    +------------ Data item 2
  | |    |    |    |                |   +----------------- Data item 1
  | |    |    |    |                +------  Processor ID (0,1,2,3,...,63)
  | |    |    |    +-------------- Normalized Time Stamp (high32bits:low32bits)
  | |    |    +--------------------------- Minor Code = 0x01
  | |    +-------------------------------- Major Code = 0x12
  | +----------------------------------- Total number of buffers = 3
  +------------------------------------- Type indicator - (SFT_HOOK_VARDATA2 | 3 )
