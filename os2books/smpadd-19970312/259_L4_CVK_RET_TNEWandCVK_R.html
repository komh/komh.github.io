CVK_RET_TNEW and CVK_RET_TEND Events

┌──────────────────────────────┬──────────┬──────────────────────────────┐
│Event                         │Code      │Description                   │
├──────────────────────────────┼──────────┼──────────────────────────────┤
│KDP_T_TASK_CREATE             │104       │Task create                   │
├──────────────────────────────┼──────────┼──────────────────────────────┤
│KDP_T_TASK_END                │105       │Task died                     │
└──────────────────────────────┴──────────┴──────────────────────────────┘

Fields returned in cvkcmd_s are as follows:

    Cmd

      CVK_RET_TNEW or CVK_RET_TEND
    Value
      Not used.
    OffV
      Not used.
    SegV
      Not used.
    MTE
      Not used.
    PID
      PID of process that was created or died.
    TID
      1 (Primary TID in process)
    DBit
      Not used.
    Reg
      Not used.
    MemCache
      Not used.
