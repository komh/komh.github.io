LISTIO Field - CmdFlag

    CmdFlag (ULONG)

      Command flag.

      Indicates the type of operation to perform. Valid values are:

    • FILE_BEGIN
      Seek from beginning of file.

    • FILE_CURRENT
      Seek from current file pointer,

    • FILE_END
      Seek from end of file.

    • FILE_SECTOR
      Indicates the value in Offset is in sectors rather than bytes. This permits access to disks up to 1 TB (terabyte) in size. This flag is only valid in the raw file system and must be ORed with either FILE_BEGIN, FILE_CURRENT, or FILE_END.

    • LISTIO_READ
      Read operation is requested.

    • LISTIO_WRITE
      Write operation is requested.

      Either LISTIO_READ or LISTIO_WRITE must be specified.
