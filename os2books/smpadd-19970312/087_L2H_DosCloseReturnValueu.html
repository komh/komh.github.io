DosClose Return Value - ulrc

    ulrc (APIRET) - returns

      Return Code.

      DosClose returns one of the following values:

    • NO_ERROR
    • ERROR_FILE_NOT_FOUND
    • ERROR_ACCESS_DENIED
    • ERROR_INVALID_HANDLE

      For a full list of error codes, see Errors.
