Finding Source Files

The search path tells the debugger where to find the source file used in the source windows. The debugger searches for the source files in the following order:

  • The path defined by the CAT_OVERRIDE environment variable, if specified.

  • The directory where the object file was compiled.

  • The path defined by the CAT_HOST_BIN_PATH environment variable, if specified.

  • The directory where the executable file is located.

  • The path defined by the CAT_HOST_SOURCE_PATH environment variable, if specified.

  • The current directory.

  • The path defined in the INCLUDE environment variable.

  • The directory of the last source file found in this debug session.

Note: The CAT_PATH_RECURSE environment variable, if specified, will cause the debugger to search recursively all subdirectories of the CAT_HOST_BIN_PATH and CAT_HOST_SOURCE_PATH environment variables.
