Scan Processor Control Blocks (CVK_CMD_SCANPCB)

Parameters:

    Cmd

      CVK_CMD_SCANPCB
    Value
      Processor number to start with. Specify 0 for processor 1.

    Returns:

    Value

      Processor number of next processor control block. 0 if no more PCBs.
    OffV
      Number of information blocks returned in current response. 
    struct {  ULONG  pNum;   /* Processor Number         */
  USHORT slot;   /* Slot index of thread running on processor.
                    (0 means processor idle) */
  USHORT flags;  /* Flags (currently unused) */
  }  PCBInfo[OffV]
      Information returned immediately following OffV.
