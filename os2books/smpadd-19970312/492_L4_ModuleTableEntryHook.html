Module Table Entry Hook (SFT_HOOK_MTE)

This hook is also produced at STRACE startup. The data is used by trace post-processors for information about loaded memory objects. 

 3401   a8    3   254:3418461893    0  -- see strace.mte --
   |    |    |    |                 |    |
   |    |    |    |                 |    +---- MTE data (not directly reported;
   |    |    |    |                 |          but see the STRACE.MTE file for an example)
   |    |    |    |                 |
   |    |    |    |                 +--- Processor ID (0,1,2,3,...,63)
   |    |    |    +-------------- Normalized Time Stamp (high32bits:low32bits)
   |    |    +---------------------------- Minor Code = 0x03
   |    +--------------------------------- Major Code = 0xa8
   +-------------------------------------- Type Indicator/# of buffers (SFT_HOOK_MTE | 1)
