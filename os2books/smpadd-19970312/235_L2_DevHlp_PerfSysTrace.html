DevHlp_PerfSysTrace

    Description

      Record data to STRACE buffer.

      This function writes software trace information to the STRACE buffer.

    Parameters
      Major code
      Data length in bytes
      Minor code
      Data to be traced

    Exit
      Return code.

    Assembly language 
       ;       dh_PerfSysTrace - Record data to STRACE buffer.
 ;
 ;       This routine records data in the STRACE buffer.
 ;
 ;       ENTRY:  AX          = major code (0-255)
 ;               BX          = data length, in bytes (0-255)
 ;               CX          = minor code (0-255)
 ;               DS:SI = pointer to data
 ;
 ;       EXIT-NORMAL:   AX = 0  SOFTWARE TRACE DATA AREA MODIFIED
 ;
 ;       EXIT-ERROR:    AX = ERROR CODE
 ;
 ;  Uses C calling convention.  See DevHlp_SysTrace for example.
 ;
 ;

   MOV     AX,MajorCode                 ; major code for trace hook
   MOV     BX,TraceSize                 ; size of data
   MOV     CX,MinorCode                 ; minor code
   LDS     SI,TraceData                 ;
   MOV     DL,DevHlp_PerfSysTrace
   CALL    DevHlp
