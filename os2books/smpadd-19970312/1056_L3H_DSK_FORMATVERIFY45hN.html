DSK_FORMATVERIFY (45h) - Number of Tracks

    Number of Tracks

      The number of tracks to format/verify on a multitrack request. This is set to zero for single track requests. On return from a multitrack request, Number of Tracks is set to one of the following values:

        Value

          Description
        8000H
          Multitrack successful
        4000H
          Multitrack not supported
        2000H
          Multitrack failed on FORMAT operation
        1000H
          Multitrack failed on VERIFY operation
