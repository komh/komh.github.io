DosGetProcessorStatus - Parameters

    ProcNum (ULONG) - input

      Processor number.

    pStatus (PULONG) - output

      Pointer to the status returned for the specified processor.

      Pointer to the status returned for the specified processor. Valid values are:

    • Processor offline.

    • Processor online.

    ulrc (APIRET) - returns

      Return Code.

      DosGetProcessorStatus returns one of the following values:

    • NO_ERROR 87
        ERROR_INVALID_PARAMETER
