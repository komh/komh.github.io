PSD_IRQ_EOI

    PSD_IRQ_EOI keywords

      Optional, Default, Can't Block

    Description
      Issue an EOI.

      This function is used to issue an End-Of-Interrupt.

    Mode
      Called in Kernel, and Interrupt mode.

    Entry
      Pointer to PSD_IRQ structure.

    Exit

      NO_ERROR

        operation completed successfully.
      -1
        operation failed.
    Structures 
      typedef struct psd_irq_s{
   ulong_t flags;    (Input)
   ulong_t data;     (Input/Output)
   ulong_t procnum;  (Input)
} PSD_IRQ;

      data

        is the interrupt level to end.
      flags
        is not used in this operation.
      procnum
        is the processor number of where the operation should take place.
    Notes
      If this function is omitted, OS/2 will perform all EOI operations for an 8259 Master/Slave based PIC system. The requests will be sent to CPU0 depending on the state of the INIT_GLOBAL_IRQ_ACCESS flag.
