WC_DBE_APPSTAT

The purpose of this control is to provide applications with a way to display application specific keyboard status such as keyboard lock or input inhibition. This control can be used by your application to display all kinds of messages. Because this control cannot be used under the SBCS PM, you may not use this under the DBCS PM.
