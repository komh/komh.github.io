OS/2 P

OS/2 P supports the country code 86 and code page 1381, 437 and 850. It provides the SBCS windows environment but does not provide the DBCS windows environment. The primary code page must be the combined code page 1381. The DBCS OS/2 (S-Chinese version) System Structure shows the system structure of OS/2 P(S-Chinese) version.
