RC Resource Compiler

The RC.EXE in the OS/2 Toolkit 2.1 of SBCS OS/2 version can handle the DBCS as well as SBCS. The resource compiler can compile the source which includes the literal with mixed SBCS and DBCS string, alternate mnemonics and other DBCS functions. You must specify the country code and code page parameters correctly for the DBCS version you need. The PM resource itself does not include the country code and code page information. These parameter are specified for the compiler. If you compile the source file without country code and code page, then the current country code and code page of the environment is used. If you have problem in using the resource compiler, check the following items:

  • Did you run the utility on an proper environment? Same code page?

  • Did you specify a proper country code?

  • Did you specify a proper code page?

  • Is the specified code page supported in the specified country code?

  • Make sure you have a correct and up-to-date COUNTRY.SYS in your system.

  • Make sure you are using a correct and up-to-date RC.EXE.

The following shows how it is done : 

RC -r -cc 88 -cp 948 sample.rc
