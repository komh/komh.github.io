GET KEY (51)

The possible combinations of positions 3-8 of Data String additionally include the following:

  • Bytes 3 and 4 contain a DBCS character.

If an undefined key is pressed, GET KEY returns nothing. For example, pressing Alt+a does not return @Aa. This is different from SBCS OS/2.

You cannot get raw keystrokes to the DBCS input method by GET KEY. You just receive the final results of the conversions done by the DBCS input method.
