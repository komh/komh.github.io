OS/2 T

OS/2 T supports the country code 88 and code page 938, 948, 437 and 850. It provides the SBCS windows environment but does not provide the DBCS windows environment. The DBCS OS/2 (T-Chinese version) System Structure shows the system structure of OS/2 T-Chinese version.
