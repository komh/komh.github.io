For Character Strings:

  • An SBCS percent sign matches any number of SBCS and/or DBCS characters.

  • A DBCS percent sign matches any number of SBCS and/or DBCS characters, i.e. a DBCS percent sign works in the same way as an SBCS percent sign.

  • An SBCS underscore matches one SBCS character.

  • A DBCS underscore matches one DBCS character.
