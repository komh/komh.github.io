EAB Session Option

An EAB (one byte) is associated with a display position (i.e. each SBCS character, each byte of a DBCS character, or each attribute byte) on a presentation space and follows each display position when EAB option is used. The bit definition of EAB's is the same as that of the SBCS version except for the following:

  • 3270, with NOXLATE option:

    • When a field has the DBCS Field attribute, bit 5-6 of the EAB which follows the attribute byte is set to 11 or the same as SBCS OS/2.

    • When a character has the DBCS Character attribute (i.e. using 3270 SA order), bit 5-6 of the EAB that follows each byte of the DBCS character is set to 11 or the same as SBCS OS/2.
