Different NLVs of Toolkit

There are different NLVs for different DBCS countries. The major differences of these NLVs are:

  • APSYMBOL.APS (Symbol table for IPF)

    The symbol tables for all the DBCS countries and their names are APS???.APS. ('???' above is the code page ID, for example, '437' for code page 437.) IPFC will apply the APSYMBOL.APS file only, so you need to rename APS437.APS to APSYMBOL.APS when you want to compile a 437 code page source file.

    Note: Most toolkit NLV packages include all the APS???.APS files for different code pages. However, the previous packages may not include all the APS???.APS files that the latest package has.

  • MRI

    MRI of the sample programs, on-line technical information and other tools in Japanese package are translated, but are kept as is in the packages of other DBCS countries.
