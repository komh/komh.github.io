ATM Outline Fonts

This is the Adobe Type Manager compatible outline fonts. These outline fonts can be installed during the system installation process. Most of them have a long font face name. This face name has special meaning to the Japanese end users. They belong to two popular Japanese font families, the Mincho and Gothic. The -H- or -V- in each font face name implies the font is used in Horizontal writing or Vertical writing. The -78 or -90 implies the font provides the character set with the sequence conformed to JIS'78 or JIS'90.

The following shows the available ATM MBCS fonts in OS/2 J2.1 version.

  • Mincho HeiseiMincho-W3-H-78

  • Mincho HeiseiMincho-W3-H-90

  • Mincho HeiseiMincho-W3-V-78

  • Mincho HeiseiMincho-W3-V-90

  • Gothic HeiseiKakuGothic-W3-H-78

  • Gothic HeiseiKakuGothic-W3-H-90

  • Gothic HeiseiKakuGothic-W3-V-78

  • Gothic HeiseiKakuGothic-W3-V-90

Note: Because the OS/2 J2.1 provides ATM font, but the other NLVs don't, the consideration for using ATM font is not to assume they are always available.
