EAD/NOEAD, SO/NOSO/SPACESO and Affected Functions

EAD/NOEAD and SO/NOSO/SPACESO session options are additionally supported. In the following figure, X indicates affected copy functions by each option and - indicates non-affected ones: 

                       ┌───────┬─────────┐
                       │ NOEAD │    NOSO │ ─ Default Value
                       │   EAD │      SO │
                       │       │ SPACESO │
┌──────────────────────┼───────┼─────────┤
│ COPY PS              │     X │       * │
│ COPY PS TO STRING    │     X │       X │
│ COPY STRING TO PS    │     X │       - │
└──────────────────────┴───────┴─────────┘
