DBCS Keyboard and Input Method Basic

In SBCS system software, DOS or OS/2, the user can input all the SBCS characters from the keyboard by one key stroke or one key stroke with the shift key (upper shift). In DBCS system, number of DBCS Character is too large, then the DBCS character set can not be mapped by the number of key even with the shift key, the control key or the alter key. Thus the DBCS keyboard function and DBCS input method function are required.

This section provides you with the general concept of the DBCS keyboard and input method. See Keyboard and Input Method in Each DBCS Country for details of implementations in each DBCS country.
