Available Outline Fonts

DBCS PM supports three kinds of outline fonts, the short vector outline fonts, the ATM outline fonts and the TrueType fonts. Short vector outline fonts have been supported since version 1.x as a separated product and only for Japanese version. From version 2.0x all the DBCS OS/2 versions provide the short vector outline fonts as a part of the system software. ATM outline fonts are provided from version 2.1 and for Japanese version only. TrueType outline fonts are provided from version 2.1 and for T-Chinese version only.
