Byte-Counting or Character-Counting

The length of a character string is the number of bytes (not the number of characters). Note that by supporting mixed strings, character does not necessarily mean byte any more.

The length of a graphic string is the number of characters (not the number of bytes). The maximum length of each type is as follows:

    GRAPHIC

      127
    VARGRAPHIC
      2000
    LONG VARGRAPHIC
      16350
