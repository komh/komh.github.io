Communication

Basically, communication functions pass through any binary data including DBCS. Such functions are APPC, ACDI, SRPI and so on. However, code translation and terminal emulator functions require special consideration for DBCS and country-unique code pages. This chapter describes DBCS considerations of Communication Manager/2 1.1 from the SBCS version.
