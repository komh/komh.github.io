Select a Default Font

This happens when an application is first executed, without INI file. The application needs to select a default font for itself because the end user did not specify it. Normally, the system default font is recommended in this case. However, for some reasons, for example, the system default font is big and window is not big enough to display the whole text... you need a smaller font in this case.
