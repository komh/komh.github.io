Select a Scalable Font

In some cases, you look for an outline font. Since you can select any size you want in an outline font. However, you need to make sure that the selected outline font is an MBCS font. To do this, check both FM_TYPE_MBCS and FM_TYPE_OUTLINE values in the fsType field of the FONTMETRICS structure.
