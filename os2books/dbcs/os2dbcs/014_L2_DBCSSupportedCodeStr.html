DBCS Supported Code Structure

This section explains the DBCS code structure, which is necessary for discussion on DBCS support in the following chapters. In IBM Personal System code structure, single-byte (SBCS-PC) code based on ASCII/ISO 646 and double-byte (DBCS-PC) code coexist.
