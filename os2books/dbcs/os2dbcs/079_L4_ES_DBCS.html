ES_DBCS

    WM_CHAR

      If the character is an SBCS character or the text length limit is exceeded, an alarm sounds and the character is ignored.
    EM_PASTE
      If the text contains an SBCS character, an alarm sounds and the text is ignored.
    WM_SETWINDOWPARMS
      Same as the above
