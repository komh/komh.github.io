Fonts for UDC

DBCS OS/2 supports UDC (User Defined Character), and end user can use the UDC font editor utility (USRFNT.EXE) to create a new font for an UDC. Currently, this UDC font editor utility can only create UDC font for the Mincho raster fonts provided by the PMNLSFD1.FDR font driver. If the user selects other MBCS fonts, the UDC characters will not be displayed even if the data are still there.
