Host File Transfer Program

The host file transfer programs for DBCS 3270 File Transfer (APVUFILE) are different from those for SBCS (IND$FILE). The following are marketed in the United States and can be used with Japanese 3270 File Transfer. 

Product Name                               Program Number
5550 File Transfer Program CICS (MVS,VSE) version 2  5799-PGZ
5550 File Transfer Program VM/CMS         version 2  5799-PGX
5550 File Transfer Program MVS/TSO        version 2  5799-PGY

You do not need to change the host command name in the file transfer profile; it is set to "APVUFILE" as the default value.
