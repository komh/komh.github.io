ES_SBCS

    WM_CHAR

      If the character is a DBCS character or the text length limit is exceeded, an alarm sounds and the character is ignored.
    EM_PASTE
      If the text contains a DBCS character, an alarm sounds and the text is ignored.
    WM_SETWINDOWPARMS
      Same as the above
