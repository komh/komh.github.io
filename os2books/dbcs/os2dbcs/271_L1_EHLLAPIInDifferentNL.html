EHLLAPI In Different NLVs

EHLLAPI (Emulator High Level Language Application Programming Interface) is an application programming interface which is provided by the following OS/2 related products:

  • Extended Edition version 1.x

  • Extended Services version 2.x

  • Communications Manager/2 1.1

The differences of the EHLLAPI among DBCS countries are:

  • API's that have code-page parameter(s)

    Supported code pages are different.

  • ASCII parameter - 3270 File Transfer

    Japanese OS/2 uses JISCII instead of ASCII but Korean and T-Chinese OS/2 use ASCII as does SBCS OS/2.
