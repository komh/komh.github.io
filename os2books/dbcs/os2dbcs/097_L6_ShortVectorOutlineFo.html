Short Vector Outline Fonts

The short vector fonts are included in the Font Diskettes of each DBCS OS/2. After the system installation. End user need to install these diskettes by the INSTAID utility. After the installation, the short vector outline fonts are available as the public fonts in the system.

These fonts can be handled in a same manner as all other SBCS outline fonts. However, do not assume they are loaded/installed.

The following shows the short vector fonts that are provided in each DBCS version of OS/2 2.x.
