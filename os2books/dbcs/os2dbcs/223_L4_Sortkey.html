Sortkey

Application developers or translators can add a phonetic reading (or whatever you like) to each translated help index topic (ICMD tags). Then, help indexes will be sorted in a meaningful sequence (typically, the order of the phonetic readings). The following example demonstrates the use of the SORTKEY tag 

:i1 id=pointsize sortkey='point size'.changing fonts
