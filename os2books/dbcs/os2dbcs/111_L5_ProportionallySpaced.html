Proportionally-Spaced Fonts

Before version 2.0, all fonts for combined code pages are monospaced ones, if you assumed this in your program then your program will have problem when run on version 2.0 or later.

You can query the sizes of characters using GpiQueryWidthTable. However, this call is of little use for DBCS since there are so many DBCS characters and you will need a large table(s). Instead, you may use GpiQueryTextBox, GpiQueryCharStringPos or GpiQueryCharStringPosAt. See also Character vs. Byte.

Note:

  • An error is returned if the range of code points specified includes invalid code points (for example, the DBCS first byte code ranges when FirstChar is an SBCS character). So, the following code works with SBCS OS/2 but not with DBCS OS/2. 
    GpiQueryWidthTable(hps, 0L, 256L, w_table);

  • You can get width table information for DBCS characters. To specify a DBCS character to FirstChar, store the first byte of the DBCS character at the lowest order byte of FirstChar and the second byte at the next higher byte.
