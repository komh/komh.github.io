Additional Error Status Indications

Some additional error status indications are supported as follows:

    X -S

      Operator attempts to input DBCS data on an SBCS field or SBCS data on a DBCS field.
    X PROG797
      Received an SO in a DBCS subfield (DBCS part of a SBCS/DBCS mixed data) or SO/SI are not paired.
    X PROG798
      Received an invalid order or control code in a DBCS field. Received an SO or SI in a DBCS field.
    X PROG799
      Specified address is the second byte of a DBCS character.

    Note: When the OLD OIA option is selected, these are not returned. It is not recommended to use the OLD OIA option.
