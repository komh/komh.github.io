Hard─Coded Modules

Most software programs consist of many modules with a big volume of source code. So it is not an easy task to find out which portions of the program have been hardcoded. Here are some hits for find out the modules which might be hard-coded:

  • Find out any data or string that include the quotation mark in the source code. The PII, MRI or message data may be hardcoded in the program.

  • Find out the numeric data, hexical data or key words like "codepage", "country" or "language"..etc in the source code. The code page or country information may be hard-coded in the program.

  • Find out the key words like "VGA", "XGA" or "Enhanced-Keyboard" in the source code. The hardware information may be hard-coded in the program.

  • Run your program on a DBCS OS/2 environment on both VGA and XGA display. Display some dialogs of the program. The dialogs or windows size may be hard-coded and do not fit the DBCS screen.
