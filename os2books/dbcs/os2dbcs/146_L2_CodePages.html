Code Pages

Each DBCS DB2/2 supports the following code pages:

    DB2/2

      Code Pages
    Japanese
      932 and 942
    Korean
      949
    Traditional Chinese
      938,948 and 950
    Simplified Chinese
      1381

    Note: The extended code pages (942, 944 and 948) are not supported by DOS Database Requester of DBCS OS/2 Version

    You can find information on how code pages are handled in DB2/2 in Appendix H. General National Language Support (NLS) in Database 2 OS/2 Guide(S62G-3663).
