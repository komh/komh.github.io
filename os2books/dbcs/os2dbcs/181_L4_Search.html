Search

In function SEARCH FIELD and SEARCH PRESENTATION SPACE:

  • SO/SI in the presentation space will be ignored.

  • If PS Position is on the second byte of a DBCS character (i.e. the search begins with the second byte of a DBCS character), the start position will be adjusted to the next position, (from the next character if SRCHFRWD is selected, up to the first byte if SRCHBKWD is selected.)

  • When the code page of the presentation space is Japanese(Katakana) which does not have lowercase alphabetic characters in it, uppercase and lowercase representations of the same letter will match.
