Code Page Supported in DBCS OS/2 2.1

The Supported code page in each NLV of OS/2 2.1 shows the code pages that are supported in the NLV of OS/2 2.1 that are available in each DBCS country. In DBCS OS/2, the primary code page must be the combined code page. In this document, only these code pages are discussed. The information of the other code page is included in the appendix for your reference. 

              ┌───────────┬───────────┬───────────┬───────────┐
              │ OS/2 J2.1 │ OS/2 H2.1 │ OS/2 P2.1 │ OS/2 T2.1 │
  ┌───────────┼───────────┼───────────┼───────────┼───────────┤
  │ combined  │ 932/942   │     949   │    1381   │   938/948 │
  │ code page │           │           │           │   950     │ *1
  ├───────────┼───────────┼───────────┼───────────┼───────────┤
  │ SBCS code │ 437/850   │   437/850 │   437/850 │   437/850 │
  │ page      │           │           │           │           │
  ├───────────┼───────────┼───────────┼───────────┼───────────┤
  │ Country   │   81      │     82    │     86    │     88    │
  │ Code      │           │           │           │           │
  └───────────┴───────────┴───────────┴───────────┴───────────┘

Note:
      *1) There are two packages of OS/2 T2.1.
          One supports 938/948 as in OS/2 T2.0 and the other supports 950.
