ATTRB Session Option

If you do not want the attribute bytes translated into blanks, you can override this translation with ATTRB option. The bit definition of the attributed bytes are the same as that of the SBCS version except for the following:

  • 5250: When a field has Katakana Field attribute, bit 4-6 is set to 100.
