Compiler Option

Like the resource compiler, you need to specify a valid country code and code page parameter when you want to have a correct compiled result of your information source file. You need to specify the language ID parameter when you execute the IPFC. The following shows how it is done: 

IPFC sample.txt   /INF
                  /COUNTRY=081
                  /CODEPAGE=942
                  /LANGUAGE=JPN
