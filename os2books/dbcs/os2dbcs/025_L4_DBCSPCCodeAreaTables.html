DBCS PC Code Area Tables

The DBCS-PC Code Area for Japanese. (code page 932, 942) shows the DBCS─PC code area of a typical mixed code page, the 932 and 942 in OS/2 Japanese version. The figcures of other DBCS code pages are included in the DBCS PC Code Area Tables for your information.
