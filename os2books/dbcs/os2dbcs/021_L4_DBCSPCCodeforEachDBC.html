DBCS PC Code for Each DBCS Language

The DBCS languages currently supported on IBM Personal System are Japanese, Korean, Traditional Chinese and Simplified Chinese. Each language has its own character set and associated code page, which are uniquely defined in IBM Corporate Standard. Number of Characters Used in Each DBCS National Language shows the number of doubleÄbyte characters accommodated on the code pages designed for each of the DBCS languages.
