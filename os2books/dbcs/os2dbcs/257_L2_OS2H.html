OS/2 H

OS/2 H had a significant support change in 1992, the character code standard that used in IBM personal system in the country was changed from the standard which had been defined by IBM to the one which conforms to the Korea Standard that is defined by the Korea Government. The DOS environment, which IBM provides to Korean customers, changed from DOS H for PS/55 DA/2 to DOS H/V for VGA. The OS/2 H the versions before 2.00.1 supported the code page 934, 944, 437 and 850, and the versions from 2.00.1 support the code page 949, 437 and 850. The applications for PC DOS and DOS H/V are supported in the MVDM environment, and the windows applications are supported only in the SBCS version of windows environment of the OS/2 H. The DBCS OS/2 (Korea Hangeul version from 2.1) System Structure shows the system structure of OS/2 Hangeul version.
