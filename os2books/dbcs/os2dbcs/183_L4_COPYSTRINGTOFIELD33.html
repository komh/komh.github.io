COPY STRING TO FIELD (33)

Writing into the field is done in the same way as the operator does and:

  • An attempt to input a DBCS character when only one display position remains will inhibit input and return code 5.

  • This function cannot input an SBCS/DBCS mixed string into a field of 5250. See the explanation on SEND KEY for more information.

If the last byte of the source string specified is the first byte of a DBCS character, it will not be written or counted.
