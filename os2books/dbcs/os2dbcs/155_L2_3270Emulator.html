3270 Emulator

The following character sets and code pages are supported: 

┌─────────────────────────────────┬──────────┬──────────┬───────────┐
│                                 │SBCS CS/CP│DBCS CS/CP│Combined CP│
├─────────────────────────────────┼──────────┼──────────┼───────────┤
│Japan Japanese(Katakana) Extended│ 1172/290 │  370/300 │      930  │
│      Japanese(Latin) Extended   │ 1172/1027│  370/300 │      939  │
│Korea Korean Extended            │ 1173/833 │  934/834 │      933  │
│ROC   T─Chinese Extended         │ 1175/037 │  935/835 │      937  │
└─────────────────────────────────┴──────────┴──────────┴───────────┘


Note: Communications manager started to support Japanese Katakana
      extended SBCS code page from OS/2 EE version J1.32.
      The combination of SBCS CS/CP previously supported as
      Japanese Katakana was 332/290.
      Also Japanese(Latin), combination of US English SBCS
      (CS=101, CP=037) and Japanese DBCS (CS=370, CP=300),
      which is supported by the Japanese 3270PC, 3270PC/G and
      3270PC/2, is not supported by the 3270 Emulator of OS/2.
