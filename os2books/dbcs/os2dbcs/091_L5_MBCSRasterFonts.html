MBCS Raster Fonts

The MBCS raster fonts are supported as a font device. There are several font device drivers that are provided in DBCS PM. They need to be loaded to the system as a public font before used. Each font device driver provides several fonts with various sizes. See the Available Fonts in DBCS PM for detail.
