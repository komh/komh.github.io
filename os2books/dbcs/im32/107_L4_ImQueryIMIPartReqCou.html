ImQueryIMIPartReqCount

This entry is called to query request count of the specified IMInstance Part.

    Format: 

    APIRET APIENTRY ImQueryIMIPartReqCount(
                HIMIP himiPart,
                PULONG pulCount )

    Parameters: 

    himiPart(input) : Part handle.

pulCount(output) : access count

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred
