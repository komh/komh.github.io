ImShowStatusWindow

This function show or hide a status window.

    Format: 

    APIRET APIENTRY ImShowStatusWindow(HIMI hImi, ULONG ulFlag)

    Parameters:

    hImi(input)

      - a handle of IM Instance
    ulFlag(input)
      - flag 
      ┌──────────────────────────────┬──────────────────────────────────────────────────┐
│Value                         │Description                                       │
├──────────────────────────────┼──────────────────────────────────────────────────┤
│SSW_SHOW                      │Show the status window                            │
├──────────────────────────────┼──────────────────────────────────────────────────┤
│SSW_HIDE                      │Hide the status window                            │
└──────────────────────────────┴──────────────────────────────────────────────────┘
    Return:

    Zero is successful. Others are errors.
