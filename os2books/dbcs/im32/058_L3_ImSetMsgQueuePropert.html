ImSetMsgQueueProperty

An application can set the default IM Instance association way.

    Format: 

    APIRET APIENTRY ImSetMsgQueueProperty( HMQ hmq, ULONG ulFlag )

    Parameters:

    hmq(input)

      - message queue handle
    ulFlag(input)
      - message queue property information to set. Please refer to ImQueryMsgQueueProperty
    Return:

    Zero is successful. Others are errors.
