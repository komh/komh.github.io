ImSetDefaultIME

Applications can set the default IME of the specified codepage.

    Format: 

    APIRET APIENTRY ImSetDefaultIME(ULONG ulCodepage, ULONG ImeId)

    Parameters:

    ulCodepage(input)

      - codepage to set the default IME
    ImeId(input)
      - IME ID of the IME to be the default
    Return:

    Zero is successful. Others are errors.
