ImEnumRegisterWord

Applications can enumerate register words by its reading, style and register string.

    Format: 

    APIRET APIENTRY ImEnumRegisterWord(HIMI hIMI,
                         REGISTERWORDENUMPROC pfnEnumProc,
                         PSZ pszReading,
                         ULONG ulType,
                         PSZ pszRegister,
                         PVOID pData)

    Parameters:

    hIMI(input)

      - a handle of IM Instance
    pfnEnumProc(input)
      - pointer to EnumRegisterWord, the application-defined function. This function will be called back and used to enumerate.
    pszReading(inputt)
      - pointer to the reading string to be enumerated. When this is NULL, all reading strings matching with ulStyle and pszRegister are enumerated.
    ulType(input)
      - type of the register word. When this is 0, all types matching with pszReading and pszRegister are enumerated.
    pszRegister(input)
      - pointer to the register string to be enumerated. When this is NULL, all register strings matching with pszReading and ulStyle are enumerated.
    pData(input)
      - Application uses this data. This is passed to the function specified by pfnEnumProc.
    Return:

    Remarks:

    EnumRegisterWord - Application-defined function.

    Format: 

            ULONG EXPENTRY EnumRegisterWord(PSZ pszReading,
                                                  ULONG ulType,
                                                  PSZ pszRegister,
