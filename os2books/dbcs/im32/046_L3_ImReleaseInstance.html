ImReleaseInstance

This function releases IM Instance. An application must call this function for each call of ImGetInstance.

    Format: 

    APIRET APIENTRY ImReleaseInstance(HWND hWnd, HIMI hIMI)

    Parameters:

    hWnd(input)

      - a window handle
    hIMI(input)
      - a handle of a IM Instance
    Return:

    Zero is successful. Others are errors.
