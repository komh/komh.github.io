ImRequestIMIPart

This entry is called to request access to IMInstance part.

    Format: 

    APIRET APIENTRY ImRequestIMIPart(
                HIMIP himiPart,
                PVOID * ppPartData )

    Parameters: 

    himiPart(input) : Part handle

ppPartData(output) : Address of Part data structure

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred

    Remarks:

    This function increments the request count of himiPart, which can be queried by ImQueryIMIPartReqCount.
