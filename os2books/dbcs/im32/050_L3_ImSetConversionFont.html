ImSetConversionFont

Applications can set font attribute information in a conversion window.

    Format: 

    APIRET APIENTRY ImSetConversionFont(HIMI hIMI, PFATTRS pFontAttr)

    Parameters:

    hIMI(input)

      - a handle of a IM Instance
    pFontAttr(input)
      - a pointer to font information, FATTRS
    Return:

    Zero is successful. Others are errors.
