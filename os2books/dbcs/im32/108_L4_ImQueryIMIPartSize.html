ImQueryIMIPartSize

This entry is called to query the IMInstance Part data size.

    Format: 

    APIRET APIENTRY ImQueryIMIPartSize(
                HIMIP himiPart,
                PULONG pulPartSize )

    Parameters: 

    himiPart(input) : Part handle.

pulPartSize(output) : Part data size.

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred
