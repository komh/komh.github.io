ImShowIMEDlg

This function make the IME display setting dialog boxes, which are the configuration dialog, the word register dialog, and the dictionary selection dialog.

    Format: 

    APIRET APIENTRY ImShowIMEDlg(HIMI hIMI, ULONG ulDlgType, PREGISTERWORDHEADER pRegWord)

    Parameters:

    hIMI(input)

      - a handle of an IM Instance
    ulDlgType(input)
      - Dialog box type 
      ┌──────────────────────────────┬──────────────────────────────────────────────────┐
│Type                          │Meaning                                           │
├──────────────────────────────┼──────────────────────────────────────────────────┤
│IME_DLG_CONFIG                │IME Configuration Dialog                          │
├──────────────────────────────┼──────────────────────────────────────────────────┤
│IME_DLG_REGWORD               │Word Register Dialog                              │
├──────────────────────────────┼──────────────────────────────────────────────────┤
│IME_DLG_DICTIONARY            │Dictionary Selection Dialog                       │
└──────────────────────────────┴──────────────────────────────────────────────────┘
    pRegWord(input)
      - a pointer to a REGISTERWORDHEADER structure. This parameter is valid only when the ulDlgType is IME_DLG_REGWORD.
    Return:

    Zero is successful. Others are errors

    Remarks:

    See REGISTERWORDHEADER structure.
