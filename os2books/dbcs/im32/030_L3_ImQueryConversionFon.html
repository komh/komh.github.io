ImQueryConversionFont

Applications can get font information in a conversion window.

    Format: 

    APIRET ImQueryConversionFont(HIMI hIMI, PFATTRS pFontAttrs)

    Parameters:

    hIMI(input)

      - a handle of a IM Instance
    pFontAttrs(output)
      - a buffer to be copied the font information, FATTRS into
    Return:

    Zero is successful. Others are errors.
