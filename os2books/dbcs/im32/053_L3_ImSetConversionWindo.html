ImSetConversionWindowPos

Applications can set position of a conversion window.

    Format: 

    APIRET APIENTRY ImSetConversionWindowPos(HIMI hIMI, PCONVERSIONPOS pConvPos)

    Parameters:

    hIMI(input)

      - a handle of a IM Instance associated with the conversion window.
    pConvPos(input)
      - a pointer to position of conversion window, CONVERSIONPOS.
    Return:

    Zero is successful. Others are errors.

    Remarks:

    See CONVERSIONPOS
