ImQueryIMInstanceReqCount

This entry is called to query access count of the specified IMInstance.

    Format: 

    APIRET APIENTRY ImQueryIMInstanceReqCount(
                HIMI hImi,
                PULONG pulCount )

    Parameters: 

    hImi(input) : IMInstance handle

pulCount(output) : access count

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred
