ImDestroyIMIPart

This entry is called to destroy a IMInstance Part handle.

    Format: 

    APIRET APIENTRY ImDestroyIMIPart( HIMIP himiPart )

    Parameters: 

    himiPart(input) : Part handle to be destroyed.

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred
