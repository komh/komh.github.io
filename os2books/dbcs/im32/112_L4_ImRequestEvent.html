ImRequestEvent

This entry is called to request the notification events to be delivered to the application in the case that IME changed some fields of IMInstance.

    Format: 

    APIRET APIENTRY ImRequestEvent(
                HIMI hImi,
                ULONG msg,
                ULONG mp1,
                ULONG mp2 )

    Parameters: 

    hImi(input) : IMInstance handle

msg(input)  : Message event type to request. This is one of the following values.

   WM_IMEREQUEST
   WM_IMENOTIFY

mp1(input) : Message parameter 1 for the message which is specified in 'msg'.

mp2(input) : Message parameter 2 for the message which is specified in 'msg'.

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred

    Remarks:

    This API generates WM_IMEREQUEST or WM_IMENOTIFY message in PM environment.
