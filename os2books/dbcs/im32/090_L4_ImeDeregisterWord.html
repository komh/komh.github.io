ImeDeregisterWord

This entry is called to De-register a previously registered word with its reading from IME.

    Format: 

    APIRET APIENTRY ImeDeregisterWord(
                HIMI hImi,
                PSZ pszWord,
                PSZ pszReading,
                ULONG ulWordType )

    Parameters: hImi(input) : IMInstance handle. pszWord(input) : Word to be De-registered. pszReading(input) : Reading for pszWord. ulWordType(input) : Word type id.

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred
        ERROR_NOT_SUPPORTED : This function is not supported.
