ImDeregisterIME

IME Install program can deregister IME which was registered with ImRegisterIME. ImDeregisterIME removes the registration from all of the code pages.

    Format: 

    APIRET APIENTRY ImDeregisterIME( PSZ pszImeName )

    Parameters: 

    pszImeName(input) - a name of IME to deregister

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred

    Remarks:

    This API is not called from IME DLL itself, but from IME install program only.
