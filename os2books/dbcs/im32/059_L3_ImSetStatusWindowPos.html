ImSetStatusWindowPos

ImSetStatusWindowPos

Applications can set position of a status window.

    Format: 

    APIRET APIENTRY ImSetStatusWindowPos(HIMI hIMI, PPOINTL pptPos)

    Parameters:

    hIMI(input)

      - a handle of a IM Instance associated with the status window.
    pptPos(input)
      - a pointer to a window position in World coordinate
    Return:

    Zero is successful. Others are errors.
