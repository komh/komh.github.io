ImeRegisterWord

This entry is called to register a new word with its reading to IME.

    Format: 

    APIRET APIENTRY ImeRegisterWord(
                HIMI hImi,
                PSZ pszWord,
                PSZ pszReading,
                ULONG ulWordType )

    Parameters: 

    hImi(input)       : IMInstance handle.

pszWord(input)    : Word to be registered.

pszReading(input) : Reading for pszWord.

ulWordType(input) : Word type id.

    Returns: 

    Success indicator

0 : Successful completion

other : Error occurred
        ERROR_NOT_SUPPORTED : This function is not supported.
