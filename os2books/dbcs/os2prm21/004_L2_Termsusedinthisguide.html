Terms used in this guide

You will see the following terms in this guide:

    NLS

      National Language Support
    SBCS
      Single-byte character set
    DBCS
      Double-byte character set
    An SBCS string
      A string made up of SBCS characters
    A DBCS string
      A string made up of DBCS characters
    An SBCS/DBCS mixed string
      A string made up of both SBCS and DBCS characters
    A mixed string
      A Short form of an SBCS/DBCS mixed string
    PS/55
      Personal System/55 developed from PS/2 to support DBCS
    DBCS OS/2
      Generic term for the Asian OS/2 supporting DBCS
    SBCS OS/2
      Generic term for the American OS/2, used in contrast to DBCS OS/2
    Combined code page
      Generic term for a combination of an SBCS and a DBCS code pages
    MRI
      Abbreviation for Machine Readable Information, which refers to all the language-sensitive information passing between the user and an application.
    SBCS Windows
      Generic term for the Microsoft Windows for US, used in contrast to Japanese Windows.
    Japanese Windows
      Generic term for the Microsoft Windows for Japan, which enhanced to support Japanese.
