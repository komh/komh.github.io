You are able to edit and compile your application programs with SBCS OS/2 with the following programming tools, though you are not able to actually perform input and output of DBCS characters.

  • OS/2 Programming Tools V2.1
  • IBM C/C++ Tools V2.0
This chapter describes how to use these utilities to generate application program for DBCS OS/2.

Note: Yet you need at least the following in order to test your application programs in a real DBCS-support environment.

  • An IBM Personal System (PS/55, PS/V, PS/2 and PS/VP) 
      IBM DBCS OS/2 V2.1 and higher
    
      IBM DBCS OS/2 Programming Tools for OS/2 V2.0 and higher
