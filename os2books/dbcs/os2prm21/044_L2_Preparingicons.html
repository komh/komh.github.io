Preparing icons

┌──────────────────────────────────────────────────────────────────────┐
│An icon should be created for a display unit of which resolution is   │
│1040 X 768 for PS/55.                                                 │
└──────────────────────────────────────────────────────────────────────┘

To create an icon for DBCS OS/2, you need to take the following into consideration:

  • Resolution of a display unit

    Resolution of a PS/55 display unit capable of running OS/2 was 1040x768 with DBCS OS/2 V1.x. However, DBCS OS/2 V2.0 supports various display resolutions from 640x480 to 1040x768.

  • The number of colors that a display unit can use

    16 colors are default for DBCS OS/2, whereas 256 colors are common for a 8514 display in USA. DBCS users can select 256 colors as a selectable option at installation of DBCS OS/2. Do not assume that 256 colors are always available to your program.

In order to create an icon which best shows your intention, you should create two types of icons for the PS/55 high resolution mode:

  • an icon for 1040x768 display resolution with 16 colors

  • an icon for 1040x768 display resolution with 256 colors

DBCS OS/2 automatically selects and displays icons best-fitted for a PS/55 display unit. If you don't provide 1040 X 768 icons, DBCS OS/2 will pick up independent form icons.

We created the following four icons and packed those into ORDENT.ICO for our sample program:

  • an icon for 1040 X 768 with 256 colors(for DBCS)

  • an icon for 1040 X 768 with 16 colors (for DBCS)

  • an icon for a 8514 display with 256 colors(for SBCS )

  • an icon for a VGA display with 16 colors (for SBCS and DBCS)

  • Hints and tips

    You don't need to create an icon for PS/55 from the scratch. It can be created from an icon for the IBM 8514/A display.

    To create an icon for PS/55, select "device" at the action bar of the OS/2 icon editor, then select "custom" from the pull-down menu. Values need to specified are:

    • 1040 X 768 as display resolution

    • 64 X 64 as image resolution

    • 16 colors or 256 colors

  • End of Hints and tips
