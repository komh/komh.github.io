Using Font Dialog control

┌──────────────────────────────────────────────────────────────────────┐
│The Font Dialog control provided by PM supports DBCS fonts as well as │
│SBCS fonts.                                                           │
└──────────────────────────────────────────────────────────────────────┘

The font dialog control, which is a standard user interface to show and let users select a font, displays all available fonts on the runtime environment regardless of SBCS font or DBCS font.

It is recommended to present a mix string as a sample text in the dialog if the environment is for DBCS, since the end user may not know that there are fonts with SBCS characters only, and may happen to select one of the SBCS fonts.
