GetOutlineTextMetrics

    Functional Difference from WIN95

    The PANOSE member in the OUTLINETEXTMETRIC structure is not supported. This member is reserved and set to zero.

    Functional Difference from SBCS Open32

    The CharSet and PitchAndFamily member value will contain appropraite values.
