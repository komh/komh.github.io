Windows Country Code v.s. OS/2 Country Code

Both Windows country code and OS/2 country code are based on international phone codes. Basically, those values should be the same for all countries. But, there's an exception as follows. So, Open32 maps country code internally between Windows one and OS/2 one whenever required. 

┌──────────┬──────────────────────┬──────────────────────┐
│Country   │Windows Country Code  │OS/2 Country Code     │
├──────────┼──────────────────────┼──────────────────────┤
│Taiwan    │886                   │88                    │
└──────────┴──────────────────────┴──────────────────────┘
