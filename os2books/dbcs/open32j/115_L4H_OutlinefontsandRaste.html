Outline fonts and Raster fonts

In Windows, True Type is the only format of outline fonts and no way to know what type of outline font is selected in OS/2. So When the flag of FM_DEFN_OUTLINE set field fsDefn of OS/2 FONTMETRICS, return as True Type font, else returns as Raster font.
