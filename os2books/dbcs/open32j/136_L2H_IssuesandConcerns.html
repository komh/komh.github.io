Issues and Concerns

  • The programming interface enhancement in the GpiQueryFonts function is required to get the Glyph List Name of OS/2 fonts, which is necessary to perform correct font mapping.

  • The OS/2 TrueType font driver is expected to return the correct IBM Font Family Class value by computing the PANOSE number values defined in the 'OS/2' table in the TrueType header in the font file. This is also necessary to perform correct font m ping.

  • The base operating system is expected to provide system GUI fonts that should be compatible with the font used in the Windows 95 as much as possible, to improve application portability.

  • The base operating system is expected to support Font Alias function to emulate platform unique typefaces. This improves the application and document portability.
