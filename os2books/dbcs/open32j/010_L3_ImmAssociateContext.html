    Implementation

    This API is translated to ImAssociateInstance(hWnd,hIMI,phIMI) of OS/2 IM. A hIMC of Win95 is used as the hIMI of OS/2.

    Behavior using 16 bit IME

    You can use this function as same as 32bit IME.
