    Implementation

    The y-coordinate is translated to Win95 coordinate from OS/2 one.

├──────────┼──────────────────────────────┼──────────────────────────────┤

└──────────┴──────────────────────────────┴──────────────────────────────┘

    See ImmGetStatusWindowPos

    Behavior using 16 bit IME

    You can use this message as same as a 32bit IME.
