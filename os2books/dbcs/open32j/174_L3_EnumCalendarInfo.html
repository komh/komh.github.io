EnumCalendarInfo

Not supported in Warp 4.

    Functional Difference from WIN95

    N/A

    Functional Difference from SBCS Open32

    New

    Implementation

    • Refer to either one of following to get the actual locale id. If it is a locale id generated by language-neutral locale or sublanguage-neutral locale, then this API converts the actual locale ID in the same way as ConvertDefaultLocale for each locale. If any locale id is specified, check whether that specified locale is valid or not.

───────────────────────────────────────────────────────────────────────────────────

    • Open32 has the hard-coded table which shows all CALIDs available for each locale. For ENUM_ALL_CALENDARS parameter, Open32 works for all available CALIDs. When the CALID is specified, Open32 confirms that it is the available CALID for the specified locale, and works for it.

    • Map WIN95 CALTYPE to ULS definition. Like as WIN95, overridden information by SetLocaleInfo also overrides the calendar information.
