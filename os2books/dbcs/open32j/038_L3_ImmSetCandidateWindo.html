ImmSetCandidateWindow

    Functional Difference from WIN95

    Nothing

    Functional Difference from SBCS Open32

    'N/A'

    Implementation

    This API is translated to ImSetCandidateWindowPos(HIMI, PCANDIDATEPOS).

    The CANDIDATEFORM of Win95 is translated to the CANDIDATEPOS of OS/2.

    The coordinates of ptCurrentPos and rcArea of Win95 are translated to OS/2's coordinate.

    Behavior using 16 bit IME

    You can use this function as same as 32bit IME.
