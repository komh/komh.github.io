GetTextAlign/SetTextAlign (Modify)

GetTextAlign/SetTextAlign

    Functional Difference from WIN95

    none.

    Functional Difference from SBCS Open32

    The text alignment option for vertical writing is supported. 

      VTA_CENTER

    Implementation

    The option value is defined in os2win.h of SBCS Open32 as follows. 

    #define VTA_CENTER_W            TA_CENTER_W

    Need to check how the Windows 95 handles this option.
