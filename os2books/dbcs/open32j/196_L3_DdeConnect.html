    Functional Difference from SBCS Open32

    Codepage check was implemented for CONVCONTEXT. Accepts CP_WINANSI and the default ANSI codepage only.

    Implementation

    CP_WINANSI is mapped to the Windows' default ANSI codepage. Windows' default ANSI codepage is mapped to OS/2 codepage as described in the Windows Codepage v.s. OS/2 Codepage in the previous section.
