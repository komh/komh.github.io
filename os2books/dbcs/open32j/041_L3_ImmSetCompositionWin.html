ImmSetCompositionWindow

    Functional Difference from WIN95

    Nothing.

    Functional Difference from SBCS Open32

    'N/A'

    Implementation

    This API is translated to ImSetConversionWindowPos(HIMI, PCONVERSIONPOS).

    A hIMC of Win95 is used as the hIMI of OS/2.

    The CANDIDATEFORM of Win95 is translated to the CANDIDATEPOS of OS/2.

    The coordinate of ptCurrentPos and rcArea of Win95 are translated to OS/2's coordinate.

    Behavior using 16 bit IME

    You can use this function as same as a 32bit IME.
