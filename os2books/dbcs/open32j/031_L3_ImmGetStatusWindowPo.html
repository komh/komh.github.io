    Functional Difference from WIN95

    Nothing.

    Functional Difference from SBCS Open32

    'N/A'

    Implementation

    This API is translated to ImQueryStatusPos(HIMI, PPOINTL).

    The lpptPos of Win95 is converted from Win95 coordinate to OS/2 coordinate.

    Behavior using 16 bit IME

    You can use this function as same as 32bit IME.
