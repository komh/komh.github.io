Neutral Language

WIN32 defines neutral language as primary language and sub-language. Such neutral language ID can be specified in MAKELANGID macro and its result can be used by MAKELCID macro and ConvertDefaultLocale API and other locale-aware APIs that takes LCID as one of parameters. Its definition is not so clear in WIN32 specification of MSDN level 1, Oct., 1995, but the following table should be valid according to on our study of Windows 95.

Legend:
