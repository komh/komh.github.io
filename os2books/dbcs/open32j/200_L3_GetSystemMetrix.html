GetSystemMetrix

Open32 NLS Extension supports the following system metrics in addition to SBCS Open32. 

SM_CYKANJIWINDOW

    Functional Difference from WIN95

    Nothing.

    Functional Difference from SBCS Open32

    N/A. The above system metrics are all new to SBCS Open32.

    Implementation

    Open32 processes the above system metric requests as follows: 

     ----------------------------------------------------------
   System Metric                System Value(OS/2)
 ----------------------------------------------------------
   SM_KANJIWINDOW               Always returns zero since the bottom line is
                                never used for Input Method program.
 ----------------------------------------------------------
