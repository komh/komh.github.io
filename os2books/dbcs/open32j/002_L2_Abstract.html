Abstract

Open32 NLS Extension provides National Language Support (NLS) functionality to Open32 application developers who want to release their applications for International Markets, or those who want to add Multilingual Content capability to their applications.

The extended functions are designed to be compatible with Windows 95 NLS functions so that the developers can share the most of their program source code between Open32 and the most recent Win32 platforms. The Open32 NLS Extension covers the following major function categories:

  • Input Language and Input Method Editor Management

  • Character Set and Writing System Support

  • Other National Language Support based on Codepage, Locale and Unicode
