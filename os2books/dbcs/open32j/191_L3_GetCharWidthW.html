GetCharWidthW

    Functional Difference from WIN95

    N/A

    Since OS/2 GPI (GRE) does not support Unicode, Open32 NLS Extension converts each Unicode character in the specified range to codepage character, queries each character width, and sum up these widths to determine the total character width.
