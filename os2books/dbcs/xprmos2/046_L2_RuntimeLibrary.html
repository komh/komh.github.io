Runtime Library

There are DBCS NLV unique APIs such as WinDBCSIMEControl(). They are special APIs to handle DBCS input method, and applications do not use them in general. If your application happens to use them, note that the import libraries, PMNLS286.LIB and PMNLS386.LIB are not contained in the IBM Visual Age C++ for OS/2. In such case, you'll need to copy the library files from IBM Visual Age C++ for OS/2 Japanese version or from Toolkit for the previous DBCS OS/2.
