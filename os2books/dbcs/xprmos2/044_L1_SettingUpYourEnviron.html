Setting Up Your Environment For Developing DBCS Application

┌──────────────────────────────────────────────────────────────────────┐
│Basically, the SBCS IBM Visual Age C++ for OS/2 can be used for       │
│building an application program which supports DBCS.                  │
└──────────────────────────────────────────────────────────────────────┘

You can edit and compile your application programs on SBCS OS/2 Warp with the IBM Visual Age C++ for OS/2, though you can not actually perform input and output of DBCS characters.

This chapter describes how to use utilities in IBM Visual Age C++ for OS/2 to build application programs for DBCS OS/2 Warp.

Note: Still, you need the following DBCS environment to test your application programs:

  • An IBM Personal System (doesn't have to be a DBCS model). A 101 keyboard is supported by DBCS OS/2 Warp.
  • One of IBM DBCS OS/2 Warp NLV.

Although all utilities provided by IBM Visual Age C++ for OS/2 are DBCS-enabled, some modifications are required to let them build application programs which support DBCS.
