Setting Up Programs and Files

There are several issues associated with setting up programs and files. Some of these issues, such as the physical location of programs and data, are also discussed in Using the Workplace Shell in a LAN Environment. This section discusses only those issues which affect all programs and files across local and remote disks, such as the use of Extended Attributes (EAs) and file associations.
