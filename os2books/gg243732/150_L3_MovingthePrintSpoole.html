Moving the Print Spooler

Moving the spooler can be done at any time. The process is very straightforward:

  • Open the OS/2 System folder

  • Find the System Setup folder and open it

  • Find the Spooler folder and open it

  • Specify the spool path in the dialog and close it

  • Reboot and your spooler will be moved.

    Remember to check that OS/2 V2.0 deleted the old SPOOL directory when it created the new one you specified.
