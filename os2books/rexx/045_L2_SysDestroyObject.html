SysDestroyObject

    Function:

      SysDestroyObject
    Syntax:
      result = SysDestroyObject(name)

      name

        The object name. This can be specified as an object ID (for example <WP_DESKTOP>) or as a fully specified file name.
      result
        The return code from WinDestroyData. This will return 1 (TRUE) if the object was destroyed and 0 (FALSE) if the object was not destroyed.
      Purpose:
        Destroy an existing Workplace Shell object. 
        Examples:
      /* Code */ If SysDestroyObject("MyProgram") Then Do Say "Myprogram object destroyed." if SysDestroyObject("MyFolder") Then Say "Myfolder object destroyed." Else Say 'Could not destroy folder "MyFolder"' End Else Say 'Could not destroy program "MyProgram"'
