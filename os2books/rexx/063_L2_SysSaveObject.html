SysSaveObject

    Function:

      SysSaveObject
    Syntax:
      rc = SysSaveObject(name, fAsync)

      name

        The object name. This can be specified as an object ID (for example <WP_DESKTOP>) or as a fully specified file name.
      fAsync
        The object to be saved Synchronously or Asynchronously.
      rc
        This will return 1 (TRUE) if the object was saved and 0 (FALSE) if the object was not saved.
      Purpose:
        Save an existing Workplace Shell object.
