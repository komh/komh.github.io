SysCurState

    Function:

      SysCurState
    Syntax:
      SysCurState state

      state

        The desired state which to set the cursor. Should be one of the following:

      • ON'
          Display the cursor.
        'OFF'
          Hide the cursor.
        Purpose:
          Use this function to hide or display the cursor.
