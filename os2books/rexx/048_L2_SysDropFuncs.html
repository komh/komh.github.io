SysDropFuncs

    Function:

      SysDropFuncs
    Syntax:
      call SysDropFuncs
    Purpose:
      Use this function to drop all the loaded RexxUtil functions. Once this function is processed by a REXX program, the RexxUtil functions are not accessible in any OS/2 sessions.
