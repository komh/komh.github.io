Advanced REXX Functions

As you become more skilled at programming, you may want to create procedures that do more and run more efficiently. Sometimes this means adding a special function to a procedure or calling a subroutine.

In this section, we shall see how these functions can help to build a better foundation in REXX.

    FunctionsReference

      Perform a computation and return a result.
    DATATYPE( )Reference
      An internal function that verifies that the data is a specific type.
    SUBSTR( )Reference
      An internal function that selects part of a string.
    CALLReference
      Causes the procedure to look for a subroutine label and begin running the instructions following the label.
    REXX.CMD File CommandsReference
      Treat commands as expressions.
    Error MessagesReference
      Tell you if the command runs correctly. If the procedure runs correctly, no message is displayed.
