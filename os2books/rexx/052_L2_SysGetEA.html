SysGetEA

    Function:

      SysGetEA
    Syntax:
      result = SysGetEA(file, name, variable)

      file

        The file containing the extended attribute.
      name
        The name of the extended attribute.
      variable
        The name of a REXX variable in which the extended attribute value is placed.
      result
        The function result. If the result is 0, the extended attribute has been retrieved and placed in a variable. A non-zero result is the OS/2 return code of the failing function.
      Purpose:
        Read a named extended attribute from a file. 
      Examples:
  /* Code    */
  if SysGetEA("C:\CONFIG.SYS", ".type", "TYPEINFO") = 0 then do
    parse var typeinfo 11 type
    say type
    end
  /* Output */
  OS/2 Command File
