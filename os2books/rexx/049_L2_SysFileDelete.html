SysFileDelete

    Function:

      SysFileDelete
    Syntax:
      rc = SysFileDelete(file)
    Purpose:
      Deletes the specified file. Does not support wildcards.
    RC:
      Return Codes

    • File deleted successfully.

    • Error. File not found.

    • Error. Path not found.

    • Error. Access denied.

      26

        Error. Not DOS disk.
      32
        Error. Sharing violation.
      36
        Error. Sharing buffer exceeded.
      87
        Error. Invalid parameter
      206
        Error. Filename exceeds range error
