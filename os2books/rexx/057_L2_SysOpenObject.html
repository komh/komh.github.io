SysOpenObject

    Function:

      SysOpenObject
    Syntax:
      rc = SysOpenObject(name, View, Flag)

      name

        The object name. This can be specified as an object ID (for example <WP_DESKTOP>) or as a fully specified file name.
      View
        The view of an object to be opened.
      Flag
        The view of an object to be opened by calling _wpOpen (FALSE) or surface an existing view by calling _wpViewObject (TRUE).
      rc
        This will return 1 (TRUE) if the view of an object was opened and 0 (FALSE) if the view of an object was not opened.
      Purpose:
        Open a view of an existing Workplace Shell object.
