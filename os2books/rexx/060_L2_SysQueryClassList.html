SysQueryClassList

    Function:

      SysQueryClassList
    Syntax:
      call SysQueryClassList stem

      stem

        The name of a stem variable in which the entire set of registered classes is placed.
      Purpose:
        Retrieve the complete list of registered object classes. 
        Examples:
      /* Code */ call SysQueryClassList "list." do i = 1 to list.0 say 'Class' i 'is' list.i end
