SysSetIcon

    Function:

      SysSetIcon
    Syntax:
      result = SysSetIcon(filename, iconfilename)

      filename

        The name of the file to have the icon set.
      iconfilename
        The name of a .ICO file containing icon data.
      result
        The return code from WinSetIcon. This returns 1 (TRUE) if the icon was set and 0 (FALSE) if the new icon was not set.
      Purpose:
        Associate an icon file with a specified file. 
        Examples:
      /* Code */ if SysSetIcon(file, "NEW.ICO") then say 'Icon "NEW.ICO" attached to' file'.'
