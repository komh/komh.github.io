SysDeregisterObjectClass

    Function:

      SysDeregisterObjectClass
    Syntax:
      result = SysDeregisterObjectClass(classname)

      classname

        The name of the object class to deregister.
      result
        The return code from WinDeregisterObjectClass. This returns 1 (TRUE) if the class was deregistered and 0 (FALSE) if the class was not deregistered.
      Purpose:
        Deregister an object class definition from the system. 
        Examples:
      /* Code */ Call SysDeregisterObjectClass "MyClass"
