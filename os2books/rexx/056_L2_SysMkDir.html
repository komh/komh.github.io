SysMkDir

    Function:

      SysMkDir
    Syntax:
      rc = SysMkDir(dirspec)

      dirspec

        The directory that should be created.
      Purpose:
        Creates a specified directory quickly. In case of failure, one of many return codes is given so that the exact reason for failure can be determined.
      RC:
        Return Codes

      • Directory creation was successful.

      • Error. File not found.

      • Error. Path not found.

      • Error. Access denied.

        26

          Error. Not a DOS disk.
        87
          Error. Invalid parameter.
        108
          Error. Drive locked.
        206
          Error. Filename exceeds range.
        Example:
          call SysMkDir 'c:\rexx'
