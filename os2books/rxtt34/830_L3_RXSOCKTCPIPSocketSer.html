Description from the author:

"rxSock is a REXX function package providing access to the OS/2 TCP/IP socket API as provided by the IBM TCP/IP product, version 1.2.1."

See the home page of Dave Briccetty (see Internet - Web Pages) for two REXX programs using this DLL:

One to get usenet news articles using NNTP and sockets and the other to send mail to an email list with SMTP and sockets.

Note: There's a complete example for using RxSock in the issue 2, volume 2, Feb 1994 of the magazine EDM/2 written by the author of RxSock.
