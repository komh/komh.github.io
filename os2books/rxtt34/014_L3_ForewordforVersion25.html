Foreword for Version 2.50

And here it is: The next version of RXT&T.

---------- * ----------

Unfortunately, IBM decided to close the OS/2 Developer Forum 1 on CompuServe in January 1997. Therefore, this forum is not an official release point for RXT&T anymore.
But there's now a new OS/2 related forum on CompuServe called OS/2 Central. This forum is not associated with IBM and thus is independent from any of the decisions IBM will make in the future (well, mostly independent :-) )

So, the REXX section of OS/2 Central is now the official release point for RXT&T on CompuServe. And, as time goes by, I'll try to upload mostly all software mentioned in RXT&T to the library of this forum.

I invite all OS/2 Users to visit the OS/2 Central.

---------- * ----------

The fourth Edition of Dick Goran's excellent REXX Reference Summary Handbook is out!

This book is a must have for every serious REXX programmer doing WPS related programs in REXX!

---------- * ----------

Digging around with OS/2 REXX I've found some further features I did not know before regarding the use of the parameter placeholder %0 to %9 in REXX programs. Take a look at the sections Get the invocation syntax, Get the name of the MAIN REXX program called, and Get the parameters as seen by CMD.EXE to see what you can do with this feature.

---------- * ----------

Also new in this release is a section with Hints for Object REXX. At this time, this section only contains hints on using Classic REXX programs under Object REXX. But I plan to add further information about the Object-Oriented features of Object REXX as soon as I've time.

---------- * ----------

Bernd Schemmer [Team OS/2, Certified LAN Server Engineer, Certified OS/2 Engineer], Frankfurt in February 1997
