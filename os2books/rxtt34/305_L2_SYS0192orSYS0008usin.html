SYS0192 or SYS0008 using PIPEs

Problem: An OS/2 command with two or more PIPEs (|) leads to the OS/2 error message SYS0192 or SYS0008.

    Hint: The number of PIPEs in one command is limited in OS/2. See Using PIPEs for a workaround.
