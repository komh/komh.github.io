Printing REXX Tips & Tricks

To get a printed copy of RXT&T you can use the Print command under the Services menu when RXT&T is displayed by the view program

An alternative method I suggest you try is to use the inf2htm package from Peter Childs (see EMail Addresses). This program will convert the INF file into an HTML file; you can then use your favorite Web browser to print the HTML file (but please observe my Distribution policy).

The package should be available at the Hobbes archive; the name of the archive is infhtm9b.zip.

Source: Thanks to Dave Hicks (see EMail Addresses) for this information.
