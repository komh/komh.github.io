Name: MAXFH.*

Description from the author:

"MAXFH will increase the file handles allocated to the present (and parent) process.
The syntax is: MAXFH n progname
where n is the number of handles to allocate and progname is the executable to call. "

Note: The DLLs RXU and the new REXXUTIL DLL also contain functions to increase the maximum number of file handles.
