This DLL provides some mathematical functions to REXX programs:

  • cos( x )
  • sin( x )
  • tan( x )
  • acos( x )
  • asin( x )
  • atan( x )
  • cosh( x )
  • sinh( x )
  • tanh( x )
  • ceil( x )
  • floor( x )
  • exp( x )
  • log( x )
  • log10( x )
  • sqrt( x )
  • pow( xy )
  • pi()
  • e()
