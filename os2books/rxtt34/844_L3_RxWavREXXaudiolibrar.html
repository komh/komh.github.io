RxWav library is a set of function that allows to manipulate RIFF wave audio file, with change of volume level, apply filter, merge different file, etc.

You can download the source from the web site of the author (see Internet - Web Pages). The documentation and a demo program is also online available.

The functions in the beta 0.9 are:

  • WavLoadFuncs
  • WavDropFuncs
  • WavAllocTrac
  • WavDropTrac
  • WavQueryFile
  • WavReadStereo
  • WavWriteStereo
  • WavReadMono
  • WavWriteMono
  • WavCopyTrac
  • WavSinOsc
  • WavTestOsc
  • WavWhiteNoiseOsc
  • WavPinkNoiseOsc
  • WavMute
  • WavFilterBand
  • WavFilterHigh
  • WavFilterLow
  • WavGraphEQ
  • WavReverb
  • WavReverse
  • WavEco
  • WavPeek
  • WavPoke
  • WavPeak
  • WavSeek
  • WavAnalyze
  • WavDyn
  • WavAvg
  • WavVocoder
  • WavConv
