RexxUni.Dll provides a part of UniCode API for Rexx programs. It includes the following functions:

  • UniLoadFuncs()
  • UniDropFuncs()
  • UniQueryCp( [ CpName ] )
  • UniXlat( Str , [ ToCp ] [, FromCp ] )
  • UniCpName( [Cp] )
