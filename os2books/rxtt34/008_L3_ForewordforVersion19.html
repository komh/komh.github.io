Foreword for Version 1.90

And here it is: The next version of RXT&T.

This version includes additional information from various OS/2 WARP APARs regarding REXX. Please be aware that APARs are living documents! So, if you want to use some information collected from an APAR, always get the newest version of the referenced APAR. There likely is already a fix or workaround for the problem. by the time you contact IBM to get a copy of the APAR.

Beginning with this version the current version of RXT&T is also available in the internet at the home page of Quercus Systems (see Internet - Web Pages). BTW: The home page of Quercus Systems contains some very useful information about REXX. So I strongly suggest you take a look at it.

This is the last version of RXT&T for this year -- Merry Christmas and a Happy New Year to all users of RXT&T.

Bernd Schemmer [Team OS/2, Certified LAN Server Engineer, Certified OS/2 Engineer], Frankfurt in December 1995
