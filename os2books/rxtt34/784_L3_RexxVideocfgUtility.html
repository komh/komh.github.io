Rexx Videocfg Utility

Name Rexx Videocfg Utility
Version 1996
Author IBM
Distrib. part of WARP 4
Type REXX DLL
Price -
Source WARP 4
Name: RXVIDCFG.DLL

This DLL exports only the REXX function SySVCFGSetcurrentCFG which seems to change the video mode. There is no documentation for the DLL in OS/2 (because it's from IBM?? :-)), but you might look at the REXX program VCFGCID.CMD (in the directory \OS2\INSTALL of WARP 4) to get the syntax for the function.
