QuerySingleSyslevel

Usage:

call QuerySingleSyslevel syslevelfile, "syslevelstem"

where:
syslevelfile = fully qualified path to the syslevelfile
(e.g. SYSLEVELFILE = "D:WARPSRV\SYSLEVEL.WSB")

syslevelstem = stem for the results

returns:

SYSLEVELSTEM.1.MAJORVERSION = "0"
SYSLEVELSTEM.1.MINORVERSION = "50"
SYSLEVELSTEM.1.SYSLEVELPATH = "D:\WARPSRV\SYSLEVEL.WSB"
SYSLEVELSTEM.1.PREVIOUSCSDLEVEL = "IP08200"
SYSLEVELSTEM.1.PRODUCT = "IBM OS/2 Warp Server with WINOS2"
SYSLEVELSTEM.1.COMPONENTID = "562285100"
SYSLEVELSTEM.1.CSDLEVEL = "IP08200"
SYSLEVELSTEM.1.TYPE = ""
SYSLEVELSTEM.0 = "1"
SYSLEVELSTEM.1.KIND = "15"

notes:
I don't know the meaning of all returned values.

(This is a function of the DLL RLANUTIL)
