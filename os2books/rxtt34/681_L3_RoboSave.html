Name: ROBOSAVE.*

RoboSave is a program to save the OS/2 configuration (WPS, INI-Files, CONFIG.SYS and also user-defined files).

It is mentioned in this document because it contains a program named ROBOSVUT.EXE. This program can copy open files.

BTW: RoboSave is a real good backup program for your WPS.
