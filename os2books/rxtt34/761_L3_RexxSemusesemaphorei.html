This package provides some functions for using semaphores in REXX (taken from an older version; maybe the current version has more functions):

Global setup

  • SemLoad
  • SemDrop
  • SemVersion
  • SemLogState
  • SemSetLanguageDll
  • SemReturnValidAlways

Mutex Semaphores

  • SemMCreate32 (was SemCreateMutex)

  • SemMCreate16 (was SemCreateSem16)

  • SemMCreate

  • SemMClaim

  • SemMRelease

  • SemMClose

  • SemMCloseAll

  • SemMLogState

  • SemMGetAllSems

  • SemMQuerySemHandle

Event Semaphores

  • SemECreate

  • SemEClose

  • SemECloseAll

  • SemEPost

  • SemEReset

  • SemEWait

  • SemEQueryPostCount

  • SemELogState

  • SemEGetAllSems

  • SemEQuerySemHandle

Window Functions

  • RxSemListWindow

  • RxSemToggleSwitchVisibility

Miscellaneous Functions

  • OpenNetscapeWindow

  • ProcessIDsOf (Registered)

  • WordWrap (Registered)

  • RxWinSetTitle (Registered)

  • PriorityQuery (Registered)

  • PrioritySet (Registered)

  • KillProcess (Registered)

  • MyProcessID (Registered)

  • SetTextScreenSize (Registered)

  • ChecksumString (Registered)

  • ChecksumFile (Registered)
