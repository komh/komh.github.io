Distribution

This document is only distributed through electronic sources like the Internet or BBSes. The name of the file containing this document is either RXTTnn.EXE or RXTTnn.ZIP (where nn is the version number).

The EXE file is an self-extracting archive; it was packed with the free pack program LH2 Version 2.22 89/09/19 (32 bit wc) from Peter Fitzsimmons. The format of the archive created by LH2 is compatible with the format of the DOS unpack program LHA.

The ZIP file was packed with the free program ZIP.EXE (version 1.9, dated Aug 20th 1992, by the Info-Zip group). Info-Zip is Copyright (C) 1990-1992 Mark Adler, Richard B. Wales, Jean-loup Gailly and Kai Uwe Rommel, et al.

Every new version of RXT&T is announced in the news groups

comp.lang.rexx

and

comp.os.os2.announce .

The current version of RXT&T is always available at the following locations:

    Internet - WWW (3)

      at Hobbes in the directory /pub/os2/dev/rexx (see Internet - Web Pages)

    I encourage all users of this document to distribute it on all BBSes, on-line services or Internet locations they think it belongs on.
