CLTRUTIL seems to be a REXXUTIL version without the WPS related functions.

Note that you can load the functions of CLTRUTIL with the function SysLoadFuncs if booted from diskette (see Using REXXUTIL)

CLTRUTIL exports (according to an ExeHdr tool):

  • SYSFILETREE
  • SYSLOADFUNCS
  • SYSCURSTATE
  • SYSDRIVEMAP
  • SYSDROPFUNCS
  • SYSDRIVEINFO
  • SYSFILEDELETE
  • SYSSEARCHPATH
  • SYSFILESEARCH
  • SYSGETMESSAGE
  • SYSTEMPFILENAME
  • SYSCLS
  • SYSWAITNAMEDPIPE
  • SYSTEXTSCREENREAD
  • SYSTEXTSCREENSIZE
  • SYSSLEEP
  • SYSGETEA
  • SYSMKDIR
  • SYSRMDIR
  • SYSPUTEA
  • SYSOS2VER
  • SYSCURPOS

Note: CLTRUTIL is part of the OS/2 WARP Server / OS/2 WARP Connect.
