Description from the author:

"MaxUserREXX is a REXX callable dynamic link library for OS/2's REXX Procedures Language/2. With MaxUserREXX and REXX you can write REXX command scripts that will allow you to manipulate your Maximus v2.xx USER.BBS and LASTUSER.BBS files. It will also allow you to query and change system information that relates to the user in the files IPCxx.BBS, BBSTATxx.BBS and MTAG.BBS. This allows you as a REXX programmer almost complete control over your user information from the command line."
