Description from the author:

"RxExtras is a set of functions to enhance OS/2's REXX programming language, and is accompanied by additional functions to be used by other PM REXX-based software (VisPro/REXX, VX-Rexx, among others). Some of the functions provided by RxExtras can be accomplished by vrious other means using 'pure' OS/2 REXX code, but RxExtras provides an easier interface and more efficient processing.

The following categories of functions are available:

  • Input/Output
  • Rexx Environment
  • Rexx Stems
  • OS/2 Environment
  • Listboxes
  • Pushbuttons
  • SpinButtons
  • Windows
  • Containers
  • Notebooks
  • Entry Fields
  • Free Format Windows
  • Clipboard
  • Printing
  • Semaphores
"
