Using TEMPLATE.CMD

Due to a limitation of the OS/2 View program I had to split TEMPLATE.CMD into four smaller parts. So, to get the complete source code for TEMPLATE.CMD you must do the following (note: to access each part, click on the corresponding link and follow the steps below):

  • Copy the first part of the source for TEMPLATE.CMD into a file (e.g., TEXT.TMP)

  • Append the second part of the source for TEMPLATE.CMD to that file

  • Append the third part of the source for TEMPLATE.CMD to that file

  • Append the fourth part of the source for TEMPLATE.CMD to that file

  • Delete the leading empty lines and blanks from the newly created file TEXT.TMP

  • Rename TEXT.TMP to TEMPLATE.CMD

That's all! But: Do not forget to read the leading comments in the file TEMPLATE.CMD - that's the documentation!
