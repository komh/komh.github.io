The keyword instruction SIGNAL

There's a bug (or a feature) in the keyword instruction SIGNAL. Just try the following code: 

 
/* show a bug in the SIGNAL instruction */
/*                                      */
  signal value "xyz"
  return

XYZ:
  say "Hello from XYZ"
