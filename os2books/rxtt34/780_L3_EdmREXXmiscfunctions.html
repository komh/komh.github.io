The DLL contains the following functions:

    EdmRexxVer

      return the version of the DLL
    EdmLoadFuncs
      register all functions from the DLL
    EdmCountryInfo
      provides read access to current country dependent information (Countrycode, CodePage, DateFormat, Currency, Thousandsseparator, ...)
    EdmDate( format, baseDate )
      an extension of the Date() function
