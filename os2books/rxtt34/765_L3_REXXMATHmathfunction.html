This DLL provides some mathematical functions to REXX programs:

ANSI Math Functions

  • acos( x )
  • asin( x )
  • atan( x )
  • atan2( x, y )
  • ceil( x )
  • cos( x )
  • cosh( x )
  • exp( x )
  • fabs( x )
  • floor( x )
  • fmod( x, y )
  • frexp( x )
  • ldexp( x, i )
  • log( x )
  • log10( x )
  • modf( x )
  • pow( x, y )
  • sin( x )
  • sinh( x )
  • sqrt( x )
  • tan( x )
  • tanh( x )
Non-ANSI Math Functions

  • hypot( x, y )
  • j0( x )
  • j1( x )
  • jn( i, x )
  • y0( x )
  • y1( x )
  • yn( i, x )
  • pi()
Available when REXXMATH is compiled with IBM C Set/2

  • erf( x )
  • erfc( x )
  • gamma( x )
