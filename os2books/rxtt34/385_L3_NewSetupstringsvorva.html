New Setup strings vor various objects in WARP 4

[Autolink] Menu

This section contains a (perhaps incomplete) list of new and changed setup strings for objects that already exist in WARP 3, listed by class.

(The fourth edition of REXX Reference Summary Handbook contains a complete list of all possible WPS Setup strings for OS/2 2+, WARP 3 and WARP 4)

Source: Personal Systems Issue 11/96, Online edition 

     
  Class WPFontPalette

  Key Name         Value          Description
  ----------------------------------------------------------------
  AUTOSETUP        YES            Reinitialize the font palette
                                  with the default set of fonts

                   NO             Do not reinitialize the font
                                  palette

  FONTS            font[,...]     Specifies the fonts to be used
                                  in the font palette.
                                  font is in the form of size.name,
                                  where size is the point size and
                                  name is the name of the font
                                  including modifiers.
                                  For example, specify
                                    9.Helv
                                  for a 9-point Helvetica font.

                                      r   g   b   set   of   numbers .

                                      Icons   View .   color   may   be   a
                                      standard   color   name   or   an   r   g   b   set

   ICONTEXTVISIBLE    YES               The   names   of   objects   contained   in   a
                                      folder   are   displayed   in   the   Icon   View .

                                      folder   are   not   displayed   in   the
