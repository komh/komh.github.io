Foreword for Version 1.70

This version of REXX Tips & Tricks (to be exact: All sections already contained in the version 1.60) was edited by Dave Hicks (see EMail Addresses) for proper, correct English. So this version should be free of grammatical errors -- assuming that I haven't inserted new bugs in the new & updated sections in this version. Dave also sends me a lot of new information and some bug reports for RXT&T.

Many thanks to Dave for his great work!

Also thanks to the other users who send me bug reports and new information for REXX Tips & Tricks. I hope that I've mentioned all of them in the section Hints & Bug reports. Please let me know if I've forgotten someone so I can correct this mistake in the next version of RXT&T.

A note to the German users of this document:

There's definitely no German version of this version of RXT&T! I mention this because some German readers of the previous version asked me for a German version of RXT&T. I've only used the German language for the version 1.1, 1.2, 1.3 and 1.4. Since version 1.5 RXT&T is only available in English language (There is some German information in this document, see Using IPF Keys for the details).

If you really want a German version of RXT&T you might check if one of the old German version is still available in one of your local BBSes. The filename of the archive with the last German version is RXTT14.EXE.
Generally I suggest you use the current English version and send me an EMail if you've got a question concerning RXT&T (You can use the German language for EMails).

Sorry about that but there are many more user understanding English than German.

Bernd Schemmer [Team OS/2], Frankfurt in August 1995 (see also the General Foreword)
