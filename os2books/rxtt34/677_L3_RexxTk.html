An extension library for use with most versions of Rexx and providing access to the Tk Graphic User Interface Toolkit.

Requirements:

  • A version of Rexx that supports use of extension libraries. (e.g. Classic Rexx, Object Rexx or Regina)
  • Version 8.0 (or later) of the freely available Tcl/Tk distribution. There's a link to TCL/TK for OS/2 on the download page for Rexx/tk

Special Note for the version rexxtk20os2 (as of 07/01/2002):

In the ZIP file (rexxtk20os2.zip) is the necessary DLL rexxtran.dll missing; the WPI file (rexxtk20os2.wpi) is okay. So you should download the WPI file!
