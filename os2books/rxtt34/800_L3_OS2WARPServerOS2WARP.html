OS/2 WARP Server / OS/2 WARP Connect

The OS/2 WARP Connect CD-ROM and OS/2 WARP Server CD-ROM (Beta version). both contain 3 REXX DLLs for the CID installation. Unfortunately there's no documentation for these DLLs on the CD-ROM. Because it seems that the DLLs may also be useful for other REXX programs, I've included the list of exported functions in RXT&T:

  • CLTRUTIL
  • RLANUTIL
  • RXUTILS

Note: Unfortunately only a few functions are used in the code. So I'm very interested in further information about these DLLs.
