Description from the author:

"APMT provides a set of executables and a REXX function package. Using the functions provided you can write a REXX program which can drive a PM application running on the same or a different PC. Thus the REXX program can act as a test script for testing the PM application.
Note however that APMT does not have any record/playback features.
Another use of APMT is to automatically run a PM program on another machine for, for example, monitoring purposes."
