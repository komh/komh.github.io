The DIR command

Use the switch /R of the DIR command to get the long name of a file in a DOS session (works only for files and directorys that have the EA .LONGNAME set)

Example output of DIR with the /R switch: 

 
BMT_MICR        134173  29.05.96 11.28 BMT Micro Shareware Catalog
