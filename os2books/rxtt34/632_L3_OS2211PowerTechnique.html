"OS/2 2.11 Power Techniques" is the best general reference about OS/2 I know.

For REXX programmers, the very detailed section about the OS/2 ini files, the RC files to create the ini files and the setup strings for the WPS objects are of special interest.

This book is a 'must have' for all people who want to do some changes to the WPS using REXX.
