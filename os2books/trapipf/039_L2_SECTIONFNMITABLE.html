----------------------------------------------------------------

----------------------------------------------------------------
          |THE SYSTEM DETECTED A HARDWARE MEMORY ERROR ON THE

  2       |OPTION CARD.  RUN OPTION CARD DIAGNOSTICS (MEMORY
          |EXPANSION DIAGNOSTICS, ETC.)

          |THE SYSTEM DETECTED A TIMEOUT ON THE DMA CHANNEL
  3       |(INDICATES A DMA TIMEOUT; MAY BE CAUSED BY FIXED DISK
          |ERROR.
