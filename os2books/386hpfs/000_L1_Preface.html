Preface


The purpose of this document is to consolidate in one reference the features and architecture of 386 HPFS, and important considerations of its implementation and usage.

The source material was gathered from several available documents, as well as from IBM internal technical databases. The source documents were:

o

    LAN Server 3.0 Network Administrator Reference Volume 1: Planning & and Installation, document S96F-8428-00
o
    LAN Server 3.0 Network Administrator Reference Volume 2: Performance Tuning, document S96F-8429-00
o
    LAN Server 3.0 Network Administrator Reference Volume 3: Network Administrator Tasks, document S96F-8430-00
o
    Network Administrator Reference Supplement for OS/2 2.0, document S04G-1080-00

Also included are sections on LAN Server 3.0 Entry and Advanced architecture, as it is important to consider the differences between the two.
