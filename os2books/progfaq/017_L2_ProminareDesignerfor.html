Prominare Designer for OS/2

Prominare Inc, of Canada have launched a development tool for creating GUI's for OS/2. The tool can generate code for multiple class libraries and multiple compilers.

Supported Platforms are OS/2, Windows 3.1 and Windows NT.

The tool supports all PM controls, MMOS/2 and Pen for OS/2.

Suggested retail price: $699 (US).

A complete list of qualified resellers can found through Prominare's WWW server at http://www.prominare.com/prominare.
