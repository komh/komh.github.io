CSD levels for IBM C Set++?

Following CSD's are current

o

    CTC0011 - IBM C/C++ Tools (Compiler) 2.0 and 2.01
o
    CTU0003 - IBM C/C++ Tools (Utilities) 2.0 and 2.01
o
    CTL0009 - IBM C/C++ Tools (Class Libraries) 2.0
o
    CTM0009 - IBM C/C++ Tools (Class Libraries) 2.01

Be careful with CTL0009 and CTM0009. Do NOT install wrong CSD. Check with the OS/2 command SYSLEVEL the version level of the class libraries. IBM C/C++ packages bought later than November 1993 should be 2.01 or higher.
