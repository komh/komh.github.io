How do I get the error message from a DOS API call?

For DOSAPI calls, you can issue a DosGetMessage as follows: 

   RC = DosXXXX(...);
   DosGetMessage(NULL,0,Msg,sizeof(Msg),RC,"OSO001.MSG",&Msg_Length);

Credit: Ken Kahn
