Lab Session 3 - The Clipboard Viewer

In this exercise, you are using the clipboard support for the VDM environment. Partial and complete copying of text and graphical screen contents is the main task of this exercise as follows:

  • Fill a VDM windowed session with text data (DIR /W) and copy the screen contents to the clipboard. Use the Clipboard Viewer to check your results.

  • Create some graphics in VDM windowed session (GRAPHIC.EXE) and copy a part of the graphics to the clipboard. Again check the Clipboard Viewer.

  • Copy text (more than one line!) from the clipboard into EDLIN.
