DOS Emulation

Subcomponent of the Multiple Virtual DOS Machines component of OS/2 Version 2.0, which provides emulation of DOS software services to applications running in virtual DOS machines. DOS Emulation either provides an equivalent service or invokes the appropriate protected mode service within the OS/2 Version 2.0 kernel. Also known as DOSEM.
