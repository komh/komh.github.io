Multiple Application VDM

Method of running Windows applications under OS/2 Version 2.0, whereby the Windows Program Manager is started in a virtual DOS machine, and Windows applications are started using the Program Manager. Errors in any Windows application may potentially result in termination of the entire VDM This is method of running Windows applications is not recommended unless applications require communication using shared memory.
