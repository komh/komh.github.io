virtual machine monitor

Routine supplied by operating systems which utilize the virtual 8086 mode of the 80386 processor, to provide emulation of interrupts and exceptions resulting when an 8086 application issues an instruction which cannot be executed in virtual 8086 mode. The 8086 Emulation component of MVDM is a virtual machine monitor.
