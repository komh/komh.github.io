high memory area

The first 64KB of extended memory above 1MB. The High Memory Area (HMA) is unique in that code can be executed in it while in real mode, using the A20 address line wraparound. The HMA officially starts at FFFF:0010h and ends at FFFF:FFFFh, making it slightly less than 64KB in length.
