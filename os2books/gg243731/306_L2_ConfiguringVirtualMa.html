Configuring Virtual Machine Boot

The DOS operating system loaded into a VDM by VMB may be:

  • An actual DOS system diskette
  • An image of a DOS system diskette saved to hard disk
  • A DOS partition on hard disk.

    For any of the alternatives, we need to do the following:

  • Set up the start-up batch file AUTOEXEC.BAT

  • Set up the configuration file CONFIG.SYS

  • Provide OS/2 V2.0 with the real DOS to boot.
