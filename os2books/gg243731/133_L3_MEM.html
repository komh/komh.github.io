MEM

MEM displays the amount of used and free memory in the DOS environment. It lists information about allocated memory areas, free memory areas, and programs that are currently loaded into memory.

The format of the MEM command is: 

    mem [/program | /debug | /classify]

where:

  • p(rogram)
      displays the status of programs that are currently loaded into memory.
    /d(ebug)
      displays the status of currently loaded programs, internal drivers and other programming information.
    /c(lassify)
      displays the status of programs loaded into conventional memory and the upper memory area.

    MEM only runs in a DOS session.
