Installation of DPMI

The OS/2 Version 2.0 installation procedure copies the DPMI API Layer virtual device driver DPM.SYS into the \OS2\MDOS directory on the user's system. If users decide to use selective install, they can choose any combination of EMM, XMS, or DPMI. When they select any of these memory options, the appropriate DEVICE= statement is added to CONFIG.SYS. The default memory statement in CONFIG.SYS is: 

DEVICE=C:\OS2\MDOS\VEMM.SYS

If the user does not select DPMI support at installation time and wishes to add it at a later time, CONFIG.SYS must be modified by adding the statement: 

DEVICE=C:\OS2\MDOS\VDPMI.SYS
