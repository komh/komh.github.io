Requirements

The following are required to do the labs:

  • OS/2 Version 2.0

  • DOS Version 5.0

  • CLIPVIEW.EXE program, in the productivity folder

  • The following programs in your C:\ITSCLABS directory:

    • ENVIRON.EXE program

    • GRAPHIC.EXE program

    • INT19.EXE program

    • QENV.BAT program

    • QCONFIG.EXE program

    • SOUND.EXE program

    • PMCHART.EXE program

    • PaintBrush program

    • WinGif program
