watchdog timer

Hardware timer provided by the 80386 processor to ensure that 8086 applications running in virtual 8086 mode do not disable interrupts for an unnecessarily long period. Such lengthy disabling may cause unrecoverable system errors. The watchdog timer generates periodic non-maskable interrupts which allow an operating system to detect an errant 8086 application and terminate it.
