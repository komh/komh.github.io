Installation Preparation

The following are required before starting the installation:

  • Basic DOS PC Support/400 Installation diskette(s) V2R1 or later

  • DOS 5.0 bootable diskette, formatted with the system (/S) option

  • If using a LAN, the LAN Support Program diskette 1.22 or later

  • DOS PC Support/400 Installation and Administration Guide (SC41-0006).
