Steps

  • Ask your instructor for a bootable DOS 5.0 diskette.

  • Create an image of this diskette, using the VMDISKS utility.

  • Copy a DOS session object onto the desktop.

  • Change the "DOS_STARTUP Drive" setting for it.

  • Try to boot this image.

  • Create another DOS object, using the C:\ITSCLABS\IBMDOS33.VMB image.

  • Boot this one as well.

  • Study the CONFIG.SYS of these DOS diskettes.

  • Check things like HPFS, XMS, EMS, mouse, etc.
