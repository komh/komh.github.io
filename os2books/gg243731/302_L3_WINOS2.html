WIN-OS/2

The following setting is available when the selected virtual DOS machine type is WIN-OS/2 full screen or WIN-OS/2 window:

WIN_RUNMODE

Function:

    OS/2 V2.0 can use two modes to run Windows programs:

    Real

    Standard

      Real mode is the mode that Windows 2.0 programs run in. Windows 3.0 programs usually run in standard mode. For a detailed discussion, see Windows Applications. Use this setting to specify WIN-OS/2 mode for your session.

    Default:
      AUTO (the system selects standard mode as long as it has the OS/2 V2.0 Virtual Device Drivers required to support a standard mode WIN-OS/2 session in the OS/2 V2.0 operating system). AUTO enables the system to automatically choose between real and standard.
    Settable:
      At VDM creation only.
    Examples:
      None.
