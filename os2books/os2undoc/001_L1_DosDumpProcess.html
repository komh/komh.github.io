DosDumpProcess

    Prototype:

      APIRET APIENTRY DosDumpProcess ( ULONG Action, ULONG Drive, PID Pid ) ;
    Linkage Definition:

      IMPORTS DosDumpProcess = DOSCALLS.113
    Parameters:

        ULONG Action Operation to perform (see comments).
        ULONG Drive Dump drive ('A'=A: and so on).
        PID Pid Process ID of process to be dumped.

    Comments:

    The Action parameter can take the following values:

        00000000 - Disable Process Dumps 00000001 - Set Dump Drive
        00000002 - Dump Process Data
