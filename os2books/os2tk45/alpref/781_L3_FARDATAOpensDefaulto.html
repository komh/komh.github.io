.FARDATA (Opens Default or Named Far Data Segment)

When used with .MODEL, starts a far data segment for initialized data.

    Syntax 

      .FARDATA [SegmentName]
    Remarks

      If the SegmentName parameter is not specified, the assembler sets it to FAR_DATA.
