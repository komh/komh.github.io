Examples

CODE     SEGMENT
         ASSUME CS:CODE, DS:CODE
Value    DW   0
         MOV  BX,offset Value          ; load the address of Value into BX
         MOV  [BX],BX                  ; store the contents of BX into the
                                       ; memory location addressed by [BX]
CODE     ENDS
