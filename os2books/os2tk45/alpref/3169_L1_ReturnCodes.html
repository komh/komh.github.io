Return Codes

When ALP completes, it passes a return code back to the program that invoked it. This return code shows whether ALP completed successfully or with an error.

The return codes are:

  • Normal program completion.
  • User-specified file not found.
  • Unexpected system error.
  • Terminated by user or operating system.
  • Syntax errors in input file.
  • Command line usage error.
  • Internal sanity check failure.
  • Error accessing ALP messages file.
