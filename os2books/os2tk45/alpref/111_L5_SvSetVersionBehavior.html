Sv - Set Version Behavior

This option controls the various modes of compatibility that the assembler is designed to emulate. The argument to the Sv option must be one of the following keywords:

    ALP

      Native operating mode; don't emulate other assemblers
    M510
      Emulate Microsoft MASM Version 5.10
    M600
      Emulate Microsoft MASM Version 6.00
    M611
      Emulate Microsoft MASM Version 6.11
    M612
      Emulate Microsoft MASM Version 6.12
    M613
      Emulate Microsoft MASM Version 6.13
    M614
      Emulate Microsoft MASM Version 6.14
