Examples

CODE     SEGMENT
         ASSUME CS:CODE, DS:CODE

Value    DB  0                     ; Value[0]
         DB  1                     ; Value[1]
         DB  2                     ; Value[2]
         DB  3                     ; Value[3]
         DB  4                     ; Value[4]

  MOV    AL, Value[3]              ; load AL with the fourth byte at Value (3)
  MOV    BX, offset Value          ; get address of Value
  MOV    AL, [BX][1][2]            ; also gets the fourth byte (3)

CODE     ENDS
