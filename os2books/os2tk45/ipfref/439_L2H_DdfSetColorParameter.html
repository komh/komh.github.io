DdfSetColor Parameter - fForColor

    fForColor (COLOR) - input

      Specifies the desired foreground color.

      The following color value constants may be used for the foreground and background colors:

        CLR_DEFAULT - used to set IPF default text color
        CLR_BLACK
        CLR_BLUE
        CLR_RED
        CLR_PINK
        CLR_GREEN
        CLR_CYAN
        CLR_YELLOW
        CLR_BROWN
        CLR_DARKGRAY
        CLR_DARKBLUE
        CLR_DARKRED
        CLR_DARKPINK
        CLR_DARKGREEN
        CLR_DARKCYAN
        CLR_PALEGRAY
