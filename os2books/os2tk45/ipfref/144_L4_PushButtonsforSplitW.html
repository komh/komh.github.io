Push Buttons for Split Windows

Be careful when using heading tags to define a control area for split windows. A control area cannot be defined in the secondary window heading tag of a split window. You must define the control area (the coverpage window) in the primary window heading tag. In the previous examples of split windows, the push button feature of IPF was disabled (see Disabling the Display of Push Buttons).
