HM_DISMISS_WINDOW - Syntax

This message tells the Help Manager to remove the active help window. 

param1
    ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
    ULONG  ulReserved  /*  Reserved value, should be 0. */

returns
    ULONG  rc          /*  Return code. */
