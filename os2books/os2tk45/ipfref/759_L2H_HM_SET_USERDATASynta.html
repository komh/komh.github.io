HM_SET_USERDATA - Syntax

The application sends this message to IPF to store data in the IPF data area. 

param1
    ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
    VOID   usrdata     /*  4-byte user data area. */

rc
    ULONG  rc          /*  Return code. */
