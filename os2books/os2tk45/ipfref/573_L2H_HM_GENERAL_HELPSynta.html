HM_GENERAL_HELP - Syntax

When the Help Manager receives this message, it displays the general help window for the active application window. 

param1
    ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
    ULONG  ulReserved  /*  Reserved value, should be 0. */

returns
    ULONG  rc          /*  Return code. */
