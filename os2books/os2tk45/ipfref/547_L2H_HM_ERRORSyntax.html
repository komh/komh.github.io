HM_ERROR - Syntax

This message notifies the application of an error caused by a user interaction. 

param1
    ULONG  ulErrorCode  /*  Error code. */

param2
    ULONG  ulReserved   /*  Reserved value, should be 0. */

returns
    ULONG  ulReserved   /*  Reserved value, should be 0. */
