DdfListItem - Errors

Possible returns from WinGetLastError

    HMERR_DDF_MEMORY (0x3001)

      Not enough memory is available.
    HMERR_DDF_LIST_UNINITIALIZED (0x3008)
      No definition list has been initialized by DdfBeginList.
