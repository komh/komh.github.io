Attribute Values for Window Origin and Size

Both the heading tags and :link. have attributes that define window origin and size. The heading tag has four attributes:

    x=

      Specifies a point on the x-axis. The x-axis runs horizontally from left to right.
    y=
      Specifies a point on the y-axis. The y-axis runs vertically from bottom to top.
    width=
      Specifies the width (horizontal space) of the window.
    height=
      Specifies the height (vertical space) of the window.

    The :link. tag also has four attributes:

    vpx=

      Specifies a point on the x-axis. The x-axis runs horizontally from left to right.
    vpy=
      Specifies a point on the y-axis. The y-axis runs vertically from bottom to top.
    vpcx=
      Specifies the width (horizontal space) of the window.
    vpcy=
      Specifies the height (vertical space) of the window.
