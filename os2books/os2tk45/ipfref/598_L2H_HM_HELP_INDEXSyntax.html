HM_HELP_INDEX - Syntax

When the Help Manager receives this message, it displays the help index window. 

param1
    ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
    ULONG  ulReserved  /*  Reserved value, should be 0. */

returns
    ULONG  rc          /*  Return code. */
