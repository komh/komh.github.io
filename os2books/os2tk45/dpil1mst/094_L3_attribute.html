attribute

  • A characteristic that identifies and describes a managed object. The characteristic can be determined, and possibly changed, through operations on the managed object.

  • Information within a managed object that is visible at the object boundary. An attribute has a type, which indicates the range of information given by the attribute, and a value, which is within that range.

  • Variable data that is logically a part of an object and that represents a property of the object. For example, a serial number is an attribute of an equipment object.

  • In the Desktop Management Interface (DMI), a piece of information about a component.
