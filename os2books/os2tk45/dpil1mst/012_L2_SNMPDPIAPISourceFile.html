SNMP DPI API Source Files

The following source files are provided:

    snmp_dpi.h

      The public SNMP DPI 2.0 API as provided to the DPI subagent programmer. The DPI subagent code must include this file.
    dpi_samp.c
      A very basic example of an SNMP DPI 2.0 subagent implementation, dpiSimp.mib.
    dpiSimp.mib
      The dpiSimple MIB that goes with the dpi_samp.c source.

    The DPI subagent programmer can use the snmp_dpi.h include file and the dpi_samp.c file as an example of using the DPI API.
