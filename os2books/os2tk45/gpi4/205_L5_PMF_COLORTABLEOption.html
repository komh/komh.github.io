PMF_COLORTABLE Option

The PMF_COLORTABLES option may be used to append or replace color table entries in the presentation space from the metafile. The PMF_COLORTABLES option can have one of the values shown in the following table.

PMF_COLORTABLES Options for GpiPlayMetaFile 

┌────────────────────┬────────────────────────────────────────┐
│Value               │Description                             │
├────────────────────┼────────────────────────────────────────┤
│CTAB_REPLACE        │Replaces the logical color table in the │
│                    │presentation space with the color table │
│                    │in the metafile.                        │
├────────────────────┼────────────────────────────────────────┤
│CTAB_REPLACEPALETTE │Replaces the current palette, if it     │
│                    │exists, in the presentation space with  │
│                    │the palette in the metafile.            │
├────────────────────┼────────────────────────────────────────┤
│CTAB_NOMODIFY       │Maintains the logical color table in the│
│CTAB_DEFAULT        │presentation space.                     │
└────────────────────┴────────────────────────────────────────┘
