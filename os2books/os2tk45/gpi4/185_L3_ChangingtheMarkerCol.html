Changing the Marker Color

The following figure shows how to set the marker foreground color to green. 

#define INCL_GPIPRIMITIVES
#include <os2.h>
void fncMARK04(void){
    HPS hps;
    MARKERBUNDLE mbnd;

    mbnd.lColor = CLR_GREEN;
    GpiSetAttrs(hps, PRIM_MARKER, MBB_COLOR, 0L, &mbnd);
} /* fncMARK04 */
