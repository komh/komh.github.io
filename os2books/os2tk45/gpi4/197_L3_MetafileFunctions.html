Metafile Functions

The OS/2 operating system provides a set of functions that allow you to:

  • Create a metafile
  • Store pictures in a metafile
  • Play a metafile
  • Save a metafile
  • Load a metafile
  • Edit a metafile
  • Copy a metafile
  • Delete a metafile
