PMF_DEFAULTS Option

The PMF_DEFAULTS option can have one of the values shown in the following table.

PMF_DEFAULTS Options for GpiPlayMetaFile 

┌────────────────────┬────────────────────────────────────────┐
│Value               │Description                             │
├────────────────────┼────────────────────────────────────────┤
│DDEF_LOADDISC       │Replaces the default attributes, default│
│                    │viewing limits, and default arc         │
│                    │parameters in the presentation space    │
│                    │with the values specified in the        │
│                    │metafile.                               │
├────────────────────┼────────────────────────────────────────┤

│DDEF_DEFAULTS       │default viewing limits, and default arc │

└────────────────────┴────────────────────────────────────────┘
