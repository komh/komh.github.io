pglCreateContext Parameter - IsDirect

    IsDirect (BOOL) - input

      Context to be created is direct.

      TRUE

        Bypass Graphics Programming Interface and render directly to a PM window. (This method is generally faster.)
      FALSE
        Use the GpiBitBlt function to blit OpenGL rendering to a PM window.
