pglQueryCapability Return Value - rc

    rc (LONG) - returns

      NULL

        No OpenGL Support on this machine.
      1
        OpenGL Support available through PM and DIVE.
      2
        OpenGL support available through PM blitting only.
      3
        Advanced OpenGL support available.
