pglQueryConfigs - Parameters

    hab (HAB) - input

      Handle to anchor block.

    pVisualConfig (PVISUALCONFIG) - returns

      A NULL-terminated array of pointers to VISUALCONFIG structures. The list of structures is not guaranteed to be in any certain order. A NULL is returned if no suitable VISUALCONFIG was found.
