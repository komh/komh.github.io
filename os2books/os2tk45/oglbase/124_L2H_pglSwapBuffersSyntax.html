pglSwapBuffers - Syntax

This function swaps the front and back buffers of hwnd. Double-buffered rendering is done when smooth animation between frames is desired. 

#include <pgl.h>

HAB     hab;
HWND    hwnd;

pglSwapBuffers(hab, hwnd);
