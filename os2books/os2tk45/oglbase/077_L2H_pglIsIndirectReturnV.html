pglIsIndirect Return Value - rc

    rc (LONG) - returns

      The following values can be returned:

    • This Context bypasses GPI and is faster.
      < 0
        Error occurred.
      1
        This Context uses conventional PM blitting methods and is slower. However it allows an application to integrate OpenGL and GPI rendering commands. See Remarks section.
