pglMakeCurrent - Syntax

This function binds an OpenGL context to an OS/2 PM window. Only one context can be bound to a window at a time. 

#include <pgl.h>

HAB     hab;
HGC     hgc;
HWND    hwnd;
BOOL    rc;

rc = pglMakeCurrent(hab, hgc, hwnd);
