pglChooseConfig - Syntax

This function returns a pointer to a visual configuration structure that meets the specified minimum requirements. A PVISUALCONFIG structure is required when creating an OpenGL rendering context. 

#include <pgl.h>

HAB                hab;
int               *attriblist;
PVISUALCONFIG     *pVisualConfig;

pVisualConfig = pglChooseConfig(hab, attriblist);
