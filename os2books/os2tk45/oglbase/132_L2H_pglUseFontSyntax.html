pglUseFont - Syntax

This function creates the specified number (count) of OpenGL display lists containing bitmaps of the named logical character set identifier (llcid). 

#include <pgl.h>

HAB       hab;
HPS       hps;
FATTRS    fattrs;
LONG      llcid;
int       first;
int       count;
int       listbase;
BOOL      rc;

rc = pglUseFont(hab, hps, fattrs, llcid, first,
       count, listbase);
