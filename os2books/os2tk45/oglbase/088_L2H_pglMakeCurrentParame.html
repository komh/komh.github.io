pglMakeCurrent - Parameters

    hab (HAB) - input

      Handle to anchor block.

    hgc (HGC) - input

      Handle to an OpenGL context.

    hwnd (HWND) - input

      Handle to a PM window.

    rc (BOOL) - returns

      The following values may be returned:

      TRUE

        Context was successfully destroyed, or context was bound to a window.
      FALSE
        Error occurred.
