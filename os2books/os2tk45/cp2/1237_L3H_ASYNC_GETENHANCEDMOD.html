ASYNC_GETENHANCEDMODEPARMS (74h) - Description

This function returns the Enhanced mode parameters for a COM port. It also returns the state of the current DMA-operation mode and user-requested settings for DMA receive/transmit operations. This IOCtl is not currently supported by COM.SYS.
