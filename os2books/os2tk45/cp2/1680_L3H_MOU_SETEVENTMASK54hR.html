MOU_SETEVENTMASK (54h) - Remarks

The physical mouse device driver gets the new mask for enabled/disabled mouse device events from the caller's Parameter Packet. This mask determines which events are to be queued (that is, to be read by calling MOU_READEVENTQUE).
