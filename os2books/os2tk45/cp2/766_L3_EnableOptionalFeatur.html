Enable Optional Features (CVK_CMD_ENA)

Parameters:

    Cmd

      CVK_CMD_ENA
    Value
      Index of feature to enable.

    • CVK_RET_TNEW and CVK_RET_TEND notifications.

    Returns:

    Cmd

      Return code.

    Note: All optional features are disabled when a CVK_CMD_RUN command is issued with a Value of -CVK_CMD_RUN.
