PRT_VERIFYFONT (6Ah) - Font ID

    Font ID

      The Font ID value to validate. Values can be 0 - 65535. The Font ID is contained in the font file for downloadable fonts. For cartridge fonts, Font IDs are the numbers on the cartridge label, and are entered in the DEVINFO statement for the printer.

      Note: A value of 0 for both the Code Page number and Font ID indicates the default hardware code page and font. This combination is always valid.
