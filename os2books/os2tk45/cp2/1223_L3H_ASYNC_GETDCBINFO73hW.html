ASYNC_GETDCBINFO (73h) - Write Timeout

    Write Timeout

      Specifies the time period used for Write Timeout processing. See Note 8 of ASYNC_SETDCBINFO. The value is in .01 second units based on zero (where 0 = .01 seconds).
