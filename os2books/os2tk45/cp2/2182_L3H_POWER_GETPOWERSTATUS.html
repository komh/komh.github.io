POWER_GETPOWERSTATUS (60h) - BatteryStatus

    BatteryStatus

      Contains the status value for the DC battery power source. Supported values are:

    • High
    • Low
    • Critical
    • Charging
      255
        = Unknown
      All other values are reserved.
