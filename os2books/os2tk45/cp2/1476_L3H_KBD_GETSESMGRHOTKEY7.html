KBD_GETSESMGRHOTKEY (76h) - Remarks

This request obtains the scan code the physical keyboard device driver uses as the Session Manager hot key. It should first be called with parameter list subtype=0 to determine the maximum number of hot keys the physical device driver can support. The value returned is used to determine the required size of the data buffer on a subsequent call to return the hot key data structures (parameter list subtype=1).
