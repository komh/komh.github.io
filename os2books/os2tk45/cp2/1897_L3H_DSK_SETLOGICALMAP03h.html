DSK_SETLOGICALMAP (03h) - Logical Drive Number

    Logical Drive Number

      On entry, a logical drive number (1=A, 2=B, and so forth) is specified. On return, this byte specifies the logical drive currently mapped to the drive that the specified file handle is opened on. A zero is returned if there is only one logical drive mapped onto this physical drive.
