OEMHLP_GETOEMADAPTIONINFO (00h) - Remarks

OEMs can add nonstandard OEMHLP IOCtls to the OS/2 operating system if they market the OS/2 operating system with their own product name. Programs that use these IOCtls work only with that OEM's adaptation of the OS/2 operating system. Before issuing nonstandard calls, your device driver initialization code should issue the Query OEM Adaptation Information IOCtl routine to verify the OEM name field.
