CDROMAUDIO_GETSUBCHANNELQ (63h) - Description

This function reads and returns the address information from a channel of the current head location. The function does not interrupt the present state of the drive. One of its intended purposes is to monitor the information of the read head while the drive is playing audio tracks.

Note: The Running Time within a track and Running Time on a disc are returned in binary.
