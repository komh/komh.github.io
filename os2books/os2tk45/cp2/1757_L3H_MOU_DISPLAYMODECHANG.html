MOU_DISPLAYMODECHANGE (5Dh) - Remarks

This call does not provide return parameters or support input parameters. If the pointer was hidden on the mode switch start (Function 51h), then it is redrawn.

This function is the complement to MOU_UPDATEDISPLAYMODE.
