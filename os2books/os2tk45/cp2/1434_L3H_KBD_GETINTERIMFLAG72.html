KBD_GETINTERIMFLAG (72h) - Flags

    Flags

      A 1-byte field containing flag bits. A bit set to 1 indicates the states listed below:

      Bit 7

        Interim console flag on
      Bit 6
        Reserved = 0
      Bit 5
        Program requested on-the-spot conversion
      Bit 4
        Reserved = 0
      Bit 3
        Reserved = 0
      Bit 2
        Reserved = 0
      Bit 1
        Reserved = 0
      Bit 0
        Reserved = 0
