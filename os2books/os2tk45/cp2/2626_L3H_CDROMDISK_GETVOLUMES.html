CDROMDISK_GETVOLUMESIZE (78h) - Description

This function returns the total number of accessible sectors on the disc. The value returned is the same as that returned in the extended BIOS parameter block by DSK_GETDEVICEPARAMS. This function is performed regardless of whether the drive is already playing.
