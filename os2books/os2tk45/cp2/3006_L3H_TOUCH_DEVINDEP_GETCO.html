TOUCH_DEVINDEP_GETCOORDSYS (60h) - Data Packet Format

The data packet is a location in application storage where the Touch Display driver returns to the caller the coordinate system being used to report X,Y,Z data report values, including maximum and minimum limits and change report hysteresis values. The format is as follows: 

┌─────────────────────────────────┐
│Field     Length     C Datatype  │
├─────────────────────────────────┤
│X min     WORD       USHORT      │
├─────────────────────────────────┤
│X max     WORD       USHORT      │
├─────────────────────────────────┤
│Y min     WORD       USHORT      │
├─────────────────────────────────┤
│Y max     WORD       USHORT      │
├─────────────────────────────────┤
│Z min     WORD       USHORT      │
├─────────────────────────────────┤
│Z max     WORD       USHORT      │
├─────────────────────────────────┤
│X hys     WORD       USHORT      │
├─────────────────────────────────┤
│Y hys     WORD       USHORT      │
└─────────────────────────────────┘
