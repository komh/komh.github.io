MOU_UPDATEDISPLAYMODE (51h) - Length

    Length

      An input parameter to VioSetMode. Length specifies the length of the data structure in bytes including Length itself. The minimum structure size required is 3 bytes; the maximum structure size required is 34 bytes. OS/2 2.1 sets the first mode (in the list of modes supported by this display configuration) with a data structure matching the mode data specified.
