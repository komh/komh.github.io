MOU_GETMICKEYCOUNT (61h) - Mickeys/Centimeter

    Mickeys/Centimeter

      The number of mickeys/centimeter supported by the mouse device. Return values are in the range of: 
          0 < number of mickeys/centimeter <= (32K - 1)
