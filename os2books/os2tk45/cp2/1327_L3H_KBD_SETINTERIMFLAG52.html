KBD_SETINTERIMFLAG (52h) - Flag

    Flag

      A 1-byte field containing bit flags. A bit set to 1 indicates the state listed below:

      Bit 7

        Interim console flag on.
      Bit 6
        Reserved. Must equal 0.
      Bit 5
        Program requested on-the-spot conversion.
      Bit 4
        Reserved. Must equal 0.
      Bit 3
        Reserved. Must equal 0.
      Bit 2
        Reserved. Must equal 0.
      Bit 1
        Reserved. Must equal 0.
      Bit 0
        Reserved. Must equal 0.
