DSK_WRITETRACK (44h) - Track Layout Table

    Track Layout Table

      Has the following: 
      ┌─────────────────────────┬───────┐
│Field                    │Length │
├─────────────────────────┼───────┤
│Sector Number for Sector │WORD   │
│1                        │       │
├─────────────────────────┼───────┤
│Sector Size for Sector 1 │WORD   │
├─────────────────────────┼───────┤
│Sector Number for Sector │WORD   │
│2                        │       │
├─────────────────────────┼───────┤
│Sector Size for Sector 2 │WORD   │
├─────────────────────────┼───────┤
│Sector Number for Sector │WORD   │
│3                        │       │
├─────────────────────────┼───────┤
│Sector Size for Sector 3 │WORD   │
│   .                     │  .    │
│   .                     │  .    │
│   .                     │  .    │
├─────────────────────────┼───────┤
│Sector number for Sector │WORD   │
│n                        │       │
├─────────────────────────┼───────┤
│Sector size for Sector n │WORD   │
└─────────────────────────┴───────┘

      The sector table that is specified provides information that is used during the WRITE track operation.
