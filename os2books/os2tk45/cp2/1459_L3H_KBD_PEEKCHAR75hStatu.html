KBD_PEEKCHAR (75h) - Status

    Status

      A 1-WORD field that contains 0 to indicate no keystroke is available or 1 to indicate that a Character data Record is being returned. The sign bit is set to 0 if the current input mode is ASCII or 1 if the input mode is binary.
