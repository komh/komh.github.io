COUNTRYINFO Field - fsCurrencyFmt

    fsCurrencyFmt (UCHAR)

      Bit fields for currency format.

      This field contains the following bit fields:

      Bit

        Description
      0
        Placement of the currency indicator.

      • currency indicator precedes money value
      • currency indicator follows money value
      1
        Number of spaces (0 or 1) between currency indicator and money value.
      2
        When this bit is set, ignore the first two bits; the currency indicator replaces the decimal indicator.
