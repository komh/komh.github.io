ASYNC_SETLINECTRL (42h) - Data Bits

    Data Bits

      Has the following value and meaning:

      00h-04h

        Reserved
      05h
        5 data bits
      06h
        6 data bits
      07h
        7 data bits (initial value)
      08h
        8 data bits
      09h-FFh
        Reserved
