KBD_GETINPUTMODE (71h) - Mode

    Mode

      A 1-byte field containing one of the following values:

      Bit 1xxxxxx1

        Shift report
      Bit 0xxxxxx0
        ASCII mode
      Bit 1xxxxxxx
        BINARY mode
