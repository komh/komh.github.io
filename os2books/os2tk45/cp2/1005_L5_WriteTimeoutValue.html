Write Timeout Value

The user-specific value, in .01 seconds units (based on 0, where 0 = .01 seconds), is used for the Write Timeout processing, if Normal Write Timeout processing is enabled.

    Initial Value

      1 minute
    First Level Open
      Set to 1 minute
    Mode Utility
      No effect
