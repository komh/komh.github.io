KBD_GETKEYBDTYPE (77h) - Keyboard Type

    Keyboard Type

      A 1-WORD field containing:

      High Byte

        Reserved. Set to 0.
      Low Byte
        Has the following bit values:

        00h

          Personal Computer AT keyboard
        01h
          Enhanced Keyboard
        02H-FFh
          Reserved. Set to 0
