VIOMODEINFO Field - fbType

    fbType (UCHAR)

      Bit mask of mode being set.

      Mode characteristics bit mask:

        Bit

          Description
        7-4
          Reserved
        3
          0 = VGA BIOS compatible modes
          1 = Native mode
        2
          0 = Enable color burst
          1 = Disable color burst
        1
          0 = Text mode
          1 = Graphics mode
        0
          0 = Monochrome compatible mode
          1 = Other
