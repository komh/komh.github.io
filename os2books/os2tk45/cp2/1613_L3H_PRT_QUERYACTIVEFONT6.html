PRT_QUERYACTIVEFONT (69h) - Returns

The error return codes for this function are as follows:

    00H

      Completed successfully
    03H
      No code page function because spooler not started
    09H
      Error caused by switcher error, not by input parameters
    0AH
      Error caused by invalid printer name as input
    0DH
      Received code page request when code page switcher not initialized
    10H
      Received request for SFN, not in SFN table
