POWER_GETPOWEREVENT (61h) - Remarks

On input, the ParmLength field indicates the size of the Parameter Packet in bytes. On output, the ParmLength field indicates either the number of bytes written (without error) or the minimum required size for the function (with the PowerBadLength error code value) to be completed.
