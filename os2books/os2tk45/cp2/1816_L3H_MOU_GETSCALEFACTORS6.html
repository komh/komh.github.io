MOU_GETSCALEFACTORS (66h) - Remarks

Scaling factors are ratio values that determine how much relative movement is necessary before the physical mouse device driver reports a mouse event. The ratios specify the number of mickeys per 8 pels. The default ratio values are: Vertical/Row ratio - 16 mickeys per 8 pels
Horizontal/Column ratio - 8 mickeys per 8 pels

This is the complement function for MOU_SETSCALEFACTORS.
