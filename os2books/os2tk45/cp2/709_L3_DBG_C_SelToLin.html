DBG_C_SelToLin

Debug Command 31 - Translate Selector:Offset to Linear Address Command

Parameters

    Pid

      = Process ID of debuggee
    Cmd
      = DBG_C_SelToLin
    Value
      = Selector from address to be translated
    Index
      = Offset from address to be translated

    Returns

    Addr

      = Linear address

    The Value and Index fields will be left unchanged.
