TOUCH_DEVINDEP_GETEMULSTATE (65h) - Data Packet Format

The data packet is a location in application storage where the Touch Display driver will return the caller to the Touch Display mouse emulation state. The format is as follows: 

┌──────────────────────────────────────┐
│Field          Length     C Datatype  │
├──────────────────────────────────────┤
│Emul State     WORD       USHORT      │
└──────────────────────────────────────┘
