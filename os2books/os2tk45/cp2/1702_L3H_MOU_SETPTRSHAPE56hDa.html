MOU_SETPTRSHAPE (56h) - Data Packet Format

The Data Packet is a location in application storage that contains the Pointer Image buffer. The Pointer Image buffer contains a new pointer shape defined by the user for the physical Mouse device driver. The format of this buffer is dependent on the display mode. The buffer always consists of the and pointer image data followed by the XOR pointer image data. The buffer always describes only one display plane.
