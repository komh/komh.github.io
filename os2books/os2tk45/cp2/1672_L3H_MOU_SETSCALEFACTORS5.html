MOU_SETSCALEFACTORS (53h) - Remarks

This function reassigns the current pointing device scaling factors. Scaling factors are ratio values that determine how much relative movement is necessary before the physical Mouse device driver reports a mouse event. These ratios specify the number of mickeys per 8 pels. The default ratio values are: Vertical/Row ratio - 16 mickeys per 8 pels
Horizontal/Column ratio - 8 mickeys per 8 pels
