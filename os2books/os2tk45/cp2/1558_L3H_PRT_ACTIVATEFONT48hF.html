PRT_ACTIVATEFONT (48h) - Font ID

    Font ID

      The ID value of the font to make currently active.

      0000H

        If the Code Page value and Font ID are specified as zero, set printer to hardware default code page and font. If Font ID is zero and the code page is a valid nonzero, then any font within the specified code page is acceptable.
      0001H-FFFFH
        Valid Font ID numbers; font types defined by the font file definitions for downloadable fonts. For cartridge fonts, Font IDs are the numbers on the cartridge label and are also entered in the DEVINFO statement for the printer.
