CDROMDISK_READLONG (72h) - Description

This function reads 2352 bytes of the specified sectors in the data buffer. The returned sectors include the 12 bytes of sync, 4 bytes of header, 2048 bytes of data, and 288 bytes of EDC/ECC at the end.
