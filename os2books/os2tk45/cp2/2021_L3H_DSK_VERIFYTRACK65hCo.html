DSK_VERIFYTRACK (65h) - Command Information

    Command Information

      A byte with bit 0 defined as follows:

      Bit 0

        If clear (0), the track layout contains non-consecutive sectors or does not start with Sector 1. If set (1), the track layout starts with Sector 1 and contains only consecutive sectors.
      All other bits are reserved and must be set to 0.
