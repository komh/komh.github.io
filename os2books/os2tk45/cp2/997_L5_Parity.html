Parity

Determines whether a parity bit exists and, if appropriate, what algorithm determines its value. See IOCtls ASYNC_SETLINECTRL and ASYNC_GETLINECTRL.

    Initial Value

      Even parity
    First Level Open
      No effect
    Mode Utility
      User interface to change the parity characteristics
