SCR_ABIOSPASSTHRU (74h) - Remarks

This function is supported by the physical screen device driver. The parameter packet is copied to a 64-byte work area that is used as an ABIOS request block. The LID and UNIT fields are overwritten with the Logical ID of the ABIOS video device opened by SCREEN$ and 0, respectively. This request block is used with the DevHlp_ABIOSCall and the resulting request block is copied back to the caller's Parameter Packet.
