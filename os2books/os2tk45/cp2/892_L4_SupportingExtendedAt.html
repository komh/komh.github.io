Supporting Extended Attributes

To support extended attributes, applications should do the following:

  • Fill in the .ASSOCTABLE extended attribute for all major file types that the application recognizes or uses.

  • Fill in the .ICON extended attribute for executable files.

  • Set the .TYPE extended attribute for data files the application creates.

  • Fill in and use the .LONGNAME extended attribute as appropriate.

  • Support .HISTORY and .VERSION.

  • Support the other Standard Extended Attributes as appropriate.
