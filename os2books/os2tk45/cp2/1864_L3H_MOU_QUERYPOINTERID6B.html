MOU_QUERYPOINTERID (6Bh) - Data Packet Format

The Data Packet is a location in application storage with the following format: 

┌───────────────────────────────────────────────┐
│Field                   Length     C Datatype  │
├───────────────────────────────────────────────┤
│Device ID               WORD       USHORT      │
├───────────────────────────────────────────────┤
│COM Port Number         WORD       USHORT      │
├───────────────────────────────────────────────┤
│Secondary Device ID     WORD       USHORT      │
└───────────────────────────────────────────────┘
