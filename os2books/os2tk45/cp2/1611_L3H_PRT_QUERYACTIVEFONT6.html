PRT_QUERYACTIVEFONT (69h) - Font ID

    Font ID

      On return, is the ID value of the font that is currently active.

      0000H

        If the Code Page value and Font ID are specified as zero, the printer is set to the hardware default code page and font. If Font ID is zero and code page is nonzero, no error is returned if any Font ID is available for the specified code page.
      0001H-FFFFH
        Valid Font ID numbers; font types defined by the font file definitions for downloadable fonts. For cartridge fonts, Font IDs are the numbers on the cartridge label, and are also entered in the DEVINFO statement for the printer.
