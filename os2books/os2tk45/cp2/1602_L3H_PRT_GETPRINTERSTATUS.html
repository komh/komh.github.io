PRT_GETPRINTERSTATUS (66h) - Data

    Data

      On return, Data byte is set to:

      Bit 7

        1 = Not Busy
      Bit 6
        1 = Acknowledge
      Bit 5
        1 = Out of Paper
      Bit 4
        1 = Selected
      Bit 3
        1 = I/O Error
      Bit 2
        Unused
      Bit 1
        Unused
      Bit 0
        1 = Timeout
