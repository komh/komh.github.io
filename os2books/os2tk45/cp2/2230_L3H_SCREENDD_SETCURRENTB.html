SCREENDD_SETCURRENTBANK (01h) - Current Video Mode Type

    Current Video Mode Type

      Defines the adapter video mode type to the device driver, and has one of the following values:

    • Text Mode
    • Planar Mode
    • Linear Mode

      This field is required.
