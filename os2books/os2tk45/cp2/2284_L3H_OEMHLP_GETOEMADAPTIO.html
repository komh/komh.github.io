OEMHLP_GETOEMADAPTIONINFO (00h) - OEM Name

    OEM Name

      Contains the name of the OEM stored as an ASCIIZ string. Use this field when this is not an IBM version of the OS/2 operating system and you have added additional OEMHLP functions to the system.
