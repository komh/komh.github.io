DCBINFO Field - usReadTimeout

    usReadTimeout (USHORT)

      Time period used for Read Timeout processing.

      Specifies the time period used for Read Timeout processing. See Note 9 of ASYNC_SETDCBINFO. The value is in .01 second units based on zero (where 0 =.01 seconds).
