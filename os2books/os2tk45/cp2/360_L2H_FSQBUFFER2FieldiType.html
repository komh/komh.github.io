FSQBUFFER2 Field - iType

    iType (USHORT)

      Type of item.

      Possible values are described in the following list:

    • FSAT_CHARDEV
      Resident character device

    • FSAT_PSEUDODEV
      Pseudocharacter device

    • FSAT_LOCALDRV
      Local drive

    • FSAT_REMOTEDRV
      Remote drive attached to the file-system driver.
