SCR_ALLOCLDTBGVAL (76h) - Options

    Options

      Characteristics of Video RAM (VRAM) address space are customized with the Options parameter. The following values are valid:

    • Make segment readable code; no validation in background
    • Make segment writable data; no validation in background
    • Free selector
    • Make segment readable IOPL code
    • Make segment Read/Write IOPL data
    • Tag selectors Physical Video Buffer (PVB); make segment writable data with validation in background
      Other
        Reserved
