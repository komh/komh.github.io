CDROMAUDIO_GETAUDIODISK (61h) - Description

This function returns the first and last track numbers and the CD Redbook address for the lead-out track. This information is read from the TOC information on the subchannel Q lead-in track. The first and last track numbers are binary values, not BCD. This command is performed regardless of whether the play audio is processing.
