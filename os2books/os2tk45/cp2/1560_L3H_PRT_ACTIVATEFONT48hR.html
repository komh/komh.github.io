PRT_ACTIVATEFONT (48h) - Returns

The error return codes for this function are as follows:

    00H

      Completed successfully
    0AH
      Write fault, if monitor registered and the DevHlp_MonWrite fails
    02H
      Code page is not available
    03H
      No code page function because spooler not started
    04H
      Font ID is not available (verify)
    09H
      Error caused by switcher error, not by input parameters
    0AH
      Error caused by invalid printer name as input
    0DH
      Received code page request when code page switcher not initialized
    0FH
      SFN table full: cannot activate another entry
    13H
      DASD error reading font file
    15H
      DASD error reading font file definition block
    17H
      DASD error while writing to temporary spool file
    18H
      Disk full error while writing to temporary spool file
    19H
      Spool file handle was bad
