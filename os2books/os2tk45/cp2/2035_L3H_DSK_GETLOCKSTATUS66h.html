DSK_GETLOCKSTATUS (66h) - Drive Unit

    Drive Unit

      This field is used only when the IOCtl is issued without using a previously allocated file handle. A file handle of -1 must be used. Notice that media in the drive is not required. Drive Unit values are 0=A, 1=B, 2=C, and so forth.
