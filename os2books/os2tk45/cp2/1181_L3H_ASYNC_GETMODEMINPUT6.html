ASYNC_GETMODEMINPUT (67h) - Modem Control Input Status

    Modem Control Input Status

      If a bit has a value of 1, the condition is on. If a bit has a value of 0, the condition is off.

      Bit 0-3

        Undefined
      Bit 4
        Clear To Send (CTS)
      Bit 5
        Data Set Ready (DSR)
      Bit 6
        Ring Indicator (RI)
      Bit 7
        Data Carrier Detect (DCD)
