VIOMODEINFO Field - color

    color (UCHAR)

      Number of colors (power of 2).

      Number of colors defined as a power of 2. This is equivalent to the number of color bits that define the color, as in this example:

        Value

          Definition
        0
          Monochrome
        1
          2 colors
        2
          4 colors
        4
          16 colors
        8
          256 colors
        16
          64K colors
        24
          16M colors
