SCREENDD_SVGA_OEM (09h) - Manufacturer Specific Data

    Manufacturer Specific Data

      Manufacturer-specific data is in a format designated by the manufacturer. For Diamond Computer Systems, values returned are BIOS major value (high WORD) and BIOS minor value (low WORD). For Orchid Technology, the value returned represents the Micro Channel slot number.
