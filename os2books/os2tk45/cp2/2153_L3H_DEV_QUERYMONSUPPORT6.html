DEV_QUERYMONSUPPORT (60h) - Remarks

This request is used to query a device driver for monitor support. The physical device driver returns the system error MONITORS_NOT_SUPPORTED if it does not support character monitors. If monitors are supported, then it returns NO_ERROR (00H).
