TOUCH_DEVDEP_SETTOUCHTHRESHOLD (56h) - Push hysteresis threshold

    Push hysteresis threshold

      Sets the value of Z, when Z is decreasing on the chosen Z scale, at which the Touch display resets the selection detection bit in the status returned by Function 53h.

      Note: Any parameter value of -1 causes the driver default value to be substituted.
