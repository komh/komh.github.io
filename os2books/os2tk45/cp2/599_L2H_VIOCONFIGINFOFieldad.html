VIOCONFIGINFO Field - adapter

    adapter (ULONG)

      Display adapter type

      Display adapter type.

        Value

          Definition
        0-3
          Reserved
        3
          VGA Display Adapter
        4-6
          Reserved
        7
          8514/A
        8
          Image Adapter/A
        9
          XGA Display Adapter
        Values ranging from 0-4095 are reserved for IBM.
