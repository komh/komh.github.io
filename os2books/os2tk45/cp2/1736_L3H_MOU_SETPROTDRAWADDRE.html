MOU_SETPROTDRAWADDRESS (5Ah) - Pointer DS

    Pointer DS

      Contains two 1-WORD fields whose format is as follows:

      WORD 0

        Pointer Draw device driver data segment selector
      WORD 1
        Reserved=0.
