KBD_CREATE (5Dh) - Remarks

The desired initial code page should be either the default code page for the system or one of the two optional code pages that can be prepared in the CODEPAGE= statement in the CONFIG.SYS file.

This function returns an UNKNOWN_COMMAND error if the caller is in the Presentation Manager session.
