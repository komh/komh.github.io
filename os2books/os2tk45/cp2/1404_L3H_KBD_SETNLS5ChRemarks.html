KBD_SETNLS (5Ch) - Remarks

This request installs one of two possible Code Page Translation Tables into the device driver. This IOCtl updates the number 1 (or 2) entry of the Code Page Control Block. Entry zero is the device-driver-resident Code Page Translation Table.

Notice that this IOCtl is similar to KBD_SETTRANSTABLE, except that different entries in the Code Page Control Block are updated. This IOCtl can be called from a DOS session.
