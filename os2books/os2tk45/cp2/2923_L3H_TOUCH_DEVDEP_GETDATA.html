TOUCH_DEVDEP_GETDATAREPORTRATE (65h) - Data Packet Format

The data packet is a location in application storage where the Touch Display driver returns to the caller the Touch Display maximum data reporting rate. The format is as follows: 

┌───────────────────────────────────────┐
│Field           Length     C Datatype  │
├───────────────────────────────────────┤
│Report rate     WORD       SHORT       │
└───────────────────────────────────────┘
