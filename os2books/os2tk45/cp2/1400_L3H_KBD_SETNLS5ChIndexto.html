KBD_SETNLS (5Ch) - Index to Load

    Index to Load

      1-WORD identifying the number of the Code Page Control Block to load (1 or 2). A -1 indicates a custom code page for which the segment containing the custom code page is locked. This option is not valid for a DOS session.
