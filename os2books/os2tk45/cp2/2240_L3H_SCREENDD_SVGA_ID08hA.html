SCREENDD_SVGA_ID (08h) - Adapter Type

    Adapter Type

      Returns the code value of the chip set manufacturer, and has one of the following values:

    • Headland Technology
    • Trident Microsystems
    • Tseng Laboratories
    • Western Digital
    • ATI Technologies
    • IBM
    • Cirrus Logic
    • S3 Incorporated
    • Chips and Technologies
