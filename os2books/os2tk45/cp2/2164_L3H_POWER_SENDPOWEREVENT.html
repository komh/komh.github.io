POWER_SENDPOWEREVENT (40h) - Remarks

The SubId values permitted in the ulParm1 field are limited to values 03H through 06H. If the caller requests a SubId value outside this supported range, the request returns with a PowerBadSubId error code.

When Power Management is disabled (SubId=0004H), all requests except Power Management Enable (SubId=0003H) fail and are returned to the caller with a ReturnCode value of PowerDisabled.
