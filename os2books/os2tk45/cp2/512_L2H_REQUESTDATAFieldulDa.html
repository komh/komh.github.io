REQUESTDATA Field - ulData

    ulData (ULONG)

      An event code that is specified by the application.

      This data is understood by both the thread that is adding the element to the queue (the client thread) and the thread that receives the queue element (server thread). It has no special meaning, and is not altered by the operating system.
