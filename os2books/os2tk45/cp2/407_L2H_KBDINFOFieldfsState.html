KBDINFO Field - fsState

    fsState (USHORT)

      Current shift state.

      Bit 15

        SysReq key down.
      Bit 14
        CapsLock key down.
      Bit 13
        NumLock key down.
      Bit 12
        ScrollLock key down.
      Bit 11
        Right Alt key down.
      Bit 10
        Right Ctrl key down.
      Bit 9
        Left Alt key down.
      Bit 8
        Left Ctrl key down.
      Bit 7
        Insert on.
      Bit 6
        CapsLock on.
      Bit 5
        NumLock on.
      Bit 4
        ScrollLock on.
      Bit 3
        Either Alt key down.
      Bit 2
        Either Ctrl key down.
      Bit 1
        Left Shift key down.
      Bit 0
        Right Shift key down.
