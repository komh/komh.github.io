MOU_UPDATEDISPLAYMODE (51h) - Adapter Type

    Adapter Type

      The display adapter type, with values set as follows:

      • Monochrome-compatible
      • Color graphics adapter
      • Enhanced graphics adapter
      • VGA or PS/2* display adapter
      • Reserved
      • PS/2 Display Adapter 8514/A
      • PS/2 Image Adapter/A
      • PS/2 XGA Display Adapter
        10 = PS/2 SVGA Display Adapter
      Note: Values ranging from 0-4095 are reserved for IBM.
