SCR_GETROMFONTADDR (7Fh) - Returns

The error return codes for this function are as follows:

    AX = 0

      Successful.

      When the GDT Selector is allocated correctly, the value of the GDT selector allocated will be returned. If the GDT Selector is not allocated correctly, 0 will be returned.

    AX != 0
      Error
