Scan Module Table (CVK_CMD_SCANMTE)

Parameters:

    Cmd

      CVK_CMD_SCANMTE

    Returns:

    Cmd

      CVK_RET_LIB if information was returned, otherwise returns CVK_RET_SUC if end of module table reached.
    Value
      MTE handle of module if Cmd = CVK_RET_LIB.
    UCHAR NAME[Value]
      Null-terminated full pathname of module (immediately follows Value)

      Not present if SMTE or name is not paged in. Packet length is CVK_RETSIZE_SCANMTE in this case.

      The module table scan ignores resource modules, which contain only resource objects and segments, and forwarder modules, which contain no objects or segments.
