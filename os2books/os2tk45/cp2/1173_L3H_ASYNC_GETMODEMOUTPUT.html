ASYNC_GETMODEMOUTPUT (66h) - Modem Control Output Status

    Modem Control Output Status

      If a bit has a value of 1, the condition is on. If a bit has a value of 0, the condition is off.

      Bit 0

        Data Terminal Ready (DTR)
      Bit 1
        Request To Send (RTS)
      Bits 2-7
        Undefined
