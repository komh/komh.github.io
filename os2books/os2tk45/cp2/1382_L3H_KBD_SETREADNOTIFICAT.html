KBD_SETREADNOTIFICATION (59h) - Read/Peek Notification Flags

    Read/Peek Notification Flags

      Contains the following flags:

      Bits 15-1

        Reserved. Set to 0.
      Bit 0
        Mask indicating whether to enable or disable Read/Peek notification:

      • Disable Read/Peek notification
      • Enable Read/Peek notification
