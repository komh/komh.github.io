Perform Kernel Debugger Command (CVK_CMD_RAW)

Parameters:

    Cmd

      CVK_CMD_RAW
    OffV
      Null-terminated text of command to perform (overlays OffV)

    Returns:

    Cmd

      Indicates whether this is the last response.

      CVK_CMD_RAW

        More response packets coming.
      CVK_RET_SUC
        Last response packet.
      Value
        Length of command response.
      OffV
        Null-terminated response to command (overlays OffV)

      Note: This command may return multiple responses. The last event packet response will have Cmd set to CVK_RET_SUC instead of CVK_CMD_RAW.
