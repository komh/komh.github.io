KBD_XLATESCAN (79h) - Xlate State1

    Xlate State1

      Identifies the state of translation across successive calls. Initially this WORD should be zero and must be reset to 0 when the caller wants a new start to translation. Notice that it can take several calls to this IOCtl to complete a character, so this field should not be touched unless a fresh start to translation is desired. Also, this field is set to 0 at the completion of translation.
