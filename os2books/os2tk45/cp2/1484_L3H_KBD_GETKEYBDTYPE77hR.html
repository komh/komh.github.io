KBD_GETKEYBDTYPE (77h) - Remarks

This request returns the keyboard type. The 101- and 102-Key Enhanced keyboards, 88- and 89-Key Enhanced keyboards, 122-Key Enhanced keyboards, and MFI keyboards all return the value 01H to indicate an Enhanced keyboard is attached. For specific keyboard Hardware IDs, see KBD_QUERYKBDHARDWAREID.
