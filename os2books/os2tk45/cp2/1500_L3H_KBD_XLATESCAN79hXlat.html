KBD_XLATESCAN (79h) - Xlate Flags

    Xlate Flags

      Has the following:

      High Byte

        Bit settings are as follows:

        Bits 8-15

          Reserved. Set to 0.
        Low Byte
          Bit settings are as follows:

          Bits 1-7

            Reserved. Set to 0.
          Bit 0
            Translation complete.
