TOUCH_DEVINDEP_GETEVENTMASK (63h) - Data Packet Format

The data packet is a location in application storage where the Touch Display driver will return to the caller the Touch Display event mask, which controls the type of events that are to be added to the event queue. The format is as follows: 

┌──────────────────────────────────────┐
│Field          Length     C Datatype  │
├──────────────────────────────────────┤
│Event Mask     WORD       USHORT      │
└──────────────────────────────────────┘
