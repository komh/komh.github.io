DSK_UNLOCKEJECTMEDIA (40h) - Length Command Information

    Length Command Information

      Bits 0-1

        Description
      00
        Unlock.
      01
        Lock - drive can be locked with or without media in it. If the drive is locked without media in it, the media will be ejected when it is inserted until an UnlockIOCtl is issued.
      10
        Eject Media - media will be ejected if the drive is not locked.
      11
        Loads Media.
