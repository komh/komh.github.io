Read NPX Information (CVK_CMD_RNPX)

Parameters:

    Cmd

      CVK_CMD_RNPX
    TID
      Slot of thread whose NPX state is desired.

    Returns:

    Cmd

      Return code.

    • NPX state read and returned 1
        NPX is being emulated.
      2
        Invalid slot.
      3
        Thread has no NPX state.
      4
        State swapped out.
      Reg
        Image of NPX state as saved by FSAVE instruction.
