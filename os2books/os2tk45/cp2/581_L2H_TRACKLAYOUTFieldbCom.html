TRACKLAYOUT Field - bCommand

    bCommand (BYTE)

      Command information.

      Bit 0

        If clear (0), the track layout contains nonconsecutive sectors or does not start with Sector 1. If set (1), the track layout starts with Sector 1 and contains only consecutive sectors.
      All other bits are reserved and must be set to 0.
