PIPESEMSTATE Field - fFlag

    fFlag (BYTE)

      Additional information about the state of the named pipe.

      This parameter contains the following bit fields:

      Bit

        Description
      7-1
        Reserved
      0
        NPSS_WAIT
        If set, a thread is waiting at the other end of the pipe.
