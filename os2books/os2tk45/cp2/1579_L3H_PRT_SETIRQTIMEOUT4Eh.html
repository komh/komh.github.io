PRT_SETIRQTIMEOUT (4Eh) - Remarks

This IOCtl informs the physical parallel port device driver of the length of time, in seconds, to wait for a hardware interrupt before a write request is canceled. Valid timeout values range between 0 and 65535 seconds.
