MOU_GETBUTTONCOUNT (60h) - Data Packet Format

The Data Packet is a location in application storage where the physical mouse device driver returns to the caller the number of buttons supported by the physical device driver. The format of the Data Packet is as follows: 

┌────────────────────────────────────────────┐
│Field                Length     C Datatype  │
├────────────────────────────────────────────┤
│Number Supported     WORD       USHORT      │
└────────────────────────────────────────────┘
