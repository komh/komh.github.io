POWER_SETPOWEREVENTRES (41h) - ReturnCode

    ReturnCode

      The function-specific completion value. The valid return values are:

      PowerNoError

        (0000H)
      PowerBadReserved
        (0002H)
      PowerSemAlreadySetup
        (0005H)
      PowerBadFlags
        (0006H)
      PowerBadSemHandle
        (0007H)
      PowerDisabled
        (0009H)
      PowerTooManyQueues
        (000BH)
      PowerBadSemaphore
        (000DH)
