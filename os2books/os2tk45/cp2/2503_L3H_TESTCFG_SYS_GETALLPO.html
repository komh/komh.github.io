TESTCFG_SYS_GETALLPOSIDS (61h) - Data Packet Format

The data packet is an array of ten POS IDs as follows: 

┌─────────────────────────┬───────┐
│Field                    │Length │
├─────────────────────────┼───────┤
│POS ID for slot 0        │WORD   │
├─────────────────────────┼───────┤
│POS ID for slot 1    .   │WORD   │
│   .    .                │   .   │
│                         │   .   │
│                         │   .   │
├─────────────────────────┼───────┤
│POS ID for slot 8        │WORD   │
└─────────────────────────┴───────┘
