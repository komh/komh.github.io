CDROMDISK_GETVOLUMESIZE (78h) - Returns

The error return codes for this function are as follows:

    02h

      Device not ready: The drive is not ready, or the controller does not respond.
    07h
      Unknown media: The media is a compact disc that is not formatted according to CD Redbook specifications.
    10h
      Uncertain media.
    13h
      Unsupported parameter: The first 4 bytes of the parameter packet are not "Signature".
