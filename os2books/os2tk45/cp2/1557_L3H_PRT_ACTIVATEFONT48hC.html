PRT_ACTIVATEFONT (48h) - Code Page

    Code Page

      The value of the code page to make active.

      0000H

        If the Code Page value and Font ID are specified as zero, set printer to hardware default code page and font.
      0001H-FFFFH
        Valid code page numbers.
