MOU_UPDATEDISPLAYMODE (51h) - Text Rows

    Text Rows

      Number of text rows. Twenty-five rows are supported for the color graphics adapter. Twenty-five and forty-three rows are supported for the VGA adapter. Twenty-five and fifty rows are supported for the IBM Personal System/2* Display Adapter.
