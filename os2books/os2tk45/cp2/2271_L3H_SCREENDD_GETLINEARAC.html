SCREENDD_GETLINEARACCESS (0Bh) - Linear Flags

    Linear Flags

      Valid flags are defined in the following list:

        GETLINEAR_FLAG_MAPPHYSICAL 0x00000010L
        GETLINEAR_FLAG_MAPPROCESS 0x00000020L
        GETLINEAR_FLAG_MAPSHARED 0x00000400L
        GETLINEAR_FLAG_MAPATTACH 0x80000000L
