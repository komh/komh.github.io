TOUCH_DEVDEP_SETTOUCHTHRESHOLD (56h) - Push Threshold

    Push Threshold

      Sets the value of Z, when Z is increasing on the chosen Z scale, at which the Touch display sets the selection detection bit in the status returned by Function 53h.
