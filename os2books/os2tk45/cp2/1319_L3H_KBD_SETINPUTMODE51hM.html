KBD_SETINPUTMODE (51h) - Mode

    Mode

      A 1-byte field containing:

      Bit 1xxxxxx1

        Shift Report
      Bit 0xxxxxx0
        ASCII mode
      Bit 1xxxxxxx
        BINARY mode
