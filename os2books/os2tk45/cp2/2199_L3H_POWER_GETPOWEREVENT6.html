POWER_GETPOWEREVENT (61h) - Returns

Both Parameter and Data Packets contain return information.

When an error condition is encountered with an input packet field (Parameter or Data) the INVALID_PARAMETER IOCtl error code (13H) is returned to the caller. The caller must then inspect the Data Packet's ReturnCode field to determine the specific nature of the failure.
