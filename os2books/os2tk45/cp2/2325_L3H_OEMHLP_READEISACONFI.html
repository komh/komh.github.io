OEMHLP_READEISACONFIGINFO (04h) - Function Number

    Function Number

      For the Read EISA Function Configuration Information (subfunction 01) only, this is the sub-subfunction to read. Refer to the technical reference documentation for your personal computer hardware for details on the meaning of this parameter.
