CDROMDISK_GETHEADLOC (70h) - Returns

The error return codes for this function are as follows:

    02h

      Device not ready: The drive is not ready, or the controller does not respond.
    07h
      Unknown Media: The media is a compact disc that is not formatted according to the CD Redbook specification.
    10h
      Uncertain media.
    13h
      Unsupported parameter: The addressing mode is out of range or not supported, or the first 4 bytes of the parameter packet are not "Signature".
