TOUCH_DEVDEP_SETCALIBCONST (52h) - Parameter Packet Format

The parameter packet is a location in application storage that contains the following data: 

┌──────────────────────────────────────────────────────┐
│Field                          Length     C Datatype  │
├──────────────────────────────────────────────────────┤
│Channel 1 gain constant        DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│Channel 2 gain constant        DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│Channel 3 gain constant        DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│Channel 4 gain constant        DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│X as a function of X C[xx]     DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│X as a function of Y C[xy]     DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│Y as a function of X C[yx]     DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│Y as a function of Y C[yy]     DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│X offset C[x0]                 DWORD      ULONG       │
├──────────────────────────────────────────────────────┤
│Y offset C[y0]                 DWORD      ULONG       │
└──────────────────────────────────────────────────────┘
