SCREENDD_SVGA_OEM (09h) - Remarks

In some instances, it might be necessary to provide additional data to applications and the system that is unique to a specific adapter. This function facilitates this processing.

The manufacturer-specific data field is open to future expansion.

If the Manufacturer Name ID returned is a zero, the identification for this manufacturer is not currently implemented. This does not imply that the adapter is not supported. Special identification is provided for the adapters in addition to SVGA that require special handling.
