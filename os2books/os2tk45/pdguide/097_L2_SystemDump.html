System Dump

A System dump requires that the following steps are performed in order:

  • You set up for a system dump by adding statements to the CONFIG.SYS file and adding instrumentation to your code
  • The system performs the dump
  • You use the PM Dump Facility dump formatter to format and display dump information in a way that you can analyze it.
