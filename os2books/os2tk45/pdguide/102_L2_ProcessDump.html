Process Dump

A process dump file contains very basic unformatted system and user storage data related to the process that encountered the error and does not contain any data pertaining to main memory.

A process dump requires that the following steps are performed in order:

  • You set up for a process dump by adding statements to the CONFIG.SYS file and adding instrumentation to your code
  • The system performs the dump
  • You use the PM Dump Facility dump formatter to format and display dump information in a way that you can analyze it.
