FFSTQueryConfiguration - Syntax

FFSTQueryConfiguration queries the FFST configuration parameters. 

#define INCL_FFST
#include <os2.h>

PULONG          buffer_length;
PCONFIGPARMS    pConfigParms;
APIRET          rc;             /*  Return code. */

rc = FFSTQueryConfiguration(buffer_length,
       pConfigParms);
