Creating the Open32 UDMD

To analyze and migrate Windows 32-bit code, you need to create the Open32 UDMD as follows:

  • Double-click on the SMART2 Toolset object to start SMART.

  • SMART displays the SMART2 Toolset - Icon View window. Double-click on the SMART2 Source Migration object.

  • SMART displays the SMART2 Source Migration window. Select the Tables pull-down. Select Create User UDMD from the pull-down.

  • SMART displays the Create User Defined Migration Data Base dialog. Click on the Select1 push button to choose the directory name in which to store the Open32 UDMD.

  • SMART displays the Select User UDMD Directory dialog. In the New Path: entry field, type the subdirectory: 
    SMART\SMARTAUX\DAPIE32

    Click on the Ok push button.

  • SMART returns to the Create User Defined Migration Data Base dialog. Click on the Select2 push button.

  • SMART displays the Select Input Dictionary Filename dialog. In the File: entry field, type the dictionary filename: 
    DAPIE32.DCT

    Click on the Ok push button.

  • SMART displays the Create User Defined Migration Data Base dialog. Click on the Process push button.

  • On the SMART2 Source Migration window. To load the correct table and Open32 UDMD, select the Tables pull-down. Select the Select Tables pull-down.
