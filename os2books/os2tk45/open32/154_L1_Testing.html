Testing

To test Open32 programs, do the following:

  • Rewrite the makefile for the OS/2 build environment.
  • Use the VisualAge C++ compiler to compile the code.
  • Use the IPF compiler included in the Toolkit to compile the help files.
  • Use the RC compiler included in the Toolkit to compile the resource files.
  • Perform testing as you do normally.
