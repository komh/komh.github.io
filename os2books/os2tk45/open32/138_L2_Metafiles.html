Metafiles

There are two different kinds of Open32 metafiles:

  • A true Open32 metafile that contains a stream of GDI commands.

  • A metafile that has been converted into an Open32 metafile. A metafile of this type is created by obtaining a metafile handle through WinTranslateGraphicsObject. These metafiles are wrappers for an OS/2 metafile.

You can use the GetMetaFileBitsEx function with the first metafile type only. GetMetaFileBitsEx does not work the the second type of metafile: a converted metafile.
