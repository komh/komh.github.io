WinInsertLboxItem - Parameters

    hwndLbox (HWND) - input

      List box handle.

    index (LONG) - input

      Index of the list box item.

    psz (PSZ) - input

      Text to be inserted.

    lRetIndex (LONG) - returns

      Actual index where it was inserted.
