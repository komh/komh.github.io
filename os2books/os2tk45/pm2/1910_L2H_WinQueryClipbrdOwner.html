WinQueryClipbrdOwner Return Value - hwndClipbrdOwner

    hwndClipbrdOwner (HWND) - returns

      Window handle of the current clipboard owner.

      NULLHANDLE

        If the clipboard is not owned by any window, or if an error occurred.
      Other
        Window handle of the current clipboard owner.
