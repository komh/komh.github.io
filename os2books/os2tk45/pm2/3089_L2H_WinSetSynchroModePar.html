WinSetSynchroMode - Parameters

    hab (HAB) - input

      Anchor-block handle.

    lMode (LONG) - input

      Synchronization mode.

      SSM_SYNCHRONOUS

        Synchronous mode
      SSM_ASYNCHRONOUS
        Asynchronous mode
      SSM_MIXED
        Mixed mode.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
