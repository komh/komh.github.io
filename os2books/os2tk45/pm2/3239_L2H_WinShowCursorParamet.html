WinShowCursor - Parameters

    hwnd (HWND) - input

      Handle of window to which the cursor belongs.

    fShow (BOOL) - input

      Show indicator.

      TRUE

        Make cursor visible
      FALSE
        Make cursor invisible.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred, or an attempt was made to show the cursor when it was already visible.
