WinStopTimer - Parameters

    hab (HAB) - input

      Anchor-block handle.

    hwnd (HWND) - input

      Window handle.

    ulTimer (ULONG) - input

      Timer identifier.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred, or timer did not exist.
