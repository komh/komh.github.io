WinSetMenuItemText - Parameters

    hwndMenu (HWND) - input

      Menu window handle.

    usId (USHORT) - input

      Identity of the menu item.

    pText (PSZ) - input

      Text for the menu item.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
