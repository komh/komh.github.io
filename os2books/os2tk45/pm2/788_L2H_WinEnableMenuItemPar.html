WinEnableMenuItem - Parameters

    hwndMenu (HWND) - input

      Menu window handle.

    usId (USHORT) - input

      Item identifier.

    fEnable (BOOL) - input

      Enable flag.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
