WinQuerySwitchHandle Return Value - hswitchSwitch

    hswitchSwitch (HSWITCH) - returns

      Switch list handle for the specified application.

      NULLHANDLE

        Application is not in the switch list, or an error occurred
      Other
        Switch list handle.
