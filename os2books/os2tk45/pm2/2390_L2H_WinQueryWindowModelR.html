WinQueryWindowModel Return Value - ulModel

    ulModel (ULONG) - returns

      Memory model associated with the window.

      PM_MODEL_1X

        The 16-bit memory model of the 80386 processor.
      PM_MODEL_2X
        The 32-bit memory model of the 80386 processor.
