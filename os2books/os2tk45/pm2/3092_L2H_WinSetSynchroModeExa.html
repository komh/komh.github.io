WinSetSynchroMode - Example Code

This function is intended for use in an application with a distributed queue. 

#define INCL_WINMESSAGEMGR
#include <OS2.H>
HAB hab;
WinSetSynchroMode(hab,
                  SSM_SYNCHRONOUS);  /* synchronous mode. */
