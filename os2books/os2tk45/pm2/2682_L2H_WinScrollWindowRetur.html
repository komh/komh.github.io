WinScrollWindow Return Value - lComplexity

    lComplexity (LONG) - returns

      Complexity of resulting region/error indicator.

      RGN_NULL

        NULL rectangle invalid
      RGN_RECT
        Simple rectangle invalid
      RGN_COMPLEX
        Complex rectangle invalid
      RGN_ERROR
        Error.
