WinIsWindowVisible - Parameters

    hwnd (HWND) - input

      Window handle.

    rc (BOOL) - returns

      Visibility-state indicator.

      TRUE

        Window and all its parents have the WS_VISIBLE style bit set on
      FALSE
        Window or one of its parents have the WS_VISIBLE style bit set off.
