WinSetLboxItemText - Remarks

This macro is defined as: 

#define WinSetLboxItemText(hwndLbox, index, psz)  \
        ((BOOL)WinSendMsg(hwndLbox,               \
                          LM_SETITEMTEXT,         \
                          MPFROMLONG(index),      \
                          MPFROMP(psz)))

This macro requires the existence of a message queue.
