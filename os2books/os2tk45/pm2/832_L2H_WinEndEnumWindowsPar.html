WinEndEnumWindows - Parameters

    henum (HENUM) - input

      Enumeration handle.

      Returned by previous call to the WinBeginEnumWindows call.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
