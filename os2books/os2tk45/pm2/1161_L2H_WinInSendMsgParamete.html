WinInSendMsg - Parameters

    hab (HAB) - input

      Anchor-block handle.

    rc (BOOL) - returns

      Message-processing indicator.

      TRUE

        Current thread is processing a message sent by another thread
      FALSE
        Current thread is not processing a message, or an error occurred.
