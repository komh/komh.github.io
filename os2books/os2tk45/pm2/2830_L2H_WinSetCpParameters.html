WinSetCp - Parameters

    hmq (HMQ) - input

      Message-queue handle.

    ulCodePage (ULONG) - input

      Code page.

      Either of the two ASCII code pages specified in CONFIG.SYS can be selected.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
