WinDeleteLboxItem - Remarks

This macro is defined as: 

#define WinDeleteLboxItem(hwndLbox, index)    \
        ((LONG)WinSendMsg(hwndLbox,           \
                          LM_DELETEITEM,      \
                          MPFROMLONG(index),  \
                          (MPARAM)NULL))

This macro requires the existence of a message queue.
