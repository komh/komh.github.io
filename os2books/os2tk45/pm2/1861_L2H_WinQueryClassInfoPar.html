WinQueryClassInfo - Parameters

    hab (HAB) - input

      Anchor-block handle.

    PSZClassName (PSZ) - input

      Class name.

    PclsiClassInfo (PCLASSINFO) - output

      Class information structure.

    rc (BOOL) - returns

      Class-exists indicator.

      TRUE

        Class does exist
      FALSE
        Class does not exist.
