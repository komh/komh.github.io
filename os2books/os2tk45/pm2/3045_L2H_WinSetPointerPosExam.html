WinSetPointerPos - Example Code

This example calls WinSetPointer to set the pointer at 50, 50 in Screen coordinates. 

#define INCL_WINPOINTERS
#include <OS2.H>

WinSetPointerPos(HWND_DESKTOP,
                 (LONG)50,
                 (LONG)50);
