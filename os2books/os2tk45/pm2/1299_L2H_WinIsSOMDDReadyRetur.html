WinIsSOMDDReady Return Value - fReady

    fReady (BOOL) - returns

      SOMDD-state indicator.

      TRUE

        SOMDD has been started by the Workplace Shell process.
      FALSE
        SOMDD has not been started by the Workplace Shell process.
