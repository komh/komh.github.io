WinQueryPointerInfo - Parameters

    hptr (HPOINTER) - input

      Pointer handle.

    pptriPointerInfo (PPOINTERINFO) - output

      Pointer-information structure.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
