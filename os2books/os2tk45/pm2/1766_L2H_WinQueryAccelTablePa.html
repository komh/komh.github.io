WinQueryAccelTable Parameter - hwndFrame

    hwndFrame (HWND) - input

      Frame-window handle.

      NULLHANDLE

        Return queue accelerator.
      Other
        Return the window accelerator table, by sending the WM_QUERYACCELTABLE message to hwndFrame.
