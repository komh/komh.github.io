WinQueryActiveDesktopPathname - Parameters

    pszPathName (PSZ) - output

      Memory allocated by caller in which directory specification is written.

    ulSize (ULONG) - input

      Number of bytes pointed to by pszPathName.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
