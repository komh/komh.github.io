WinQueryWindowThunkProc - Example Code

This example shows how to get pointer conversion procedure associated with the frame window. 

#define INCL_WINTHUNKAPI
#include <OS2.H>

HWND hwndFrame;
PFN  pthnkproc;

pthnkproc = WinQueryWindowThunkProc(hwndFrame);
