WinShowPointer - Parameters

    hwndDeskTop (HWND) - input

      Desktop-window handle.

      HWND_DESKTOP

        The desktop-window handle
      Other
        The specified desktop-window handle.

      fShow (BOOL) - input

        Level-update indicator.

        TRUE

          Decrement pointer display level by one. (The pointer level is not decremented to a negative value.)
        FALSE
          Increment pointer display level by one.

        rc (BOOL) - returns

          Display-level-updated indicator.

          TRUE

            Pointer display level successfully updated.
          FALSE
            Pointer display level not successfully updated
