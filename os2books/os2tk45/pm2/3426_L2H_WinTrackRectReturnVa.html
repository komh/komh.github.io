WinTrackRect Return Value - rc

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Tracking successful.
      FALSE
        Tracking canceled, or the pointing device was already captured when this function was called.

        Only one tracking rectangle can be in use at one time.
