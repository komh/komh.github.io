WinIsControlEnabled - Parameters

    hwndDlg (HWND) - input

      Dialog window handle.

    usId (USHORT) - input

      Identity of the specified item.

    rc (BOOL) - returns

      Enabled-state indicator.

      TRUE

        Specified item is enabled.
      FALSE
        Specified item is disabled.
