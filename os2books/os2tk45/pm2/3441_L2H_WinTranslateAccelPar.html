WinTranslateAccel - Parameters

    hab (HAB) - input

      Anchor-block handle.

    hwnd (HWND) - input

      Destination window.

    haccelAccel (HACCEL) - input

      Accelerator-table handle.

    pQmsg (PQMSG) - in/out

      Message to be translated.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
