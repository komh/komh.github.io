WinEnableControl - Parameters

    hwndDlg (HWND) - input

      Dialog window handle.

    usId (USHORT) - input

      Identity of the item in the dialog template (button id).

    fEnable (BOOL) - input

      Enable flag.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
