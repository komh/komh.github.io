WinQueryWindowDC - Parameters

    hwnd (HWND) - input

      Window handle.

    hdc (HDC) - returns

      Device-context handle.

      NULLHANDLE

        Either WinOpenWindowDC has not been called for this window, or an error has occurred.
      Other
        Device context handle.
