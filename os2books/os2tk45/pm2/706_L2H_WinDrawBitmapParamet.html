WinDrawBitmap Parameter - clrFore

    clrFore (LONG) - input

      Foreground color.

      This is used if hbm refers to a monochrome bit map. In this instance, bit-map bits that are set to 1 are drawn using clrFore. Ignored if DBM_IMAGEATTRS is specified.
