WinLoadMenu Parameter - hmod

    hmod (HMODULE) - input

      Resource identifier.

      NULLHANDLE

        The resource is in the .EXE file of the application.
      Other
        The module handle returned by the DosLoadModule or DosQueryModuleHandle call.
