WinSetDesktopBkgnd - Errors

Possible returns from WinGetLastError

    PMERR_INVALID_HPTR (0x101B)

      An invalid pointer handle was specified.
    PMERR_INVALID_HWND (0x1001)
      An invalid window handle was specified.
