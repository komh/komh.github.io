WinDdePostMsg Parameter - usMsgId

    usMsgId (ULONG) - input

      Message identifier.

      Identifies the message to be posted.

      The following messages are valid:

        WM_DDE_ACK
        WM_DDE_ADVISE
        WM_DDE_DATA
        WM_DDE_EXECUTE
        WM_DDE_POKE
        WM_DDE_REQUEST
        WM_DDE_TERMINATE
        WM_DDE_UNADVISE.
