WinPostQueueMsg - Parameters

    hmq (HMQ) - input

      Message-queue handle.

    msg (ULONG) - input

      Message identifier.

    mp1 (MPARAM) - input

      Parameter 1.

    mp2 (MPARAM) - input

      Parameter 2.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred, or the queue was full.
