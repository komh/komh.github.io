WinDestroyAtomTable Parameter - hatomtblAtomTbl

    hatomtblAtomTbl (HATOMTBL) - input

      Atom-table handle.

      This is the handle returned from a previous call to the WinCreateAtomTable function. If NULL then this function does nothing.
