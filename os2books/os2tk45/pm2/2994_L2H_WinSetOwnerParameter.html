WinSetOwner - Parameters

    hwnd (HWND) - input

      Window handle whose owner window is to be changed.

    hwndNewOwner (HWND) - input

      Handle of the new owner.

      NULLHANDLE

        The window becomes "disowned"
      Other
        Handle of the new owner window.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
