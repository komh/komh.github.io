WinReleasePS - Parameters

    hps (HPS) - input

      Handle of the cache presentation space to release, as returned by the WinGetPS, the WinGetScreenPS, or the WinGetClipPS function.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
