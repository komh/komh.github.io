WinInsertLboxItem - Remarks

This macro is defined as: 

#define WinInsertLboxItem(hwndLbox, index, psz)  \
        ((LONG)WinSendMsg(hwndLbox,              \
                          LM_INSERTITEM,         \
                          MPFROMLONG(index),     \
                          MPFROMP(psz)))

This macro requires the existence of a message queue.
