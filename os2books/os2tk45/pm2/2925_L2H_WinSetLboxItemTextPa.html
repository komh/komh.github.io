WinSetLboxItemText - Parameters

    hwndLbox (HWND) - input

      List box handle.

    index (LONG) - input

      Index of the list box item.

    psz (PSZ) - input

      Pointer to a null terminated string.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
