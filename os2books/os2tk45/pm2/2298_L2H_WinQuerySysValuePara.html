WinQuerySysValue Parameter - hwndDeskTop

    hwndDeskTop (HWND) - input

      Desktop-window handle.

      HWND_DESKTOP

        Return the system values for the desktop-window handle.
      Other
        Return the system values for the specified desktop-window handle.
