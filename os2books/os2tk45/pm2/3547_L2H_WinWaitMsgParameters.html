WinWaitMsg - Parameters

    hab (HAB) - input

      Anchor-block handle.

    ulFirst (ULONG) - input

      First message identity.

    ulLast (ULONG) - input

      Last message identity.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
