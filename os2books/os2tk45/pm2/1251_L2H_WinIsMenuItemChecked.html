WinIsMenuItemChecked - Parameters

    hwndMenu (HWND) - input

      Menu window handle.

    usId (USHORT) - input

      Identity of the menu item.

    rc (BOOL) - returns

      Checked-state indicator.

      TRUE

        Menu item is checked.
      FALSE
        Menu item is not checked.
