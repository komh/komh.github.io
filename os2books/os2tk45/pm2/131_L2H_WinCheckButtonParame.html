WinCheckButton - Parameters

    hwndDlg (HWND) - input

      Dialog window handle.

    usId (USHORT) - input

      Button control identity.

    usChkstate (USHORT) - input

      Indicates the current checked state of the button.

    usRetChkst (USHORT) - returns

      Returns the previous checkstate.
