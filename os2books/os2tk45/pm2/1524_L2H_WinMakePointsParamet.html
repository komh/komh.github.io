WinMakePoints - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pwpt (PPOINTL) - in/out

      Points to be converted.

      The data type of these points after conversion is POINTL.

    cwpt (ULONG) - input

      Number of points to be converted.

      Must be positive.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
