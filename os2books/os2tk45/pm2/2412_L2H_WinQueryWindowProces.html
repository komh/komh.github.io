WinQueryWindowProcess - Parameters

    hwnd (HWND) - input

      Window handle.

    ppid (PPID) - output

      Process identity of the thread that created the window.

    ptid (PTID) - output

      Thread identity of the thread that created the window.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
