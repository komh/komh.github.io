WinDrawBitmap Parameter - clrBack

    clrBack (LONG) - input

      Background color.

      This is used if hbm refers to a monochrome bit map. In this instance, bit-map bits that are set to zero are drawn using clrBack. Ignored if DBM_IMAGEATTRS is specified.
