WinGetPS Parameter - hwnd

    hwnd (HWND) - input

      Handle of window for which the presentation space is required.

      HWND_DESKTOP

        The desktop-window handle; a presentation space for the whole of the desktop window is returned
      Other
        Handle of window for which the presentation space is required.
