WinQueryPointer - Example Code

This example obtains the pointer handle from the desktop window handle. 

#define INCL_WINPOINTERS
#define INCL_WINDESKTOP
#include <OS2.H>

HAB      hab;
HPOINTER hpointer;

hpointer = WinQueryPointer(HWND_DESKTOP);
