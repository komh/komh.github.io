WinIsThreadActive - Parameters

    hab (HAB) - input

      Anchor-block handle of calling thread.

    rc (BOOL) - returns

      Active-window indicator.

      TRUE

        Active window belongs to the calling thread
      FALSE
        Active window does not belong to the calling thread.
