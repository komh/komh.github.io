WinUnlockSystem - Parameters

    hab (HAB) - input

      The application anchor block.

    pszPassword (PSZ) - input

      Password string.

    rc (BOOL) - returns

      Return value.

      TRUE

        The system was successfully unlocked.
      FALSE
        An error occurred or the system was already in an unlocked state.
