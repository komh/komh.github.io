WinDlgBox Parameter - hmod

    hmod (HMODULE) - input

      Resource identity containing the dialog template.

      NULLHANDLE

        Use the application's .EXE file.
      Other
        Module handle returned from the DosLoadModule or DosQueryModuleHandle call.
