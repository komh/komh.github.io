WinUpperChar - Example Code

This example shows how the WinUpperChar call can be used to convert a characters in NLS languages to uppercase. 

#define INCL_WINCOUNTRY
#include <OS2.H>
#include <stdio.h>

main()
{
  HAB hab;
  char szString[] = "ö";
  hab = WinInitialize(0);

  WinUpper(hab,
           850,
           49,
           szString);
  WinTerminate(hab);
}
