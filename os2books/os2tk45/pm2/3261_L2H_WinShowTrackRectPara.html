WinShowTrackRect - Parameters

    hwnd (HWND) - input

      Window handle.

      Passed to the WinTrackRect function.

    fShow (BOOL) - input

      Show indicator.

      TRUE

        Show the tracking rectangle
      FALSE
        Hide the tracking rectangle.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
