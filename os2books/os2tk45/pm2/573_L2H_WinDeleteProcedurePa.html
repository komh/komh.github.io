WinDeleteProcedure - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pwndproc (PFNWP) - input

      Window procedure identifier to be deleted.

    rc (BOOL) - returns

      Procedure-deleted indicator.

      TRUE

        Procedure successfully deleted
      FALSE
        Procedure not successfully deleted.
