WinIsWindow - Parameters

    hab (HAB) - input

      Anchor-block handle.

    hwnd (HWND) - input

      Window handle.

    rc (BOOL) - returns

      Validity indicator.

      TRUE

        Window handle is valid
      FALSE
        Window handle is invalid.
