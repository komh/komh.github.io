WinEndPaint - Parameters

    hps (HPS) - input

      Presentation-space handle.

      Handle of the presentation space that is used for drawing and that is returned by a previous call to the WinBeginPaint function.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
