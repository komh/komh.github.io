WinSetControlColors Parameter - clrType

    clrType (LONG) - input

      Control color index.

      Must be one of the following control color indexes.

      CCT_STATIC

        Static bitmap.
      CCT_STATICTEXT
        Static text.
      CCT_GROUPBOX
        Group Box.
      CCT_PUSHBUTTON
        Push button.
      CCT_CHECKBOX
        Check box.
      CCT_RADIOBUTTON
        Radio button.
      CCT_ENTRYFIELD
        Entry field.
      CCT_LISTBOX
        List box.
      CCT_COMBOBOX
        Combo box.
      CCT_SCROLLBAR
        Scroll bar.
      CCT_FRAME
        Window frame.
      CCT_MENU
        Menu.
      CCT_TITLEBAR
        Title bar.
      CCT_SPINBUTTON
        Spin button.
      CCT_SLIDER
        Slider.
      CCT_CIRCULARSLIDER
        Circular slider.
      CCT_VALUESET
        Value set.
      CCT_MLE
        Multiple line entry field.
      CCT_CONTAINER
        Container.
      CCT_NOTEBOOK
        Notebook.
