WinCompareStrings Return Value - ulResult

    ulResult (ULONG) - returns

      Comparison result.

      WCS_EQ

        Strings are equal
      WCS_LT
        String 1 is less than string 2
      WCS_GT
        String 1 is greater than string 2
      WCS_ERROR
        Error occurred.
