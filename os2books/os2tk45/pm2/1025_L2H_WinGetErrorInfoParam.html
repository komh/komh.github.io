WinGetErrorInfo - Parameters

    hab (HAB) - input

      Anchor-block handle.

    perriErrorInfo (PERRINFO) - returns

      Error information.

      This structure contains information about the previous error code for the current thread:

      NULL

        No error information available
      Other
        Error information.
