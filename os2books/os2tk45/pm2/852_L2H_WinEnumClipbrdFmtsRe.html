WinEnumClipbrdFmts Return Value - ulNext

    ulNext (ULONG) - returns

      Next clipboard-data format index.

      Possible values include:

    • Enumeration is complete; that is, there are no more clipboard formats available.
      Other
        Index of the next available clipboard-data format in the clipboard.
