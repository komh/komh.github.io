WinQueryWindowText - Parameters

    hwnd (HWND) - input

      Window handle.

      If hwnd is a frame-window handle, the title-bar window text is copied.

    lLength (LONG) - input

      Length of pun.

      It must be greater than 0.

    pun (PUN) - output

      Window text.

    lRetLen (LONG) - returns

      Length of returned text not including the null terminator.
