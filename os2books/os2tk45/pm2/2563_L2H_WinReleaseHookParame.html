WinReleaseHook Parameter - lHook

    lHook (LONG) - input

      Type of hook chain.

      This parameter can have one of the following HK_* values:

      HK_CHECKMSGFILTER

        See CheckMsgFilterHook.
      HK_CODEPAGECHANGED
        See CodePageChangedHook.
      HK_DESTROYWINDOW
        See DestroyWindowHook.
      HK_FINDWORD
        See FindWordHook
      HK_FLUSHBUF
        See FlushBufHook
      HK_HELP
        See HelpHook.
      HK_INPUT
        See InputHook.
      HK_JOURNALPLAYBACK
        See JournalPlaybackHook.
      HK_JOURNALRECORD
        See JournalRecordHook.
      HK_LOADER
        See LoaderHook.
      HK_LOCKUP
        See LockupHook
      HK_MSGCONTROL
        See MsgControlHook.
      HK_MSGFILTER
        See MsgFilterHook.
      HK_MSGINPUT
        See MsgInputHook
      HK_PLIST_ENTRY
        See ProgramListEntryHook
      HK_PLIST_EXIT
        See ProgramListExitHook
      HK_REGISTERUSERMSG
        See RegisterUserHook
      HK_SENDMSG
        See SendMsgHook.
      HK_WINDOWDC
        See WindowDCHook
