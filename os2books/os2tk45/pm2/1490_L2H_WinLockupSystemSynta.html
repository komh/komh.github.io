WinLockupSystem - Syntax

This function is specific to OS/2 Version 2.1 or higher.

This function locks up the system. 

#define INCL_WINMESSAGEMGR
#include <os2.h>

HAB     hab;  /*  The application anchor block. */
BOOL    rc;   /*  Return code. */

rc = WinLockupSystem(hab);
