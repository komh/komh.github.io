WinMakeRect - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pwrc (PRECTL) - in/out

      Rectangle to be converted.

      The data type of the rectangle after conversion is RECTL.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
