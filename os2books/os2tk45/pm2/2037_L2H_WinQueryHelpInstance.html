WinQueryHelpInstance - Parameters

    hwndApp (HWND) - input

      Handle of the application window.

    hwndHelp (HWND) - returns

      Help Manager window handle.

      NULLHANDLE

        No Help Manager instance is associated with the application window.
      Other
        Help Manager window handle.
