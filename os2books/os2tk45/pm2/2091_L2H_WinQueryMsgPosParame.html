WinQueryMsgPos - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pptl (PPOINTL) - output

      Pointer position in screen coordinates.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
