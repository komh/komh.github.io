WinFlashWindow - Parameters

    hwndFrame (HWND) - input

      Handle of window to be flashed.

    fFlash (BOOL) - input

      Start-flashing indicator.

      TRUE

        Start window flashing
      FALSE
        Stop window flashing.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
