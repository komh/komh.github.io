WinQueryVisibleRegion Return Value - ulretn

    ulretn (ULONG) - returns

      RGN_ERROR

        An error occurred.
      RGN_NULL
        The window currently has no visible area on screen.
      RGN_RECT
        The region contains a series of rectangular areas.
      RGN_COMPLEX
        The region contains a complex shape.
