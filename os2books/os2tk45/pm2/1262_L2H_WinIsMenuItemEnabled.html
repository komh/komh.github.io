WinIsMenuItemEnabled - Parameters

    hwndMenu (HWND) - input

      Menu window handle.

    usId (USHORT) - input

      Identity of the menu item.

    rc (BOOL) - returns

      Enabled-state indicator.

      TRUE

        Menu item is enabled.
      FALSE
        Menu item is disabled.
