WinSetMsgInterest - Parameters

    hwnd (HWND) - input

      Window handle.

    ulMsgClass (ULONG) - input

      Message class to have interest level set.

      msgid

        A single message identity (for example, WM_SHOW)
      SMIM_ALL
        All messages (except for WM_QUIT if lControl is SMI_AUTODISPATCH or SMI_NOINTEREST).

      lControl (LONG) - input

        Interest-identifier for the message class.

        SMI_RESET

          Revert to interest specified for the window class.
        SMI_INTEREST
          Interested in the messages.
        SMI_NOINTEREST
          Not interested in the messages.
        SMI_AUTODISPATCH
          Interested in the message or messages, but they are to be automatically dispatched to the window procedure.

        rc (BOOL) - returns

          Interest-changed indicator.

          TRUE

            Interest successfully changed
          FALSE
            Interest not successfully changed.
