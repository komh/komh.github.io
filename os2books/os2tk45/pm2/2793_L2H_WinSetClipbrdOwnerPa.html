WinSetClipbrdOwner - Parameters

    hab (HAB) - input

      Anchor-block handle.

    hwnd (HWND) - input

      Window handle of the new clipboard owner.

      NULLHANDLE

        Clipboard-owner window is released and no new clipboard-owner window is established.
      Other
        Window handle of the new clipboard owner.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
