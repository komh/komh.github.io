WinQueryClipbrdFmtInfo Return Value - rc

    rc (BOOL) - returns

      Format-exists indicator.

      TRUE

        fmt exists in the clipboard and prgfFmtInfo is set
      FALSE
        fmt does not exist in the clipboard and prgfFmtInfo is not set.
