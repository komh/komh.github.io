identifier

  • One or more characters used to identify or name a data element and possibly to indicate certain properties of that data element. (A).

  • In programming languages, a token that names a data object such as a variable, an array, a record, a subprogram, or a function. (A).

  • A sequence of letters, digits, and underscores used to identify a data object or function. IBM.
