built-in

built-in

  • A function that the compiler will automatically inline instead of making the function call, unless the programmer specifies not to inline.

  • In programming languages, pertaining to a language object that is declared by the definition of the programming language; for example the built-in function SIN in PL/I, the predefined data type INTEGER in FORTRAN. (I). Synonymous with predefined. IBM.
