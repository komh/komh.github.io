_rmtmp - Remove Temporary Files

    Syntax 

      #include <stdio.h>
int _rmtmp(void);
    Description

    • rmtmp closes and deletes all temporary files in all directories that are held open by the calling process.
    Returns

    • rmtmp returns the number of temporary files deleted.
    Example Code

      This example uses _rmtmp to remove a number of temporary files. 

      #include <stdio.h>

int main(void)
{
   int num;
   FILE *stream;

   if (NULL == (stream = tmpfile()))
      printf("Could not open new temporary file\n");
   else {
      num = _rmtmp();
      printf("Number of temporary files removed = %d\n", num);
   }
   return 0;

   /****************************************************************************
      The output should be:

      Number of temporary files removed = 1
   ****************************************************************************/
}
