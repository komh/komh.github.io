character set

  • A finite set of different characters that is complete for a given purpose; for example, the character set in ISO Standard 646, 7-bit Coded Character Set for Information Processing Interchange. (T).

  • All the valid characters for a programming language or for a computer system. IBM.

  • A group of characters used for a specific reason; for example, the set of characters a printer can print. IBM.

  • See also portable character set.
