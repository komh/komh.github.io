_cputs - Write String to Screen

    Syntax 

      #include <conio.h>
int _cputs(char *str);
    Description

    • cputs writes the string that str points to directly to the screen. The string str must end with a null character (\0). The _cputs function does not automatically add a carriage return followed by a line feed to the string.
    Returns

      If successful, _cputs returns 0. Otherwise, it returns a nonzero value.

    Example Code

      This example outputs a prompt to the screen. 

      #include <conio.h>

int main(void)
{
   char *buffer = "Insert data disk in drive a: \r\n";

   _cputs(buffer);
   return 0;

   /****************************************************************************
      The output should be:

      Insert data disk in drive a:
   ****************************************************************************/
}
