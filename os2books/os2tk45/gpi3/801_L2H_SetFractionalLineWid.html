Set Fractional Line Width/Push and Set Fractional Line Width - Parameters

    LEN (GLENGTH1)

      Length of following data.

      0x02

        Only valid value.

      LINEWIDTH (GROUFS)

        Value for the line-width attribute.

        The nonzero value is an integral and fractional multiplier of the normal line width:

        0x0000

          Drawing default
        0x0001-0xFFFF
          Multiplier of normal line width.
