FOCAMETRICS Field - yStrikeoutSize

    yStrikeoutSize (SHORT)

      Strikeout size.

      The width of the strikeout stroke, in world coordinate units. This value describes the actual strikeout in the font if FM_SEL_STRIKEOUT is also set. Otherwise, it describes what the graphics engine will simulate if overstrike is requested in GpiCreateLogFont.
