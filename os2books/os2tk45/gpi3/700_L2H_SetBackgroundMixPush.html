Set Background Mix/Push and Set Background Mix - Syntax

These orders set, or push and set, the value of the current background mix attribute.

Set Background Mix (GSBMX)
X'0D' (MODE)

Push and Set Background Mix (GPSBMX)
X'4D' (MODE)
