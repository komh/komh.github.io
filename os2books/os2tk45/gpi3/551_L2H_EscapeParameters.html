    LEN (GLENGTH1)

    TYPE (GBIT8)

      Type identifier:

      80

        Registered value
      Other
        All other values are unregistered.

      RID (GBIT8)

        Registered identifier:

        01

          Set pel.
        02
          BITBLT function.
        03
          Flood fill function.
        04
          Draw bits function.
