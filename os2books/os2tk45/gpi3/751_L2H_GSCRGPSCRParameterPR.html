GSCR/GPSCR Parameter - PREC

    PREC (GBIT8)

      Value for character-precision attribute:

      All other values are reserved.

      0x00

        Drawing default
      0x01
        String precision
      0x02
        Character precision
      0x03
        Stroke precision
