Modify Path - Parameters

    LEN (GLENGTH1)

      Length of following data.

      0x06

        Only valid value.

      MODE (GBIT8)

        Mode of path modification:

        0x06

          Stroke the path
        Other
          All other values are reserved.

        RES (GBIT8)

          Reserved value, must be 0.

        PTHID (GLONG)

          Path identifier.

          0x00000001-0xFFFFFFFF

            Valid path identifiers.
