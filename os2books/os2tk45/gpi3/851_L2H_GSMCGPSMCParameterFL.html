GSMC/GPSMC Parameter - FLAGS

    FLAGS

      This is an optional extension.

      Values:

      NOTDEFLT (GBIT1)

        Options:

      • A cell size of zero sets drawing default
      • A cell size of zero sets to zero.
      RES (GBIT7)
        Reserved value, must be 0.
