Set Character Shear/Push and Set Character Shear - Syntax

These orders set, or push and set, the value of the current character shear attribute.

Set Character Shear (GSCH)
X'35' (LEN, HX, HY)

Push and Set Character Shear (GPSCH)
X'75' (LEN, HX, HY)
