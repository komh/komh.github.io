GBBLT Parameter - MIX

    MIX (GBIT16)

      Mix mode.

      Values are:

      0x00CC

        Source.
      0x00C0
        Source and pattern.
      0x00CA
        Source where pattern1
      0x000C
        Source where pattern0
      0x00E2
        Pattern where source1
      0x00B8
        Pattern where source0
      Other
        Reserved values.
