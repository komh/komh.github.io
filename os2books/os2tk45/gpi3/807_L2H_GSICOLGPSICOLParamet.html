GSICOL/GPSICOL Parameter - FLAGS

    FLAGS

      Values:

      DEFAULT (GBIT1)

        Options:

      • Use specified index
      • Use drawing default color.
      SPEC (GBIT1)
        Options:

      • Use index directly
      • Special value.
      RES (GBIT6)
        Reserved value, must be 0.
