Custom Fill Patterns from a Bit Map

To create a custom pattern from a hard-coded bit map, an application must use the following steps:

  • Create or load a bit map to obtain a bit-map handle; for example, use GpiLoadBitmap.

  • Call GpiSetBitmapId (ID) to tag the bit map with a local identifier (lcid) from 1 to 254.

  • Call GpiSetPatternSet (ID) or GpiSetAttrs (ABB_SET) to set the current fill pattern.

The application can now draw the area.
