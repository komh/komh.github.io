Set Pattern Symbol/Push and Set Pattern Symbol - Syntax

These orders set, or push and set, the value of the current pattern-symbol attribute.

Set Pattern Symbol (GSPT)
X'28' (PATT)

Push and Set Pattern Symbol (GPSPT)
