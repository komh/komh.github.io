Line at Given Position/Line at Current Position - Syntax

This order defines one or more connected straight lines, drawn from the given position.

Line at Given Position (GLINE)
X'C1' (LEN, P0, P1, PN)

Line at Current Position (GCLINE)
X'81' (LEN, P1, PN)
