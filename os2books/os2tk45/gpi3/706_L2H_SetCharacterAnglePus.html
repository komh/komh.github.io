Set Character Angle/Push and Set Character Angle - Syntax

These orders set, or push and set, the value of the current character angle attribute.

Set Character Angle (GSCA)
X'34' (LEN, AX, AY)

Push and Set Character Angle (GPSCA)
X'74' (LEN, AX, AY)
