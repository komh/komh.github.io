FOCAMETRICS Field - usCodePage

    usCodePage (USHORT)

      Code page supported by the font.

      Defines the registered code page supported by the font. A value of zero (0) implies that the font may be used with any of the code pages supported by OS/2.

      When a font contains special symbols for which there is no registered code page, then code page 65400 is used.
