Set Line Width/Push and Set Line Width - Syntax

These orders set, or push and set, the value of the current line-width attribute to the value specified in the order.

Set Line Width (GSLW)
X'19' (LINEWIDTH)

Push and Set Line Width (GPSLW)
X'59' (LINEWIDTH)
