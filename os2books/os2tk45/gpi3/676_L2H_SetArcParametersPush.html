Set Arc Parameters/Push and Set Arc Parameters - Syntax

These orders set, or push and set, the values of the current arc parameters.

Set Arc Parameters (GSAP)
X'22' (LEN, P, Q, R, S)

Push and Set Arc Parameters (GPSAP)
X'62' (LEN, P, Q, R, S)
