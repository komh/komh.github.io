Set Model Transform/Push and Set Model Transform - Parameters

    LEN (GLENGTH1)

      Length of following data.

    RES (GBIT8)

      Reserved value, must be 0.

    FLAGS

      Values:

      RES (GBIT6)

        Reserved value, must be 0.
      CM (GBIT2)
        Matrix control bits:

        B"00"

          Unity matrix
        B"01"
          Concatenate after
        B"10"
          Concatenate before
        B"11"
          Overwrite.

        MASK (GBIT16)

          Load mask.

        MX[LEN] (GROSOL)

          Matrix values.

          The matrix size is based on the number of bits set in MASK.
