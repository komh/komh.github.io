FOCAMETRICS Field - usNominalPointSize

    usNominalPointSize (SHORT)

      Nominal point size.

      For a bit-map font, this field contains the height of the font.

      For an outline font, this field contains the height that the font designer intended for this font. For example, some fonts are designed for text use, in which case a value of 120 (12 point) would probably be placed in this field. Other fonts are designed for "display" use ("display" is a typographer's term for larger point sizes). This is not the only size at which the font can be used.

      Measured in decipoints (a decipoint is 1/720th of an inch).
