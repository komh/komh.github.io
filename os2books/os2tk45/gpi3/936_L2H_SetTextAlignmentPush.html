Set Text Alignment/Push and Set Text Alignment - Syntax

These orders set, or push and set, the value of the current text alignment attribute.

Set Text Alignment (GSCSA)
X'10' (LEN, HORIZ, VERT)

Push and Set Text Alignment (GPSCSA)
X'50' (LEN, HORIZ, VERT)
