GSCC/GPSCC Parameter - FLAGS

    FLAGS

      Internal flags.

      This parameter is optional.

      NOTDEFLT (GBIT1)

        Values:

      • A cell size of zero sets drawing default
      • A cell size of zero sets to zero.
      RES (GBIT7)
        Reserved.

        0000000

          Only valid value.
