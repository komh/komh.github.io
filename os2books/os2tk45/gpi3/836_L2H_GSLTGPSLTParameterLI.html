GSLT/GPSLT Parameter - LINETYPE

    LINETYPE (GBIT8)

      Value for line-type attribute.

      The value is an index into a notational line-type table:

      0x00

        Drawing default
      0x01
        Dotted line
      0x02
        Short dashed line
      0x03
        Dash-dot line
      0x04
        Double dotted line
      0x05
        Long dashed line
      0x06
        Dash-double-dot line
      0x07
        Solid line
      0x08
        Invisible line
      Other
        Reserved values.
