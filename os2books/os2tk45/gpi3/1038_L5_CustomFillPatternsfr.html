Custom Fill Patterns from a Font Symbol

To create a custom pattern from a character or a symbol in a font:

  • Create a logical image font and assign it an ID; for example use GpiCreateLogFont.

  • Call GpiSetPatternSet (ID) or GpiSetAttrs (ABB_SET), passing it the local identifier for the font.

  • Call GpiSetPattern, passing the value of the code point for a character or symbol in the font.

The application can now draw the area.
