GSCE/GPSCE Parameter - FLAGS

    FLAGS

      Values as follows:

      DEFAULT (GBIT1)

        Values as follows:

        B"0"

          Set to specified value.
        B"1"
          Set to drawing default.
        RES1 (GBIT7)
          Reserved value, must be 0.
