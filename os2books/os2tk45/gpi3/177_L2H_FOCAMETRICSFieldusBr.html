FOCAMETRICS Field - usBreakChar

    usBreakChar (SHORT)

      Break character.

      The code point that represents the "space" or "break" character for this font, expressed as an offset from usFirstChar. For example, if the first character is the space in code page 850, usFirstChar = 32, and usBreakChar = 0.
