Extended Escape - Parameters

    LEN (GLENGTH2)

      Length of following data.

    TYPE (GBIT8)

      Type identifier:

      0x80

        Registered value
      Other
        All other values are unregistered.

      RID (GUCHAR)

        Registered identifier.

        No registered extended escapes are used by the OS/2* operating system.

      PARMS (GSTR)

        Parameters of escape.
