Set Line Type/Push and Set Line Type - Syntax

These orders set, or push and set, the value of the current line-type attribute.

Set Line Type (GSLT)
X'18' (LINETYPE)

Push and Set Line Type (GPSLT)
X'58' (LINETYPE)
