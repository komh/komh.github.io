GFPTH Parameter - FLAGS

    FLAGS

      Extra functions:

      RES1 (GBIT1)

        Reserved value, must be 0.
      INSIDE (GBIT1)
        Values:

      • Alternate mode
      • Winding mode.
      MOD (GBIT1)
        Values:

      • Do not modify before filling
      • Modify path before filling.
      RES2 (GBIT5)
        Reserved value, must be 0.
