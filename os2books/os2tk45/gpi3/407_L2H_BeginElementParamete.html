Begin Element - Parameters

    LEN (GLENGTH1)

      Length of following data.

    TYPE (GLONG)

      Element type code.

      Possible values are described in the following list:

      0x0000FD01

        Line bundle
      0x0000FD02
        Character bundle
      0x0000FD03
        Marker bundle
      0x0000FD04
        Area bundle
      0x0000FD05
        Image bundle
      0x00000007
        Call segment
      0x00000081
        Polyline
      0x00000085
        Polyfillet
      0x000000A4
        Polyfillet sharp
      0x000000A5
        Polyspline
      0x00000082
        Polymarker
      0x00000087
        Full arc
      0x00000091
        Image
      0x000000B1
        Character string at current position
      0x000000F1
        Character string at given position
      0x81xxxxxx-0xFFxxxxxx
        Indicates user defined elements
      Other
        Reserved values.

      DESCR (GUNDF)

        Element description data.

        This is optional.
