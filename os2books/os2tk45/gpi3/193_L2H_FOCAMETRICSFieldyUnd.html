FOCAMETRICS Field - yUnderscoreSize

    yUnderscoreSize (SHORT)

      Underscore size.

      The width (thickness) of the underscore stroke, in world coordinate units. This describes the actual underscore in the font if FM_SEL_UNDERSCORE is also set. Otherwise, it describes what the graphics engine will simulate if underscore is requested in GpiCreateLogFont.
