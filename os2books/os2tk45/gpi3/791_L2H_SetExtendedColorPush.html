Set Extended Color/Push and Set Extended Color - Syntax

These orders set, or push and set, the value of the current color attribute.

Set Extended Color (GSECOL)
X'26' (LEN, COLOR)

Push and Set Extended Color (GPSECOL)
X'66' (LEN, COLOR)
