FOCAMETRICS Field - yUnderscorePosition

    yUnderscorePosition (SHORT)

      Underscore position.

      The position of the underscore stroke from the baseline, in world coordinate units. This value describes the actual underscore in the font if FM_SEL_UNDERSCORE is also set. Otherwise, it describes what the graphics engine will simulate if underscore is requested in GpiCreateLogFont.

      Note: Positive values mean below the baseline.
