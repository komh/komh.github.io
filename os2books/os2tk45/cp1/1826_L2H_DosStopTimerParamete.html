DosStopTimer - Parameters

    htimer (HTIMER) - input

      The handle of the timer to stop.

    ulrc (APIRET) - returns

      Return Code.

      DosStopTimer returns one of the following values:

    • NO_ERROR 326
        ERROR_TS_HANDLE
      For a full list of error codes, see Errors.
