DosErrClass Parameter - pLocus

    pLocus (PULONG) - output

      A pointer to the ULONG in which the error origination is returned.

      The following values are possible:

    • ERRLOC_UNK
      Unknown

    • ERRLOC_DISK
      Random-access device such as a disk

    • ERRLOC_NET
      Network

    • ERRLOC_SERDEV
      Serial device

    • ERRLOC_MEM
      Memory
