VioSavRedrawUndo Parameter - KillIndic

    KillIndic (ULONG) - input

      Terminate indicator

      Indicates whether the thread (with the outstanding VioSavRedrawUndo) should return an error code or be terminated.

        Value

          Definition
        0
          Return error code.
        1
          Terminate thread.
