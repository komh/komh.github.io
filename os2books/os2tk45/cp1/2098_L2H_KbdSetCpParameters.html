KbdSetCp - Parameters

    ulReserved (ULONG) - input

      Reserved. Must be 0.

    pidCP (USHORT) - input

      Code page ID.

      The code page ID must be the ID of a code page installed on the system or zero.

    hkbd (HKBD) - input

      Reserved. Must be 0.

    rc (APIRET) - returns
