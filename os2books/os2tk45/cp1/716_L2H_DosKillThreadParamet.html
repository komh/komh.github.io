DosKillThread - Parameters

    tid (TID) - input

      Thread identifier within the current process to be ended.

    ulrc (APIRET) - returns

      Return Code.

      DosKillThread returns one of the following values:

    • NO_ERROR 170
        ERROR_BUSY
      309
        ERROR_INVALID_THREADID
      For a full list of error codes, see Errors.
