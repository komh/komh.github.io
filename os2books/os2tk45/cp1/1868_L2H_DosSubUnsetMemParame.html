DosSubUnsetMem - Parameters

    pbBase (PVOID) - input

      The offset of the memory pool whose use is being terminated.

    ulrc (APIRET) - returns

      Return Code.

      DosSubUnsetMem returns one of the following values:

    • NO_ERROR 532
        ERROR_DOSSUB_CORRUPTED
      For a full list of error codes, see Errors.
