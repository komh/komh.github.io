DosQueryProcType Parameter - pulproctype

    pulproctype (PULONG) - output

      The address of a ULONG in which the procedure type is returned.

      The value returned in this field is one of the following:

    • PT_16BIT
      Procedure is 16-bit.
    • PT_32BIT
      Procedure is 32-bit.
