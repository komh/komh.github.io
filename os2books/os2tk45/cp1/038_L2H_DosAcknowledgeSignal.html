DosAcknowledgeSignalException Parameter - ulSignalNum

    ulSignalNum (ULONG) - input

      The number of the signal to be acknowledged.

      Valid signals are shown in the following list:

    • XCPT_SIGNAL_INTR
    • XCPT_SIGNAL_KILLPROC
    • XCPT_SIGNAL_BREAK
