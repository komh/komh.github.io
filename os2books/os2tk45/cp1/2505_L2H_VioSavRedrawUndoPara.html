VioSavRedrawUndo Parameter - OwnerIndic

    OwnerIndic (ULONG) - input

      Ownership indicator

      Indicates whether the thread issuing VioSavRedrawUndo wants ownership of VioSavRedrawUndo to be reserved for its process.

        Value

          Definition
        0
          Reserve ownership.
        1
          Give up ownership.
