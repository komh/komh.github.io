VioQueryFonts Parameter - Options

    Options (ULONG) - input

      Controls which fonts are selected.

      Valid values are:

      VQF_PUBLIC

        Return only public fonts.
      VQF_PRIVATE
        Return only private fonts.
      VQF_ALL
        Return both public and private fonts.
