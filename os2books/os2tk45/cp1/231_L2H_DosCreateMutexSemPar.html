DosCreateMutexSem Parameter - fState

    fState (BOOL32) - input

      Initial state of the semaphore.

      Possible values are shown in the following list:

    • FALSE
      The initial state of the semaphore is "unowned."

    • TRUE
      The initial state of the semaphore is "owned."
