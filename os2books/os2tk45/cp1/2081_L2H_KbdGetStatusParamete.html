KbdGetStatus - Parameters

    pkbdinfo (PKBDINFO) - output

      Pointer to keyboard data.

      A pointer to a KBDKEYINFO structure in which the keyboard status is returned.

    hkbd (HKBD) - input

      Reserved. Must be 0.
