KbdGetHWID - Parameters

    pkbdhwid (PKBDHWID) - output

      Pointer to hardware ID.

      A pointer to a KBDHWID structure in which the keyboard hardware ID is returned.

    hkbd (HKBD) - input

      Reserved. Must be 0.
