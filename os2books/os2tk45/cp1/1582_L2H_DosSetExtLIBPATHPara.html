DosSetExtLIBPATH Parameter - pszExtLIBPATH

    pszExtLIBPATH (PSZ) - input

      Extended LIBPATH string.

      This buffer has a maximum size of 1024 bytes.

      A pointer to a NULL string removes the extended LIBPATH.
