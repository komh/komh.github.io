KbdSetStatus - Parameters

    pkbdinfo (PKBDINFO) - output

      Pointer to keyboard status.

      A pointer to a KBDINFO structure in which the keyboard status is returned.

    hkbd (HKBD) - input

      Reserved. Must be 0.
