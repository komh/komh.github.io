DosTmrQueryTime - Syntax

Provides a snapshot of the time from the IRQ0 high resolution timer (Intel 8254). 

#define INCL_DOSPROFILE
#include <os2.h>

PQWORD    pqwTmrTime;  /*  Time. */
APIRET    rc;          /*  Return Code. */

rc = DosTmrQueryTime(pqwTmrTime);
