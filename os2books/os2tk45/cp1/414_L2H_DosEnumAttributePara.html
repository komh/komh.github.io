DosEnumAttribute Parameter - pvBuf

    pvBuf (PVOID) - output

      Address of the buffer where EA information is returned.

      Level 1 information (ulInfoLevel == ENUMEA_LEVEL_NO_VALUE) is returned in a data structure of type DENA2.
