DosFreeResource - Parameters

    pb (PVOID) - input

      The address of the resource to be freed.

    ulrc (APIRET) - returns

      Return Code.

      DosFreeResource returns one of the following values:

    • NO_ERROR 5
        ERROR_ACCESS_DENIED
      For a full list of error codes, see Errors.
