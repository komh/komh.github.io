KbdGetLayoutUni - Parameters

    name (USHORT *) - output

      Keyboard layout name.

      A pointer to the location to return the keyboard layout name in unicode. This must be at least 9 UniChars long.

    hkbd (HKBD) - input

      Reserved. Must be 0.
