DosFSAttach Parameter - flag

    flag (ULONG) - input

      Type of operation to be performed.

      Possible values shown in the following list:

    • FS_ATTACH
      Attach file server

    • FS_DETACH
      Detach file server

    • FS_SPOOLATTACH
      Register a spooler device

    • FS_SPOOLDETACH
      De-register a spooler device
