DosQueryResourceSize Parameter - idt

    idt (ULONG) - input

      The type identifier of the resource.

      Possible values are between 1 and 0xFFFE inclusive. Values from 1 to 255 are reserved for predefinition. Values from 256 and on can be type defined for the resource. The fist 21 values are predefined as follows:

    • RT_POINTER
      Mouse pointer shape

    • RT_BITMAP
      Bit map

    • RT_MENU
      Menu template

    • RT_DIALOG
      Dialog template

    • RT_STRING
      String tables

    • RT_FONTDIR
      Font directory

    • RT_FONT
      Font

    • RT_ACCELTABLE
      Accelerator tables

    • RT_RCDATA
      Binary data
