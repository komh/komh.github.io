VioDeleteSetId - Parameters

    lcid (ULONG) - input

      Local ID for the font.

    hvps (HVIO) - input

      VIO presentation space handle.

      This is either 0 to indicate the default VIO session, or a value returned by VioCreatePS.

    rc (APIRET) - returns

      Return code.
