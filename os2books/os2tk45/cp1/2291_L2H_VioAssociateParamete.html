VioAssociate - Parameters

    hdc (HDC) - input

      Device-context handle.

      If this is NULL, a disassociation occurs.

    hvps (HVIO) - input

      VIO presentation space handle.

      This is returned from VioCreatePS.

    rc (APIRET) - returns

      Return code.

      VioAssociate returns one of the following values:
