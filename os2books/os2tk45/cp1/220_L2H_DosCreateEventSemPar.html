DosCreateEventSem Parameter - fState

    fState (BOOL32) - input

      Initial state of the semaphore.

      Possible values are defined in the list below:

    • FALSE
      The initial state of the semaphore is "reset."

    • TRUE
      The initial state of the semaphore is "posted."
