KbdSetLayoutUni - Parameters

    name (USHORT *) - input

      Keyboard layout name.

      Address of the unicode name of the keyboard layout. The name cannot exceed 8 UniChars in length.

    hkbd (HKBD) - input

      Reserved. Must be 0.
