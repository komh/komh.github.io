DosSetPriority Parameter - ulClass

    ulClass (ULONG) - input

      Priority class of a process.

      The values of this field are shown in the following list:

    • PRTYC_NOCHANGE
      No change, leave as is

    • PRTYC_IDLETIME
      Idle-time

    • PRTYC_REGULAR
      Regular

    • PRTYC_TIMECRITICAL
      Time-critical

    • PRTYC_FOREGROUNDSERVER
      Fixed high
