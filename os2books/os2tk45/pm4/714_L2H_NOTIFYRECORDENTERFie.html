NOTIFYRECORDENTER Field - fKey

    fKey (ULONG)

      Flag.

      Flag that determines whether the Enter key was pressed or the select button was double-clicked.

      TRUE

        The Enter key was pressed.
      FALSE
        The select button was double-clicked.
