PRJINFO4 Field - pszSpoolFileName

    pszSpoolFileName (PSZ)

      Path to spool file if it is not in the standard spool directory.

      This field is null except when an application wants to submit a print job without spooling it (due to the size of the print file). pszSpoolFileName can be set with SplSetJob using the PRJ_SPOOLFILENAME_PARMNUM(19) parameter.
