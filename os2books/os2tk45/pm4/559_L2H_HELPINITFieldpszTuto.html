HELPINIT Field - pszTutorialName

    pszTutorialName (PSZ)

      Indicates to the Help Manager that the application has a tutorial program.

      NULL

        The application either does not have a tutorial program, or the tutorial name is specified in each help panel definition.
      Other
        Default tutorial name.
