ICONINFO Field - fFormat

    fFormat (ULONG)

      Indicates where the icon resides.

      Possible values are:

      ICON_FILE

        Icon file supplied.
      ICON_RESOURCE
        Icon resource supplied.
      ICON_DATA
        Icon data supplied.
      ICON_CLEAR
        Go back to default icon.
