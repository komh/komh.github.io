WM_MATCHMNEMONIC - Parameters

    usmatch (USHORT)

      Match character.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Match indicator.

      TRUE

        Mnemonic found
      FALSE
        Mnemonic not found, or an error occurred.
