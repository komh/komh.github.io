DM_RENDERPREPARE - Syntax

This message tells a source to prepare for the rendering of an object. 

param1
     PDRAGTRANSFER  pDxfer      /*  Pointer to a DRAGTRANSFER structure. */

param2
     ULONG          ulReserved  /*  Reserved value, should be 0. */
