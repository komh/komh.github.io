BKM_SETTABCOLOR Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Tab color was successfully set.
      FALSE
        Unable to set tab color. If the page ID specified in the ulPageId parameter is invalid, FALSE is returned and no action is taken.
