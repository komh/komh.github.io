WM_QUERYCTLTYPE - Syntax

This message is sent by an application to a window to determine the type of control window it is (if any). 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
