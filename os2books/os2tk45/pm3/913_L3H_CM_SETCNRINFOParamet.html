CM_SETCNRINFO Parameter - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Container data was successfully set.
      FALSE
        Container data was not set. The WinGetLastError function may return the following errors:
          
            PMERR_INVALID_PARAMETERS
          
            PMERR_INSUFFICIENT_MEMORY.
