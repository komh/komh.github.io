BKM_INSERTPAGE - Syntax

This message inserts the specified page into the notebook data list. 

param1
     ULONG   ulPageId     /*  Page ID for placement. */

param2
     USHORT  usPageStyle  /*  Style attributes. */
     USHORT  usPageOrder  /*  Order attributes. */
