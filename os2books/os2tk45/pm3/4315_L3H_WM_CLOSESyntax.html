WM_CLOSE - Syntax

This message is sent to a frame window to indicate that the window is being closed by the user. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
