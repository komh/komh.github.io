WM_SETWINDOWPARAMS Field - pwndparams

    pwndparams (PWNDPARAMS)

      Window parameter structure.

      This points to a window parameter structure; see WNDPARAMS.

      The valid values of fsStatus are WPM_TEXT and WPM_CTLDATA.
