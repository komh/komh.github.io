CN_SCROLL - Parameters

    id (USHORT)

      Container control ID.

    CN_SCROLL (USHORT)

      Notification code.

    pNotifyScroll (PNOTIFYSCROLL)

      Pointer to the NOTIFYSCROLL structure.

    rc (ULONG)

      Reserved value, should be 0.
