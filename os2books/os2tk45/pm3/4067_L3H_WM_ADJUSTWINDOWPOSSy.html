WM_ADJUSTWINDOWPOS - Syntax

This message is sent by the WinSetWindowPos call to enable the window to adjust its new position or size whenever it is about to be moved. 

param1
     PSWP   pswp      /*  SWP structure pointer. */

param2
     ULONG  flzero    /*  Zero. */
