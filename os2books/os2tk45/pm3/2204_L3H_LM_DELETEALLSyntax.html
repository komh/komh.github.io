LM_DELETEALL - Syntax

This message is sent to a list box control to delete all the items in the list box. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
