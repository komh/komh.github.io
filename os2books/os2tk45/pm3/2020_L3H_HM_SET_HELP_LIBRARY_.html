HM_SET_HELP_LIBRARY_NAME - Syntax

This message identifies a list of help window library names to the Help Manager instance. 

param1
     PSZ    pszHelpLibraryName  /*  Library name. */

param2
     ULONG  ulReserved          /*  Reserved value, should be 0. */
