MLM_QUERYFONT - Parameters

    pFattrs (PFATTRS)

      Font attribute structure.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      System font indicator.

      TRUE

        The system font is in use.
      FALSE
        The system font is not in use.
