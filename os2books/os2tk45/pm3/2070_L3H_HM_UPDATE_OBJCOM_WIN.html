HM_UPDATE_OBJCOM_WINDOW_CHAIN - Parameters

    hwnd (HWND)

      The handle of the object to be withdrawn from the communication chain.

    hwnd (HWND)

      Window containing the handle of the object to be replaced.

    ulReserved (ULONG)

      Reserved value, should be 0.
