LM_SETITEMTEXT - Parameters

    sItemIndex (SHORT)

      Item index.

    pszItemText (PSZ)

      Item text.

      This points to a string containing the text to set the list-box item to.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
