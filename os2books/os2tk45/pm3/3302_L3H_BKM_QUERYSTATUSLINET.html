BKM_QUERYSTATUSLINETEXT - Syntax

This message queries the status line text, text size, or both for the specified page. 

param1
     ULONG      ulPageId       /*  Page ID. */

param2
     PBOOKTEXT  pBookText      /*  Pointer to a BOOKTEXT data structure. */
