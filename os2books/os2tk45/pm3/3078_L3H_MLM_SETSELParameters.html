MLM_SETSEL - Parameters

    iptAnchor (IPT)

      Insertion point for new anchor point.

    iptCursor (IPT)

      Insertion point for new cursor point.

    rc (BOOL)

      Success indicator.

      TRUE

        Selection successfully set
      FALSE
        An error occurred.
