MLM_EXPORT - Parameters

    pBegin (PIPT)

      Starting point.

      Updated to follow the last character exported.

    pCopy (PULONG)

      Number of bytes being exported.

      Decremented by the number of bytes actually exported.

    ulSuccess (ULONG)

      Number of bytes successfully exported.
