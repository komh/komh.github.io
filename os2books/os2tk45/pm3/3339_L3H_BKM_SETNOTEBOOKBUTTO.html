BKM_SETNOTEBOOKBUTTONS - Syntax

This message creates a set of buttons in the common area of the notebook page, 

param1
     ULONG            ulCount       /*  Number of buttons to set. */

param2
     PNOTEBOOKBUTTON  pButtonArray  /*  Notebook button array. */
