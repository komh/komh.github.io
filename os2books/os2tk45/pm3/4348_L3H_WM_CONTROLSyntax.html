WM_CONTROL - Syntax

This message occurs when a control has a significant event to notify to its owner. 

param1
     USHORT  id             /*  Control-window identity. */
     USHORT  usnotifycode   /*  Notify code. */

param2
     ULONG   ulcontrolspec  /*  Control-specific information. */
