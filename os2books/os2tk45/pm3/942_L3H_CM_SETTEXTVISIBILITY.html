CM_SETTEXTVISIBILITY - Parameters

    bVisible (BOOL)

      Text visibility state.

      TRUE

        Text is visible.
      FALSE
        Text is not visible.

      Reserved (PVOID)

        Reserved value, 0.

      rc (BOOL)

        Success indicator.

        TRUE

          Text visibility state was successfully set.
        FALSE
          Error occurred.
