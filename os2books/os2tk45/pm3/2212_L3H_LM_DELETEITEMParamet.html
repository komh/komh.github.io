LM_DELETEITEM - Parameters

    sItemIndex (SHORT)

      Item index.

      The zero-based index of the item to be deleted.

    ulReserved (ULONG)

      Reserved value, should be 0.

    sItemsLeft (SHORT)

      Number remaining.

      The number of items in the list after the item is deleted.
