DM_RENDERFILE - Syntax

This message is sent to the caller of DrgDragFiles to tell it to render a file. 

param1
     PRENDERFILE  rndf        /*  Pointer to a RENDERFILE structure. */

param2
     ULONG        ulReserved  /*  Reserved value, should be 0. */
