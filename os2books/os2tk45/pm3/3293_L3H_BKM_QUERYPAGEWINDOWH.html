BKM_QUERYPAGEWINDOWHWND - Syntax

This message queries the application page window handle associated with the specified page. 

param1
     ULONG  ulPageId    /*  Page ID. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
