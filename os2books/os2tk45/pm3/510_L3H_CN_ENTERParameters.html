CN_ENTER - Parameters

    id (USHORT)

      Container control ID.

    CN_ENTER (USHORT)

      Notification code.

    pNotifyRecordEnter (PNOTIFYRECORDENTER)

      Pointer to the NOTIFYRECORDENTER structure.

    ulReserved (ULONG)

      Reserved value, should be 0.
