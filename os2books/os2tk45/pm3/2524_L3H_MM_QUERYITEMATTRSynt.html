MM_QUERYITEMATTR - Syntax

This message returns the attributes of a menu item. 

param1
     USHORT  usitem             /*  Item identity. */
     USHORT  usIncludeSubmenus  /*  Include submenus indicator. */

param2
     USHORT  usattributemask    /*  Attribute mask. */
