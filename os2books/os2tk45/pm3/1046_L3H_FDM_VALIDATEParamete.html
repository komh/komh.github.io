FDM_VALIDATE - Parameters

    pFileName (PSZ)

      Pointer to the fully-qualified file name.

    usSeltype (USHORT)

      Selection type.

    rc (BOOL)

      Validity indicator.

      TRUE

        File name is valid.
      FALSE
        File name is not valid.
