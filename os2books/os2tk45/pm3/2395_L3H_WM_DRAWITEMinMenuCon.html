WM_DRAWITEM (in Menu Controls) - Syntax

This notification is sent to the owner of a menu control each time an item is to be drawn. 

param1
     USHORT      idMenu      /*  Window identifier. */

param2
     POWNERITEM  pOwnerItem  /*  Owner-item structure. */
