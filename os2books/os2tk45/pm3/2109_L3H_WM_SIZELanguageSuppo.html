WM_SIZE (Language Support Dialog) - Remarks

The Language Support Dialog Procedure responds to this message by issuing the WinDefDlgProc function, then posting a WM_PSIZE message to the application queue andsetting ulReserved to the result of the WinDefDlgProc function.
