MLM_CUT - Syntax

This message copies the text that forms the current selection to the clipboard and then deletes it from the MLE control. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
