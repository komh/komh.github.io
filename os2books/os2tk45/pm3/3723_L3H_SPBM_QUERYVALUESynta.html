SPBM_QUERYVALUE - Syntax

This message causes the component to show the value in the spin field. 

param1
     PVOID   pStorage   /*  Place for returned value. */

param2
     USHORT  usBufSize  /*  Buffer size. */
     USHORT  usValue    /*  Update/reset value. */
