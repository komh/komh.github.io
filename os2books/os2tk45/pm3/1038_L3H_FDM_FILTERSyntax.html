FDM_FILTER - Syntax

This message is sent before a file that meets the current filter criteria is added to the File list box. 

param1
     PSZ   pFilename  /*  Pointer to the file name. */

param2
     PSZ   pEAType    /*  Pointer to the .TYPE EA extended attribute. */
