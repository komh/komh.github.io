WM_TRACKFRAME (in Frame Controls) - Syntax

This message is sent to a frame window whenever it is to be moved or sized. 

param1
     USHORT  fsTrackFlags  /*  Tracking flags. */

param2
     ULONG   ulReserved    /*  Reserved value, should be 0. */
