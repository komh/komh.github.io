WM_FLASHWINDOW - Parameters

    usFlash (USHORT)

      Flash indicator.

      TRUE

        Start the window border flashing
      FALSE
        Stop the window border flashing.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
