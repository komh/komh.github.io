LM_QUERYITEMTEXTLENGTH - Syntax

This message returns the length of the text of the specified list box item. 

param1
     SHORT  sItemIndex   /*  Item index. */

param2
     ULONG  ulReserved   /*  Reserved value, should be 0. */
