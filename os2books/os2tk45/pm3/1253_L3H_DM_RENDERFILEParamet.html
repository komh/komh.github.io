DM_RENDERFILE - Parameters

    rndf (PRENDERFILE)

      Pointer to a RENDERFILE structure.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Render handling.

      TRUE

        The receiver handled the rendering.
      FALSE
        DrgDragFiles should render this file.
