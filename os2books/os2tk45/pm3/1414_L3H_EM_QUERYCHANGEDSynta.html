EM_QUERYCHANGED - Syntax

This message enquires if the text of the entry field control has been changed since the last enquiry. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
