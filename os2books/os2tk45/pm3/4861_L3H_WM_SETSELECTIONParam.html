WM_SETSELECTION - Parameters

    usselection (USHORT)

      Selection flag.

      TRUE

        The window is selected.
      FALSE
        The window is deselected.

      ulReserved (ULONG)

        Reserved value, should be 0.

      ulReserved (ULONG)

        Reserved value, should be 0.
