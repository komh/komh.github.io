WM_UPDATEFRAME - Parameters

    flCreateFlags (ULONG)

      Frame-creation flags.

      Contains the FCF_* flags that indicate which frame controls have been added or removed.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Processed indicator.

      TRUE

        Message processed
      FALSE
        Message ignored.
