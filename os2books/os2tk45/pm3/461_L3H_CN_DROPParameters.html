CN_DROP - Parameters

    id (USHORT)

      Container control ID.

    CN_DROP (USHORT)

      Notification code.

    pCnrDragInfo (PCNRDRAGINFO)

      Pointer to a CNRDRAGINFO structure.

    ulReserved (ULONG)

      Reserved value, should be 0.
