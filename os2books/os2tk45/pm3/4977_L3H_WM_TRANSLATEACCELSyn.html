WM_TRANSLATEACCEL - Syntax

This message is sent to the focus window whenever a WM_CHAR message occurs. 

param1
     PQMSG  pqmsg       /*  Pointer to a QMSG structure. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
