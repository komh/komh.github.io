WM_OWNERPOSCHANGE - Syntax

This message is sent by a frame window processing the WM_ADJUSTFRAMEPOS message. 

param1
     PSWP   ppswp       /*  Owned window state. */

param2
     PSWP   ppswpOwner  /*  Owner window state. */
