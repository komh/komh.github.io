SPBM_SETARRAY - Parameters

    pStrl (PSZ)

      Pointer to the new array of values.

    usItems (USHORT)

      Number of items in the array.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
