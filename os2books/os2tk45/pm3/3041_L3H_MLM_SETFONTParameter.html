MLM_SETFONT - Parameters

    pFattrs (PFATTRS)

      Font attribute structure.

      NULL

        The system font is set.
      other
        The specified font is set.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Success indicator.

        TRUE

          The font was successfully set.
        FALSE
          An error occurred.
