CN_DRAGAFTER Field - usDefaultOp

    usDefaultOp (USHORT)

      Default operation.

      Target-defined default operation.

      DO_COPY

        Operation is a copy.
      DO_DEFAULT
        Operation is the default drag operation. No modifier keys are pressed.
      DO_LINK
        Operation is a link.
      DO_MOVE
        Operation is a move.
      DO_UNKNOWN
        Operation is application-defined.
