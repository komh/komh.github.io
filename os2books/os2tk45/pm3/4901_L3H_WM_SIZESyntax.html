WM_SIZE - Syntax

This message occurs when a window changes its size. 

param1
     SHORT  scxold      /*  Old horizontal size. */
     SHORT  scyold      /*  Old vertical size. */

param2
     SHORT  scxnew      /*  New horizontal size. */
     SHORT  scynew      /*  New vertical size. */
