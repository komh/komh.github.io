MLM_SETWRAP - Parameters

    usWrap (USHORT)

      New value for wrap flag.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        An error occurred.
