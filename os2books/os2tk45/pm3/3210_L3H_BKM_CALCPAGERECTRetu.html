BKM_CALCPAGERECT Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Coordinates were successfully calculated.
      FALSE
        Unable to calculate coordinates. This is returned if an invalid RECTL structure is specified in the pRectl parameter.
