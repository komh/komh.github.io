MLM_SETTEXTLIMIT Return Value - ulFit

    ulFit (ULONG)

      Success indicator.

    • Successful completion. Current text fits within the new limit.
      Other
        The number of bytes by which the current text exceeds the proposed limit. The limit is not changed.
