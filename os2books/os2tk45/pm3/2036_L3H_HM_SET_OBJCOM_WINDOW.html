HM_SET_OBJCOM_WINDOW - Parameters

    objcomhwnd (HWND)

      Handle of the communication object window to be set.

    ulReserved (ULONG)

      Reserved value, should be 0.

    hwndprevioushwnd (HWND)

      The handle of the previous communication object window.
