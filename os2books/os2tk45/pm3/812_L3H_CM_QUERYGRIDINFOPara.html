CM_QUERYGRIDINFO - Parameters

    PGRIDINFO (PGRIDINFO)

      Pointer to a GRIDINFO structure.

    ulReserved (ULONG)

      This value is NULL.

    rc (BOOL)

      Success indicator.

      TRUE

        The GRIDINFO structure has been updated with the grid information.
      FALSE
        An error has occurred.
