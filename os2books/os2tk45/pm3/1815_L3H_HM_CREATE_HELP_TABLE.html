HM_CREATE_HELP_TABLE - Syntax

This message is sent by the application to give the Help Manager a new help table. 

param1
     PHELPTABLE  pHELPTABLE  /*  Help table. */

param2
     ULONG       ulReserved  /*  Reserved value, should be 0. */
