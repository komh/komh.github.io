BM_AUTOSIZE - Syntax

This message causes a button to be resized based on the size of its contents. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
