HM_EXT_HELP - Syntax

When the Help Manager receives this message, it displays the extended help window for the active application panel. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
