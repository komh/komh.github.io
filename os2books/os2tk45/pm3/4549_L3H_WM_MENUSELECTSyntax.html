WM_MENUSELECT - Syntax

This message occurs when a menu item has been selected. 

param1
     USHORT  usItem         /*  Identifier of selected item. */
     USHORT  usPostCommand  /*  Post-command flag. */

param2
     HWND    hwnd           /*  Menu-control window handle. */
