WM_QUERYFRAMECTLCOUNT - Syntax

This message is sent to the frame window in response to the receipt of a WM_SIZE or a WM_UPDATEFRAME (in Frame Controls) message. 

param1
     ULONG  ulReserved     /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved     /*  Reserved value, should be 0. */
