CSM_QUERYINCREMENT - Parameters

    ScrollIncre (PUSHORT)

      The increment value added or subtracted for the value of the control when scrolling.

    TickIncr (PUSHORT)

      The increment value used to draw the tick marks.

    rc (ULONG)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Errors occurred.
