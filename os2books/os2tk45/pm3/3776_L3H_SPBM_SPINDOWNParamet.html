SPBM_SPINDOWN - Parameters

    ulItem (ULONG)

      Number of values to spin down.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
