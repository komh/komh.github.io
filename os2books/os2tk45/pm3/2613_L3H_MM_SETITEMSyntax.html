MM_SETITEM - Syntax

This message sets the definition of a menu item. 

param1
     USHORT     usReserve          /*  Reserved value, should be 0. */
     USHORT     usincludesubmenus  /*  Include submenus indicator. */

param2
     PMENUITEM  pmenuitem          /*  Menu-item data structure. */
