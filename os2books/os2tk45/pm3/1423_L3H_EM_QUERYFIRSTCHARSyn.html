EM_QUERYFIRSTCHAR - Syntax

This message returns the zero-based offset of the first character visible at the left edge of an entry-field control. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
