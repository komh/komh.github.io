MLM_LINEFROMCHAR - Syntax

This message returns the line number corresponding to a given insertion point. 

param1
     IPT    iptFirst    /*  Insertion point of interest. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
