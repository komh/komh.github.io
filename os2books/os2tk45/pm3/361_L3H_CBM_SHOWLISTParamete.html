CBM_SHOWLIST - Parameters

    usShowing (USHORT)

      Showing indicator.

      TRUE

        Show the list box control.
      FALSE
        Do not show the list box control.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Changed indicator.

        TRUE

          The list box showing state has been changed.
        FALSE
          The list box showing state has not been changed.
