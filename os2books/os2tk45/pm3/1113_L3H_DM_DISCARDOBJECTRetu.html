DM_DISCARDOBJECT Return Value - ulAction

    ulAction (ULONG)

      Flag.

      DRR_SOURCE

        The source window procedure accepts responsibility for the operation.
      DRR_TARGET
        The target window procedure is to accept responsibility for the operation. The OS/2 shell supports the discarding of dragitems that can be rendered by the DRM_OS2FILE method.
      DRR_ABORT
        Abort the entire DM_DROP action.
