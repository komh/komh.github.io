LM_SETTOPINDEX - Syntax

This message is used to scroll a particular item to the top of the list box. 

param1
     SHORT  sItemIndex  /*  Index of the item to be made top. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
