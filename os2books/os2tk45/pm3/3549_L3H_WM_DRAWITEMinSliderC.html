WM_DRAWITEM (in Slider Controls) Return Value - rc

    rc (BOOL)

      Item-drawn indicator.

      TRUE

        The owner draws the item.
      FALSE
        If the owner does not draw the item, the owner returns this value and the slider control draws the item.
