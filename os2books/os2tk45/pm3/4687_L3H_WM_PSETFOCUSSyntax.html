WM_PSETFOCUS - Syntax

This message is posted when the Language Support Window or Dialog Procedure processes a WM_SETFOCUS message. 

param1
     HWND    hwnd        /*  Focus-window handle. */

param2
     USHORT  usfocus     /*  Focus flag. */
