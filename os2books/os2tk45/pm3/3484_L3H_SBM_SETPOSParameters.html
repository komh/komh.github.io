SBM_SETPOS - Parameters

    sslider (SHORT)

      Position of slider.

      If this value is outside the scroll-bar range, the slider is moved to the nearest valid position within the range.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred
