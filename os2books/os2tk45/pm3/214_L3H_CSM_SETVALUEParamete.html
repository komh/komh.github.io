CSM_SETVALUE - Parameters

    Value (SHORT)

      The new value to which to set the circular slider.

    ulReserved (ULONG)

      Reserved value.

    rc (ULONG)

      Success indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
