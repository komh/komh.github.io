WM_SAVEAPPLICATION - Syntax

This message is sent by the system to notify an application to save its current state. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
