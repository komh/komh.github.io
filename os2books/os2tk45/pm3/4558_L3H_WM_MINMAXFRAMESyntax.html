WM_MINMAXFRAME - Syntax

This message is sent to a frame window that is being minimized, maximized, or restored. 

param1
     PSWP   pswp        /*  Set window position structure. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
