BKM_SETTABCOLOR Field - rgbColor

    rgbColor (ULONG)

      Tab color

      Color that the tab should be set to. Additionally, the following value may be used:

      BKV_AUTOCOLOR

        Automatically paint the tabs with a default set of colors.
