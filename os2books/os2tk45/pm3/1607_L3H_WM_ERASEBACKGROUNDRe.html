WM_ERASEBACKGROUND Return Value - rc

    rc (BOOL)

      Processed indicator.

      TRUE

        If a FID_CLIENT window exists, the area of the frame covered by the FID_CLIENT window is erased in the system-window background color.

        If no FID_CLIENT window exists, the entire frame window is erased in the system-window background color.

      FALSE
        The client window did process the message.
