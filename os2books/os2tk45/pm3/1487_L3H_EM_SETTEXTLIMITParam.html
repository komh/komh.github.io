EM_SETTEXTLIMIT - Parameters

    sTextLimit (SHORT)

      Maximum number of characters in the entry field control.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred. For example, because not enough storage can be allocated.
