WM_BUTTON1UP - Syntax

This message occurs when the operator releases button 1 of the pointing device. 

param1
     POINTS  ptspointerpos  /*  Pointer position. */

param2
     USHORT  fsHitTestres   /*  Hit-test result. */
     USHORT  fsflags        /*  Keyboard control codes. */
