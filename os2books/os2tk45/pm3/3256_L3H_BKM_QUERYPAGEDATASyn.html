BKM_QUERYPAGEDATA - Syntax

This message queries the 4 bytes of application reserved storage associated with the specified page. 

param1
     ULONG  ulPageId    /*  Page ID. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
