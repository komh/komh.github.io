MM_QUERYITEM - Syntax

This message returns the definition of the specified menu item. 

param1
     USHORT     usitem             /*  Item identifier. */
     USHORT     usincludesubmenus  /*  Include submenus flag. */

param2
     PMENUITEM  pmenuitem          /*  Menu-item data structure. */
