WM_SHOW - Parameters

    usshow (USHORT)

      Show indicator.

      TRUE

        Show the window
      FALSE
        Hide the window.

      ulReserved (ULONG)

        Reserved value, should be 0.

      ulReserved (ULONG)

        Reserved value, should be 0.
