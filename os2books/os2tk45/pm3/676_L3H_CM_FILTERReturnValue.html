CM_FILTER Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        A subset was successfully created.
      FALSE
        An error occurred. The WinGetLastError function may return the following errors:

        • PMERR_NO_FILTERED_ITEMS
        • PMERR_INSUFFICIENT_MEMORY.
