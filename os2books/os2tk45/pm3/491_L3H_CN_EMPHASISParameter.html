CN_EMPHASIS - Parameters

    id (USHORT)

      Container control ID.

    CN_EMPHASIS (USHORT)

      Notification code.

    pNotifyRecordEmphasis (PNOTIFYRECORDEMPHASIS)

      Pointer to the NOTIFYRECORDEMPHASIS structure.

    ulReserved (ULONG)

      Reserved value, should be 0.
