MM_SETDEFAULTITEMID - Parameters

    ulDefItemID (ULONG)

      The menu id of the item to become the new default.

    ulReserved (ULONG)

      Reserved value, must be 0.

    rc (BOOL)

      Success of failure indicator.

      TRUE

        The conditional cascade default was set.
      FALSE
        The conditional cascade default was not set.
