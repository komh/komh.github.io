WM_SETWINDOWPARAMS - Syntax

This message occurs when an application sets or changes the window parameters. 

param1
     PWNDPARAMS  pwndparams  /*  Window parameter structure. */

param2
     ULONG       ulReserved  /*  Reserved value, should be 0. */
