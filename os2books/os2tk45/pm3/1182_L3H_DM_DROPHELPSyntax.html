DM_DROPHELP - Syntax

This message requests help for the current drag operation. 

param1
     PDRAGINFO  pDraginfo   /*  Pointer to the DRAGINFO structure used in the drag operation. */

param2
     ULONG      ulReserved  /*  Reserved value, should be 0. */
