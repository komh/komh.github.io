DM_DRAGOVERNOTIFY - Parameters

    pDraginfo (PDRAGINFO)

      Pointer to the DRAGINFO structure that represents the object being dragged.

    usDrop (USHORT)

      Drop indicator.

    usDefaultOp (USHORT)

      Default operation.

      Target-defined default operation.

    ulReserved (ULONG)

      Reserved value.
