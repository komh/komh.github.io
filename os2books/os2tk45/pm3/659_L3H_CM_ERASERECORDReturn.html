CM_ERASERECORD Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        The record was successfully erased.
      FALSE
        The record was not erased. The WinGetLastError function may return the following errors:
          
            PMERR_INVALID_PARAMETERS
          
            PMERR_INSUFFICIENT_MEMORY.
