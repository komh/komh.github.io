WM_DRAWITEM (in Font Dialog) Return Value - rc

    rc (BOOL)

      Item-drawn indicator.

      TRUE

        The owner draws the item.
      FALSE
        If the owner does not draw the item, the owner returns this value and the font dialog draws the item.
