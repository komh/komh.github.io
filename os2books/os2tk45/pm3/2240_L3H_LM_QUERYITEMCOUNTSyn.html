LM_QUERYITEMCOUNT - Syntax

This message returns a count of the number of items in the list box control. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
