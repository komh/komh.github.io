LM_SEARCHSTRING - Syntax

This message returns the index of the list box item whose text matches the string. 

param1
     USHORT  uscmd            /*  Command. */
     SHORT   sItemStart       /*  Index of the start item. */

param2
     PSZ     pszSearchString  /*  Search string. */
