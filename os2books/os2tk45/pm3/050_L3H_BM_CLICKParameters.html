BM_CLICK - Parameters

    usUp (USHORT)

      Up and down indicator.

      TRUE

        Perform the default upclick action
      FALSE
        Perform the default downclick action.

      ulReserved (ULONG)

        Reserved value, should be 0.

      ulReserved (ULONG)

        Reserved value, should be 0.
