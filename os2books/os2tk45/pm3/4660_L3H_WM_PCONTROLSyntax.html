WM_PCONTROL - Syntax

This message is posted when the Language Support Window or Dialog Procedure processes a WM_CONTROL message. 

param1
     USHORT  id            /*  Control-window identity. */
     USHORT  usnotifycode  /*  Notify code. */

param2
     ULONG   ulZero        /*  Zero. */
