WM_DDE_POKE - Syntax

This message requests an application to accept an unsolicited data item. It is always posted. 

param1
     HWND        hwnd        /*  Window handle of the sender. */

param2
     PDDESTRUCT  pDdeStruct  /*  DDE structure. */
