LM_QUERYTOPINDEX - Syntax

This message obtains the index of the item currently at the top of the list box. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
