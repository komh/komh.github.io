SPBM_SETCURRENTVALUE - Parameters

    lValue (LONG)

      Array value or index.

      Current value or index of array.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
