HM_KEYS_HELP - Syntax

This message is sent by the application and informs the help manager to display the keys help window. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
