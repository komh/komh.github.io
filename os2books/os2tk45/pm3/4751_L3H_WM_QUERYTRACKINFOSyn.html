WM_QUERYTRACKINFO - Syntax

The frame control generates this message on receiving a WM_TRACKFRAME (in Frame Controls) message. 

param1
     USHORT      ustflags    /*  Tracking flags. */

param2
     PTRACKINFO  ptrackinfo  /*  Track information structure. */
