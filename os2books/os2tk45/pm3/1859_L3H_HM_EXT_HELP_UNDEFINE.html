HM_EXT_HELP_UNDEFINED - Syntax

This message is sent to the application by the Help Manager to notify it that an extended help window has not been defined. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
