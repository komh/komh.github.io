LM_QUERYITEMTEXTLENGTH Return Value - sTextLength

    sTextLength (SHORT)

      Length of item text.

      The length of the text string, excluding the null termination character.

      LIT_ERROR

        Error occurred. For example, the item specified by its index does not exist.
      Other
        Length of item text.
