SBM_QUERYRANGE - Syntax

This message returns the scroll bar range minimum and maximum values. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */

returns
     SHORT  sfirst      /*  First bound. */
     SHORT  slast       /*  Last bound. */
