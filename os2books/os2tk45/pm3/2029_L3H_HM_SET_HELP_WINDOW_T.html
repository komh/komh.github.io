HM_SET_HELP_WINDOW_TITLE - Syntax

This message allows the application to change the window text of a help window title. 

param1
     PSZ    pszHelpWindowTitle  /*  Help window title. */

param2
     ULONG  ulReserved          /*  Reserved value, should be 0. */
