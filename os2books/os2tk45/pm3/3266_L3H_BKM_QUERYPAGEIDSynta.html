BKM_QUERYPAGEID - Syntax

This message queries the page identifier for the specified page. 

param1
     ULONG   ulPageId      /*  Location page ID. */

param2
     USHORT  usQueryOrder  /*  Page ID query order. */
     USHORT  usPageStyle   /*  Page style. */
