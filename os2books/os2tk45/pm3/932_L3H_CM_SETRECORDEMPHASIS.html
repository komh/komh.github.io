CM_SETRECORDEMPHASIS Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        An error occurred.

        The WinGetLastError function may return the following errors:

        PMERR_INVALID_PARAMETERS (1208)

        PMERR_INSUFFICIENT_MEMORY (203E)
