MM_STARTMENUMODE - Syntax

This message is used to begin menu selection. 

param1
     USHORT  usshowsubmenu  /*  Show submenu flag. */
     USHORT  usresumemenu   /*  Resume menu mode flag. */

param2
     ULONG   ulReserved     /*  Reserved value, should be 0. */
