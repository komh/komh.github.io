BKM_SETDIMENSIONS Field - usType

    usType (USHORT)

      Notebook region.

      Notebook region for which the dimensions are to be set. Valid values are:

      • BKA_MAJORTAB
      • BKA_MINORTAB
      • BKA_PAGEBUTTON.
