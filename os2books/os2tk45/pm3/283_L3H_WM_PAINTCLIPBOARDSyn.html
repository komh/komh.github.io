WM_PAINTCLIPBOARD - Syntax

This message is sent when the clipboard contains a data handle with the CFI_OWNERDISPLAY information flag set. 

param1
     HWND   hwndViewer  /*  Handle. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
