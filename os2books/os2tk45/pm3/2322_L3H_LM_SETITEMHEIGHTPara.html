LM_SETITEMHEIGHT - Parameters

    flNewHeight (ULONG)

      Height of items in list box.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful operation
      FALSE
        Error occurred.
