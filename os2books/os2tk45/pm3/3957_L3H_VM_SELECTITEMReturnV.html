VM_SELECTITEM Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Item was successfully selected.
      FALSE
        An error occurred. The WinGetLastError function may return the following errors:

        • PMERR_INVALID_PARAMETERS
        • PMERR_PARAMETER_OUT_OF_RANGE.
