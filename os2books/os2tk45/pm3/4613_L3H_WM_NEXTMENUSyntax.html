WM_NEXTMENU - Syntax

This message occurs when either the beginning or the end of the menu is reached by use of the cursor control keys. 

param1
     HWND    hwndMenu     /*  Menu-control window handle. */

param2
     USHORT  usPrev       /*  Previous-menu indicator. */
