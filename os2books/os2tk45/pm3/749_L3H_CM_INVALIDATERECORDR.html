CM_INVALIDATERECORD Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Records were successfully refreshed.
      FALSE
        An error occurred. The WinGetLastError function may return the following errors:

        • PMERR_INVALID_PARAMETERS
        • PMERR_INSUFFICIENT_MEMORY.
