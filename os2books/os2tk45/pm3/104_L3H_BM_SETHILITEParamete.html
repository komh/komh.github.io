BM_SETHILITE - Parameters

    ushilite (USHORT)

      Highlight indicator.

      TRUE

        Display the button control in the highlighted state
      FALSE
        Display the button control in the unhighlighted state.

      ulReserved (ULONG)

        Reserved value, should be 0.

      foldstate (BOOL)

        Old highlight state.

        TRUE

          The button control was in highlighted state
        FALSE
          The button control was in unhighlighted state.
