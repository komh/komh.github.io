FDM_FILTER - Parameters

    pFilename (PSZ)

      Pointer to the file name.

    pEAType (PSZ)

      Pointer to the .TYPE EA extended attribute.

    rc (BOOL)

      Success indicator.

      TRUE

        Add the file.
      FALSE
        Do not add the file.
