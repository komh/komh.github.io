TBM_SETHILITE - Syntax

This message is used to highlight or unhighlight a title-bar control. 

param1
     USHORT  usHighlighted  /*  Highlighting indicator. */

param2
     ULONG   ulReserved     /*  Reserved value, should be 0. */
