WM_CONTEXTMENU - Syntax

This message occurs when the operator requests a pop-up menu. 

param1
     POINTS  ptspointerpos  /*  Pointer position. */

param2
     USHORT  usReserved     /*  Reserved value, 0. */
     USHORT  fPointer       /*  Input device flag. */
