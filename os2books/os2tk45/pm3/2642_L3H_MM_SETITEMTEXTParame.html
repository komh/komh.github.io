MM_SETITEMTEXT - Parameters

    usitem (USHORT)

      Item identifier.

    pszItemText (PSZ)

      Item text.

      This points to a string containing the text to set the menu item to.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
