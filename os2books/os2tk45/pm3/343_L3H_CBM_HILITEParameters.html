CBM_HILITE - Parameters

    usHilite (USHORT)

      Highlighting indicator.

      TRUE

        Highlight the entry field control.
      FALSE
        Do not highlight the entry field control.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Changed indicator.

        TRUE

          The highlighting state of the entry field has been changed.
        FALSE
          The highlighting state of the entry field has not been changed.
