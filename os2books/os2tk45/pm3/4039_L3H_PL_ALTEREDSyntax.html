PL_ALTERED - Syntax

This message is broadcast to all frame windows when the PrfReset function is issued. 

param1
     HINI   hiniUser    /*  Handle of the new user profile. */

param2
     HINI   hiniSystem  /*  Handle of the new system profile. */
