CN_GRIDRESIZED - Parameters

    id (USHORT)

      Container control ID.

    CN_GRIDRESIZED (USHORT)

      Notification code.

    sGridRows (SHORT)

      New number of rows.

    sGridCols (SHORT)

      New number of columns.

    ulReserved (ULONG)

      Reserved value, should be 0.
