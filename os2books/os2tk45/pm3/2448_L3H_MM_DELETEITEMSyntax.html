MM_DELETEITEM - Syntax

This message deletes a menu item. 

param1
     USHORT  usitem             /*  Item identifier. */
     USHORT  usincludesubmenus  /*  Include submenus indicator. */

param2
     ULONG   ulReserved         /*  Reserved value, should be 0. */
