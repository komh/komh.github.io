EM_SETINSERTMODE - Parameters

    usInsert (USHORT)

      Insert mode indicator.

      TRUE

        Enable insert mode.
      FALSE
        Enable overtype mode.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Previous insert mode indicator.

        TRUE

          Insert mode was previously enabled.
        FALSE
          Overtype mode was previously enabled.
