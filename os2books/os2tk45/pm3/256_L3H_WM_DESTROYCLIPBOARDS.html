WM_DESTROYCLIPBOARD - Syntax

This message is sent to the clipboard owner when the clipboard is emptied through a call to WinEmptyClipbrd. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
