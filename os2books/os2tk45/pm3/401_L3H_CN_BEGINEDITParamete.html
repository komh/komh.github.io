CN_BEGINEDIT - Parameters

    id (USHORT)

      Container control ID.

    CN_BEGINEDIT (USHORT)

      Notification code.

    pCnrEditData (PCNREDITDATA)

      Pointer to the CNREDITDATA structure.

    ulReserved (ULONG)

      Reserved value, should be 0.
