SM_SETHANDLE - Syntax

This message sets the icon or bit-map handle of a static control. 

param1
     HBITMAP  hbmHandle   /*  Icon or bit-map handle of a static control. */

param2
     ULONG    ulReserved  /*  Reserved value, should be 0. */
