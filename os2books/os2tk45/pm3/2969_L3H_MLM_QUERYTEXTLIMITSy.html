MLM_QUERYTEXTLIMIT - Syntax

This message queries the maximum number of bytes that a multi-line entry field control can contain. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
