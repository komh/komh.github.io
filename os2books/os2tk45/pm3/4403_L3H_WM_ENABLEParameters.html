WM_ENABLE - Parameters

    usnewenabledstate (USHORT)

      New enabled state indicator.

      TRUE

        The window was set to the enabled state.
      FALSE
        The window was set to the disabled state.

      ulReserved (ULONG)

        Reserved value, should be 0.

      ulReserved (ULONG)

        Reserved value, should be 0.
