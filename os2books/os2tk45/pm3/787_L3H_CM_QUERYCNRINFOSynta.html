CM_QUERYCNRINFO - Syntax

This message returns the container's CNRINFO structure. 

param1
     PCNRINFO  pCnrInfo  /*  Pointer to a buffer into which the CNRINFO structure is copied. */

param2
     USHORT    cbBuffer  /*  Number of bytes. */
