EM_SETREADONLY - Parameters

    usReadOnly (USHORT)

      Read only state indicator.

      TRUE

        Enable read only state
      FALSE
        Disable read only state.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Previous read only state indicator.

        TRUE

          Read only state was previously enabled.
        FALSE
          Read only state was previously disabled.
