SM_QUERYHANDLE Return Value - hbmHandle

    hbmHandle (HBITMAP)

      Icon or bit-map handle of the static control.

      NULLHANDLE

        No icon or bit-map handle of the static control exists, or an error occurred.
      Other
        Icon or bit-map handle of the static control.
