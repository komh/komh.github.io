EM_SETFIRSTCHAR - Parameters

    sOffset (SHORT)

      Zero-based offset of the first character to be displayed.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred. For example, because sOffset is not valid.
