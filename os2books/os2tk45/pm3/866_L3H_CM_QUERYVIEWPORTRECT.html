CM_QUERYVIEWPORTRECT Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        The client area rectangle was returned successfully.
      FALSE
        An error occurred. The WinGetLastError function may return the following error:

          PMERR_INVALID_PARAMETERS.
