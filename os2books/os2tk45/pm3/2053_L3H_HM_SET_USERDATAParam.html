HM_SET_USERDATA - Parameters

    ulReserved (ULONG)

      Reserved value, should be 0.

    usrdata (VOID)

      4-byte user data area.

    rc (ULONG)

      Return code.

      TRUE

        The user data was successfully stored.
      FALSE
        The call failed.
