LM_QUERYSELECTION - Syntax

This message is used to enumerate the selected item, or items, in a list box. 

param1
     SHORT  sItemStart     /*  Index of the start item. */

param2
     ULONG  ulReserved     /*  Reserved value, should be 0. */
