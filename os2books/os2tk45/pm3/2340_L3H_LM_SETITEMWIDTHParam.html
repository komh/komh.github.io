LM_SETITEMWIDTH - Parameters

    lNewWidth (ULONG)

      Width of items in list box.

    reserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
