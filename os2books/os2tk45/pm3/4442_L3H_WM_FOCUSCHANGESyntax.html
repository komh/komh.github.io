WM_FOCUSCHANGE - Syntax

This message occurs when the window possessing the focus is changed. 

param1
     HWND    hwndFocus      /*  Focus window handle. */

param2
     USHORT  usSetFocus     /*  Focus flag. */
     USHORT  fsFocusChange  /*  Focus changing indicators. */
