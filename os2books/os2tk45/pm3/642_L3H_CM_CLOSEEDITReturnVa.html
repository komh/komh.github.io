CM_CLOSEEDIT Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        The direct editing of container item text was successfully ended.
      FALSE
        The direct editing of container item text was not successfully ended. The WinGetLastError function may return the following error:
          PMERR_INSUFFICIENT_MEMORY.
