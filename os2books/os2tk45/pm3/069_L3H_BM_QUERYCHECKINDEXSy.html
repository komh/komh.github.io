BM_QUERYCHECKINDEX - Syntax

This message returns the zero-based index of a checked radio button. 

param1
     ULONG  ulReserved  /*  Reserved value, should be 0. */

param2
     ULONG  ulReserved  /*  Reserved value, should be 0. */
