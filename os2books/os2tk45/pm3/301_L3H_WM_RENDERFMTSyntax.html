WM_RENDERFMT - Syntax

This message is a request to the clipboard owner to render the data of the format specified in usfmt. 

param1
     USHORT  usfmt       /*  Data format. */

param2
     ULONG   ulReserved  /*  Reserved value, should be 0. */
