SPBM_SETTEXTLIMIT - Parameters

    usLimit (USHORT)

      Character limit.

      Number of characters to allow.

    ulReserved (ULONG)

      Reserved value, should be 0.

    rc (BOOL)

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
