BKM_SETNOTEBOOKCOLORS Return Value - rc

    rc (BOOL)

      Success indicator.

      TRUE

        Colors were successfully set.
      FALSE
        Unable to set colors. Returned if an invalid notebook attribute is specified for the usBookAttr parameter.
