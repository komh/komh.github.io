BM_SETDEFAULT - Parameters

    usdefault (USHORT)

      Default state.

      TRUE

        Display the button control in the default state
      FALSE
        Display the button control in the nondefault state.

      ulReserved (ULONG)

        Reserved value, should be 0.

      rc (BOOL)

        Success indicator.

        TRUE

          Successful operation
        FALSE
          Error occurred.
