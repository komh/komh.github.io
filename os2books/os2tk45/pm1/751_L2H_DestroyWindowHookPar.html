DestroyWindowHook - Parameters

    hab (HAB) - input

      Anchor-block handle.

    hwnd (HWND) - input

      The handle of the window being destroyed.

    ulReserved (ULONG) - input

      Reserved.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
