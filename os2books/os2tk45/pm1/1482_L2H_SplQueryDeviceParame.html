SplQueryDevice Parameter - pBuf

    pBuf (PVOID) - in/out

      Buffer.

      The returned contents of the buffer depend on the value specified in ulLevel as follows:

        ulLevel

          Buffer Contents
        0
          A port name consisting of 9 characters, including a null terminator.
        2
          A print device name of type PSZ.
        3
          A PRDINFO3 structure.
