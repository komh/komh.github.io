SplQmAbort - Syntax

Description:

SplQmAbort stops the generation of the spool file. It automatically closes the spool file (see SplQmClose). 

#define INCL_SPL /* Or use INCL_PM, */
#include <os2.h>

HSPL    hspl;  /*  Spooler handle. */
BOOL    rc;    /*  Success indicator. */

rc = SplQmAbort(hspl);
