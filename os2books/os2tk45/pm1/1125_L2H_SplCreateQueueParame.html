SplCreateQueue Parameter - pbBuf

    pbBuf (PVOID) - input

      Data structure.

      It points to a data structure depending on the value specified in ulLevel as follows:

        ulLevel

          Buffer Content.
        3
          a PRQINFO3 data structure
        6
          a PRQINFO6 data structure
