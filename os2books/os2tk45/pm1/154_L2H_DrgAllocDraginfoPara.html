DrgAllocDraginfo - Parameters

    cDitem (ULONG) - input

      Number of objects being dragged.

      This number must be greater than 0.

    Draginfo (PDRAGINFO) - returns

      Pointer to the DRAGINFO structure.

      NULL

        Error occurred.
      Other
        The DRAGINFO structure.
