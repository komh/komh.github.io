SplQmClose - Syntax

Description:

SplQmClose closes the spool file. It corresponds to the DevCloseDC function. 

#define INCL_SPL /* Or use INCL_PM, */
#include <os2.h>

HSPL    hspl;  /*  Spooler handle. */
BOOL    rc;    /*  Success indicator. */

rc = SplQmClose(hspl);
