DrgAddStrHandle - Syntax

This function creates a handle to a string. 

#define INCL_WINSTDDRAG
#include <os2.h>

PSZ     pString;  /*  String for which a handle is to be created. */
HSTR    hstr;     /*  String handle. */

hstr = DrgAddStrHandle(pString);
