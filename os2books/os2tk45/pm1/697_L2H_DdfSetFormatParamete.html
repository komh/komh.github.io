DdfSetFormat - Parameters

    hddf (HDDF) - input

      Handle to DDF returned by DdfInitialize.

    fFormatType (ULONG) - input

      Formatting-activation flag.

      Only the following constants may be used in this parameter:

      TRUE

        Turn formatting on.
      FALSE
        Turn formatting off.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
