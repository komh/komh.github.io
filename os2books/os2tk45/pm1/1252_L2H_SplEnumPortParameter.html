SplEnumPort Parameter - pBuf

    pBuf (PVOID) - output

      Buffer.

      The returned content in the buffer depends on the value specified in ulLevel as follows:

        ulLevel

          Buffer Contents
        0
          An array of PRPORTINFO structures
        1
          An array of PRPORTINFO1 structures
