SplQmStartDoc - Parameters

    hspl (HSPL) - input

      Spooler handle.

    pszDocName (PSZ) - input

      Document name.

      This is part of the job description that is displayed to the end user by the spooler.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
