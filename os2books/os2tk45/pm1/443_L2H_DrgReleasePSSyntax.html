DrgReleasePS - Syntax

This function releases a presentation space obtained by using the DrgGetPS function. 

#define INCL_WINSTDDRAG
#include <os2.h>

HPS     hps;  /*  Handle of the presentation space to release. */
BOOL    rc;   /*  Success indicator. */

rc = DrgReleasePS(hps);
