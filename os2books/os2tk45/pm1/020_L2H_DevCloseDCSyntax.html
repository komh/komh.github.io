DevCloseDC - Syntax

This function closes a device context. 

#define INCL_DEV /* Or use INCL_PM, Also in COMMON section */
#include <os2.h>

HDC    hdc;  /*  Device-context handle. */
HMF    hmf;  /*  Error indicator metafile handle (for a metafile device context) */

hmf = DevCloseDC(hdc);
