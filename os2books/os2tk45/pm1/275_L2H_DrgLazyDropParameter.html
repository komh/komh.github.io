DrgLazyDrop Parameter - ulOperation

    ulOperation (ULONG) - input

      Drop operation code.

      Possible values are shown in the following list:

      DO_DEFAULT

        Default operation.
      DO_COPY
        Operation is a copy.
      DO_MOVE
        Operation is a move.
      DO_LINK
        Operation is a link.
