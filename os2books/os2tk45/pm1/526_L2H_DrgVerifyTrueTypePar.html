DrgVerifyTrueType - Parameters

    pDragitem (PDRAGITEM) - input

      Pointer to the DRAGITEM structure whose true type is to be verified.

    pType (PSZ) - input

      String specifying a type.

      This string is in the format: 

      type[,type...]

    rc (BOOL) - returns

      Validity indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
