DrgLazyDrop - Parameters

    hwndTarget (HWND) - input

      Handle of the target window receiving the drop.

    ulOperation (ULONG) - input

      Drop operation code.

      Possible values are shown in the following list:

      DO_DEFAULT

        Default operation.
      DO_COPY
        Operation is a copy.
      DO_MOVE
        Operation is a move.
      DO_LINK
        Operation is a link.

      pptlDrop (PPOINTL) - input

        Pointer to the drop location in desktop coordinates.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Objects are successfully dropped. .
        FALSE
          An error occurred.
