SplEnumJob Parameter - pBuf

    pBuf (PVOID) - output

      Buffer.

      The returned contents in the buffer depend on the value specified in ulLevel as follows:

        ulLevel

          Buffer Contents
        0
          An array containing a uJobId for each of pcReturned jobs.
        2
          An array containing a PRJINFO2 structure for each of pcReturned jobs.
