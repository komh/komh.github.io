DrgSetDragPointer - Parameters

    pdinfo (PDRAGINFO) - input

      Pointer to the DRAGINFO structure to be used for this drag.

    hptr (HPOINTER) - input

      Handle to the pointer to use.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
