HelpHook Parameter - usMode

    usMode (ULONG) - input

      Help mode.

      This has one of the following values, indicating the mode from which help has been requested:

      HLPM_FRAME

        Standard (standard window)
      HLPM_WINDOW
        Standard (standard window)
      HLPM_MENU
        Menu mode
