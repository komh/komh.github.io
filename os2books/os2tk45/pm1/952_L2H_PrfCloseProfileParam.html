PrfCloseProfile - Parameters

    hini (HINI) - input

      Initialization-file handle.

      After this function, the handle is no longer valid.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
