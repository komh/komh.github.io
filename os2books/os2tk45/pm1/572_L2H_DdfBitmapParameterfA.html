DdfBitmap Parameter - fAlign

    fAlign (ULONG) - input

      Text justification flag.

      Any of the following values can be specified:

      ART_LEFT

        Left-justify the bit map.
      ART_RIGHT
        Right-justify the bit map.
      ART_CENTER
        Center the bit map.
      ART_RUNIN
        Allow the bit map to be reflowed with text.
