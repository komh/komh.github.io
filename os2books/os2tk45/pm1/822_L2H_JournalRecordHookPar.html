JournalRecordHook - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pQmsg (PQMSG) - input

      Data structure that contains the message to be recorded.

      The field of the QMSG structure is also set when the hook is called.

      There is no return value for this hook.
