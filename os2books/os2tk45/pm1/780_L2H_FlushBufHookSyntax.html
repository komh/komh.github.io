FlushBufHook - Syntax

This function is specific to OS/2 Version 2.1 or higher.

This hook allows applications to save data before the system reboots. 

#define INCL_WINHOOKS /* Or use INCL_WIN, INCL_PM, */
#include <os2.h>

HAB     hab;  /*  Application anchor block. */
BOOL    rc;   /*  Return code. */

rc = FlushBufHook(hab);
