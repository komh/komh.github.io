WindowDCHook - Parameters

    hab (HAB) - input

      The application anchor block.

    hdc (HDC) - input

      The current device-context handle.

    hwnd (HWND) - input

      The current window handle.

    flAssociate (BOOL) - input

      Association flag.

      TRUE

        Device context has been allocated.
      FALSE
        Device context has been freed.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion.
        FALSE
          Errors occurred.
