NetBIOS Data Structures

The NetBiosOpen and NetBiosClose APIs require no data structure. NetBiosSubmit requires the standard ncb data structure used to define network control blocks (NCBs).

The NetBiosEnum and NetBiosGetInfo APIs return data at either level 0 or level 1 of detail, depending on the value of the sLevel parameter.
