7802H (INT 2AH) Get User ID and Logon Status

This API provides the user ID and logon status information for the current user of the DLS program. ┌───────────────┬──────────────────────────────────────────────────────────────┐

│ ENTRY │ CONTENTS │

│ AX │ 7802H │
├───────────────┼──────────────────────────────────────────────────────────────┤
│ ES:DI │ Address of 8 byte buffer │
└───────────────┴──────────────────────────────────────────────────────────────┘
┌───────────────┬──────────────────────────────────────────────────────────────┐
│ ON │ REGISTER │
│ RETURN │ CONTENTS │
├───────────────┼──────────────────────────────────────────────────────────────┤
│ AL │ "Zero ; No user logged on" │
