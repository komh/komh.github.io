Spooler Category

Spooler APIs give an application access to spooler queue manager operations. These functions are provided by the base operating system and supported by LAN Server across the network.

The following table summarizes the functions used to print jobs, manage jobs, and manipulate the spooler function. They are defined in the base operating system header file, PMSPL.H.
