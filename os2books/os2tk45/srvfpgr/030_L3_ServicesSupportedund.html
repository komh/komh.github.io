Services Supported under DLS

DOS LAN Services supports the following services, which are installed during installation of the redirector program. ┌──────────────┬───────────────────────────────────────────────────────────────┐
│ SERVICE │ PURPOSE │
├──────────────┼───────────────────────────────────────────────────────────────┤
│ Messenger │ Enables DLS workstations to send and receive messages across │
│ │ the LAN │
├──────────────┼───────────────────────────────────────────────────────────────┤
│ Netpopup │ Enables messages to be displayed as pop-up messages on the │
│ │ screen │
├──────────────┼───────────────────────────────────────────────────────────────┤
│ Requester │ Enables DOS computers (attached to the LAN) to be configured │
│ │ as requesters, thus enabling them to access resources on │
│ │ remote OS/2 LAN Servers │
├──────────────┼───────────────────────────────────────────────────────────────┤
│ Peer │ Enables DLS workstations to share printers and drives │
└──────────────┴───────────────────────────────────────────────────────────────┘
