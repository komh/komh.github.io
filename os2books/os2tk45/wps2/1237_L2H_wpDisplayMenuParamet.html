wpDisplayMenu Parameter - ulMenuType

    ulMenuType (ULONG) - input

      Type of menu to display.

      Possible values are described in the following list:

      MENU_EDITPULLDOWN

      MENU_FOLDERMENUBAR
      MENU_FOLDERPULLDOWN
      MENU_HELPPULLDOWN
      MENU_NODISPLAY
      MENU_OBJECTPOPUP
      MENU_OPENVIEWPOPUP
      MENU_SELECTEDPULLDOWN
      MENU_TITLEBARPULLDOWN
      MENU_TITLEBARPULLDOWNINT
      MENU_USER
      MENU_VIEWPULLDOWN
