wpQueryDefStatusView - Parameters

    somSelf (WPPower *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPPower.

    ulDefStatus (ULONG) - returns

      Flag indicating the default status view.

      Possible values are described in the following list:

      OPEN_STATUS

        The full status, including the battery life, power source, and battery state, is shown.
      OPEN_BATTERY
        Only the battery life indicator is shown.
