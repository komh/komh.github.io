wpSetLockupOnStartup - Example Code

This example lets you find the current Desktop object and sets the current lockup at startup option setting to FALSE. 

  WPDesktop    *Desktop;

  Desktop = _wpclsQueryObjectFromPath("<WP_DESKTOP>");
  _wpSetLockupOnStartup(Desktop,FALSE);
