wpDoesObjectMatch - Syntax

This instance method is called to allow the object to determine if it matches the specified criteria. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;                /*  Pointer to the object on which the method is being invoked. */
PVOID         pvoidExtendedCriteria;  /*  Pointer to class-specific extended search criteria. */
BOOL          rc;                     /*  Success indicator. */

rc = _wpDoesObjectMatch(somSelf, pvoidExtendedCriteria);
