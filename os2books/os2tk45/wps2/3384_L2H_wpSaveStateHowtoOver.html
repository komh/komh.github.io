wpSaveState - How to Override

This method should be overridden by all classes that provide savable settings. An override of the wpRestoreState method is a prerequisite.

Override processing of this method typically includes a series of calls to any combination of the save state methods:

  • wpSaveData
  • wpSaveLong
  • wpSaveString
