wpIsDetailsColumnVisible - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method determines whether any hidden column data exists for the current object.

Note: This method is used only in details view. 

    #define INCL_WINWORKPLACE
#include <os2.h>

WPFolder     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
ULONG         index;    /*  Column index for the current details view. */
BOOL          Visible;  /*  Flag indicating whether the details column is visible. */

Visible = _wpIsDetailsColumnVisible(somSelf,
            index);
