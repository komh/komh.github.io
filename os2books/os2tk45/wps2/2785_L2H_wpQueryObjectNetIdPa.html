wpQueryObjectNetId - Parameters

    somSelf (WPNetLink *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPNetLink.

    pszNetIdentity (PSZ) - returns

      Pointer to the network ID string.

      The network name is in the following format: 

      "<Network>\<Server>\<Share Name>"
      Where:

      <Network>

        Network identification
      <Server>
        Server identification
      <Share Name>
