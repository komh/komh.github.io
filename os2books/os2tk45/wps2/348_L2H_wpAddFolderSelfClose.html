wpAddFolderSelfClosePage - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method adds the Folder Automatic Close page to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPFolder     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND          hwndNotebook;  /*  Handle of the Settings notebook. */
ULONG         ulPageId;      /*  Identifier for the inserted page. */

ulPageId = _wpAddFolderSelfClosePage(somSelf,
             hwndNotebook);
