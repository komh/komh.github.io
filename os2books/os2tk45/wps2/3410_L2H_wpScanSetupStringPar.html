wpScanSetupString - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    pszSetupString (PSZ) - input

      Class-specific setup parameters for an object.

    pszKey (PSZ) - input

      Key to scan for.

    pszValue (PSZ) - input

      Buffer for the value.

      If NULL, the size of the buffer required is returned in pcbValue.

    pcbValue (PULONG) - in/out

      Buffer length.

      input

        The length of the buffer pointed to by pszValue. If pszValue is not NULL, this parameter must be initialized with the size of the buffer string (pszValue).
      output
        The actual length of the returned string, plus one for the NULL terminator, or the size of the buffer required if pszValue is NULL or the input value of pcbValue is too small.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion; key found.
        FALSE
          Error occurred; key not found or buffer too small.
