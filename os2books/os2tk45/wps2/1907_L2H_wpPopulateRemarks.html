wpPopulate - Remarks

The wpPopulate method sets the folder flags depending upon the value of fFoldersOnly.

The folder flags indicate the current population state of the folder:

    FOI_POPULATEDWITHALL

      Folder is completely populated.
    FOI_POPULATEDWITHFOLDERS
      Folder is populated only with subfolders.

    If the folder is re-populated when it has already been populated (as determined by inspecting the folder flags), no action is taken on this message apart from sending the notification message back. This method locks each object it instantiates as a result of the call.
