wpHoldJob - Syntax

This instance method is called to hold a job object. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPJob     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
BOOL       rc;       /*  Success indicator. */

rc = _wpHoldJob(somSelf);
