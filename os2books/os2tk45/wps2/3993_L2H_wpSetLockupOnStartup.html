wpSetLockupOnStartup - Parameters

    somSelf (WPDesktop *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPDesktop.

    bLockupOnStartup (BOOL) - input

      Lockup value.

      TRUE

        Automatically locks up the system when the system is started.
      FALSE
        The system is not locked up when started.

        There is no return value for this method.
