wpAddTitleConfirmationPage - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method is called to add the Title Confirmation page to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPSystem     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND          hwndNotebook;  /*  Window handle for the Settings notebook. */
ULONG         ulPageId;      /*  Identifier for the inserted page. */

ulPageId = _wpAddTitleConfirmationPage(somSelf,
             hwndNotebook);
