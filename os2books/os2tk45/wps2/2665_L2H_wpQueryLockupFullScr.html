wpQueryLockupFullScreen - Example Code

This example lets you find the current Desktop object and retrieves the current lockup, full-screen option setting. 

  WPDesktop    *Desktop;
  BOOL         bFullScreen;

  Desktop     = _wpclsQueryObjectFromPath("<WP_DESKTOP>");
  bFullScreen = _wpQueryLockupFullScreen(Desktop);
