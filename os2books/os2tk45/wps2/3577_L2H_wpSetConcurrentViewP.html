wpSetConcurrentView - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    ulCCView (ULONG) - input

      Flag indicating the concurrent view behavior of the object.

      Possible values are described in the following list:

      CCVIEW_DEFAULT

        Concurrent view mode is set to the system default.
      CCVIEW_ON
        Concurrent view mode is on.
      CCVIEW_OFF
        Concurrent view mode is off.

        There is no return value for this method.
