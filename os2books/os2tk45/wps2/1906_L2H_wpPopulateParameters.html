wpPopulate - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    ulReserved (ULONG) - input

      Reserved value; must be 0.

    pszPath (PSZ) - input

      Reserved value; must be NULL.

    fFoldersOnly (BOOL) - input

      Populate folder.

      The ored flag indicating the type of contents with which to populate the folder:

      TRUE

        Populate with folder objects only.
      FALSE
        Populate with entire contents of folder.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion.
        FALSE
          Error occurred.
