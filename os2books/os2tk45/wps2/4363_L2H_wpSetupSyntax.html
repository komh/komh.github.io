wpSetup - Syntax

This instance method is called to allow the newly created object to initialize itself based on an input setup string. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;         /*  Pointer to the object on which the method is being invoked. */
PSZ           pszSetupString;  /*  Class-specific setup parameters for an object. */
BOOL          rc;              /*  Success indicator. */

rc = _wpSetup(somSelf, pszSetupString);
