wpSetLockupFullScreen - Parameters

    somSelf (WPDesktop *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPDesktop.

    bLockupFullScreen (BOOL) - input

      Value of the lockup full-screen option.

      TRUE

        Displays the full-screen lockup bitmap when locked.
      FALSE
        Displays the lockup window when locked.

        There is no return value for this method.
