wpReadImageFile - Usage

This method reads the image data from the image data file and converts it to the format used for an OS/2 Bitmap. This method stores the data in a buffer pointed to by the instance data for the object.

Various WPImageFile methods call the wpReadImageFile method to read the data from an image file and convert it to a bitmap.
