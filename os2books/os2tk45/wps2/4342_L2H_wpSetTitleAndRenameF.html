wpSetTitleAndRenameFile Parameter - fConfirmations

    fConfirmations (ULONG) - input

      Confirmation flag. .

      Possible values are described in the following list:

      CONFIRM_RENAMEFILEWITHEXT

        Prompt the user to confirm renaming the file to match the title.
      CONFIRM_KEEPASSOC
        Prompt for confirmation when renaming files with extensions that are used to associate a file with an application.

        For example, if the user has an association between an editor and the file extension DOC, and the user renames a DOC file to have a TXT extension, the file is no longer associated with the editor. The user should be prompted to confirm the name change.

      0
        Do not prompt the user to confirm renaming the file to match the title.
