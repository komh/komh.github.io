wpSetFloatOnTop - Parameters

    somSelf (WPLaunchPad *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPLaunchPad.

    fState (BOOL) - input

      Flag indicating whether to force the Toolbar to float on top of all windows.

      TRUE

        Force the Toolbar to float on top.
      FALSE
        Do not force the Toolbar to float on top.

        There is no return value for this method.
