wpSetDefStatusView Parameter - ulDefStatus

    ulDefStatus (ULONG) - input

      Setting for the default status view.

      Possible values are described in the following list:

      OPEN_STATUS

        Full status of the object, including the battery life, power source, and battery state.
      OPEN_BATTERY
        Battery life indicator only.
