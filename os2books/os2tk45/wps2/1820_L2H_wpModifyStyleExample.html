wpModifyStyle - Example Code

The following code changes the style of an object to 'not moveable', 'not copyable', 'deleteable'. 

_wpModifyStyle (self, OBJSTYLE_NOMOVE |
                      OBJSTYLE_NOCOPY |
                      OBJSTYLE_NODELETE,

                      OBJSTYLE_NOMOVE |
                      OBJSTYLE_NOCOPY);
