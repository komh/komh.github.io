wpSetAssociationType - Syntax

This instance method is called to set an association of the program object to a data file object based on a file type. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPProgramFile     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
PSZ                pszType;  /*  Pointer to file types. */
BOOL               rc;       /*  Success indicator. */

rc = _wpSetAssociationType(somSelf, pszType);
