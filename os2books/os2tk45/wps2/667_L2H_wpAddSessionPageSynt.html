wpAddSessionPage - Syntax

This method is specfic to Version 3, or higher, of the OS/2 operating system.

This instance method is called to add the 3.0 Session Details page to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPWinConfig     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND             hwndNotebook;  /*  Handle to the Settings notebook. */
ULONG            rc;            /*  Page identifier. */

rc = _wpAddSessionPage(somSelf, hwndNotebook);
