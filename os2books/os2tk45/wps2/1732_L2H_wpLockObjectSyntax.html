wpLockObject - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method locks an object by incrementing the lock count of the receiving object so that it cannot be made dormant. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;  /*  Pointer to the object on which the method is being invoked. */

wpLockObject(somSelf);
