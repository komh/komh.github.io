wpDeleteFromContent - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method removes an object from a folder's Content List. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPFolder     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
WPObject     *Object;   /*  Pointer to the object to be deleted from the folder's content list. */
BOOL          rc;       /*  Success indicator. */

rc = _wpDeleteFromContent(somSelf, Object);
