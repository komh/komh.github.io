wpSetIconTextVisibility Parameter - ulOption

    ulOption (ULONG) - input

      The visibility state.

    • Makes the icon text not visible.
      ICON_TEXT_VISIBLE
        Makes the icon text visible.
      USE_GLOBAL_VISIBILITY
        Uses the global value.
