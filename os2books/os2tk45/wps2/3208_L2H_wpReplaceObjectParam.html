wpReplaceObject - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    targetObject (WPObject *) - input

      Object to replace.

    fMove (BOOL32) - input

      Type of move flag.

      TRUE

        Move and replace object.
      FALSE
        Copy and replace object.

      rc (BOOL32) - returns

        Success indicator

        TRUE

          Successful completion.
        FALSE
          Error occurred.
