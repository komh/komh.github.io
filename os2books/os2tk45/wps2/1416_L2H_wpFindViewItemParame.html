wpFindViewItem - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    flViews (ULONG) - input

      Flag indicating the views to search for.

      VIEW_CONTENTS

        Icon view.
      VIEW_SETTINGS
        Settings notebook.
      VIEW_HELP
        Help view.
      VIEW_RUNNING
        Program running.
      VIEW_DETAILS
        Details view.
      VIEW_TREE
        Tree view.
      VIEW_ANY
        Any view item.

      pCurrentItem (PVIEWITEM) - input

        Pointer to the previous view item in the in-use list.

        A value of NULL indicates that the current view item is the first item in the list.

      pNextItem (PVIEWITEM) - returns

        Pointer to the view item that was found.
