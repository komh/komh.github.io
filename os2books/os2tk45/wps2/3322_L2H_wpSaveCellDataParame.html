wpSaveCellData - Parameters

    somSelf (WPPalette *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPPalette.

    pCell (PCELL) - input

      Pointer to the cell whose data is to be saved.

    ulIndex (ULONG) - input

      Cell index.

      This is the number of the cell. It must be within the range 0 to xCellCount*yCellCount-1.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
