wpAppendObject - Example Code

This example appends the file "c:\file2" to the end of "c:\file1" and preserves the original "c:\file2" file. 

 WPObject     *File1,File2;

 File1 = _wpQueryObjectFromPath(_WPFileSystem,"c:\\file1");
 File2 = _wpQueryObjectFromPath(_WPFileSystem,"c:\\file2");
 _wpAppendObject(File2,File1,FALSE);
