wpQueryAssociationType - Parameters

    somSelf (WPProgram *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPProgram.

    rc (PSZ) - returns

      Success indicator.

      NULL

        Error occurred.
      Other
        filedesc. A pointer to a string containing the file type. This string can contain several file title filters separated by commas. For example: 
        "Plain Text,C Code"
