wpSetIconTextColor Parameter - ulView

    ulView (ULONG) - input

      Specifies which view to query.

      OPEN_CONTENTS

        Open contents view.
      OPEN_DEFAULT
        Open default view (same as double-click).
      OPEN_DETAILS
        Open details view.
      OPEN_HELP
        Display HelpPanel.
      OPEN_RUNNING
        Execute object.
      OPEN_SETTINGS
        Open Settings notebook.
      OPEN_TREE
        Open tree view.
      OPEN_USER
        Class-specific views have a greater value than this.
