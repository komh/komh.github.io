wpQueryDefaultView - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    ulView (ULONG) - returns

      Flag indicating the default open view.

      OPEN_CONTENTS

        Open contents view.
      OPEN_DEFAULT
        Open default view (same as double-click).
      OPEN_DETAILS
        Open details view.
      OPEN_HELP
        Display HelpPanel.
      OPEN_RUNNING
        Execute object.
      OPEN_SETTINGS
        Open Settings notebook.
      OPEN_TREE
        Open tree view.
      OPEN_UNKNOWN
        Unknown view.
      OPEN_USER
        Class-specific views have a greater value than this.
