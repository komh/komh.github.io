wpQueryObjectNetId - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method returns the network ID of the Shared Directory object that this object is linked to. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPNetLink     *somSelf;         /*  Pointer to the object on which the method is being invoked. */
PSZ            pszNetIdentity;  /*  Pointer to the network ID string. */

pszNetIdentity = _wpQueryObjectNetId(somSelf);
