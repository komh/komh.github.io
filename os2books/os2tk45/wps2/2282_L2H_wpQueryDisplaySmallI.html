wpQueryDisplaySmallIcons - Parameters

    somSelf (WPLaunchPad *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPLaunchPad.

    fState (BOOL) - returns

      Flag indicating whether small icons are displayed.

      TRUE

        Small icons are displayed.
      FALSE
        Normal-sized icons are displayed.
