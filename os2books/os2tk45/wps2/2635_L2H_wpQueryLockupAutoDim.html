wpQueryLockupAutoDim Return Value - rc

    rc (BOOL) - returns

      Current value of the lockup auto-dim option.

      TRUE

        Clears the screen and displays the floating lock after the timeout.
      FALSE
        Does not clear the screen or display the floating lock after the timeout.
