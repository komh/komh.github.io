wpDeleteContents - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    fConfirmations (ULONG) - input

      Confirmation flags.

      CONFIRM_DELETE

        Prompt for confirmation for all objects.
      CONFIRM_DELETEFOLDER
        Prompt for confirmation for folder objects only.
      NULL
        No confirmations.

      rc (ULONG) - returns

        Success indicator.

        OK_DELETE

          All objects were deleted.
        NO_DELETE
          Error occurred. At least one object was not deleted.
        CANCEL_DELETE
          User canceled the operation.
