wpSetDefStatusView - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method is called to set the default status view when Advanced Power Management (APM) is enabled. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPPower     *somSelf;      /*  Pointer to the object on which the method is being invoked. */
ULONG        ulDefStatus;  /*  Setting for the default status view. */

wpSetDefStatusView(somSelf, ulDefStatus);
