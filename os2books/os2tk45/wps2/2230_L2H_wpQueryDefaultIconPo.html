wpQueryDefaultIconPos - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method returns the default icon position of the current object within a folder. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
PPOINTL       pPointl;  /*  Pointer to the coordinates of the default icon position. */
BOOL          rc;       /*  Success indicator. */

rc = _wpQueryDefaultIconPos(somSelf, pPointl);
