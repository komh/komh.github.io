wpQueryRealName - Remarks

This method returns the fully qualified file name for this object. Generally, the object's real name and an object's title are the same. For file systems that don't support the features of a title, for example, characters, mixed case, and spaces, the title is stored in the .LONGNAME extended attribute and then the title and real name may differ. The real name of the file object can be used with any of the DOSXXX functions that act on file names.
