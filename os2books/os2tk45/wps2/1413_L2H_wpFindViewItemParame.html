wpFindViewItem Parameter - flViews

    flViews (ULONG) - input

      Flag indicating the views to search for.

      VIEW_CONTENTS

        Icon view.
      VIEW_SETTINGS
        Settings notebook.
      VIEW_HELP
        Help view.
      VIEW_RUNNING
        Program running.
      VIEW_DETAILS
        Details view.
      VIEW_TREE
        Tree view.
      VIEW_ANY
        Any view item.
