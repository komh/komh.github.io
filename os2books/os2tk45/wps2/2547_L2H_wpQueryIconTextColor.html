wpQueryIconTextColor - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    ulView (ULONG) - input

      Specifies which view to query.

      OPEN_CONTENTS

        Open contents view.
      OPEN_DEFAULT
        Open default view (same as double-click).
      OPEN_DETAILS
        Open details view.
      OPEN_HELP
        Display HelpPanel.
      OPEN_RUNNING
        Execute object.
      OPEN_SETTINGS
        Open Settings notebook.
      OPEN_TREE
        Open tree view.
      OPEN_USER
        Class-specific views have a greater value than this.

      rc (ULONG) - returns

        The RGB color value of the current color being used for the icon text background.

        The RGB value is presented as a 6-digit hex value in the format 0xRRGGBB where RR, GG, and BB are the red, green, and blue values ranging between 0x00 and 0xFF (0-255).
