wpSetFldrFlags - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    ulFlags (ULONG) - input

      Folder flags to be set.

      FOI_POPULATEDWITHALL

        This flag is set if the folder was populated with all its contents.
      FOI_POPULATEDWITHFOLDERS
        This flag is set if the folder was populated only with folders that it contains.
      FOI_WORKAREA
        This flag is set if the user sets the work-area property.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion.
        FALSE
          Error occurred.
