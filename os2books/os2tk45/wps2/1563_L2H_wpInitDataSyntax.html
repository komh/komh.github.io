wpInitData - Syntax

This instance method is called to allow the object to initialize its instance data. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;  /*  Pointer to the object on which the method is being invoked. */

wpInitData(somSelf);
