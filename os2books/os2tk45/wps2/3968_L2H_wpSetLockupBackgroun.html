wpSetLockupBackground Parameter - ulBackgroundType

    ulBackgroundType (ULONG) - input

      Can be one of the following background types:

      BACKGROUND_COLOR_ONLY

        Color only, no image.
      BACKGROUND_IMAGE
        Type of image (see image modes).
