wpFilterMenu Parameter - hwndCnr

    hwndCnr (HWND) - input

      Handle to container control window.

      Possible values are described in the following list:

      NULLHANDLE

        A pop-up request is made on the "whitespace" of an open view.
      Other
        The handle of the container requesting the pop-up menu.
