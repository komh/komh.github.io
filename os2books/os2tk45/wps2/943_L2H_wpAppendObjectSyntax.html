wpAppendObject - Syntax

This method is specific to Version 4, or higher, of the OS/2 operating system.

This instance method append a file system object to another file system object. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
WPObject     *targetObject;  /*  Object to append to. */
BOOL32        fMove;         /*  Type of move flag. */
BOOL32        rc;            /*  Success indicator */

rc = _wpAppendObject(somSelf, targetObject,
       fMove);
