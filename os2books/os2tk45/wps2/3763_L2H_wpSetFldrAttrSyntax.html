wpSetFldrAttr - Syntax

This instance method is called to allow the folder to change its current view attributes. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPFolder     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
ULONG         Attr;     /*  Flag indicating the object's attributes. */
ULONG         ulView;   /*  Flag indicating an object's open view. */
BOOL          rc;       /*  Success indicator. */

rc = _wpSetFldrAttr(somSelf, Attr, ulView);
