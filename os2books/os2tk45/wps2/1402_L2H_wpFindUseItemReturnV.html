wpFindUseItem Return Value - rc

    rc (PUSEITEM) - returns

      Success indicator.

      NULL

        No items matching the specified find criteria were found in the in-use list for this object.
      Other
        A pointer to a USEITEM structure that matches the specified find criteria.
