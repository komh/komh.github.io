wpSetMenuBarVisibility - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    ulVisibility (ULONG) - input

      Visibility state.

      MENUBAR_ON

        Makes the menu bar visible.
      MENUBAR_OFF
        Makes the menu bar invisible.
      MENUBAR_DEFAULT
        Sets the menu bar visibility to the default state.

      rc (BOOL) - returns

        Success indicator

        TRUE

          Successful completion.
        FALSE
          Error occurred.
