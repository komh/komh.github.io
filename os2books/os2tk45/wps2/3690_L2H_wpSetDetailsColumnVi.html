wpSetDetailsColumnVisibility - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    index (ULONG) - input

      Column index for current details view.

    Visible (BOOL) - input

      Flag indicating whether to show or hide the details column.

      Possible values are described in the following list:

      TRUE

        Show the details column.
      FALSE
        Hide the details column.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion.
        FALSE
          Error occurred.
