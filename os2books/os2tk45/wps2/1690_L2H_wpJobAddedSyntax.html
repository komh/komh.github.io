wpJobAdded - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method notifies the printer object that a job has been added to the print queue. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPPrinter     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
ULONG          ulJobId;  /*  ID of the job added to the print queue. */
BOOL           rc;       /*  Success indicator. */

rc = _wpJobAdded(somSelf, ulJobId);
