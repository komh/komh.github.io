wpFindUseItem Parameter - pCurrentItem

    pCurrentItem (PUSEITEM) - input

      Pointer to a USEITEM structure.

      NULL

        Retrieve the first item in the in-use list that has a usage type of type.
      Other
        Retrieve the next item in the in-use list, following the item specified by pCurrentItem, that has a usage type of type.
