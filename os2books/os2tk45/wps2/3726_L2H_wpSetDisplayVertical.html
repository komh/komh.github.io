wpSetDisplayVertical - Parameters

    somSelf (WPLaunchPad *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPLaunchPad.

    fState (BOOL) - input

      Flag indicating the orientation of the Toolbar.

      TRUE

        Display the Toolbar vertically.
      FALSE
        Display the Toolbar horizontally.

        There is no return value for this method.
