wpQueryAssociatedProgram Parameter - pulHowMatched

    pulHowMatched (PULONG) - input

      Flag indicating the association type.

      Possible values are described in the following list:

    • Associated by a program added to the menu page.
    • Associated by a match from a name filter.
    • Associated by a match of types.
