wpSetSortAttribAvailable - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method sets the availability of sort attributes for the current folder. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPFolder     *somSelf;    /*  Pointer to the object on which the method is being invoked. */
ULONG         index;      /*  Column index for the current details view. */
BOOL          Available;  /*  Flag indicating whether to show or hide the sort attributes. */
BOOL          rc;         /*  Success indicator. */

rc = _wpSetSortAttribAvailable(somSelf, index,
       Available);
