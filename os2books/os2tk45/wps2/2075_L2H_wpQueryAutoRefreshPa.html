wpQueryAutoRefresh - Parameters

    somSelf (WPPower *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPPower.

    rc (BOOL) - returns

      Automatic refresh of the power status window.

      TRUE

        Automatic refresh of the power status window enabled.
      FALSE
        Automatic refresh of the power status window disabled.
