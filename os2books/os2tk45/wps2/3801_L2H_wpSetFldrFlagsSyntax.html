wpSetFldrFlags - Syntax

This instance method is called to allow the folder to change its current flags. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPFolder     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
ULONG         ulFlags;  /*  Folder flags to be set. */
BOOL          rc;       /*  Success indicator. */

rc = _wpSetFldrFlags(somSelf, ulFlags);
