wpQueryDrawerHWND - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method returns the handle associated with the open Toolbar or drawer. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPLaunchPad     *somSelf;   /*  Pointer to the object on which the method is being invoked. */
ULONG            ulDrawer;  /*  Toolbar or drawer being queried. */
HWND             hwnd;      /*  Handle of the frame window for the Toolbar or drawer. */

hwnd = _wpQueryDrawerHWND(somSelf, ulDrawer);
