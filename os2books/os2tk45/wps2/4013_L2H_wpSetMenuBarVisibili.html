wpSetMenuBarVisibility Parameter - ulVisibility

    ulVisibility (ULONG) - input

      Visibility state.

      MENUBAR_ON

        Makes the menu bar visible.
      MENUBAR_OFF
        Makes the menu bar invisible.
      MENUBAR_DEFAULT
        Sets the menu bar visibility to the default state.
