wpAddDeviceDevice2Page - Syntax

This method is specific to Version 4, or higher, of the OS/2 operating system.

This instance method is called to allow an object to add the second Device page to the settings notebook. 

#define INCL_WORKPLACE
#include <os2.h>

WPDevice     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND          hwndNotebook;  /*  Settings notebook handle. */
ULONG         rc;            /*  Page identifier. */

rc = _wpAddDeviceDevice2Page(somSelf, hwndNotebook);
