wpDeleteFromObjUseList - Syntax

This instance method is called to remove an item type from an object's in-use list. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;   /*  Pointer to the object on which the method is being invoked. */
PUSEITEM      pUseItem;  /*  Pointer to a USEITEM structure. */
BOOL          rc;        /*  Success indicator */

rc = _wpDeleteFromObjUseList(somSelf, pUseItem);
