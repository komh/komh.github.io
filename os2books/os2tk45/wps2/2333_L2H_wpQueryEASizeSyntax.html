wpQueryEASize - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method returns the size of a file's extended attributes (EAs). 

#define INCL_WINWORKPLACE
#include <os2.h>

WPFileSystem     *somSelf;   /*  Pointer to the object on which the method is being invoked. */
ULONG             ulEASize;  /*  Size of the file's extended attributes. */

ulEASize = _wpQueryEASize(somSelf);
