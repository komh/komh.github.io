wpAddDMQSDisplayTypePage - Parameters

    somSelf (WPSystem *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPSystem.

    hwndNotebook (HWND) - input

      Window handle to the Settings notebook.

    ulPageId (ULONG) - returns

      Identifier for the inserted page.

      A return value of 0 indicates that the DMQS Display Type page was not added to the Settings notebook.
