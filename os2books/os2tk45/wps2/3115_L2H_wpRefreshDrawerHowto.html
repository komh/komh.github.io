wpRefreshDrawer - How to Override

This method can be overridden to know when the drawer is being refreshed.

During the processing of wpRefreshDrawer, wpSetDrawerHWND is called to set hwnd to NULLHANDLE while the objects are being recreated. It then calls wpSetDrawerHWND to set hwnd back to handle of the frame window for the drawer.
