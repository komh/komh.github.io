wpJobAdded - Parameters

    somSelf (WPPrinter *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPPrinter.

    ulJobId (ULONG) - input

      ID of the job added to the print queue.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion.
      FALSE
        Error occurred.
