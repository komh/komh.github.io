wpSetFloatOnTop - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method sets whether the Toolbar is forced to float on top of all other windows. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPLaunchPad     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
BOOL             fState;   /*  Flag indicating whether to force the Toolbar to float on top of all windows. */

wpSetFloatOnTop(somSelf, fState);
