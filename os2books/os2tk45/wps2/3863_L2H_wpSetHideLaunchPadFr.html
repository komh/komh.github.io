wpSetHideLaunchPadFrameCtls - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method hides the system menu and the title bar for the Toolbar. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPLaunchPad     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
BOOL             fState;   /*  Flag indicating whether to hide frame controls. */

wpSetHideLaunchPadFrameCtls(somSelf, fState);
