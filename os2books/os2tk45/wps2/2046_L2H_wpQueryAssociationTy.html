wpQueryAssociationType Return Value - rc

    rc (PSZ) - returns

      Success indicator.

      NULL

        Error occurred.
      Other
        A pointer to a string containing the file types. This string can contain several file title filters separated by commas. For example: 
        "Plain Text,C Code"
