wpSetLockupAutoDim - Example Code

This example lets you find the current Desktop object and sets the current lockup automatic dim option setting to FALSE. 

  WPDesktop    *Desktop;

  Desktop = _wpclsQueryObjectFromPath("<WP_DESKTOP>");
  _wpSetLockupAutoDim(Desktop,FALSE);
