wpQueryCoreRecord - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method returns a pointer to a MINIRECORDCORE structure associated with a given object. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject          *somSelf;      /*  Pointer to the object on which the method is being invoked. */
PMINIRECORDCORE    pMiniRecord;  /*  Pointer to an object's container control record. */

pMiniRecord = _wpQueryCoreRecord(somSelf);
