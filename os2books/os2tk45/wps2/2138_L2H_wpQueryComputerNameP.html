wpQueryComputerName - Parameters

    somSelf (WPPrinter *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPPrinter.

    pszComputerName (PSZ) - output

      Returned computer name of the object queried.

    rc (ULONG) - returns

      Success indicator.

    • An error occurred. 1
        Successful - ComputerName is local (NULL).
      2
        Successful - ComputerName is on a network. See pszComputerName for values.
