wpAddSoundSchemePage - Syntax

This method is specific to Version 4, or higher, of the OS/2 operating system.

This instance method adds the Sound Scheme page to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPSound     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND         hwndNotebook;  /*  Window handle for the Settings notebook. */
ULONG        rc;            /*  Page identifier. */

rc = _wpAddSoundSchemePage(somSelf, hwndNotebook);
