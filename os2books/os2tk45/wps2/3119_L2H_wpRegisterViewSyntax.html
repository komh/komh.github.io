wpRegisterView - Syntax

This instance method is called to allow the object to register a new open view. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND          hwndFrame;     /*  Handle to the frame window containing the new view. */
PSZ           pszViewTitle;  /*  Pointer to a string containing the name of the view. */
BOOL          rc;            /*  Success indicator. */

rc = _wpRegisterView(somSelf, hwndFrame, pszViewTitle);
