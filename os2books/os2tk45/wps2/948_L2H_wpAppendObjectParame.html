wpAppendObject - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    targetObject (WPObject *) - input

      Object to append to.

    fMove (BOOL32) - input

      Type of move flag.

      TRUE

        Moves and appends an object.
      FALSE
        Copies and appends an object.

      rc (BOOL32) - returns

        Success indicator

        TRUE

          Successful completion.
        FALSE
          Error occurred.
