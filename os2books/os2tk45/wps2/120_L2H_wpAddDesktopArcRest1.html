wpAddDesktopArcRest1Page - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method is called to allow the object to add the Archive page to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPDesktop     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND           hwndNotebook;  /*  Settings notebook handle. */
ULONG          rc;            /*  Page identifier. */

rc = _wpAddDesktopArcRest1Page(somSelf, hwndNotebook);
