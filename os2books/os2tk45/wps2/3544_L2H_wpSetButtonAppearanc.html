wpSetButtonAppearance - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method sets the appearance of an object's Frame Control Button. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
ULONG         ulButtonType;  /*  Flag indicating the type of control button. */

wpSetButtonAppearance(somSelf, ulButtonType);
