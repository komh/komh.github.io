wpIsLocked - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method indicates whether the object can be made dormant. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPObject     *somSelf;  /*  Pointer to the object on which the method is being invoked. */
BOOL          rc;       /*  A flag indicating whether the object can be made dormant. */

rc = _wpIsLocked(somSelf);
