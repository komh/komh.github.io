wpAddDDEPage - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method is called to add the DDE Details page to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPWinConfig     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND             hwndNotebook;  /*  Handle to the Settings notebook. */
ULONG            ulPageid;      /*  Identifier for the inserted page. */

ulPageid = _wpAddDDEPage(somSelf, hwndNotebook);
