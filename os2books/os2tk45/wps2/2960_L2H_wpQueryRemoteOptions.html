wpQueryRemoteOptions Parameter - pflAllJobs

    pflAllJobs (PULONG) - output

      Pointer to the flag indicating which jobs to display.

      TRUE

        Display all jobs waiting in the network queue.
      FALSE
        Display only the current user's jobs waiting in the network queue.
