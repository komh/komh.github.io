wpQueryRefreshFlags - Parameters

    somSelf (WPFileSystem *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFileSystem.

    rc (ULONG) - returns

      Refresh flags.

      DIRTYBIT

        (0x80000000) Used for refreshing the views of pre-existing objects.
      FOUNDBIT
        (0x40000000) Used for refreshing the views of new and deleted objects.
