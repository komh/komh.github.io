wpHideFldrRunObjs - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    fHide (BOOL) - input

      Flag indicating whether to hide or show open objects.

      TRUE

        Hide all objects opened from this folder.
      FALSE
        Show all objects opened from this folder.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion.
        FALSE
          Error occurred.
