wpContainsFolders - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    pfSubFolders (BOOL *) - output

      Flag indicating whether the folder contains subfolders.

      Possible values are described in the following list:

      TRUE

        The folder contains subfolders.
      FALSE
        The folder does not contain subfolders.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion.
        FALSE
          Error occurred.
