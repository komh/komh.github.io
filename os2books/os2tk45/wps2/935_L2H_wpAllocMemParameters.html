wpAllocMem - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    cbBytes (ULONG) - in/out

      Specifies the size, in bytes, of memory required.

    prc (PULONG) - in/out

      Pointer to error code.

      NULL

        If an error occurs, the system displays an appropriate message.
      prc
        If an error occurs, one of the following is returned in prc:

      • ERROR_INVALID_HANDLE 8
          ERROR_NOT_ENOUGH_MEMORY
        95
          ERROR_INTERRUPT
        103
          ERROR_TOO_MANY_SEM_REQUESTS
        105
          ERROR_SEM_OWNER_DIED
        640
          ERROR_TIMEOUT

        pbNewMem (PBYTE) - returns

          Success indicator.

          NULL

            Memory allocation failed.
          Other
            A pointer to the newly allocated memory.
