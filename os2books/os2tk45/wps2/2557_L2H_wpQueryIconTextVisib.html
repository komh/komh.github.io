wpQueryIconTextVisibility Return Value - rc

    rc (ULONG) - returns

      The current state of icon text visibility.

      Byte 1

        Has the following values:

        0x00

          Icon text is not visible.
        0x01
          Icon text is visible (ICON_TEXT_VISIBLE).
        Byte 4
          Has the following values:

          0x00

            Instance setting
          0x40
            Global setting (USE_GLOBAL_VISIBILITY).
