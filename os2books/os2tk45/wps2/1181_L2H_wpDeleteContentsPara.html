wpDeleteContents Parameter - fConfirmations

    fConfirmations (ULONG) - input

      Confirmation flags.

      CONFIRM_DELETE

        Prompt for confirmation for all objects.
      CONFIRM_DELETEFOLDER
        Prompt for confirmation for folder objects only.
      NULL
        No confirmations.
