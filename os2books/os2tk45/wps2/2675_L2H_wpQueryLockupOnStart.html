wpQueryLockupOnStartup - Example Code

This examples lets you find the current Desktop object and retrieves the current lockup at startup option setting. 

  WPDesktop    *Desktop;
  BOOL         bLockupAtStartup;

  Desktop = _wpclsQueryObjectFromPath("<WP_DESKTOP>");
  bLockupAtStartup = _wpQueryLockupOnStartup(Desktop);
