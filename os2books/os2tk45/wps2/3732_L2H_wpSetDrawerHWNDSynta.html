wpSetDrawerHWND - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method stores the handle associated with the Toolbar or drawer into the instance data. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPLaunchPad     *somSelf;   /*  Pointer to the object on which the method is being invoked. */
ULONG            ulDrawer;  /*  Toolbar or drawer being set. */
HWND             hwnd;      /*  Handle of the frame window for the drawer. */

wpSetDrawerHWND(somSelf, ulDrawer, hwnd);
