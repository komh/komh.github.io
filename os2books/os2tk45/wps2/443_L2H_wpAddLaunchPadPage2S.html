wpAddLaunchPadPage2 - Syntax

This method is specific to version 3, or higher, of the OS/2 operating system.

This instance method is called to add the Toolbar page 2 to the Settings notebook. 

#define INCL_WINWORKPLACE
#include <os2.h>

WPLaunchPad     *somSelf;       /*  Pointer to the object on which the method is being invoked. */
HWND             hwndNotebook;  /*  Handle to the Settings notebook. */
ULONG            rc;            /*  Page identifier. */

rc = _wpAddLaunchPadPage2(somSelf, hwndNotebook);
