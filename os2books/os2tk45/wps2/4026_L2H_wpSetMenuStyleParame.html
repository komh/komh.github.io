wpSetMenuStyle - Parameters

    somSelf (WPObject *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPObject.

    ulStyle (ULONG) - input

      Menu style.

      MENUS_LONG

        Long menus.
      MENUS_SHORT
        Short menus.
      MENUS_DEFAULT
        Sets the menu style to the default style.

      rc (BOOL) - returns

        Success indicator

        TRUE

          Successful completion.
        FALSE
          Error occurred.
