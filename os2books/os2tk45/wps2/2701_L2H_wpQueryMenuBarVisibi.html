wpQueryMenuBarVisibility - Parameters

    somSelf (WPFolder *) - input

      Pointer to the object on which the method is being invoked.

      Points to an object of class WPFolder.

    rc (ULONG) - returns

      Visibility state.

      MENUBAR_ON

        Menu bar is visible.
      MENUBAR_OFF
        Menu bar is invisible.
