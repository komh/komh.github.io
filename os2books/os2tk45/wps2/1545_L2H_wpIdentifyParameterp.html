wpIdentify Parameter - pszIdentity

    pszIdentity (PSZ) - output

      A unique string that identifies the specified object.

      The string is of the form "CCCCIIII" where CCCC is the class name and IIII is a unique string identifier provided by the base class (the real file name in the case of WPFileSystem).
