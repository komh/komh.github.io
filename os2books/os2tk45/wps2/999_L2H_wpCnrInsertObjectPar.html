wpCnrInsertObject Parameter - pRecInsert

    pRecInsert (PRECORDINSERT) - input

      Record position.

      This parameter specifies how this record is to be inserted relative to other records in the same container.

      NULL

        Insert a record into the next available position.
      Other
        Insert a record into the position specified by pRecInsert.
