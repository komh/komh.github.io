wpQueryLockupBackground Parameter - pulBackgroundType

    pulBackgroundType (ULONG *) - in/out

      Pointer to one of the following background types:

      BACKGROUND_COLOR_ONLY

        Color only, no image.
      BACKGROUND_IMAGE
        Type of image (see image modes).
