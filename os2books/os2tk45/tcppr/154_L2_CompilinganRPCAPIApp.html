Compiling an RPC API Application

Follow these steps to compile and link the RPC API application using an IBM 32-bit compiler for OS/2:

  • To compile your program, enter: 
    icc /C myprog.c

  • To create an executable program, you can enter:

    For VisualAge C++ 

    ilink /NOFREEFORMAT myprog,myprog.exe,NULL, rpc32dll.lib

Notes:

  • The RPC API is not re-entrant. If you are using a multithreaded program, you must serialize the access to the APIs.

  • For more information about the compile and link options, and dynamic link libraries, see the User's Guide provided with your compiler.
