xprt_register()

The xprt_register() call registers service transport handles with the RPC service package. This routine also modifies the global variable svc_socks[ ].

Syntax 

#include <rpc\rpc.h>

void
xprt_register(xprt)
SVCXPRT *xprt;

Parameters

xprt

    Pointer to the service transport handle.
