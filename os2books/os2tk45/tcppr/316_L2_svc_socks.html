svc_socks[ ]

The array svc_socks[ ] is an array of socket descriptors being serviced. The integer noregistered specifies the number of socket descriptors in svc_socks[ ].

Syntax 

#include <rpc\rpc.h>

int svc_socks[ ];

#include <rpc\rpc.h>

int noregistered;
