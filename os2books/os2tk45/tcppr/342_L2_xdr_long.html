xdr_long()

The xdr_long() call translates between C long integers and their external representations.

Syntax 

#include <rpc\rpc.h>

bool_t
xdr_long(xdrs, lp)
XDR *xdrs;
long *lp;

Parameters

xdrs

    Pointer to an XDR stream.
lp
    Pointer to the long integer.

Return Values

The value 1 indicates success; the value 0 indicates an error.
