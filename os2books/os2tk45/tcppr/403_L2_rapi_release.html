rapi_release()

The rapi_release() call ends an RSVP session.

Syntax 

#include <rsvprapi.h>

int  rapi_release(
       rapi_sid_t sid);

Parameters

sid

    Session identifier of an open session.

Description

The rapi_release() call closes an open session. The session ID will no longer be valid.

Return Values and Descriptions

RAPI_ERR_BADSID

    The session identifier is invalid, or the session is not open
RAPI_ERR_NORSVP
    The API could not communicate with the RSVP daemon.
RAPI_ERR_OK
    The session was closed successfully.
