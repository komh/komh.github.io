xdr_callhdr()

The xdr_callhdr() call translates between an RPC message header and its external representation.

Syntax 

#include <rpc\rpc.h>

void
xdr_callhdr(xdrs, chdr)
XDR *xdrs;
struct rpc_msg *chdr;

Parameters

xdrs

    Pointer to the XDR stream.
chdr
    Pointer to the call header.
