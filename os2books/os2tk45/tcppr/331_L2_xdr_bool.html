xdr_bool()

The xdr_bool() call translates between a Boolean and its external representation.

Syntax 

#include <rpc\rpc.h>

bool_t
xdr_bool(xdrs, bp)
XDR *xdrs;
bool_t *bp;

Parameters

xdrs

    Pointer to an XDR stream.
bp
    Pointer to the Boolean.

Return Values

The value 1 indicates success; the value 0 indicates an error.
