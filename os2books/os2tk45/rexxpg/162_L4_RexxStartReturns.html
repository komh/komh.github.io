RexxStart - Returns

The RexxStart return values are:

    negative

      Interpreter errors. See Appendix A in the Object REXX Reference for the list of REXX errors.
    0
      No errors occurred. The REXX procedure ran normally.
    positive
      A system return code indicating problems finding or loading the interpreter. See the return codes for the OS/2 functions DosLoadModule and DosQueryProcAddr for details.

    When a macrospace REXX procedure (see Macrospace Interface) is not loaded in the macrospace, the return code is -3 ("Program is unreadable").
