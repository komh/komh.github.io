Halt and Trace Returns

The return codes for the halt and trace functions are as follows: 

────────────────────────────────────────────────────────────────────────
 0      RXARI_OK                        The  function completed
                                        successfully.
 ──────────────────────────────────────────────────────────────────────
 1      RXARI_NOT_FOUND                 The target REXX procedure was
                                        not found.
 ──────────────────────────────────────────────────────────────────────
 2      RXARI_PROCESSING_ERROR          A failure in REXX processing
                                        occurred.
────────────────────────────────────────────────────────────────────────
