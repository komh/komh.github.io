RexxDeregisterFunction - Returns

The RexxDeregisterFunction return values are: 

────────────────────────────────────────────────────────────────────────
 0      RXFUNC_OK                  The call to the function completed
                                   successfully.
 ──────────────────────────────────────────────────────────────────────
 30     RXFUNC_NOTREG              The requested function is not
                                   registered.
────────────────────────────────────────────────────────────────────────
