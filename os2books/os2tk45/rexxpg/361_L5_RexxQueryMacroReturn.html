RexxQueryMacro - Returns

The RexxQueryMacroSpace return values are: 

────────────────────────────────────────────────────────────────────────
 0      RXMACRO_OK                           The call to the function
                                             completed successfully.
 ──────────────────────────────────────────────────────────────────────
 2      RXMACRO_NOT_FOUND                    The requested function was
                                             not found in the
                                             macrospace.
────────────────────────────────────────────────────────────────────────
