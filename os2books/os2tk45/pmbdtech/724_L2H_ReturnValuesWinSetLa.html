Return Values - WinSetLangViewer

    hwnd (HWND) - return

      Result indicator.

      Returns the previous Language Viewer handle for this codepage.

      NULLHANDLE

        No Language Viewer was previously registered for this codepage.
      Other
        Previous Language Viewer handle for this codepage.
