BIDIFLAGS

    ulFlags (ULONG)

      Contains one of the following flag values:

    OL
      When this is specified the function sends (internally) a WM_QUERYBIDIATTR (or WM_QUERYBIDISTAT) message to the return value procedure. If the return value of the message is other than zero this is the value Returned.
    LIF_NO_SENDMSG
      When this value is specified, the WinQueryBidiInfo function, does not send a WM_QUERYBIDIATTR (or WM_QUERYBIDISTAT) message to the window procedure of hwnd. Instead, it returns the Bidi attributes/status as it is stored in the window internal strucrure.
