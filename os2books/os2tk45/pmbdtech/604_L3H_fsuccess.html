fsuccess

    fSuccess(BOOL)

      Success indicator.
      TRUE
        Changes in the bidi viewer succeeded.
      FALSE
        An error occurred. The WinGetLastError function may return the following error:

        • PMERR_INVALID_PARAMETERS
