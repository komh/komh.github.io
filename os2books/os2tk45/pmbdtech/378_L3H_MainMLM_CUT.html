Main - MLM_CUT

Cause:

This message copies the text that forms the currently selected text to the clipboard and then deletes it from the MLE control. When the text type of the MLE is implicit, the characters affected may not necessarily be adjacent in the text buffer : the action is taken on the visually selected text which is continous on the screen.

Parameters: 

param1
    ULONG param1 Reserved.

param2
    ULONG param2 Reserved.

returns
    ULONG ulCopy Number of bytes transferred, counted in CF_TEXT format.
