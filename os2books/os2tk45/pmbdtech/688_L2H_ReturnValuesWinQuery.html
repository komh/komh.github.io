Return Values - WinQueryLangInfo

    ulData (ULONG) - return

      Bidirectional value indicator.

      BIDIATTR

        Bidi attribute information.
      BIDISTAT
        Bidi status information.
      NULL
        Query of the bidi attributes was unsuccessful.
