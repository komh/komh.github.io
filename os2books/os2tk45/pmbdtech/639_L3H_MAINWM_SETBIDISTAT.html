MAIN - WM_SETBIDISTAT

Cause:

This message is sent to a window procedure to notify it that the bidi status of that window is about to be changed.

Parameters: 

param1
    ULONG    ulData     Attribute data.

param2
    ULONG    ulMask     Value mask.

returns
    ULONG    fresult    Result value.
