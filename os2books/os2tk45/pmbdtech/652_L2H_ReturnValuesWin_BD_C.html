Return Values - Win_BD_ClassifyCodepage

    cp (ULONG) - return

      Code page indicator.
    OL
      A none Bidi code page.
    CP_IS_BIDI
      The code page is bidi. The following are possible values:

      CP_IS_ARABIC

        The code page is Arabic.
      CP_IS_ARABIC2
        The code page is Arabic 2.
