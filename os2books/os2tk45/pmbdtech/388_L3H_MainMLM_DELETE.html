Main - MLM_DELETE

Cause:

This message deletes text based on the VISUAL selection.

Parameters: 

param1
    IPT   iptBegin  VISUAL Starting point of deletion.

param2
    ULONG ulDel     Number of bytes to delete.

returns
    ULONG ulSuccess Number of bytes successfully deleted.
