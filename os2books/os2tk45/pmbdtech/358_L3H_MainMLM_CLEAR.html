Main - MLM_CLEAR

Cause:

This message clears the currently selected text. When the text type of the MLE is implicit, the characters affected may not necessarily be adjacent in the text buffer : the action is taken on the visually selected text which is continous on the screen.

Parameters: 

param1
    ULONG param1  Reserved.

param2
    ULONG param2  Reserved.

returns
    ULONG ulClear Number of bytes deleted, counted in CF_TEXT format.
