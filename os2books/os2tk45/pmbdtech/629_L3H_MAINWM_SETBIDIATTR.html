MAIN - WM_SETBIDIATTR

Cause:

This message is sent to a window procedure to notify it that the bidi attributes of that window are about to be changed.

Parameters: 

param1
    ULONG   ulData     Attribute data.

param2
    ULONG   ulMask     Value mask.

returns
    ULONG   fresult    Result value.
