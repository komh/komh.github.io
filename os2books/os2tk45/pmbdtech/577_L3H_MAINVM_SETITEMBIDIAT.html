MAIN - VM_SETITEMBIDIATTR

Cause:

This message sets the bidi attributes for each text item in an application.

Parameters: 

param1
     USHORT    usRow     Row index.
     USHORT    usColumn  Column index.

param2
     ULONG     param2    Reserved.

returns
     BOOL      fsuccess  Success indicator.
