Main - MLM_PASTE

Cause:

This message replaces the text that forms the current VISUAL selection, with text from the clipboard. When the text type of the MLE is implicit, the characters affected may not necessarily be adjacent in the text buffer : the action is taken on the visually selected text which is continous on the screen.

Parameters: 

param1
    ULONG param1 Reserved.

param2
    ULONG param2 Reserved.

returns
    ULONG ulCopy Number of bytes transferred, counted in CF_TEXT format.
