Return Values - WinSetLangInfo

    fSuccess (ULONG) - return

      Success indicator.

      TRUE

        Setting of the Bidi attribute succeeded.
      FALSE
        An error occurred. The WinGetLAstError function may return the following errors:

        • PMERR_INVALID_HWND
        • PMERR_INVALID_VALUE
