fsuccess

    fSuccess(BOOL)

      Success indicator.
      TRUE
        Setting of the notebook tab text bidirectional attributes succeeded.
      FALSE
        An error occurred. The WinGetLastError function may return the following error:

        • PMERR_INVALID_PARAMETERS
