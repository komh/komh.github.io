Main - MLM_EXPORT

Cause:

This message exports VISUALLY selected text to a buffer.

Parameters: 

param1
    PIPT   pBegin    Visual starting point.

param2
    PULONG pCopy     Number of bytes being exported.

returns
    ULONG  ulSuccess Number of bytes successfully exported.
