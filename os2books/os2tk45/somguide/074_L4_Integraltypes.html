Integral types

IDL supports only the integral types short, long, unsigned short, and unsigned long, which represent the following value ranges:

    short

      -2[15] .. 2[15]-1
    long
      -2[31] .. 2[31]-1
    unsigned short
      0 .. 2[16]-1
    unsigned long
      0 .. 2[32]-1
