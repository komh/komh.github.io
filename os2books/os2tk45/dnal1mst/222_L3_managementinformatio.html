management information base (MIB)

  • A standard used to define SNMP objects, such as packet counts and routing tables, that are in a TCP/IP environment.

  • All of the accessible information on a managed network.

  • A collection of managed-object definitions.

  • A set of variable bindings that reflect the current state of an SNMP agent. Extensions to the MIB can be added by a business enterprise.
