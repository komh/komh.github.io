Getting Attribute Information

The difference between List and Get commands is that a List command returns MIF information about an object, and a Get command requests the actual value for the object.

The DmiGetAttributeCmd command is used to request the current values of attributes within groups. For a description of the DmiGetAttributeReq block used to issue this command, read DmiGetAttributeReq Command Block.
