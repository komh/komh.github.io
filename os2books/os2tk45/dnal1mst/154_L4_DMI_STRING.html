DMI_STRING

The syntax for using DMI_STRING is: 

typedef struct {
        unsigned long length;
        char body[1];
} DMI_STRING;

    Parameter

      Description
    length
      The length of the string
    body
      The data for the string. The string is not null-terminated. The 1 in the definition is strictly for ANSI C compliance. The actual size will be the size of the string.
