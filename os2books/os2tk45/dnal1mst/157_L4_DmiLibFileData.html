DmiLibFileData

The syntax for using DmiLibFileData is: 

typedef struct {
        unsigned long iFileType;
        DMI_STRING *pFileData;
} DmiLibFileData_t;

    Parameter

      Description
    iFileType
      The type of the referenced file:

    • MIF file name
    • MIF file pointer
    pFileData
      A DMI_STRING pointer to the file data. For type 2, this pointer references the operating system path name to the file. For type 3, the pointer references a DMI_STRING object that contains the actual file data.
