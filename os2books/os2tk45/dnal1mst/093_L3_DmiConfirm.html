DmiConfirm

The DmiConfirm block is sent as the result of a request message to indicate that processing of the request has completed.

The format for the command block is: ┌──────────────────────────────────────────────────────────────────────────────┐
│ Table 16. DmiConfirm Command Block │
├───────────────┬───────────────┬───────────────┬──────────────────────────────┤
│ OFFSET │ SIZE │ TYPE │ VARIABLE NAME │
├───────────────┼───────────────┼───────────────┼──────────────────────────────┤
│ 0 │ 4 │ INT │ iLevelCheck │
├───────────────┼───────────────┼───────────────┼──────────────────────────────┤
│ 4 │ 4 │ PTR │ pDmiMgmtCommand │
├───────────────┼───────────────┼───────────────┼──────────────────────────────┤
│ 8 │ 4 │ INT │ iStatus │
└───────────────┴───────────────┴───────────────┴──────────────────────────────┘

    Variable Name

      Variable Description
    iLevelCheck
      A 4-octet integer with a code value of 0x444d3131 and an ISO 8859-1 value of DMI1. This value indicates the revision level of the DMI specification used by the code issuing the command and also functions as a check, so that the receiver can be confident the pointer is pointing to a valid confirmation block.
    pDmiMgmtCommand
      Pointer to a command block.
    iStatus
      A status code indicating an error condition or other information.
