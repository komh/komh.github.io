GpiQueryColor - Remarks

This function is invalid when the drawing mode (see GpiSetDrawingMode) is set to retain.

Valid CLR_* values are listed in the following list:

    CLR_BACKGROUND (0L)
    CLR_BLUE (1L)
    CLR_RED (2L)
    CLR_PINK (3L)
    CLR_GREEN (4L)
    CLR_CYAN (5L)
    CLR_YELLOW (6L)
    CLR_NEUTRAL (7L)
    CLR_DARKGRAY (8L)
    CLR_DARKBLUE (9L)
    CLR_DARKRED (10L)
    CLR_DARKPINK (11L)
    CLR_DARKGREEN (12L)
    CLR_DARKCYAN (13L)
    CLR_BROWN (14L)
    CLR_PALEGRAY (15L)
