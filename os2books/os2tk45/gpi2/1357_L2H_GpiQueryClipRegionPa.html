GpiQueryClipRegion - Parameters

    hps (HPS) - input

      Presentation-space handle.

    hrgn (HRGN) - returns

      Clip-region handle (if any).

      NULLHANDLE

        Null handle (no region is selected)
      HRGN_ERROR
        Error
      Otherwise
        Clip region handle.
