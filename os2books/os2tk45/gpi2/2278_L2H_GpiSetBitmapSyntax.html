GpiSetBitmap - Syntax

This function sets a bit map as the currently selected bit map in a memory device context. 

#define INCL_GPIBITMAPS /* Or use INCL_GPI, INCL_PM, Also in COMMON section */
#include <os2.h>

HPS        hps;     /*  Presentation-space handle. */
HBITMAP    hbm;     /*  Handle of the bit map to be set. */
HBITMAP    hbmOld;  /*  Old bit-map handle. */

hbmOld = GpiSetBitmap(hps, hbm);
