GpiSetPatternSet - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lSet (LONG) - input

      Pattern-set local identifier.

      LCID_DEFAULT

        Default (can be set explicitly with GpiSetDefAttrs).
      1-254
        Identifies a logical font or a bit map.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
