GpiQueryBitmapInfoHeader - Parameters

    hbm (HBITMAP) - input

      Bit-map handle.

    pbmpData (PBITMAPINFOHEADER2) - in/out

      Bit-map information header.

      This is a structure, that on return, is filled with data for the specified bit map.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
