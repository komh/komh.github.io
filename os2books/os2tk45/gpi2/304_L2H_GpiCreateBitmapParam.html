GpiCreateBitmap Parameter - flOptions

    flOptions (ULONG) - input

      Options.

      If the bit map is stored on a device, the flOptions parameter is passed to the device. Bits 16 through 31 can be used for special features known to be supported by the particular device driver.

      The following option is supported:

      CBM_INIT

        Initialize the bit map with pbInitData.
