GpiMove - Parameters

    hps (HPS) - input

      Presentation-space handle.

    pptlPoint (PPOINTL) - input

      Position to which to move.

      This position is in world coordinates.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
