GpiConvert Parameter - lTarg

    lTarg (LONG) - input

      Target coordinate space.

      CVTC_WORLD

        World coordinates
      CVTC_MODEL
        Model space
      CVTC_DEFAULTPAGE
        Page space before default viewing transform
      CVTC_PAGE
        Page space after default viewing transform
      CVTC_DEVICE
        Device space.
