GpiQueryStopDraw - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lValue (LONG) - returns

      Stop draw condition indicator.

      SDW_OFF

        No "stop draw" condition currently exists
      SDW_ON
        The "stop draw" condition currently exists
      SDW_ERROR
        Error.
