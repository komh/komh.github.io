GpiQueryFontAction Parameter - options

    options (ULONG) - input

      Options.

      The following may be OR'ed together if required:

      QFA_PUBLIC

        Interested in any change of public fonts.
      QFA_PRIVATE
        Interested in any change of private fonts for the current process.
