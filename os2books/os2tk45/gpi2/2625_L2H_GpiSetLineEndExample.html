GpiSetLineEnd - Example Code

This function sets the line end to be square (as opposed to round for example). 

#define INCL_GPIPRIMITIVES
#include <OS2.H>

HPS hps;         /*   Presentation-space */
                 /*   handle.            */

GpiSetLineEnd(hps,
              LINEEND_SQUARE);
