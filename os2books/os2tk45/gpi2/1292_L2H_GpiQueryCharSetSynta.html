GpiQueryCharSet - Syntax

This function returns the character-set local identifier (lcid), as set by the GpiSetCharSet function. 

#define INCL_GPIPRIMITIVES /* Or use INCL_GPI, INCL_PM, */
#include <os2.h>

HPS     hps;    /*  Presentation-space handle. */
LONG    lLcid;  /*  Character-set local identifier. */

lLcid = GpiQueryCharSet(hps);
