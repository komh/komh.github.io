GpiQueryDefAttrs Parameter - lPrimType

    lPrimType (LONG) - input

      Primitive type.

      This is the type of primitive for which default attribute values are to be queried, as follows:

      PRIM_LINE

        Line and arc primitives
      PRIM_CHAR
        Character primitives
      PRIM_MARKER
        Marker primitives
      PRIM_AREA
        Area primitives
      PRIM_IMAGE
        Image primitives.
