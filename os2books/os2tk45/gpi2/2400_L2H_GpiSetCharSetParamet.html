GpiSetCharSet - Parameters

    hps (HPS) - input

      Presentation-space handle.

    llcid (LONG) - input

      Character-set local identifier.

      LCID_DEFAULT

        Default (can be set explicitly with GpiSetDefAttrs).
      1-254
        Identifies a logical font.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
