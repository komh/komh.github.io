GpiQueryPalette Return Value - hpal

    hpal (HPAL) - returns

      Palette handle.

      NULLHANDLE

        Null handle (no palette is selected)
      PAL_ERROR
        Error occurred
      Otherwise
        Handle of the palette currently selected into this presentation space.
