GpiSetLineJoin - Example Code

This function sets the line-join to be round (as opposed to bevel or miter). 

#define INCL_GPIPRIMITIVES
#include <OS2.H>

HPS hps;         /*   Presentation-space */
                 /*   handle.            */

GpiSetLineEnd(hps,
              LINEJOIN_ROUND);
