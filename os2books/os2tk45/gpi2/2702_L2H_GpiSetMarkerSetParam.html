GpiSetMarkerSet Parameter - lSet

    lSet (LONG) - input

      Marker-set local identifier.

      The identity (lcid) of the required marker set:

      LCID_DEFAULT

        Default (can be set explicitly with GpiSetDefAttrs)
      1-254
        Identifies a logical font.
