GpiQueryColorData - Syntax

Information about the current logical color table or the selected palette is returned by this function. 

#define INCL_GPILOGCOLORTABLE /* Or use INCL_GPI, INCL_PM, */
#include <os2.h>

HPS      hps;      /*  Presentation-space handle. */
LONG     lCount;   /*  Number of elements supplied in alArray. */
PLONG    alArray;  /*  Array. */
BOOL     rc;       /*  Success indicator. */

rc = GpiQueryColorData(hps, lCount, alArray);
