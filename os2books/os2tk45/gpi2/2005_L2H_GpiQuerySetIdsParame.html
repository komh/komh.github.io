GpiQuerySetIds Parameter - alTypes

    alTypes (PLONG) - output

      Object types.

      Elements indicate whether the corresponding allcids element refers to a logical font or a tagged bit map.

      LCIDT_FONT

        Font object
      LCIDT_BITMAP
        Bit map.
