GpiQueryDeviceBitmapFormats Parameter - lCount

    lCount (LONG) - input

      Number of elements.

      Number of elements in alArray (must be a positive even number). For the complete set of formats returned, the value of this parameter must be at least double the number of device formats returned by DevQueryCaps
