GpiQueryLineType Return Value - lLineType

    lLineType (LONG) - returns

      Line type.

      LINETYPE_DEFAULT

        (0L) Default
      LINETYPE_DOT
        (1L) Dotted line
      LINETYPE_SHORTDASH
        (2L) Short-dashed line
      LINETYPE_DASHDOT
        (3L) Dash-dotted line
      LINETYPE_DOUBLEDOT
        (4L) Double-dotted line
      LINETYPE_LONGDASH
        (5L) Long-dashed line
      LINETYPE_DASHDOUBLE DOT
        (6L) Dash-double dotted line
      LINETYPE_SOLID
        (7L) Solid line
      LINETYPE_ALTERNATE
        (9L) Alternate pale on
      LINETYPE_INVISIBLE
        (8L) Invisible line
      LINETYPE_ERROR
        (-1L) Error.
