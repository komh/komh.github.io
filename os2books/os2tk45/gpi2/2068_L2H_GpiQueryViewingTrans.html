GpiQueryViewingTransformMatrix - Syntax

This function returns the current viewing transform (see GpiSetViewingTransformMatrix).. 

#define INCL_GPITRANSFORMS /* Or use INCL_GPI, INCL_PM, */
#include <os2.h>

HPS          hps;          /*  Presentation-space handle. */
LONG         lCount;       /*  Number of elements. */
PMATRIXLF    pmatlfArray;  /*  Transform matrix. */
BOOL         rc;           /*  Success indicator. */

rc = GpiQueryViewingTransformMatrix(hps, lCount,
       pmatlfArray);
