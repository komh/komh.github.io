GpiPtVisible - Parameters

    hps (HPS) - input

      Presentation-space handle.

    pptlPoint (PPOINTL) - input

      Point to be checked.

      The point is given in world coordinates.

    lVisibility (LONG) - returns

      Visibility indicator.

      PVIS_INVISIBLE

        Not visible
      PVIS_VISIBLE
        Visible
      PVIS_ERROR
        Error.
