GpiSetAttrMode - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lMode (LONG) - input

      Attribute mode.

      AM_PRESERVE

        Preserve attributes
      AM_NOPRESERVE
        Do not preserve attributes.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
