GpiDrawSegment - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lSegment (LONG) - input

      Segment to be drawn; it must be greater than 0.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
