GpiQueryTextBox Parameter - lCount2

    lCount2 (LONG) - input

      Number of points.

      Contains the number of points to be returned in the aptlPoints array. Specify TXTBOX_COUNT to get the maximum information. This parameter must be greater or equal to 0.
