GpiQueryClipRegion Return Value - hrgn

    hrgn (HRGN) - returns

      Clip-region handle (if any).

      NULLHANDLE

        Null handle (no region is selected)
      HRGN_ERROR
        Error
      Otherwise
        Clip region handle.
