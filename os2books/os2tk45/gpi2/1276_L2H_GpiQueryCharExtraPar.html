GpiQueryCharExtra - Parameters

    hps (HPS) - input

      Presentation-space handle.

    Extra (PFIXED) - output

      Character-extra attribute value.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
