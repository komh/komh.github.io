GpiSetStopDraw - Example Code

This example shows how to stop drawing. 

#define INCL_GPICONTROL
#include <OS2.H>
HPS hps;         /*   Presentation-space */
                 /*   handle.            */

GpiSetStopDraw(hps,SDW_OFF);
