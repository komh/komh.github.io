GpiRemoveDynamics - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lFirstSegid (LONG) - input

      First segment in the section.

      It must be greater than 0.

    lLastSegid (LONG) - input

      Last segment in the section.

      It must be greater than 0.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
