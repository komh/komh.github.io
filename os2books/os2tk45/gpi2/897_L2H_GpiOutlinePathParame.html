GpiOutlinePath - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lPath (LONG) - input

      Identifier of path to be outlined; it must be 1.

    lOptions (LONG) - input

      Options.

      Reserved; must be 0.

    lHits (LONG) - returns

      Correlation and error indicators.

      GPI_OK

        Successful
      GPI_HITS
        Correlate hits
      GPI_ERROR
        Error.
