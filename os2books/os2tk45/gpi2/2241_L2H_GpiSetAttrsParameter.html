GpiSetAttrs Parameter - lPrimType

    lPrimType (LONG) - input

      Primitive type.

      The primitive type for which attributes are to be set:

      PRIM_LINE

        Line and arc primitives
      PRIM_CHAR
        Character primitives
      PRIM_MARKER
        Marker primitives
      PRIM_AREA
        Area primitives
      PRIM_IMAGE
        Image primitives.
