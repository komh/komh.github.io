GpiCombineRegion Return Value - lComplexity

    lComplexity (LONG) - returns

      Complexity of resulting region and error indicators.

      RGN_NULL

        Null region
      RGN_RECT
        Rectangular region
      RGN_COMPLEX
        Complex region
      RGN_ERROR
        Error.
