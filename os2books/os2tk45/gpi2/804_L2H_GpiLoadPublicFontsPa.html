GpiLoadPublicFonts - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pszFilename (PSZ) - input

      Filename.

      This is the fully-qualified name of the font resource. The file-name extension is ".FON".

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
