GpiLoadMetaFile - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pszFilename (PSZ) - input

      Filename.

      The name of the file that is to be loaded into a metafile.

    hmf (HMF) - returns

      Metafile handle or error.

      <>0

        Metafile handle
      GPI_ERROR
        Error.
