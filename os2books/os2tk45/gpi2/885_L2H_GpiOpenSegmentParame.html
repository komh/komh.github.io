GpiOpenSegment - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lSegment (LONG) - input

      Segment identifier.

      Must be zero or a positive number.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
