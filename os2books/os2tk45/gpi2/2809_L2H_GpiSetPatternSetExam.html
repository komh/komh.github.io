GpiSetPatternSet - Example Code

This function sets the current pattern-set attribute to the logical font with id 35. 

#define INCL_GPIPRIMITIVES
#include <OS2.H>
HPS hps;         /*   Presentation-space */
                 /*   handle.            */


GpiSetPatternSet(hps, 35L);
