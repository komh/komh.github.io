GpiSetPel - Example Code

This function sets a pel, at a position specified in world coordinates, using the current (line) color and mix. 

#define INCL_GPIBITMAPS
#include <OS2.H>
HPS hps;         /*   Presentation-space */
                 /*   handle.            */


POINTL ptlPoint = {0,0};


GpiSetPel(hps, &ptlPoint);
