GpiCallSegmentMatrix Parameter - lOptions

    lOptions (LONG) - input

      Transformation options.

      Specify how the transform defined by the pmatlfArray parameter should be used to modify the existing current model transform for the duration of the function. The existing transform is the concatenation, in the current function context, of the instance, segment, and model transforms, from the root segment downwards.

      TRANSFORM_REPLACE

        The previous model transform is discarded and replaced by the specified transform.
      TRANSFORM_ADD
        The specified transform is combined with the existing model transform. The existing transform precedes the new transform. This option is most useful for incremental updates to transforms.
      TRANSFORM_PREEMPT
        The specified transform is combined with the existing model transform. The new transform precedes the existing transform.
