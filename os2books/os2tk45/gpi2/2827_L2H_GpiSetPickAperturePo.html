GpiSetPickAperturePosition - Parameters

    hps (HPS) - input

      Presentation-space handle.

    pptlPick (PPOINTL) - input

      Center of the pick aperture.

      The center is in presentation page coordinates.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
