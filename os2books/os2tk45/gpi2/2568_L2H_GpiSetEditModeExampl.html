GpiSetEditMode - Example Code

This example sets the current editing mode to insert. 

#define INCL_GPISEGEDITING
#include <OS2.H>

HPS hps;         /*   Presentation-space */
                 /*   handle.            */


GpiSetEditMode(hps, SEGEM_INSERT); /* insert mode. */
