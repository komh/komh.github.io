GpiSetLineType - Example Code

This function sets the line-type to be round (as opposed to bevel or miter). 

#define INCL_GPIPRIMITIVES
#include <OS2.H>

HPS hps;         /*   Presentation-space */
                 /*   handle.            */


GpiSetLineType(hps,
              LINETYPE_DEFAULT);
