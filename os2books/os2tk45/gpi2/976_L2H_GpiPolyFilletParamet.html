GpiPolyFillet - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lCount (LONG) - input

      Number of points.

      Must not be negative. Zero is valid but causes no output.

    aptlPoints (PPOINTL) - input

      Array of points.

    lHits (LONG) - returns

      Correlation and error indicators.

      GPI_OK

        Successful
      GPI_HITS
        Correlate hits
      GPI_ERROR
        Error.
