GpiUnloadFonts - Parameters

    hab (HAB) - input

      Anchor-block handle.

    pszFilename (PSZ) - input

      Fully qualified file name of the font resource.

      The file name extension is .FON

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
