GpiQueryMarker Return Value - lSymbol

    lSymbol (LONG) - returns

      Marker symbol.

      MARKSYM_DEFAULT

        (0L) Default
      MARKSYM_CROSS
        (1L) Cross symbol
      MARKSYM_PLUS
        (2L) Plus symbol
      MARKSYM_DIAMOND
        (3L) Diamond symbol
      MARKSYM_SQUARE
        (4L) Square symbol
      MARKSYM_SIXPOINTSTAR
        (5L) Six-point star symbol
      MARKSYM_EIGHTPOINTSTAR
        (6L) Eight-point star symbol
      MARKSYM_SOLIDDIAMOND
        (7L) Solid diamond symbol
      MARKSYM_SOLIDSQUARE
        (8L) Solid square symbol
      MARKERSYM_DOT
        (9L) Dot symbol
      MARKSYM_SMALLCIRLCE
        (10L) Small Circle symbol
      MARKSYM_BLANK
        (64L) Blank symbol
      MARKSYM_ERROR
        (-1L) Error
