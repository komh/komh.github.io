GpiFloodFill - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lOptions (LONG) - input

      Flood fill options.

      This parameter can have one of the following values:

      FF_BOUNDARY

        Fills up to the specified color devaldesc.
      FF_SURFACE
        Fills while on the specified color.

      lColor (LONG) - input

        Color.

        The boundary or surface color, depending on the value of lOptions.

        This is either a logical color index, or an RGB value, depending on the state of the color table.

      lHits (LONG) - returns

        Correlation and error indicators.

        GPI_OK

          Successful
        GPI_HITS
          Correlate hits
        GPI_ERROR
          Error.
