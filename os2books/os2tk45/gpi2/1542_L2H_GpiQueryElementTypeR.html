GpiQueryElementType Return Value - lReqLength

    lReqLength (LONG) - returns

      Size of the data required to hold the element content.

      This can be used for a subsequent GpiQueryElement function.

      >=0

        Size of data
      GPI_ALTERROR
        Error.
