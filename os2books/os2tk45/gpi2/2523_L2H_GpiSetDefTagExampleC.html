GpiSetDefTag - Example Code

This function specifies the default value of the primitive tag (see GpiSetTag). 

#define INCL_GPIPRIMITIVES
#include <OS2.H>


HPS     hps;    /* Presentation space handle     */

GpiSetDefTag(hps, 1L);
