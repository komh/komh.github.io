GpiSetCharDirection Parameter - lDirection

    lDirection (LONG) - input

      Character direction.

      CHDIRN_DEFAULT

        The default; the same as CHDIRN_LEFTRIGHT (unless changed with GpiSetDefAttrs)
      CHDIRN_LEFTRIGHT
        Character boxes are arranged parallel to, and in the same direction as, the baseline. This is the usual convention for Roman text.
      CHDIRN_TOPBOTTOM
        Character boxes are arranged in columns directed 90 degrees clockwise from the baseline. This is the usual convention for Chinese characters. This option can be used for drawing Roman text vertically (a y-axis title on a graph, for example). The reference point within the character definition is at the center of the character, along the x-direction, in this case.
      CHDIRN_RIGHTLEFT
        Character boxes are arranged parallel to, but in the reverse of, the baseline direction. This is the usual convention for Arabic text.
      CHDIRN_BOTTOMTOP
        Character boxes are arranged in columns directed 90 degrees counterclockwise from the baseline. The reference point within the character definition is at the center of the character, along the x-direction, in this case.
