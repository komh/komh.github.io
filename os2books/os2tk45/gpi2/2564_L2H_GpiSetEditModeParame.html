GpiSetEditMode - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lMode (LONG) - input

      Edit mode.

      SEGEM_INSERT

        Insert mode
      SEGEM_REPLACE
        Replace mode.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
