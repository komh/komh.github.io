GpiLabel - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lLabel (LONG) - input

      Required label.

      No check is made on the value of this parameter.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
