GpiQueryLineWidthGeom - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lLineWidth (LONG) - returns

      Geometric line width.

      If the geometric line width is currently set to the default, zero is returned.

      >=0

        Geometric line width
      LINEWIDTHGEOM_ERROR
        Error.
