GpiSelectPalette Return Value - hpalOld

    hpalOld (HPAL) - returns

      Old palette handle.

      NULLHANDLE

        Successful completion, default (or loaded) color table was in effect
      PAL_ERROR
        Error occurred
      Otherwise
        Old palette handle.
