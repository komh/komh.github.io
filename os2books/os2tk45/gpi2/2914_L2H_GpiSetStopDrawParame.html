GpiSetStopDraw - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lValue (LONG) - input

      Stop draw condition.

      SDW_OFF

        Clear the "stop draw" condition
      SDW_ON
        Set the "stop draw" condition.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
