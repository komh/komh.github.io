GpiEndPath - Syntax

This function ends the specification of a path started by GpiBeginPath. 

#define INCL_GPIPATHS /* Or use INCL_GPI, INCL_PM, */
#include <os2.h>

HPS     hps;  /*  Presentation-space handle. */
BOOL    rc;   /*  Success indicator. */

rc = GpiEndPath(hps);
