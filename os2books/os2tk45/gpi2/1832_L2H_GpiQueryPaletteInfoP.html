GpiQueryPaletteInfo Parameter - flOptions

    flOptions (ULONG) - input

      Specifies options.

      LCOLOPT_INDEX

        If this is set, the index is to be returned for each RGB value in the aulArray parameter.

        Other flags are reserved and must be 0.
