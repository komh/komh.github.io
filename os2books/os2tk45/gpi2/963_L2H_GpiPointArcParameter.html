GpiPointArc - Parameters

    hps (HPS) - input

      Presentation-space handle.

    pptl2 (PPOINTL) - input

      Intermediate and end points.

      Pointer to an array containing the intermediate and end points.

    lHits (LONG) - returns

      Correlation and error indicators.

      GPI_OK

        Successful
      GPI_HITS
        Correlate hits
      GPI_ERROR
        Error.
