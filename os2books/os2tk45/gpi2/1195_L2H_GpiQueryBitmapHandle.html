GpiQueryBitmapHandle - Example Code

This example uses GpiQueryBitmapHandle to return the handle of the bit map currently tagged with the specified local identifier (lcid) set by GpiSetBitmapId. 

#define INCL_GPIBITMAPS         /* Bit-map functions            */
#include <os2.h>

HBITMAP hbm;            /* bit-map handle                       */
HPS     hps;            /* presentation-space handle            */
LONG    lLcid;          /* local identifier                     */

hbm = GpiQueryBitmapHandle(hps, lLcid);
