GpiQueryStopDraw Return Value - lValue

    lValue (LONG) - returns

      Stop draw condition indicator.

      SDW_OFF

        No "stop draw" condition currently exists
      SDW_ON
        The "stop draw" condition currently exists
      SDW_ERROR
        Error.
