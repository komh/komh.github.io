GpiQueryCharDirection - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lDirection (LONG) - returns

      Character direction.

      This parameter can have one of the following values:

      CHDIRN_DEFAULT

        Default
      >0
        Character direction
      CHDIRN_ERROR
        Error.
