GpiSetPickApertureSize - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lOptions (LONG) - input

      Setting option.

      PICKAP_DEFAULT

        Use the default pick aperture. The value of psizlSize is ignored.
      PICKAP_REC
        Use the values specified by psizlSize.

      psizlSize (PSIZEL) - input

        Pick aperture size.

      rc (BOOL) - returns

        Success indicator.

        TRUE

          Successful completion
        FALSE
          Error occurred.
