GpiSetClipRegion - Syntax

This function defines the region to be used for clipping, when any drawing takes place through the specified presentation space. 

#define INCL_GPIREGIONS /* Or use INCL_GPI, INCL_PM, */
#include <os2.h>

HPS      hps;          /*  Presentation-space handle. */
HRGN     hrgn;         /*  Region handle. */
PHRGN    phrgnOld;     /*  Old region handle (if any). */
LONG     lComplexity;  /*  Complexity of clipping and error indicators. */

lComplexity = GpiSetClipRegion(hps, hrgn,
                phrgnOld);
