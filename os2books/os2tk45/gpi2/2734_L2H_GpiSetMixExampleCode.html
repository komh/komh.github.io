GpiSetMix - Example Code

This function sets the current foreground mix attribute for each individual primitive type. 

#define INCL_GPIPRIMITIVES
#include <OS2.H>

HPS hps;         /*   Presentation-space */
                 /*   handle.            */

GpiSetMix(hps,
          FM_LEAVEALONE);
