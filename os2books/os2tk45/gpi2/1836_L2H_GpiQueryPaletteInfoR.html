GpiQueryPaletteInfo Return Value - lRetCount

    lRetCount (LONG) - returns

      Number of elements.

      Zero is returned if no palette is selected.

      PAL_ERROR

        Error occurred
      Otherwise
        The number of elements of palette information passed back in the aulArray parameter, unless ulCount parameter is 0, in which case this is the total number of elements that are needed to hold the palette information.
