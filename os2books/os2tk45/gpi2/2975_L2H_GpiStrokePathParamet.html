GpiStrokePath - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lPath (LONG) - input

      Identifier of path to be stroked; it must be 1.

    flOptions (ULONG) - input

      Stroke option.

      Reserved; must be 0.

    lHits (LONG) - returns

      Correlation and error indicators.

      GPI_OK

        Successful
      GPI_HITS
        Correlate hits
      GPI_ERROR
        Error.
