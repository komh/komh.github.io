GpiRectVisible Return Value - lVisibility

    lVisibility (LONG) - returns

      Visibility indicator.

      RVIS_INVISIBLE

        Not visible
      RVIS_PARTIAL
        Some of the rectangle is visible
      RVIS_VISIBLE
        All of the rectangle is visible
      RVIS_ERROR
        Error.
