GpiMarker - Parameters

    hps (HPS) - input

      Presentation-space handle.

    pptlPoint (PPOINTL) - input

      Position of the marker.

    lHits (LONG) - returns

      Correlation and error indicators.

      GPI_OK

        Successful
      GPI_HITS
        Correlate hits
      GPI_ERROR
        Error.
