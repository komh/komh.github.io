GpiQueryNearestColor - Syntax

This function returns the nearest color available to the color specified on the currently associated device. Both colors are specified in RGB terms. 

#define INCL_GPILOGCOLORTABLE /* Or use INCL_GPI, INCL_PM, */
#include <os2.h>

HPS      hps;        /*  Presentation-space handle. */
ULONG    flOptions;  /*  Options. */
LONG     lRgbIn;     /*  Required color. */
LONG     lRgbOut;    /*  Nearest available color to the one specified. */

lRgbOut = GpiQueryNearestColor(hps, flOptions,
            lRgbIn);
