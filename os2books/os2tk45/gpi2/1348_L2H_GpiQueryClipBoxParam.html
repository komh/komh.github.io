GpiQueryClipBox - Parameters

    hps (HPS) - input

      Presentation-space handle.

    prclBound (PRECTL) - output

      Bounding rectangle.

      The coordinates of the bounding rectangle, in world coordinates.

    lComplexity (LONG) - returns

      Complexity and error indicators.

      RGN_NULL

        Null region
      RGN_RECT
        Rectangular region
      RGN_COMPLEX
        Complex region
      RGN_ERROR
        Error.
