GpiSetLineWidthGeom - Example Code

This function sets the line width geometry to double the default of 1. 

#define INCL_GPIPRIMITIVES
#include <OS2.H>

HPS hps;         /*   Presentation-space */
                 /*   handle.            */

GpiSetLineWidthGeom(hps,
                    2L);
