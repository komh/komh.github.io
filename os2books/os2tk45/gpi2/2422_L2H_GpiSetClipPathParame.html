GpiSetClipPath Parameter - lPath

    lPath (LONG) - input

      Path control flag.

    • The current clip path stops being the current clip path; the current clip path is to be reset to an infinite one (that is, no clipping).
    • The path that has been defined is to be intersected with the current clip path.
