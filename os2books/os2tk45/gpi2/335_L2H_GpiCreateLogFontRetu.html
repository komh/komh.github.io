GpiCreateLogFont Return Value - lMatch

    lMatch (LONG) - returns

      Match indicators.

      FONT_MATCH

        Font requirements matched successfully.
      FONT_DEFAULT
        Font requirements not matched; a default font is used.
      GPI_ERROR
        Error occurred.
