GpiPop - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lCount (LONG) - input

      Number of attributes to be restored.

      It must be greater or equal to 0.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
