GpiQueryBitmapHandle - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lLcid (LONG) - input

      Local identifier.

      It must be greater than 0 and less than 255.

    hbm (HBITMAP) - returns

      Bit-map handle.

      <>0

        Bit-map handle
      GPI_ERROR
        Error.
