GpiPaintRegion - Parameters

    hps (HPS) - input

      Presentation-space handle.

    hrgn (HRGN) - input

      Region handle.

    lHits (LONG) - returns

      Correlation and error indicators.

      GPI_OK

        Successful
      GPI_HITS
        Correlate hits
      GPI_ERROR
        Error.
