GpiQueryEditMode - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lMode (LONG) - returns

      Current editing mode.

      SEGEM_INSERT

        Insert mode
      SEGEM_REPLACE
        Replace mode
      SEGEM_ERROR
        Error.
