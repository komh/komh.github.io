GpiSetPatternRefPoint - Parameters

    hps (HPS) - input

      Presentation-space handle.

    pptlRefPoint (PPOINTL) - input

      Pattern reference point.

      The coordinates are world coordinates.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
