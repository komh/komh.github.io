GpiQueryPattern - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lPatternSymbol (LONG) - returns

      Pattern symbol.

      PATSYM_DEFAULT

        Default
      PATSYM_DENSE1 to PATSYM_DENSE8
        (1L to 8L) Solid shading with decreasing density
      PATSYM_VERT
        (9L) Vertical pattern
      PATSYM_HORIZ
        (10L) Horizontal pattern
      PATSYM_DIAG1 to PATSYM_DIAG1
        (11L to 14L) Diagonal pattern 1 to 4, bottom left to top right
      PATSYM_NOSHADE
        (15L) No shading
      PATSYM_SOLID
        (16L) Solid shading
      PATSYM_HALFTONE
        (17L) Alternate pels set on
      PATSYM_BLANK
        (18L) Blank
      PATSYM_ERROR
        (-1L) Error.
