GpiSetBitmapDimension - Parameters

    hbm (HBITMAP) - input

      Bit-map handle.

    psizlBitmapDimension (PSIZEL) - input

      Width and height of bit map.

      The width and height, respectively, of the bit map in units of 0.1 millimeter.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
