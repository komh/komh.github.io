GpiBeginPath - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lPath (LONG) - input

      Path identifier.

      This must be 1.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
