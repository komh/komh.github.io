GpiComment - Parameters

    hps (HPS) - input

      Presentation-space handle.

    lLength (LONG) - input

      Data length.

      The length of pbData in bytes. lLength must >= 0 and <= 255.

    pbData (PBYTE) - input

      Comment string.

      No conversion of any kind is performed on the data.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion
      FALSE
        Error occurred.
