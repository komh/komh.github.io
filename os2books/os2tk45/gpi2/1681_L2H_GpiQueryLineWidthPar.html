GpiQueryLineWidth - Parameters

    hps (HPS) - input

      Presentation-space handle.

    fxLineWidth (FIXED) - returns

      Line width.

      LINEWIDTH_DEFAULT

        (0L) Default
      LINEWIDTH_NORMAL
        (1L) Normal width (1.0)
      LINEWIDTH_THICK
        (2L) Thick line
      LINEWIDTH_ERROR
        (-1L) Error.
