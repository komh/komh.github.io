IMAGE32 Sample Program

IMAGE32 (located in \TOOLKIT\SAMPLES\PM\PORTING) demonstrates how to migrate an existing OS/2 16-bit application to a 32-bit application.

This sample also demonstrates how to display an image using the GpiImage function. The image data comes from a file that you must select using the standard File Open menu item.
