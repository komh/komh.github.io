IPFC Environment Variables

There are four environment variables that can be used to specify the location of source files:

    IPFC

      Specifies the location of the .IPF support files (such as APSYMBOL.APS and IPF*.NLS).
    IPFCIMBED
      Searches for files imbedded with the .im macro.
    IPFCARTWORK
      Specifies the location of artwork files and artlink files.
    TMP
      Indicates where the compiler should store the temporary files it creates during the compilation.
