Print

Print enables you to print one or more topics. You can also print a set of topics by first marking the topics in the Contents list. To print the document contents list do the following:

  • Select Print from the Services menu.

  • Click on Contents (or press the Up or Down Arrow key to select it).

  • Click on Print (or select it and press Enter).

  • The Contents list is printed on your printer.
