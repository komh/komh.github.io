ADMCT Sample Program

ADMCT (located in \TOOLKIT\SAMPLES\MM\ADMCT) is an example of a media control driver (MCD) that demonstrates how to control a streaming device. Streaming devices use the services of the sync/stream manager (SSM) of OS/2 multimedia to control the data stream from a source location to a target location.
