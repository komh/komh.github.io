Toolkit Desktop Objects

To restore your Toolkit Desktop objects, insert the CD or diskette that contains the Toolkit Installation program (TKINSTAL.EXE), start the program, and follow these steps:

  • Click on Options to open Installation options.

  • Deselect Install selected files and Write CONFIG.SYS updates to: (only Register WPS classes and Create desktop objects should be selected).

  • Click on OK to close Installation options.

  • Highlight the root (top-most) component in the component tree, and then fill in the Destination: field with the location of your Toolkit subdirectory.

  • Click on Install.
