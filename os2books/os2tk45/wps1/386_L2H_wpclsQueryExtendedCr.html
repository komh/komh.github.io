wpclsQueryExtendedCriteria Parameter - ulSearchType

    ulSearchType (ULONG) - input

      Flag indicating type of search to be performed.

      SEARCH_ALL_FOLDERS

        Search all folders.
      SEARCH_THIS_FOLDER
        Search current folder only.
      SEARCH_THIS_TREE
        Search the current folder tree only.
