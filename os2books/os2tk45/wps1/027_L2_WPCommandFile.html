Description

This is the command file object class. All DOS and OS/2 command/batch files are of this class. These program files are templatable. An instance of this class can be created as a Workplace object. Instances of this class are created initially by the system for each command file in the system. Each instance has a title corresponding to the file name of the command file it represents and resides in a folder corresponding to the physical directory in which the program file resides.
