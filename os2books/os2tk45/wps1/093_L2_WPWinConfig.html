Description

This class is specific to Version 3, or higher, of the OS/2 operating system.

This is the WIN-OS/2* configuration object class. This object stores and allows the user to modify the default settings for WIN-OS/2 applications. It initially resides in the system setup folder. An instance of this class can be created as a Workplace object.
