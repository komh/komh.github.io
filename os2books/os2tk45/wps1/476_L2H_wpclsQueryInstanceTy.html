wpclsQueryInstanceType Return Value - psuccess

    psuccess (PSZ) - returns

      Success indicator.

      NULL

        Error occurred.
      Other
        A pointer to a string containing file types. This string can contain several file types separated by commas. For example: 
        "Plain Text,C Code"
