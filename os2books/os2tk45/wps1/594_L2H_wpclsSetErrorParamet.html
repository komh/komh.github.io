wpclsSetError - Parameters

    somSelf (M_WPObject *) - input

      Pointer to the WPObject class object.

    ulErrorId (ULONG) - input

      Error code.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        The error was successfully stored.
      FALSE
        The error was not successfully stored.
