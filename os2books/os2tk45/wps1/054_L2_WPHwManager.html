Description

This class is specific to Version 4, or higher. of the OS/2 operating system.

This is the Hardware manager folder object class. An instance of this class can be created as a Workplace object. An instance of this class is created initially by the system. It has the title "Hardware Manager" and resides in the "OS/2 System" folder.
