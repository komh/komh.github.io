wpclsUnInitData - Syntax

This method is called to allow the class object to free allocated resources. 

#define INCL_WINWORKPLACE
#include <os2.h>

M_WPObject     *somSelf;  /*  Pointer to the WPObject class object. */

wpclsUnInitData(somSelf);
