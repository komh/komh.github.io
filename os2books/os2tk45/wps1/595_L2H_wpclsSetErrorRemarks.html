wpclsSetError - Remarks

This method can be used when writing class methods for workplace objects that return BOOLEAN values. When a given class method fails, the class method can log its error code in the class object so that the caller can later retrieve it using the wpclsQueryError class method. The function is analogous to the WinSetErrorInfo function call that is used by Presentation Manager functions to log their error return codes.
