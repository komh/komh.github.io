wpclsQueryInstanceFilter Return Value - psuccess

    psuccess (PSZ) - returns

      Success indicator.

      NULL

        Error occurred.
      Other
        A pointer to a string containing file title filters. This string can contain several file title filters separated by commas. For example: 
        "*.TXT, *.DOC"
