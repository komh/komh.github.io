wpclsFindObjectEnd - Parameters

    somSelf (M_WPObject *) - input

      Pointer to the WPObject class object.

    hfind (HFIND) - input

      Handle associated with a previous wpclsFindObjectFirst or wpclsFindObjectNext method call.

    rc (BOOL) - returns

      Success indicator.

      TRUE

        Successful completion.
