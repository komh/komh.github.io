Description

This class is specific to version 2.1, or higher, of the OS/2 operating system.

This is the power object class that represents the power management support. An instance of this class can be created as a Workplace object. An instance of this class is created initially by systems that support power management. It cannot be deleted. It has the title "Power" and resides in the "System Setup" folder.
