wpclsInitData - Syntax

This method is called to allow the class object to initialize its instance data. 

#define INCL_WINWORKPLACE
#include <os2.h>

M_WPObject     *somSelf;  /*  Pointer to the WPObject class object. */

wpclsInitData(somSelf);
