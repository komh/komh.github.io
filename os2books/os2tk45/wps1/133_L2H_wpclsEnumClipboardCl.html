wpclsEnumClipboardClasses - Remarks

If the second parameter is zero this method returns the first atom in list. Otherwise, it returns the next class name after the one passed in. If the atom passed in is the last atom in the list the method will return 0.

The class names are returned and passed as atoms. Use WinQueryAtomName to get the real name from the atom.
