Description

This class is specific to Version 4, or higher, of the OS/2 operating system.

This is the WPUrlFolder object class. An instance of this class can be created as a Workplace object. This folder is a subclass of WPFolder and differs from a regular folder in the following ways:

  • Has a different default icon, or icons, and title.
  • Opens in the Details view by default.
  • Shows details of the WPFolder class by default.
  • Sorts its contents by default.

A WPUrlFolder object is intended to be a convenient place to store and browse for URLs, but it will accept drops of any WPObject.
