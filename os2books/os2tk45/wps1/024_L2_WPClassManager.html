Class definition file: wpabs.idl

Class hierarchy SOMObject
SOMClassMgr
WPClassManager

Description

This is the Workplace class manager object class. It is responsible for modifying SOMClassMgr to support object class replacement, and to utilize the Workplace class registration mechanism so that a Workplace class can be found using somFindClass. See the online book System Object Model Programming Reference for more information on SOMClassMgr.
