Description

This class is specific to Version 3, or higher, of the OS/2 operating system.

This is the network link object class. This object handles all saving and restoring of data for the shared directory object by providing a persistent link or reference to that shared directory. The purpose of this link is to locally store all the instance data of the shared directory that this object is linked to and then reroute all requests for help, context menus, open views, and so forth, to this object. An instance of this class can be created as a Workplace object.
