WinIsWPDServerReady - Parameters

    fReady (BOOL) - returns

      DSOM server-state indicator. DSOM Server status.

      TRUE

        Workplace Shell DSOM Server is ready.
      FALSE
        Workplace Shell DSOM Server is not ready.
