WinRestartWPDServer Parameter - fAction

    fAction (BOOL) - input

      Flag indicating the action to perform on the Workplace Shell DSOM server.

      TRUE

        Start the Workplace Shell DSOM server.
      FALSE
        Stop the Workplace Shell DSOM server.
