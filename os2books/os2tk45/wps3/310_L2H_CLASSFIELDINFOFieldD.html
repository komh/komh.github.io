CLASSFIELDINFO Field - DefaultComparison

    DefaultComparison (ULONG)

      Default comparison operator in the Include Page Criteria dialog box (in the Settings notebook).

      For example, CMP_LESS_OR_EQUAL or CMP_NOT_EQUAL.
