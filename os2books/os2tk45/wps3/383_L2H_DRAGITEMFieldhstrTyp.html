DRAGITEM Field - hstrType

    hstrType (HSTR)

      String handle of the object type.

      The string handle must be created using the DrgAddStrHandle function. The string is of the form: 

      type[,type...]
      The first type in the list must be the true type of the object. The following types are used by the OS/2* shell:

      DRT_ASM

        Assembler code
      DRT_BASIC
        BASIC code
      DRT_BINDATA
        Binary data
      DRT_BITMAP
        Bit map
      DRT_C
        C code
      DRT_COBOL
        COBOL code
      DRT_DLL
        Dynamic link library
      DRT_DOSCMD
        DOS command file
      DRT_EXE
        Executable file
      DRT_FONT
        Font
      DRT_FORTRAN
        FORTRAN code
      DRT_ICON
        Icon
      DRT_LIB
        Library
      DRT_METAFILE
        Metafile
      DRT_OS2CMD
        OS/2 command file
      DRT_PASCAL
        Pascal code
      DRT_RESOURCE
        Resource file
      DRT_TEXT
        Text
      DRT_UNKNOWN
        Unknown type.
