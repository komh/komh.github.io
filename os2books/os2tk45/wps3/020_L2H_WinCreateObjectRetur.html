WinCreateObject Return Value - rc

    rc (HOBJECT) - returns

      Handle to the created object.

      NULLHANDLE

        Error occurred.
      Other
        A handle to the object created. This handle is persistent and can be used for the WinSetObjectData and WinDestroyObject function calls.
