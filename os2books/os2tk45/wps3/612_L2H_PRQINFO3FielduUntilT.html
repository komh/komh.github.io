PRQINFO3 Field - uUntilTime

    uUntilTime (USHORT)

      Minutes after midnight. when queue ceases to be active.

      For example, the value 1200 represents 8 p.m.

      If uUntilTime and uStartTime are both 0, the print queue is always available.
