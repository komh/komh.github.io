CLASSFIELDINFO Field - pfnOwnerDraw

    pfnOwnerDraw (PFNOWNDRW)

      Ownerdraw procedure for Details View column.

      You would use this if you wanted to draw the field in the Details View of your object. An example of this might be if you wanted to draw a graphic image in the field.

      This field can be NULL except if ownerdraw is being used (that is, if the object is choosing to draw the field itself).
