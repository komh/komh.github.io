WinIsSOMDDReady - Syntax

This function returns the state of the DSOM daemon (SOMDD), started by the Workplace Shell process using WinRestartSOMDD. 

#define INCL_WPCLASS
#include <os2.h>

BOOL    fReady;  /*  SOMDD-state indicator. */

fReady = WinIsSOMDDReady();
