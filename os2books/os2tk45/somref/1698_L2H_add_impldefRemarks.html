add_impldef - Remarks

Adds the specified ImplementationDef object to the Implementation Repository.

Note: Unless the impl_flags attribute of the given ImplementationDef object contains the IMPLDEF_IMPLID_SET flag, the impl_id attribute of the ImplementationDef object is ignored, and a new impl_id value is created for the newly added ImplementationDef object.
