somdSOMObjFromRef - Syntax

This method returns the SOM object corresponding to the specified object reference. 

SOMDServer       receiver;
Environment     *env;
SOMDObject       objref;
SOMObject        rc;

rc = somdSOMObjFromRef(receiver, env, objref);
