somdNewObject - Remarks

The somdNewObject method returns a new object of the class specified by objclass. The hints field is currently ignored by the SOMDObjectMgr. (This field is designed so that application-specific creation options can be supplied in a later SOMobjects release.)

In DSOM, the SOMDObjectMgr selects a random server which has advertised knowledge of the desired class objclass, and forwards the creation request to that server. If multiple capable servers are registered in the Implementation Repository, subsequent calls will be routed to different servers (starting them automatically if necessary) in order to distribute the load across several servers where possible.
