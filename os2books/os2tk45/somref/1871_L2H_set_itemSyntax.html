set_item - Syntax

This method sets the contents of an item in a list. 

NVList           receiver;
Enviornment     *env;
long             item_number;
Identifier       item_name;
TypeCode         item_type;
void            *value;
long             value_len;
Flags            item_flags;
ORBStatus        rc;

rc = set_item(receiver, env, item_number,
       item_name, item_type, value,
       value_len, item_flags);
