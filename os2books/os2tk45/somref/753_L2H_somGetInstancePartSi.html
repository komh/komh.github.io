somGetInstancePartSize Parameter - rc

    rc (long)

      bytes

        Returns the size, in bytes, of the instance variables introduced by this class. This does not include the size of instance variables introduced by this class's ancestor or descendent classes.
      0
        Returns 0 if a class introduces no instance variables.
