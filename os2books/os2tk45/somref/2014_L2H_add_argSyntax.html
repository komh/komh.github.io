add_arg - Syntax

This method incrementally adds an argument to a Request object. 

Request          receiver;
Environment     *env;
Identifier       name;
TypeCode         arg_type;
void            *value;      /*  A pointer to the argument value to be added. */
long             len;
Flags            arg_flags;
ORBStatus        rc;

rc = add_arg(receiver, env, name, arg_type,
       value, len, arg_flags);
