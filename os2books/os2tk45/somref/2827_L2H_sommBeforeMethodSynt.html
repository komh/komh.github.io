sommBeforeMethod - Syntax

Specifies a method that is automatically called before execution of each client method. 

SOMMBeforeAfter    receiver;
Environment       *env;
SOMObject          object;
somID              methodId;
va_list            ap;
boolean            rc;

rc = sommBeforeMethod(receiver, env, object,
       methodId, ap);
