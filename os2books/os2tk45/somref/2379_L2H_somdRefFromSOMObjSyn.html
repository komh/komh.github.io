somdRefFromSOMObj - Syntax

This method returns an object reference corresponding to the specified SOM object. 

SOMDServer       receiver;
Environment     *env;
SOMObject        somobj;
SOMDObject       rc;

rc = somdRefFromSOMObj(receiver, env, somobj);
