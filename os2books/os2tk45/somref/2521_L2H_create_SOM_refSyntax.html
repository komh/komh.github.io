create_SOM_ref - Syntax

This method creates a simple, transient DSOM reference to a SOM object. 

SOMOA                receiver;
Environment         *env;
SOMObject            somobj;
ImplementationDef    impl;
SOMDObject           rc;

rc = create_SOM_ref(receiver, env, somobj,
       impl);
