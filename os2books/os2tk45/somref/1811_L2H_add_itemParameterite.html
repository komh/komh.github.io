add_item Parameter - item_flags

    item_flags (Flags)

      A Flags bitmask (unsigned long). The item_flags can be one of the following values to indicate parameter direction:

      ARG_IN

        The argument is input only.
      ARG_OUT
        The argument is output only.
      ARG_INOUT
        The argument is input/output.
      IN_COPY_VALUE
        An internal copy of the argument is made and used.
      DEPENDENT_LIST
        Indicates that a specified sublist must be freed when the parent list is freed.
