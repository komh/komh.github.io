somParentNumResolve Parameter - parentMtab

    parentMtab (somMethodTabs)

      A list of method tables for the parents of the class being implemented. The SOM API specifies that the list of parent method tables for a given class be stored in the auxiliary class data structure of the class, in the parentMtab component. Thus, for the class "XYZ", the parent method table list is found in location XYZCClassData.parentMtab. Parent method table lists are available from class objects via the method call somGetPClsMtabs.
