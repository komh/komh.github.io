create_operation_list - Syntax

This method creates an NVList initialized with the argument descriptions for a given operation. 

ORB              receiver;
Environment     *env;
OperationDef     oper;
NVList           new_list;
ORBStatus        rc;

rc = create_operation_list(receiver,
       env, oper, new_list);
