somFindClass - Syntax

This method finds the class object for a class.

For backward compatibility, this method does not take an Environment parameter. 

SOMClassMgr    receiver;
somId          classId;
long           majorVersion;
long           minorVersion;
SOMClass       rc;

rc = _somFindClass(receiver, classId, majorVersion,
       minorVersion);
