somApply - Syntax

This function invokes an apply stub. Apply stubs are never invoked directly by SOM users. The somApply function must be used instead. 

SOMObject           objPtr;
somToken           *retVal;
somMethodDataPtr    mdPtr;
va_list             args;
boolean             rc;

rc = somApply(objPtr, retVal, mdPtr, args);
