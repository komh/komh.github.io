send Parameter - invoke_flags

    invoke_flags (Flags)

      A Flags bitmask (unsigned long) containing send method control information. The argument invoke_flags can have the following value.

      INV_NO_RESPONSE

        Indicates that the invoker does not intend to wait for a response, nor does it expect any of the output arguments (inout or out) to be updated.
