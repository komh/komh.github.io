SOMLoadModule - Syntax

This function loads the dynamically linked library (DLL) containing a SOM class. 

string        className;
string        fileName;
string        functionName;
int           majorVersion;
long          minorVersion;
somToken     *modHandle;
int           rc;

rc = (*SOMLoadModule)(className, fileName,
       functionName, majorVersion, minorVersion,
       modHandle);
