execute_request_loop - Parameters

    receiver (SOMOA)

      A pointer to the SOMOA object managing the implementation.

    env (Environment *)

      A pointer to the Environment structure for the method caller.

    waitFlag (Flags)

      A Flags bitmask (unsigned long) indicating whether the method should block (SOMD_WAIT) or return to the caller (SOMD_NO_WAIT) when there is no request message pending.

    rc (ORBStatus)

      May return an OBJ_ADAPTER exception which contains an DSOM error code for the operation. SOMDERROR_NoMessages is returned as an ORBStatus code if the method is invoked with SOMD_NO_WAIT and no message is pending.
