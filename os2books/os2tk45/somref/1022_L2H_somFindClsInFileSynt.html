somFindClsInFile - Syntax

This method finds a class object for a class.

For backward compatibility, this method does not take an Environment parameter. 

SOMClassMgr    receiver;
somId          classId;
long           majorVersion;
long           minorVersion;
string         file;
SOMClass       rc;

rc = somFindClsInFile(receiver, classId, majorVersion,
       minorVersion, file);
