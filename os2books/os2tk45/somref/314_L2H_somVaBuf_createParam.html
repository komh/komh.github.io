somVaBuf_create - Parameters

    vb (char *)

      Pointer to user-allocated memory or NULL.

    size (int)

      Size of memory pointed at by vb, or else zero.

    rc (somVaBuf)

      If successful, somVaBuf is returned; otherwise, a NULL value is returned.
