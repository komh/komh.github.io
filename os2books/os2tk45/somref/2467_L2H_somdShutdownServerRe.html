somdShutdownServer - Remarks

The somdShutdownServer method is invoked to stop a server process. If the server process corresponding to the server alias exists, it will be stopped and a code indicating success is returned.

Note: If the designated server is registered in the Implementation Repository (on the server's machine) as "nonstoppable" (via the regimpl "-n" option), then this method will return an error.

Note: On AIX, this method will fail to stop the server process if the process owner executing this method is not the same as that of either the server process or root.
