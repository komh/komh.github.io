somBuildClass - Syntax

This function automates the process of building a new SOM class object. 

unsigned long            inheritVars;
somStaticClassInfoPtr    sciPtr;
long                     majorVersion;
long                     minorVersion;
SOMClass                 rc;

rc = somBuildClass(inheritVars, sciPtr, majorVersion,
       minorVersion);
