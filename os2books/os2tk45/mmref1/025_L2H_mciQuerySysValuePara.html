mciQuerySysValue Parameter - pValue

    pValue (PVOID) - in/out

      Pointer to the return field. The type of data object this field points to is dependent on the attribute requested:

        System Attribute

          Data Type
        MSV_CLOSEDCAPTION
          BOOL
        MSV_MASTERVOLUME
          ULONG
        MSV_HEADPHONES
          ULONG
        MSV_SPEAKERS
          ULONG
        MSV_WORKPATH
          PSZ
        MSV_SYSQOSVALUE
          ULONG
        MSV_SYSQOSERRORFLAG
          ULONG
