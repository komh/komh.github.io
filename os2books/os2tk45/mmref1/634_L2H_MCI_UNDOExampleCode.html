MCI_UNDO - Example Code

The following code illustrates undoing the last operation. 

    USHORT                usDeviceID;
    MCI_EDIT_PARMS        mep;

    mep.hwndCallback = hwndMyWindow;

    mciSendCommand( usDeviceID,
                    MCI_UNDO,
                    MCI_NOTIFY,
                    &mep,
                    0 );
