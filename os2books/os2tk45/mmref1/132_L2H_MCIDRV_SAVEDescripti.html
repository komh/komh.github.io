MCIDRV_SAVE - Description

This message is sent from MDM to MCDs to save the state of an active device context. 




    

    
ulParam1 (ULONG)

       This
parameter can contain the following standard flags:

      
MCI_WAIT

         Control is not to be returned until the
action indicated by this message is completed.

      

      
pParam2 (PVOID)

         Not
used.
