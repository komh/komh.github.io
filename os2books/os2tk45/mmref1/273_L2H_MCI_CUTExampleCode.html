MCI_CUT - Example Code

The following code illustrates removing five seconds of a file. 

    USHORT                usDeviceID;
    MCI_EDIT_PARMS        mep;

    mep.hwndCallback =  hwndMyWindow;
    mep.ulFrom = 0;
    mep.ulTo = 5000;

    mciSendCommand( usDeviceID,
                    MCI_CUT,
                    MCI_NOTIFY | MCI_FROM | MCI_TO,
                    &mep,
                    0 );
