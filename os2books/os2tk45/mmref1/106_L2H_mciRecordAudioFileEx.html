mciRecordAudioFile - Example Code

The following code illustrates how to record an audio file. 

#define INCL_MCIOS2
#define INCL_MACHDR
#include <os2me.h>

ULONG rc;
HWND  hwnd;
rc=mciRecordAudioFile (hwnd, "SOUND.WAV", "TITLE", 0);
