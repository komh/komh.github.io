MCI_REDO - Example Code

The following code illustrates redoing the last editing action most recently undone. 

    USHORT               usDeviceID;
    MCI_EDIT_PARMS       mep;

    mep.hwndCallback = hwndMyWindow;

    mciSendCommand(usDeviceID,
                    MCI_REDO,
                    MCI_NOTIFY,
                    &mep,
                    0 );
