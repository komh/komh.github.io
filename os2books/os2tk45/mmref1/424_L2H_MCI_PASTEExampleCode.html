MCI_PASTE - Example Code

The following code illustrates pasting data from the clipboard into the current file position. 

    USHORT                usDeviceID;
    MCI_EDIT_PARMS        mep;

    mep.hwndCallback = hwndMyWindow;

    mciSendCommand( usDeviceID,
                    MCI_PASTE,
                    MCI_NOTIFY,
                    &mep,
                    0 );
