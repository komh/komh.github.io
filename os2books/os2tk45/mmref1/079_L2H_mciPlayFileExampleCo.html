mciPlayFile - Example Code

The following code illustrates how to play a digital audio file. 

#define INCL_MCIOS2
#define INCL_MACHDR
#include <os2me.h>         /* Play a wave file           */
                           /* set to valid window handle */
ULONG   rc;
HWND    hwnd;
rc=mciPlayFile ( hwnd, "GONG.WAV", 0,0,0);
