IMPLIB Help

To display IMPLIB help, type the following at the command prompt: 

    IMPLIB /?

The appropriate copyright statement is displayed along with a list of IMPLIB options. 

    Usage: IMPLIB [options] implibname {deffile... | dllfile...}
    Valid options are:
        /?
        /HELP
        /IGNORECASE
        /NOIGNORECASE
        /NOLOGO
