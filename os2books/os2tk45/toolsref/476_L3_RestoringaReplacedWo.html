Restoring a Replaced Workplace Object Class

To return a replaced class to its original class:

  • Select the replaced class from the list in the Workplace Object Class window.
  • Display the pop-up menu by clicking mouse button 2.
  • Select the Unreplace choice. Note that only classes that have already been replaced are valid.
  • Complete the following input fields:

Original class

    The name of the replaced object class being returned to its original object class in the Workplace.
Replacement class
    The name of the replaced object class being returned to its original object class.
