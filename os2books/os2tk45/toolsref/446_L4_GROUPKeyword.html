GROUP Keyword

The GROUP parameter is an optional keyword parameter that defines the group this tracepoint belongs to. For more description and examples of groups see the online help for the trace command. 

        GROUP=groupnam,

where:

    groupnam

      is an ASCII string specifying which group this tracepoint belongs. The groupname symbol must have been previously defined by the GROUPLIST statement. If an invalid groupname is given, the tracepoint will be discarded and a message issued.

      If GROUP is omitted, the trace statement will have a group value of 0.
