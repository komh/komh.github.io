Example

[nmake]
CFLAGS=/ss /ms /Gd-
.C.OBJ:
  $(CC) -c $(CFLAGS) $*.C

These lines in the TOOLS.INI file do the following:

  • Define the CFLAGS macro as "/ss /ms /Gd-"
  • Redefine the predefined inference rule to build .OBJ files from .C source files
