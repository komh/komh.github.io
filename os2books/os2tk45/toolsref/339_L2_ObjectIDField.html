Object ID Field

The Object ID must be enclosed in angle brackets (<>). This field is required when you modify an existing object that was not dropped on Object Utility/2. If you try to create an instance that is not a template, without an Object ID, you will receive a warning. You can create a new instance without an object ID. The Object ID must be unique, if specified, when creating an instance. Templates cannot have an object ID. Instances with an object ID cannot be made into a template.
