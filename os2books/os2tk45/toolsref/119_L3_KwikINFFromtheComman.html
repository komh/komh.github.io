KwikINF From the Command Line

You can start, terminate, and configure KwikINF from the command line in an OS/2 Window by entering: 

KwikINF [no options] [/C] [/T] [/?]

where:

    no options

      starts KwikINF. After entering this command, the default KwikINF hot key (ALT + Q) is enabled.
    /T
      terminates KwikINF and disables the KwikINF hot key.
    /C
      opens the Configure KwikINF window. Use this window to select another KwikINF hot key, to select a default online document from the BOOKSHELF to search, and to select the activation behavior of the KwikINF window.
    /?
      displays the following information. 
       Usage: KwikINF [Option]
  Option    Description
    /C      Configure KwikINF
    /T      Terminate KwikINF
    /?      This short help list
