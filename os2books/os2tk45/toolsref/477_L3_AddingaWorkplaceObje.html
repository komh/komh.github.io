Adding a Workplace Object Class

To add a class to the Workplace Shell:

  • Display the pop-up menu by clicking mouse button 2.
  • Select the Add Class choice.
  • Complete the following input fields:

New class name

    The name of object class you want to add to the Workplace. Type the class name exactly as it is built; it is case-sensitive.
Library module
    The name of the dynamic link library (DLL) that holds the object definition. Type the library name with complete file specification information.

    Note: The DLL must be created by the IBM System Object Model.
