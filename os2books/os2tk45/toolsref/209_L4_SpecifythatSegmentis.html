Specify that Segment is Shared

These attributes determine if the segment can be shared by other processes.

    SHARED

      One copy of the data segment is loaded and shared among all processes accessing the module (default for dynamic-link libraries).
    NONSHARED
      The segment cannot be shared, and must be loaded separately for each process (default for applications).
