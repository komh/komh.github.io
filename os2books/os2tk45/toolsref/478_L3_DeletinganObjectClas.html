Deleting an Object Class

To delete a class from the Workplace Shell:

  • Select the class you want to delete from the list in the Workplace Object Class window.
  • Display the pop-up menu by clicking mouse button 2.
  • Select the Delete a Class choice.
  • Complete the name of the class you want to delete from the Workplace.

Note: You cannot delete classes that are predefined by the operating system, such as WPObject or WPClass.
