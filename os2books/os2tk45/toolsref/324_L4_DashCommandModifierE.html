Dash-Command Modifier Examples

LIGHT.LST : LIGHT.TXT
  - FLASH LIGHT.TXT

In the example above, NMAKE never ends, regardless of the exit code returned by FLASH. 

LIGHT.LST : LIGHT.TXT
  -1 FLASH LIGHT.TXT

In the example above, NMAKE ends if the exit code returned by FLASH is greater than 1.
