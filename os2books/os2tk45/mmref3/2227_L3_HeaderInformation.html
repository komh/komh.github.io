Header Information

The header information section defines general information about the CTOC header and about the entire compound file. It contains the following fields:

  • ulHeaderSize
  • ulEntriesTotal
  • ulEntriesDeleted
  • ulEntriesUnused
  • ulBytesTotal
  • ulBytesDeleted
  • ulHeaderFlags
