SETUP_BLITTER Field - lDstPosX

    lDstPosX (LONG)

      The X-coordinate position of the rectangle in the destination output buffer into which the output image is to be placed by DiveBlitImage. This is the position of the lower-left corner of the output image rectangle relative to the lower-left corner of the output image buffer or, if specified, relative to the lScreenPosX, lScreenPosY position.
