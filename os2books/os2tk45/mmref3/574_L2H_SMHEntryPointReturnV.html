SMHEntryPoint Return Value - rc

    rc (ULONG) - returns

      Return codes indicating success or failure:

      NO_ERROR

        Successful.
      ERROR_INVALID_FUNCTION
        Invalid function requested.
      FAILURE
        An SMH message-specific error return code.
