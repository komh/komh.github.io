SMH_DEREGISTER - Description

This message disconnects a stream handler from the Sync/Stream Manager. 




    

    
pParmIn (PPARM_DEREG)

       A
pointer to a PARM_DEREG data structure.


    

    
rc (ULONG)

       Return
codes indicating success or the type of failure:

      
NO_ERROR

         Function performed.

       ERROR_INVALID_FUNCTION


        Invalid function requested.

       ERROR_HNDLR_NOT_FOUND


        Stream-handler registration information was not
found.

       ERROR_INVALID_HNDLR_NAME

         Invalid
stream-handler name.

       ERROR_INVALID_BLOCK

         Invalid
pointer.
