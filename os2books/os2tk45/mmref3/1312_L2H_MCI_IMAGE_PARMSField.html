MCI_IMAGE_PARMS Field - pPelBuffer

    pPelBuffer (PVOID)

      If pPelBuffer is equal to 0 on an MCI_GETIMAGEBUFFER message, the message will be considered a query for the buffer length and attributes of the current image element.
