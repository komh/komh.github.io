SpiDisableEvent - Syntax

This function disables event notification for a particular event. 

#include <os2.h>

HEVENT    hevent;  /*  Event. */
ULONG     rc;      /*  Return codes indicating success or the type of failure: */

rc = SpiDisableEvent(hevent);
