SMH_LOCKMEM - Description

This message locks a memory object (available at Ring 3 only). 




    

    
pParmIn (PPARM_LOCKM)

       A
pointer to a PARM_LOCKM data structure.


    

    
rc (ULONG)

       Return
codes indicating success or the type of failure:

      
NO_ERROR

         Function performed.

       ERROR_INVALID_FUNCTION


        Invalid function requested.

       ERROR_INVALID_BLOCK


        Invalid pointer.
