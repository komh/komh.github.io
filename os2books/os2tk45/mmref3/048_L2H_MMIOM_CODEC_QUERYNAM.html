MMIOM_CODEC_QUERYNAMELENGTH - Description

This message is sent to the CODEC procedure by mmioQueryCODECNameLength to request the length of the CODEC name. 




    

    
phCODEC (PHCODEC)

       Pointer
to CODEC handle.

    

    
usMsg (USHORT)

       Set
to MMIOM_CODEC_QUERYNAMELENGTH.

    

    
lParam1 (LONG)

       Pointer
to the length in bytes of the CODEC name string.

    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

       Return
codes indicating success or failure:

      
MMIO_SUCCESS

         The information is requested successfully.


      MMIO_ERROR

         An error code
is returned.
