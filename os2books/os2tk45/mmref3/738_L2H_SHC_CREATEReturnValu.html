SHC_CREATE Return Value - rc

    rc (ULONG)

      Return codes indicating success or the type of failure:

      NO_ERROR

        Success.
      ERROR_INVALID_FUNCTION
        Invalid function requested.
      ERROR_INVALID_STREAM
        Invalid stream handle.
      ERROR_INVALID_PARAMETER
        Invalid parameter.
      ERROR_INVALID_HID
        Invalid handler ID.
      ERROR_INVALID_SPCBKEY
        Invalid SPCBKEY.
      ERROR_DEVICE_NOT_FOUND
        Invalid device driver name.
      ERROR_STREAM_CREATION
        Error creating the stream..
      ERROR_INVALID_BLOCK
        Invalid DCB pointer.
      ERROR_ALLOC_RESOURCES
        Error allocating a system resource. Out of memory or similar error.
      FAILURE
        Stream-handler-specific error return code.
