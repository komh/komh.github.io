MIDIQueryVersion - Syntax

This function queries the current RTMIDI device driver and DLL version numbers. 

#include <mididll.h>

PULONG    pulVersion;  /*  Version number. */
ULONG     rc;          /*  Return code. */

rc = MIDIQueryVersion(pulVersion);
