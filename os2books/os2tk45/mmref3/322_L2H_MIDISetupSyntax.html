MIDISetup - Syntax

This function provides a generic setup for an application. 

#include <mididll.h>

PMIDISETUP    pMidiSetup;  /*  Pointer to setup values. */
ULONG         ulFlag;      /*  Not used. */
ULONG         rc;          /*  Return code. */

rc = MIDISetup(pMidiSetup, ulFlag);
