JPEGOPTIONS Field - usQuantization[4]

    usQuantization[4] (USHORT)

      An array of quantization values used for compression only. Each value has a range of 1-65535, where a value of 1 is the highest quality and a value of 65535 is the lowest quality. The default value is is 100.
