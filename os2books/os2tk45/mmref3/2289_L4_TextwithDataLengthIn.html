Text with Data Length Information

The 'ltxt' chunk contains text that is associated with a data segment of specific length. The chunk fields are as follows:

    Field

      Description
    ulName
      Specifies the cue point name. This value must match one of the names listed in the <cue-ck> cue-point table.
    ulSampleLength
      Specifies the number of samples in the segment of waveform data.
    ulPurpose
      Specifies the type or purpose of the text. For example, ulPurpose can specify a FOURCC code like "scrp" for script text or "capt" for close-caption text.
    usCountry
      Specifies the country code for the text. See Country Codes for a current list of country codes.
    usLanguage, usDialect
      Specify the language and dialect codes for the text. See Language and Dialect Codes for a current list of language and dialect codes.
    usCodePage
      Specifies the code page for the text.
