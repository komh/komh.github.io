Notation Conventions

The following table lists some of the notation conventions used to describe RIFF and the multimedia file formats. Further conventions and the notation for documenting RIFF forms are presented later in Notation for Representing Sample RIFF Files.

    Notation

      Description
    <element label>
      RIFF file element with the label "element label"
    <element label: TYPE>
      RIFF file element with data type "TYPE"
    [<element label>]
      Optional RIFF file element
    <element label>...
      One or more copies of the specified element
    [<element label>]...
      Zero or more copies of the specified element
