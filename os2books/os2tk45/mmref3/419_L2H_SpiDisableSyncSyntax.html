SpiDisableSync - Syntax

This function removes a grouping of streams. 

#include <os2.h>

HSTREAM    hstreamMaster;  /*  Stream grouping. */
ULONG      rc;             /*  Return codes indicating success or the type of failure: */

rc = SpiDisableSync(hstreamMaster);
