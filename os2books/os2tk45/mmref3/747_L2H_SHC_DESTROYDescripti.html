SHC_DESTROY - Description

This message removes a stream instance for a stream handler. 




    

    
pParmIn (PPARM_DESTROY)

       A
pointer to a PARM_DESTROY data structure.


    

    
rc (ULONG)

       Return
codes indicating success or the type of failure:

      
NO_ERROR

         Success.

       ERROR_INVALID_FUNCTION


        Invalid function requested.

       ERROR_INVALID_STREAM


        Invalid stream handle.

       ERROR_INVALID_HID


        Invalid handler ID.

       FAILURE


        Stream handler-specific error return code.
