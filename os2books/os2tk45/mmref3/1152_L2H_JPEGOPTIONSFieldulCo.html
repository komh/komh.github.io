JPEGOPTIONS Field - ulColorOrder

    ulColorOrder (ULONG)

      A color order value (used for both compression and decompression) which can be set to one of the following values:

      YUV_YVU

        YCbCr color order (default)
      ~YUV_YVU
        YCrCb color order
