TimerGetPointer Return Value - rc

    rc (ULONG) - returns

      Returns 0 if the command is completed successfully.

      TIMERERR_INVALID_PARAMETER

        Invalid parameter (ulFlag not set to 0) or invalid pointer.
      TIMERERR_INTERNAL_SYSTEM
        System problem.
