MCI_MIXSETUP_PARMS Field - ulDeviceType

    ulDeviceType (ULONG) - input

      Device type can be one of the following values:

      • MCI_DEVTYPE_WAVEFORM_AUDIO
      • MCI_DEVTYPE_SEQUENCER
      If ulDeviceType is MCI_DEVTYPE_WAVEFORM_AUDIO, the following digital audio fields must be specified: ulBitsPerSample, ulFormatTag, ulSamplesPerSec, and ulChannels.
