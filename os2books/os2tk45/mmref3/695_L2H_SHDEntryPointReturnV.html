SHDEntryPoint Return Value - rc

    rc (ULONG) - returns

      Return codes indicating success or the type of failure:

      NO_ERROR

        Successful.
      ERROR_INVALID_FUNCTION
        Invalid function requested.
      FAILURE
        An SHD message-specific error return code.
