MIDIINSTANCEINFO Field - ulAttributes

    ulAttributes (ULONG)

      Instance attributes.

      MIDI_INST_ATTR_CAN_RECV

        Instance can accept messages.
      MIDI_INST_ATTR_CAN_SEND
        Instance can send messages.
      MIDI_INST_ATTR_ENABLE_R
        Receive is enabled.
      MIDI_INST_ATTR_ENABLE_S
        Send is enabled.
