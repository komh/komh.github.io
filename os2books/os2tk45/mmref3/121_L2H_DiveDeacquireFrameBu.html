DiveDeacquireFrameBuffer - Syntax

This function releases exclusive access to the frame buffer. 

#include <dive.h>

HDIVE    hDiveInst;  /*  Display engine DIVE instance. */
ULONG    rc;         /*  Return codes. */

rc = DiveDeacquireFrameBuffer(hDiveInst);
