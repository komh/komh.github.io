MMIOM_CODEC_OPEN - Description

This message requests a CODEC procedure instance be opened. On completion, it returns a CODEC handle in phCODEC. 




    

    
phCODEC (PHCODEC)

       Pointer
to CODEC handle.

    

    
usMsg (USHORT)

       Set
to MMIOM_CODEC_OPEN.

    

    
lParam1 (LONG)

       Pointer
to an CODECOPEN structure.

    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

       Return
codes indicating success or failure:

      
MMIO_SUCCESS

         The specified instance is opened successfully.


      MMIO_ERROR

         An error code
is returned.
