SpiEnumerateProtocols Return Value - rc

    rc (ULONG) - returns

      Return codes. Return codes indicating success or the type of failure:

      NO_ERROR

        Success.
      ERROR_INVALID_FUNCTION
        Invalid function requested.
      ERROR_INVALID_HID
        Invalid handler ID.
      ERROR_INVALID_BUFFER_SIZE
        Array passed is too small to hold all the SPCB keys. The size of the buffer needed is returned in pulNumSPCBKeys.
      ERROR_INVALID_BLOCK
        Invalid pointer.
      FAILURE
        Stream handler-specific error return code.
