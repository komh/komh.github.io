MCI_PALETTE_PARMS Field - ulPalEntries

    ulPalEntries (ULONG)

      Number of RGB entries to be set on an MCI_SETIMAGEPALETTE message, or number of entries placed in pPalette on an MCI_GETIMAGEPALETTE message.
