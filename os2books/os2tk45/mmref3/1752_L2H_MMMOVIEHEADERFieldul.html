MMMOVIEHEADER Field - ulSuggestedBufferSize

    ulSuggestedBufferSize (ULONG)

      Suggested buffer size in bytes for reading the file. If the value specified in this field is greater than the stream buffer specified in the SPI.INI file for the stream protocol, the MCD will use the ulSuggestedBufferSize value as the buffer size for creating streams.
