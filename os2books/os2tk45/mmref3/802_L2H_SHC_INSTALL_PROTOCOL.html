SHC_INSTALL_PROTOCOL - Description

This message installs or removes a specified stream protocol for a stream handler. 




    

    
pParmIn (PPARM_INSTPROT)


      A pointer to a PARM_INSTPROT
data structure.

    

    
rc (ULONG)

       Return
codes indicating success or the type of failure:

      
NO_ERROR

         Success.

       ERROR_INVALID_SPCBKEY


        Invalid spcbkey.

       ERROR_INVALID_PROTOCOL


        Invalid stream protocol control block.

       ERROR_ALLOC_RESOURCES


        Out of memory or some other resource.

       FAILURE


        Stream handler-specific error return code.
