MMDECOMPRESS Field - pSrcBuf

    pSrcBuf (PVOID)

      Pointer to the source compresssed data. On return the pointer is adjusted to point to the data not yet compressed. The NULL must be returned when all data in the pSrcBuf field is decompressed.
