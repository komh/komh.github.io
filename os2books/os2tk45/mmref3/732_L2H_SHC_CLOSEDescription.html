SHC_CLOSE - Description

This message closes a stream handler. 




    

    
pParmIn (PPARM_CLOSE)

       A
pointer to a PARM_CLOSE data structure.


    

    
rc (ULONG)

       Return
codes indicating success or the type of failure:

      
NO_ERROR

         Success.

       ERROR_INVALID_FUNCTION


        Invalid function requested.

       ERROR_INVALID_HID


        Invalid handler ID.

       FAILURE


        Stream-handler-specific error return code.
