Register Review

Registers discussed so far:

    CR3

      32-bit physical address of the Page Directory
    IDTR
      32-bit linear address of IDT, 16-bit size of IDT
    GDTR
      32-bit linear address of GDT, 16-bit size of GDT
    LDTR
      16-bit selector for an entry ( type 2 ) in the GDT
    SS
      16-bit selector, used for stack operations
    CS
      16-bit selector, used to locate instructions
    DS
      16-bit selector, used to locate data, generally the default
    ES
      16-bit selector, used to locate data, string destination
    FS
      16-bit selector, used to locate data explicitly
    GS
      16-bit selector, used to locate data explicitly
