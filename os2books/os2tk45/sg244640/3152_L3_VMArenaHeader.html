VM Arena Header

    Locations

      _ahvmSys locates the System Arena VMAH.

      _ahvmShr locates the Shared Arena VMAH.

      PTDA field ptda_ah locates each Private Arena VMAH.

    VM Owner
      For shared and system arenas: os2krnl (0xffaa)

      For private arenas: ptda (0xffcb)
