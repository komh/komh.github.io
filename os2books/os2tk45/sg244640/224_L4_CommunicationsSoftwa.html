Communications Software

Any terminal software that can communicate at 9600 bps will do. OS/2 2.0 comes with a program (Softerm Custom) that is adequate. PMDF, which is part of the OS2PDP package on the CDROM that accompanies this book, also provides a terminal emulation facility but in addition provides REXX support that allows Kernel Debugger command sequences to be automated.
