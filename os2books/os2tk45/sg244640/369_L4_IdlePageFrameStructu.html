Idle Page Frame Structures.

Notes

  • Idle Page Frame Structures are chained in a double linked-list. The head of this list may be located as follows:

    • Locate list structure whose address is given by _pgIdleList

    • The first double-word of the list structure points to the pseudo-page frame structure that heads the idle list.

    • The second double-word contains the pseudo-frame number of the pseudo-PF. N.B. This marks the end of the linked list only.

    • The backward pointer to the first true idle PF is given by the 5 low order digits of the second double-word of the pseudo-PF. This value may be used with the .MP command to format the first idle PF provided it is not equal to the pseudo-frame number, which would imply no idle PFs on the idle list.

    • Subsequent idle PFs are found by following the BLink until the pseudo-frame is encountered.

    For a description of the fields formatted by .MP select .MP Output Field Descriptions
