From more to less privileged, for example, Ring 0 to Ring 3

  • Verify that all steps below will work. If not, general protection fault.

  • Pop IP, CS

  • Add immediate value to 'old' SP

  • Pop SP, SS

  • Add immediate value to 'new' SP

  • Zero every selector which has PL more privileged than the new CPL. This is required to maintain integrity because access validation is done only when a selector register is changed - not when it is used.
