16 Bit Error Logging API's for IBM OS/2 Version 2.1

This section describes the "Logging Facility for OS/2 2.1". This comprises a set of 3 APIs, the logging deamon (LOG.SYS) and the log formatter (SYSLOG).

Both the Logging Deamon and Log Formatter are described in the OS/2 Command Reference - see LOG.SYS under DEVICE statement of CONFIG.SYS and the SYSLOG command.

Note:

C Language prototype definitions for the Error Logging APIs may be found under RAS API Prototypes.

    • DosLogRegister API

    • DosLogEntry API

    • DosLogRead API

    • Error Log Entry Formatting DLL Routines
