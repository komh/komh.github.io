A Job File Number Table (JFT) entry is assigned to each open file system object within a process. The JFT provides a cross-reference to the handle for the corresponding SFT. The JFT is locatable from the PTDAfield JFN_pTable (PTDA +0x5b8 (H/R: +0x5b0)) for each process.

The JFT is initially allocated within the PTDA at label JFN_Table (PTDA +0x35e) with 20 entries. If this is expanded by use of the DosSetMaxFH then JFN_pTable is updated to point to the new table.
