Parameter Format Notation

The Parameter portion of a tracepoint listing typically includes both fixed descriptive ASCII strings and variable format descriptors. The following list enumerates the classes of format descriptors that are encountered in the tracepoint listings:

  • A
      Output a segmented (16:16) address (32 bits)
    %B
      Output a byte (8 bits) of data
    %D
      Output A double word (32 bits) of data
    %F
      Output a flat (0:32) address (32 bits)
    %Q
      Output a quad word (64 bits) of data
    %S
      Output an ASCIIZ string (which is "n" bytes terminated by 00h)
    %U
      Format the remainder of the trace record as a sequence of bytes
    %W
      Output a word (16 bits) of data
    %X
      Output the major event code
    %Y
      Output the minor event code

    A number preceding one of the above specifies a sequence of that type.
