The term loader applies to two distinct components under OS/2:

    OS2LDR

      This is the OS2KRNL loader. One of its functions is to load the OS2KRNL module at boot time. After the system has booted OS2LDR provides the CBIOS layer for the kernel.
    System Loader
      This is a component of the kernel. It is responsible for loading program modules, DLLs, Device Drivers and File System Drivers.

    The logging facility discussed in this section applies to the System Loader.
