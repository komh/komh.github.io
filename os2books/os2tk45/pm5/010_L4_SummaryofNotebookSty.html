Summary of Notebook Styles for Old Notebook

The old notebook control provides style bits so that your application can specify or change the default style settings. One style bit from each of the following groups can be specified. If you specify more than one style bit, you must use an OR operator (|) to combine them.

  • Type of binding

      BKS_SOLIDBIND

        Solid (default)
      BKS_SPIRALBIND
        Spiral
      

          Top-right corner
        BKS_BACKPAGESTL
          Top-left corner
        
          Location of major tabs

            Right edge (default)

            Bottom edge

              BKS_TABTEXTCENTER

                Alignment of status-line text

                BKS_STATUSTEXTLEFT

                  Left-justified (default)

                  Right-justified

                  Centered
                
                  Styles not applicable to an Old Notebook

                  BKS_TABBEDDIALOG

                    Indicates a new-style notebook is desired.
                  BKS_BUTTONAREA
                    Creates common button area for a new notebook. (This style uses the same area as BKS_POLYGONTABS, so it may generate unexpected results if used.)
