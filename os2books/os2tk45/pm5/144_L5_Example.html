Example

This example creates a menu for the window created with the WINDOW statement. 

  WINDOWTEMPLATE 1
  BEGIN
      WINDOW "Sample", 1, 0, 0, 100, 100, "MYCLASS", 0, FCF_STANDARD
      CTLDATA MENU
      BEGIN
          MENUITEM "Exit", 101
      END
  END
