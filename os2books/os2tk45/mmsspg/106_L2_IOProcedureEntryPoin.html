I/O Procedure Entry Point

The following example illustrates the the entry point used to access the functionality of an I/O procedure. 

LONG APIENTRY IOProc_Entry ( PVOID   pmmioStr,
                             USHORT  usMessage,
                             LONG    lParam1,
                             LONG    lParam2)
