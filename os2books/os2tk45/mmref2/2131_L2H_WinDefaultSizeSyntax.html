WinDefaultSize - Syntax

This function sizes the window to its default size-the optimal size of the dialog window. 

#define INCL_SW
#include <os2.h>

HWND    hwnd;  /*  Secondary window handle. */
BOOL    rc;    /*  Return codes. */

rc = WinDefaultSize(hwnd);
