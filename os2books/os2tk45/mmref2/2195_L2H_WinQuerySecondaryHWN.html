WinQuerySecondaryHWND - Parameters

    hwnd (HWND) - input

      Secondary window handle.

    ulFlag (ULONG) - input

      Flags.

      QS_FRAME

        Returns the HWND for the outer standard frame window.
      QS_DIALOG
        Returns the HWND for the inner dialog frame window.

      hwnd (HWND) - returns

        Returns the requested window handle.
