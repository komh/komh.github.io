mmioGetLastError - Parameters

    hmmio (HMMIO) - input

      The open file handle returned by mmioOpen.

    rc (ULONG) - returns

      Return code indicating success or type of failure:

      MMIOERR_INVALID_HANDLE

        An invalid handle was passed.
