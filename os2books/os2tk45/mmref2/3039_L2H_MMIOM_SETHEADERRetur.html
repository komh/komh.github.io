MMIOM_SETHEADER Return Value - rc

    rc (ULONG)

      • Upon successful completion, the number of bytes written is returned.

      • For a failure, 0 is returned.

      • If the header is invalid, MMIOERR_INVALID_STRUCTURE is set in ulErrorRet.
