GBM_SETTEXTPOSITION - Parameters

    usTextPos (USHORT)

      Text position.

      GB_TEXTBELOW

        Position text below the bitmap.
      GB_TEXTABOVE
        Position text above the bitmap.

      ulReserved (ULONG)

        Reserved.

      rc (USHORT)

        Return codes indicating success or failure.

        TRUE

          Success.
        FALSE
          Failure or not recognized.
