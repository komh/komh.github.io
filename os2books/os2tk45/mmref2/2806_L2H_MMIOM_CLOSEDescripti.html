MMIOM_CLOSE - Description

This message is sent to an IOProc by mmioClose to request that a file be closed. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_CLOSE.

    

    
lParam1 (LONG)

       The
usFlags parameter of mmioClose.


    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

       Return
codes indicating success or failure:

      
MMIO_SUCCESS

         The specified file was closed successfully.


      MMIO_ERROR

         An error code
is returned.

       MMIO_WARNING

         The
file was closed, but the IOProc expected additional data.
