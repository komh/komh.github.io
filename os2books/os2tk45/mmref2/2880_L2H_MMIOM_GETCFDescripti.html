MMIOM_GETCF - Description

This message can be sent by an application to request the return of a handle hmmcf to the RIFF compound file that contains the element associated with hmmio. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_GETCF.

    

    
lParam1 (LONG)

       This
parameter is not used.

    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

      

        

        

      • Returns a handle (type HMMCF) to the CTOC table that
contains the element associated with hmmio.

        

      • On error, NULL is returned.
