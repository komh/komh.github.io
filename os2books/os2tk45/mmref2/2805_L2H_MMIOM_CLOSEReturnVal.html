MMIOM_CLOSE Return Value - rc

    rc (ULONG)

      Return codes indicating success or failure:

      MMIO_SUCCESS

        The specified file was closed successfully.
      MMIO_ERROR
        An error code is returned.
      MMIO_WARNING
        The file was closed, but the IOProc expected additional data.
