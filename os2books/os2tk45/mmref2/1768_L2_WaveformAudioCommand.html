Waveform Audio Commands

The waveform audio device supports the following device-type specific commands and extensions to the following basic and required commands:

  • CAPABILITY
  • CONNECTOR
  • COPY
  • CUE
  • CUT
  • DELETE
  • LOAD
  • PASTE
  • RECORD
  • REDO
  • SEEK
  • SET
  • STATUS
  • UNDO
