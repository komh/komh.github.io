mmioClose Parameter - usFlags

    usFlags (USHORT) - input

      Contains nothing or the following flag:

      MMIO_FHOPEN

        This flag is used to tell the I/O to not close the file or files of type FOURCC_DOS. This allows an HMMIO instance to be closed in cases where a DOS file handle was provided to the I/O procedure on an mmioOpen call.
