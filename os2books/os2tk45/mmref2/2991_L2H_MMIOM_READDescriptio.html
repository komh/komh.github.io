MMIOM_READ - Description

This message is sent to an IOProc by mmioRead to request that bytes be read from an open file. Data translation is expected during message processing when specified. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_READ.

    

    
lParam1 (LONG)

       A
(PSZ) pointer to the buffer to read to.

    

    
lParam2 (LONG)

       The
number of bytes to read.

    

    
rc (ULONG)

       Returns
the number of bytes actually read. Returns 0L if no more bytes can be read.


      
MMIO_ERROR

         READ was not successful.
