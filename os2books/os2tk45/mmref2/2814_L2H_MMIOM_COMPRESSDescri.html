MMIOM_COMPRESS - Description

This message is sent to an I/O procedure to compress the uncompressed data in the buffer. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_COMPRESS.

    

    
lParam1 (LONG)

       Pointer
to the MMCOMPRESS structure.

    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

       Return
codes indicating success or failure:

      
MMIO_SUCCESS

         The data is decompressed successfully.


      MMIO_ERROR

         An error code
is returned.
