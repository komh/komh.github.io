MMIOM_DECOMPRESS - Description

This message is sent to the IOProc to decompress the compressed data in the buffer. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to the MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_DECOMPRESS.

    

    
lParam1 (LONG)

       Pointer
to the MMDECOMPRESS structure.


    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

       Return
codes indicating success or failure:

      
MMIO_SUCCESS

         The data is decompressed successfully.


      MMIO_ERROR

         An error code
is returned.
