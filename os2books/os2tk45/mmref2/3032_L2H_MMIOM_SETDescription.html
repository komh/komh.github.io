MMIOM_SET - Description

This message is sent to the IOProc to set or query extended file information. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_SET.

    

    
lParam1 (LONG)

       Pointer
to the MMEXTENDINFO structure.


    

    
lParam2 (LONG)

       Contains
one of the following flags:

      
MMIO_SET_EXTENDEDINFO

         Sets extended information.


      MMIO_QUERY_EXTENDEDINFO_BASE

         Query
only the information of MMEXTENDINFO
structure.

       MMIO_QUERY_EXTENDEDINFO_ALL

         Query
all extended information including the CODEC associated information.

      

      
rc (ULONG)

         Return
codes indicating success or failure:

        
MMIO_SUCCESS

           The set is completed successfully.


        MMIO_ERROR

           An error code
is returned.
