MMIOM_GETHEADER Return Value - rc

    rc (ULONG)

      • Upon successful completion, the number of bytes copied to the header structure.

      • For a failure, 0 is returned.

      • If the length passed in was not enough to hold the header, MMIOERR_INVALID_BUFFER_LENGTH is set in ulErrorRet.

      • If the header is bad, MMIOERR_INVALID_STRUCTURE is set in ulErrorRet.
