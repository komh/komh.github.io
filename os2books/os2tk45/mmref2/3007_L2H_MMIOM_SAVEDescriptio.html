MMIOM_SAVE - Description

This message is sent to an I/O procedure to request temporary changes in a file be made permanent. A new file name can be supplied to save the changes. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_SAVE.

    

    
lParam1 (LONG)

       Optional
pszFileName parameter.

    

    
lParam2 (LONG)

       This
parameter is not used.

    

    
rc (ULONG)

       Return
codes indicating success or failure:

      
MMIO_SUCCESS

         The file has been saved.

       MMIO_ERROR


        An error code is returned.
