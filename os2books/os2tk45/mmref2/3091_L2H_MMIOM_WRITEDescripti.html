MMIOM_WRITE - Description

This message is sent to an IOProc by mmioWrite to request that bytes be written to an open file. 




    

    
pmmioinfo (PMMIOINFO)

       A
pointer to an MMIOINFO data structure.


    

    
usMsg (USHORT)

       Set
to MMIOM_WRITE.

    

    
lParam1 (LONG)

       A
pointer (PCHAR) to the buffer to write form.

    

    
lParam2 (LONG)

       The
number of bytes to write.

    

    
rc (ULONG)

       Returns
the number of bytes actually written or MMIO_ERROR if the write was not
successful.
