GBM_QUERYTEXTPOSITION Return Value - rc

    rc (ULONG)

      Return codes indicating the position of the graphic button.

      GB_TEXTBELOW

        The graphic button text is located below the bitmap.
      GB_TEXTABOVE
        The graphic button text is located above the bitmap.
