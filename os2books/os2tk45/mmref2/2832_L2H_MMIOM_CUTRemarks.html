MMIOM_CUT - Remarks

If data is already in the clipboard, it is overwritten with this message call. The media position will be at the position corresponding to the ulStartTime field of the MMIO_EDIT_PARMS structure passed in the lParam1 parameter.

The ulDuration field of the MMIO_EDIT_PARMS structure cannot be set to 0.
