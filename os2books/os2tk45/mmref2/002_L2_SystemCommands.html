System Commands

System commands are interpreted directly by the media device manager (MDM), and are not passed to media control interface drivers. The following commands are system commands:

  • ACQUIRE
  • CONNECTION
  • CONNECTORINFO
  • DEFAULTCONNECTION
  • GROUP
  • MASTERAUDIO
  • RELEASE
  • SYSINFO
