mmioGetLastError - Remarks

The user can call mmioGetLastError for those functions that return only MMIO_ERROR or MMIO_CF_FAILURE, and obtain additional information about the failing condition from the ulErrorRet field of the MMIOINFO structure. If ulErrorRet does not contain an MMIO error code, it contains an OS/2 error code or 0.
