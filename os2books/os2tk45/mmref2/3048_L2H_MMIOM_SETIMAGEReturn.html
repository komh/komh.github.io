MMIOM_SETIMAGE Return Value - rc

    rc (ULONG)

      Return codes indicating success or failure.

      MMIO_SUCCESS

        The request was successful.
      MMIO_ERROR
        An error code is returned.
      MMIOERR_UNSUPPORTED_FUNCTION
        The IOProc does not support multiple images.
