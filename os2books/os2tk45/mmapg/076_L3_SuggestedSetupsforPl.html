Suggested Setups for Playlists

The following tables provide guidelines for setting up playlists to utilize memory more efficiently.

│8         │1         │4KB         │30          │3           │

├──────────┼──────────┼────────────┼────────────┼────────────┤

│16        │2         │16KB        │20          │3           │
└──────────┴──────────┴────────────┴────────────┴────────────┘
