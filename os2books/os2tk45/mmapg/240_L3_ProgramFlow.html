Program Flow

The following figure illustrates the interaction between OS/2 multimedia system components and the AVC I/O Procedure Installation sample program. Source code is located in the \TOOLKIT\SAMPLES\MM\AVCINST subdirectory. 

┌──────────────────────────────┐
│AVO I/O Procedure Installation│
│       Sample Program         │
└──────────────┬───────────────┘
       (1)     
    ┌─────────────────────┐
    │    MMIO Manager     │
    └──────────┬──────────┘
       (2)     
    ┌─────────────────────┐
    │  AVC I/O Procedure  │
    └─────────────────────┘

  • When OK is selected, the AVC I/O Procedure Installation program calls the mmioInstallfIOProc function to install or deinstall the AVC I/O procedure (AVCAPROC.DLL) in the system.

  • The MMIO Manager installs or deinstalls the AVC I/O procedure.
