Network Related Methods

A program or Workplace Shell object would call the following methods to get information about a network object. These methods are not normally overridden. 

┌──────────────────────────────┬──────────────────────────────┐
│Method Name                   │Description                   │
├──────────────────────────────┼──────────────────────────────┤
│wpQueryNetIdentity            │Returns the fully qualified   │
│                              │network name for a network    │
│                              │group object.                 │
├──────────────────────────────┼──────────────────────────────┤
│wpQueryObjectNetId            │Returns the network ID of the │
│                              │shared directory object that  │
│                              │this object is linked to.     │
├──────────────────────────────┼──────────────────────────────┤
│wpQuerySrvrIdentity           │Returns the fully qualified   │
│                              │network name for this server  │
│                              │object.                       │
└──────────────────────────────┴──────────────────────────────┘

A program or Workplace Shell object would call the following methods to set information about a network object. These methods are not normally overridden. 

┌──────────────────────────────┬──────────────────────────────┐
│Method Name                   │Description                   │
├──────────────────────────────┼──────────────────────────────┤
│wpSetNetIdentity              │Sets the fully qualified      │
│                              │network name for a network    │
│                              │group object.                 │
├──────────────────────────────┼──────────────────────────────┤
│wpSetObjectNetId              │Sets the network ID of the    │
│                              │shared directory object that  │
│                              │this object is linked to.     │
└──────────────────────────────┴──────────────────────────────┘
