Object Information: Set/Query Methods

The Workplace Shell provides several set and query methods that you can use to set or to get object information, such as the default view of the object's in-use list, the object's icon, and the object's title. This chapter provides information for using object information methods. It describes how to define the behavior of Workplace Shell objects, using applicable styles.
