Obtaining the Class Style


The wpclsQueryStyle method is called to return the class attributes for this object class:

    Class Style Name

      Description
    CLSSTYLE_NEVERCOPY
      The object cannot be copied.
    CLSSTYLE_NEVERDELETE
      The object cannot be deleted.
    CLSSTYLE_NEVERMOVE
      The object cannot be moved.
    CLSSTYLE_NEVERTEMPLATE
      This class does not have a template. Also, the Styler object will not have a Create another item in its context menu.

    Classes should override this method to get special behavior.
