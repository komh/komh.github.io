Pop-Up Menu Application Sample Code

The pop-up menu application includes the following files:

    File Name

      Description
    HELLO.IDL
      Class interface definition
    HELLO.C
      Source code
    HELLO.RC
      Resource code
    HELLO.MAK
      Hello make file for building the application

    Following these files are three REXX files that register and deregister the new class with the Workplace Shell, and create an instance of the class.
