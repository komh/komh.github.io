Using REXX Utility Functions

There are useful functions that pertain to the object oriented Workplace Shell. REXXUTIL provides useful functions that allow the creation, destruction, and the altering of the properties of objects. These functions are listed as follows:

  • SysCreateObject
  • SysDestroyObject
  • SysSetObjectData
  • SysSetIconData
  • SysRegisterObjectClass
  • SysDeregisterObjectClass
  • SysQueryClassList
  • SysGetEa
  • SysPutEa
  • SysIni
