Determining whether Data Can Be Exchanged

During the drag portion of a Drag and Drop operation, the target must determine if it can exchange or receive data from the source for each object involved in the operation. The target determines which rendering mechanism and format to use. There are four combinations of rendering mechanisms and formats supported by the Workplace Shell:

  • <DRM_OBJECT, DRF_OBJECT>

  • <DRM_OS2FILE, DRF_UNKNOWN>

  • <DRM_DISCARD, DRF_UNKNOWN>

  • DRM_OS2FILE, DRM_PRINT) X (DRF_TEXT)

The DRAGITEM structure contains the source object's rendering information, which is used to determine whether a Drop operation is valid at that point. The pDragItemhstrRMF field in the DRAGITEM data structure is used to hold a handle to a string containing the rendering mechanism and format.
