WPStyler Class Methods

WPStyler uses six class methods:

  • wpclsInitData
  • wpclsQueryIcon
  • wpclsQueryStyle
  • wpclsQueryDefaultHelp
  • wpQueryDefaultView
  • clsQueryModuleHandle.

The last one is a new class method introduced by WPStyler.

The following subsections provide explanations on each of them, such as why the class method is used and whether or not it is overridden.
