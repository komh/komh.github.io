About Set/Query Methods

The set and query methods are used to set and interrogate the properties of objects. The information associated with an object can be:

  • Default help
  • Default view
  • Error set
  • Handle
  • Icon
  • Icon data
  • Identifier
  • Settings page size
  • Title.
