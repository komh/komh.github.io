CFGFILT

    CFGFILT.EXE

      Filter configuration utility

    Syntax 

    CFGFILT───┬┬────────────┬──┬──────────────┬┬──┤
          │└─-u──┬────┬─┘  └─-d─┬───────┬─┘│
          │      └─-i─┘         ├─start─┤  │
          │                     └─stop──┘  │
          │                                │
          ├──────────────-c────────────────┤
          │                                │
          ├──────────────-f─┬──────┬───────┤
          │                 └─file─┘       │
          │                                │
          ├──────────────-m─┬───┬──────────┤
          │                 └─#─┘          │
          │                                │
          └──────────────-p─┬───┬──────────┘
                            └─#─┘

    Description

    This program is used to activate filtering and manage filter rules.

    If no parameters are given on the command line, CFGFILT will print the current status of packet filtering, followed by the currently-defined filter rules.
