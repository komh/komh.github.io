tunnel

Tunnel ID.

    Syntax

      (Decimal integer between 0 and 999999)
    Description
      This parameter indicates that the traffic should pass through the IPSec tunnel with the specified tunnel ID. This is part of the Virtual Private Network (VPN) feature.

      When this parameter is used, the scope of the rule changes from system access control to tunnel access control. A rule which contains this parameter does not permit or deny traffic like a normal rule, but simply directs the traffic through an IPSec tunnel.

      Consequently, traffic which matches a rule with a tunnel ID must also be permitted by another filter rule in order to be allowed into or out of the system.
