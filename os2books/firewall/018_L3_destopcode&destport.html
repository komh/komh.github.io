destopcode & destport

These two fields specify criteria for the IP port number to which the packet is addressed. The destopcode (operator) and destport (value) fields are used together to form a logical equation.

For example, 'eq 161' would indicate packets which are addressed to port number 161. 'ge 1024' would indicate packets which are addressed to any port greater than or equal to 1024.

destopcode

Destination port operator.

    Syntax

      any | eq | gt | lt | neq | ge | le

    destport

    Destination port value.

    Syntax

      (decimal integer between 1 and 65535)
