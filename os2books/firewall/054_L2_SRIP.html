SRIP

    SRIP.EXE

      Secure Remote IP Client

    Syntax 

    SRIP컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴

    Description

    This command starts the VPN client graphical user interface. This client allows dynamic IPSec tunnels to be established via SSL to a remote AIX server running IBM SecureWay Firewall.

    SRIP.EXE is not included with MPTS; it is available only if the VPN client option was installed as part of TCP/IP services (TCP/IP V4.1 and above only).
