Message and error codes

Every log entry should include a message code of the format: 

    ICAxxxxs
where xxxx represents the specific message number, and s indicates the message severity.

The specific message number should fall into one of the following ranges:

    0000 - 0999

      Intrusion alarm messages
    1000 - 1999
      Filter-related messages
    2000 - 2999
      Proxy-related messages
    3000 - 3999
      SOCKS-related messages
    4000 - 4999
      Pager-related messages
    5000 - 5999
      SSL-related messages
    6000 - 6999
      VPN-specific messages
    7000 - 8999
      Reserved for future use
    9000 - 9999
      Miscellaneous messages

    Note: Some of the above categories involve SecureWay Firewall features that are not available in the MPTS firewall (such as the SOCKS and pager features), and are therefore unlikely to be encountered in this environment.

    The severity level indicator will be one of the following:

    i

      Informational messages
    w
      Warning messages
    e
      Error messages
    s
      Severe error messages
