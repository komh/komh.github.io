FWD_K

    FWD_K.EXE

      IPSec tunnel definition removal utility.

    Syntax 

    FWD_K───srcip───destip────────────────────────┤

    Description

    This program is used delete all tunnel definitions from the active cache which match the given source and destination IP addresses.
