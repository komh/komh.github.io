References

The information in this book was compiled from a number of different sources:

  • IBM SecureWay Firewall for AIX User's Guide (IBM RedBook GC31-8419-03)

    Chapter 10 contains a good description of filter rules.

  • IBM SecureWay Firewall for AIX Reference (IBM RedBook SC31-8418-03)

    Chapter 1, the 'Rules' section, has some syntax information for the parameters used by filter rules.

  • Inside OS/2 Warp Server for e-business (IBM RedBook SG24-5393-00)

    Chapter 7, sections 7.5 and 7.6 provide a helpful summary of the available firewall features, along with some notes on configuration and usage.

  • A Comprehensive Guide to Virtual Private Networks, Volume I: IBM Firewall, Server and Client Solutions (IBM RedBook SG24-5201-00)

    This book discusses VPN configuration, which includes IP filtering, in excellent detail for multiple IBM platforms. Chapter 9, section 9.4 is evidently the source for the information included in SG24-5393-00, but it includes some additional notes on establishing tunnels with various types of host. Chapters 4, 7 and 8 provide some other MPTS-specific information as well.

  • Guarding the Gates Using the IBM eNetwork Firewall V3.3 for Windows NT (IBM RedBook SG24-5209-01)

    Chapters 1 and 2 provide an excellent overview of firewalls in general, and the architectural design of the IBM Firewall in particular.
