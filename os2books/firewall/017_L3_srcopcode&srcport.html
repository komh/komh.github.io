srcopcode & srcport

These two fields specify criteria for the IP port number from which the packet is being sent. The srcopcode (operator) and srcport (value) fields are used together to form a logical equation.

For example, 'eq 161' would indicate packets which have been sent from port number 161. 'ge 1024' would indicate packets which have been sent from any port greater than or equal to 1024.

srcopcode

Source port operator.

    Syntax

      any | eq | gt | lt | neq | ge | le

    srcport

    Source port value.

    Syntax

      (decimal integer between 1 and 65535)
