FSSD

    FSSD.EXE

      Filter log daemon

    Syntax 

    FSSD컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴컴

    Description

    This command starts the packet filter logging daemon. It may be also run as a detached process with the command 'detach fssd'.

    The desired level of detail for logging may be configured in FWLOG.CNF. If FSSD encounters an error, a message will be recorded in the file FWERRORS.ERR. See Configuring logging for more information.

    Since the currently-active log file is locked, the FWLSLOG command must be used to view the log entries. This command also formats the log files into a more easily-readable form.
