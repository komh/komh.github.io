Firewall Commands

This chapter describes the commands which comprise the various firewall functions.

All of these commands are executable programs which are run from a standard command prompt.

Of these commands, the following involve basic firewall functions:

  • CFGFILT

  • FSSD

  • FWLSLOG

And the following are specific to the Virtual Private Network feature:

  • ADMIN

  • FWD_K

  • FWINSERT

  • HAND_K

  • SRIP
