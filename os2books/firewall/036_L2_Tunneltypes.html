Tunnel types

There are two general categories of IPSec tunnel used by the firewall VPN client: manual tunnels and dynamic tunnels.

A graphical user interface for configuring and using dynamic tunnels is available as part of the TCP/IP product. However, there is no GUI available for setting up manual tunnels; the only current means of configuring these tunnels is through command-line utilities and the direct editing of configuration files.
