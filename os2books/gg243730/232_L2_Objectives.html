Objectives

Subjects which will be covered in this lab session are:

  • Memory allocation using the DosAllocMem() function

  • Page granularity in memory allocation

  • The general protection exception error message

  • The guard page fault exception error message

  • The new SWAPPER.DAT functionality

  • Starting multiple DOS sessions

  • Starting multiple threads.

    Software required to do these exercises:

  • OS/2 Version 2.0

  • C Set/2

  • IBM Developer's Toolkit for OS/2 2.0
