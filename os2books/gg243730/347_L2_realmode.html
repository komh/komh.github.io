real mode

Default mode of operation for the Intel 80286 and 80386 processors, and the only mode of operation for the 8086 processor. In real mode, the processor acts as a 16-bit device, its physical memory address space is limited to 1MB, and memory references translate directly to physical addresses. With the 80386, paging is not supported in real mode.
