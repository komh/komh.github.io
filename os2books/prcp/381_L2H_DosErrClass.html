DosErrClass

EXTRN  DosErrClass:FAR
INCL_DOSMISC      EQU 1

PUSH   WORD    Code          ;Error code for analysis
PUSH@  WORD    Class         ;Error classification (returned)
PUSH@  WORD    Action        ;Recommended action (returned)
PUSH@  WORD    Locus         ;Error locus (returned)
CALL   DosErrClass

Returns WORD
