DosCopy

EXTRN  DosCopy:FAR
INCL_DOSFILEMGR     EQU 1

PUSH@  ASCIIZ  SourceName    ;Source path name string
PUSH@  ASCIIZ  TargetName    ;Target path name string
PUSH   WORD    OpMode        ;Operation mode
PUSH   DWORD   0             ;Reserved (must be zero)
CALL   DosCopy

Returns WORD
