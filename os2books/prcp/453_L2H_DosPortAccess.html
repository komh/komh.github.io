DosPortAccess

EXTRN  DosPortAccess:FAR
INCL_DOSDEVICES     EQU 1

PUSH   WORD    0             ;Reserved (must be zero)
PUSH   WORD    TypeOfAccess  ;Request or release
PUSH   WORD    FirstPort     ;First port number
PUSH   WORD    LastPort      ;Last port number
CALL   DosPortAccess

Returns WORD
