DosGetPID

 
PIDINFO struc
  pidi_pid       dw  ? ;current process' process ID
  pidi_tid       dw  ? ;current process' thread ID
  pidi_pidParent dw  ? ;process ID of the parent
 
PIDINFO ends

EXTRN  DosGetPID:FAR
INCL_DOSPROCESS     EQU 1

PUSH@  OTHER   ProcessIDsArea ;Process IDs (returned)
CALL   DosGetPID
