DosQCurDir

DosQCurDir

EXTRN  DosQCurDir:FAR
INCL_DOSFILEMGR     EQU 1

PUSH   WORD    DriveNumber   ;Drive number
PUSH@  OTHER   DirPath       ;Directory path buffer (returned)
PUSH@  WORD    DirPathLen    ;Directory path buffer length (returned)
CALL   DosQCurDir

Returns WORD
