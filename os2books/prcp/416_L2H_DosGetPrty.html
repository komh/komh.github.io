DosGetPrty

EXTRN  DosGetPrty:FAR
INCL_DOSPROCESS     EQU 1

PUSH   WORD    Scope         ;Indicate scope of query
PUSH@  WORD    Priority      ;Priority (returned)
PUSH   WORD    ID            ;Process or thread ID
CALL   DosGetPrty

Returns WORD
