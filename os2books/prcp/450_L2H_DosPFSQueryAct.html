DosPFSQueryAct

EXTRN  DosPFSQueryAct:FAR
INCL_DOSPFS         EQU 1

PUSH@  ASCIIZ  PrinterName   ;Printer name string
PUSH@  WORD    CodePage      ;Code Page return
PUSH@  WORD    FontID        ;Font ID return
PUSH   WORD    SFN           ;System File Number
PUSH   DWORD   0             ;Reserved (must be zero)
CALL   DosPFSQueryAct

Returns WORD
