DosCreateQueue

EXTRN  DosCreateQueue:FAR
INCL_DOSQUEUES      EQU 1

PUSH@  WORD    RWHandle      ;Queue handle (returned)
PUSH   WORD    QueuePrty     ;Ordering to use for elements
PUSH@  ASCIIZ  QueueName     ;Queue name string
CALL   DosCreateQueue

Returns WORD
