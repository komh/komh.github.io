Help Menu Bar Item

A "Help" menu bar item, whether it is defined on the menu bar or in a "Help" pulldown menu, should be defined using the MIS_HELP style. This will cause the item to generate a WM_HELP message, rather than a WM_COMMAND message. The definition of such an item is shown in Figure "Resource Script File".
