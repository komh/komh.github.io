SRPI

Server-Requester Programming Interface; programming interface enabling master-slave communication between a workstation application (the requester) and a host application (the server) using an SNA LU2.0 communications link. SRPI is supported in the PWS under DOS, OS/2 Extended Edition V1.x and OS/2 Extended Services, and in System/370 hosts running VM/CMS and MVS/TSO.
