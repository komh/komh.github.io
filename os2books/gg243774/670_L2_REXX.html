REXX

Restructured Extended Executor; procedural language included as part of OS/2 Version 2.0, which provides batch language functions along with structured programming constructs such as loops, conditional testing and subroutines. Programs or subroutines written using the REXX language may be invoked directly from the command line, or from an application written in REXX or another programming language.
