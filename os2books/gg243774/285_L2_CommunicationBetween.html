Communication Between Objects

Objects communicate with one another in order to convey events and initiate actions on the part of other objects. Such communication is typically initiated in one of two ways:

  • By an object to convey information to another object with which it has an application-defined relationship, such as a request. This is similar to the application-initiated event discussed for Presentation Manager applications in The Presentation Manager Application Model.

  • By the user directly manipulating the objects' icons. For example, dropping one icon over another icon initiates a conversation between the two objects. This is similar to the user-initiated event discussed for Presentation Manager applications in The Presentation Manager Application Model.
