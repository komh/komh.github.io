Placing a Check Mark on a Pulldown Menu Item

hMenu = WinWindowFromID(hFrame,            /* Get menu bar handle        */
                         FID_MENU);

rc = WinCheckMenuItem(hMenu,               /* Menu bar handle            */
                      MI_OPTION1,          /* Menu item identifier       */
                      TRUE);               /* Set check mark             */
