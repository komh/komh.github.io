WinBroadcastMsg() Function

rc = WinBroadcastMsg(hwnd,                    /* Current is parent       */
                     WMP_MYMESSAGE,           /* Message identifier      */
                     mp1,                     /* 1st message parameter   */
                     mp2,                     /* 2nd message parameter   */
                     BMSG_POST);              /* Post message via queue  */
